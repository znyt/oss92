        &lt;h3&gt;第三卷第六十七章  画地图的启示&lt;/h3&gt;
<br/>
<br/>        陈子昂不知道哥哥为什么会和傅海生闹矛盾，对于自己的哥哥她还是了解的，除非惹急了他，但是傅海生和夏颜应该和他没有什么冲突，而且陈子霖还算是一个能拿得住的人，陈子昂百思不得解，想打电话再问问江素轩，想一想就算了。
<br/>
<br/>        江俞轩做事情稳重，没有电话过来证明事情已经解决了，她远水解不了近渴，着急也没有办法。
<br/>
<br/>        虽然休假了，其实她也不得闲，两个淘气宝要吃要喝，这几天才知道春姨有多么的难，嘟嘟和团团已经是小学生了，还要批改作业，这么一忙，也就忘记了给陈子霖打电话。
<br/>
<br/>        等忙完两个孩子，她也已经劳累了一天，公司发的很多邮件都没有时间看，看着两个孩子都乖巧的休息了，她才打开邮件，一封封的看着，自己公司内部的事情她是很满意，现在公司早已经步入正轨，有没有她在都是一样的。
<br/>
<br/>        现在耗材有很多型号都已经成功了，每个硒鼓的打印量都在一千四百页左右，只等几个客户单位试用后的反馈再进行合理的改进和测试，后期将会大批量的投入使用，她还想开一个新闻发布会。
<br/>
<br/>        当看到郝景文发给她的邮件时，她一连看了三遍他们的方案，寒伯安还真是人才，能从一个出租车司机的口中发现商机。
<br/>
<br/>        上次她也和那个司机聊了天，感觉目前他们的这个行业的服务态度确实需要改变，如果他们及早做出改变，后期他们永远都是独树一帜。
<br/>
<br/>        可是，世界上没有如果，你不改变自然会有人来改变。
<br/>
<br/>        郝景文标红的那句话是要她引起足够的重视：我们后期的服务将要在我国大陆各主要城市及旅游地区实行服务网络，以24小时的取还服务和配套服务，不仅要随时随地满足客户的服务需求，更要为客户的安全行车保驾护航。尤其是安全，这个需要适时掌握行车动态，请陈总酌情考察，并给予重视。
<br/>
<br/>        他们不但是要将服务做好，还要一套完整的app来做为行业的支撑，他们的方案已经出来了，但是软件支撑上还只是空白。
<br/>
<br/>        陈子昂对前期物流有了简便的app操作，而且大家也在完善及细化app的应用，但是这不等于能够运用到寒伯安说的这个事情上。
<br/>
<br/>        她给赵绾绾发了个信息：“绾绾？”
<br/>
<br/>        “子昂。”
<br/>
<br/>        他们年龄相仿，而且当初都是一起打拼过来的，所以就没有那么多客气的话语。
<br/>
<br/>        “信息化软件情况怎么样？还有没有能力开发别的软件？”赵绾绾主管信息化。
<br/>
<br/>        “我们的l系统已经趋于成熟，后期就是在售后服务中根据客户的需求进行完善和改进。物流app现在用的还不错，也会在应用的过程中进行适时完善和升级。另外，耗材现在已经投入了打量的人力来进行测试和改进。倪小美和苟江凌基本都泡在耗材上，而销售网络app软件正在紧要关头。”
<br/>
<br/>        “我想开发一种新的软件。”
<br/>
<br/>        “如果你想开发新的软件不是不可以，只是需要耗费一番时间、人力和财力，而且还需要再引进软件开发的人才，目前大家的工作已经处于饱和状态，大家都是在拼着命的干活。”
<br/>
<br/>        “我再考虑一下，明天我们见面说。”
<br/>
<br/>        “好，子昂，照顾好自己，你可是我们心尖尖上的人。你的开心是最重要的。”赵绾绾发了一个拥抱。
<br/>
<br/>        “送你小花花。你也要照顾好自己。”陈子昂发了一支玫瑰花给赵绾绾。
<br/>
<br/>        赵绾绾说得不错，他们本来写软件出身的，现在又让他们分一部分人出来做耗材研发，对大家来说都是一种挑战，虽然有借鉴，但是作用不是很大，如果再进行开发的话无形中就会给大家增加更大的工作量。
<br/>
<br/>        看看时间才晚上九点多，于是陈子昂又给郝景文发了条信息。
<br/>
<br/>        “郝总，您好！”陈子昂从来不问别人在不在的这种话题，如果人家在线会直接回复你，不在线也就不回复你，或者人家在线，正忙着别的，没有时间来及时回复你。
<br/>
<br/>        “我有事情想咨询你，关于行车系统（目前暂时这样称呼）您们打算用多久时间上线？”
<br/>
<br/>        “陈总，您好，我们的项目暂时只是初步启动，后期软件上线还得看开发商的时间，当然是越快越好。”
<br/>
<br/>        寒伯安已经出国去考察了，他也看了好几个厂家的车辆。江俞轩虽然忙着陈子昂公司的事情，但是他也没有闲着，出差的时间，他会利用空闲的时间来考察当地的情况。
<br/>
<br/>        “我明天去公司，和大家讨论一下方案，另外这个我们需要您们给我们具体的需求。”
<br/>
<br/>        “我们的设想和方案已经给你发了邮件了，对于软件开发上我们不是很懂，所以希望有合适的软件开发商给他们合理的建议。”
<br/>
<br/>        “明白，那不打扰你郝总，我们回头联系。”
<br/>
<br/>        陈子昂再一次细细研究了郝景文发的方案，他们设计的是一个全新的领域，对于陈子昂来说是这样的，对寒伯安他们来说也是这样的，一个群体的路怎么走，靠的是一个领头人，寒伯安已经将他的想法写出来了，但是只是他的想法，在具体的实施过程中会碰到各自不同的问题，俗话说计划没有变化快。
<br/>
<br/>        在软件开发上，往往提出来的是一个枝干，可是这个枝干会产生不同的枝丫，最后有可能形成千万条枝丫的参天大树，因为每个人对抽象的事物的理解是不同的。
<br/>
<br/>        陈子昂这几年也经常出差，坐车，如果以一个消费者的观念去看待问题，是不是会好一点？
<br/>
<br/>        “俞轩，休息了吗？郝总发的邮件你看了没有？”陈子昂又给江俞轩发了一个信息。
<br/>
<br/>        但是等了一会，江俞轩并没有给他回信息。
<br/>
<br/>        陈子昂拿出纸笔，开始按照郝景文发的方案思路往下走，在纸上一遍一遍的描绘着脑子里的地图。
<br/>
<br/>        地图，陈子昂看着自己画的状似地图的线路图，恍然大悟。
<br/>
<br/>        这是一个多元化的时代，每个领域都会有专业的一面，如何将他们组合到一起，拿来为自己所用呢。
<br/>
<br/>        陈子昂的app能够跑起来，不也是和别人合作了才能起来的吗？
<br/>
<br/>        陈子昂将笔丢在子桌子上，拍拍手：“睡觉，明天可以去上班了！”
