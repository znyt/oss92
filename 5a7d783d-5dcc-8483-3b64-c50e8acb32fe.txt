　　苏莫沉吟了片刻，距一个月的时间，还有三天，倒是可以尝试冲击武尊之境。
<br/>　　在魔渊之时，经过一段时间的修炼，他的修为已经颇为稳固，虽然距离冲击境界还有点距离，但这近一个月之中，他参悟五行之力的时候，也没忘记修炼。
<br/>　　所以，他现在完全可以尝试冲击武尊之境了。
<br/>　　当然，武尊之境非同小可，从武帝到武尊，是大境界的跨越，难度将会更大，苏莫并没有什么把握。
<br/>　　少倾之后，苏莫缓缓平复心境，一挥手，取出了一小堆中品灵晶。
<br/>　　这堆灵晶足有两千有余，算是一笔巨款了。
<br/>　　“吞！”
<br/>　　苏莫轻喝一声，直接释放出了吞噬战魂，瞬间便将所有灵晶直接吞噬。
<br/>　　而后，强大的灵气疯狂的涌入了他的体内。
<br/>　　大混沌五行神诀运转，苏莫极力的炼化，将灵气炼化成五行玄力，汇入五枚神丹之中。
<br/>　　他的五枚神丹，早已达到了武帝境九重的巅峰，汇入大量的灵气之后，立刻剧烈的震动了起来，自行的开始了冲击武尊之境。
<br/>　　两千块中品灵晶的灵气，非常的磅礴，说是浩荡如江河也不为过，这就为苏莫冲击境界瓶颈，提供了坚实的底子。
<br/>　　时间缓慢流过，苏莫盘膝而坐的身形，微微的颤抖，不断的颤抖。
<br/>　　这是强力冲击武尊之境，体内神丹的震动，所引起的身体颤抖。
<br/>　　苏莫的身躯，已经化为了金刚之躯，如同钢铁浇筑，他催动了血脉力量。
<br/>　　因为如此强力的冲击境界瓶颈，不动用血脉之力增强身体强度，他很容易爆体而亡。
<br/>　　一个时辰又一个时辰，约莫六个时辰之后，两千中品灵晶的灵气便消耗一空。
<br/>　　苏莫再次取出了两千中品灵晶，继续吞噬，继续冲击境界瓶颈。
<br/>　　强力的冲击境界，让得他全身大汉淋漓，连动用了金刚之体的他，身体都面临着巨大的压力。
<br/>　　这若是没有特殊体质，肉身根本无法承受如此浩大灵气的冲击。
<br/>　　不断的冲击境界瓶颈，时间的流逝仿佛都慢了下来。
<br/>　　不知道过去了多少个时辰，苏莫的体内，五枚神丹猛然一震，而后齐齐开始膨胀了起来。
<br/>　　吞噬的所有灵气，突然疯狂的向五枚神丹中汇聚，如同长鲸吸水一般。
<br/>　　这一刻，苏莫成功的冲破了境界瓶颈，跨入了武尊之境。
<br/>　　“突破了！”苏莫微微有些惊讶，因为他并没有破境的信心，只是拼命一试，没想到居然真的顺利突破了。
<br/>　　这真的是意外之喜啊！踏入了武尊之境，是一个非常大的跨越，与武帝之境不可同日而语。
<br/>　　少倾之后，所有灵气全部被神丹吸收，苏莫身上的气息平静了下来，修为停在了武尊境一重的初期。
<br/>　　“真的是可喜可贺！”
<br/>　　苏莫面上露出了一丝笑容，本以为想要突破到武尊之境，难度肯定极大，没想到这么轻易就达到了。
<br/>　　微微闭上眼眸，感受了一番体内的力量，苏莫面上笑容更甚。
<br/>　　达到武尊之境，实力增加的不是一星半点，可谓是直线暴增，远比武帝境之时，提升境界之时实力增长的大。
<br/>　　片刻之后，苏莫压下心中的激动，意念一动，便离开了玉镯空间。
<br/>　　随即，苏莫便走出了府邸，向太阴城之外走去。
<br/>　　一月之期将到，他也要离开太阴城了，再说继续留下来，也没有什么用处了。
<br/>　　九重浮屠塔太过消耗灵晶，而在外族之中，也很难见到青璇，只能过段时间再来了。
<br/>　　而且，接下来一段时间，他要好好的尝试玄力融合，还是回到道宗比较好。
<br/>　　没过多久，苏莫便离开了太阴城，身形拔空而起，向问道山的方向飞去。
<br/>　　高空之上，白云之间，苏莫身形如同疾驰的闪电，瞬息千余里。
<br/>　　飞行了一个时辰之后，苏莫突然停了下来，因为他的前方，赫然出现了一人，拦住了他的去路。
<br/>　　这是一名年轻男子，身穿月白色长衫，面容冷峻，一头长发随风飘舞，气息浑厚无比，深邃不可测。
<br/>　　“你是何人？”苏莫皱眉问道，他心中暗暗警惕了起来，此人怕是来着不善。
<br/>　　他仔细打量了此人一眼，此人武尊境四重境界，并没有多高的修为，但却是给他一种无形的压力。
<br/>　　此人仅是平淡的站在高空中，却给人一种天地之间，唯我独尊的气势。
<br/>　　“苏莫，随我去天虚宗！”冷峻青年平淡的说道。
<br/>　　“你是天虚宗的人？”苏莫闻言眸光一闪，立刻便明白了，天虚宗找他麻烦了。
<br/>　　“不要逼我出手，不然你会后悔！”冷峻青年并未回答苏莫，言语平淡，面色也是平淡无比，给人一种波澜不惊之感。
<br/>　　“是吗？”苏莫眼眸微眯，眸中爆射出一道冷冽的寒光，此人倒是非常自信，仿佛吃定了他一般。
<br/>　　不过，苏莫却是毅然无惧，他修为未突破之时，就已经能击败天虚宗真传弟子，现在突破到了武尊之境，实力不会比武尊境四重武者弱多少。
<br/>　　“你只有两个选择，要么乖乖跟我走，要么被我擒住，我带你去天虚宗，给你三个呼吸的时间考虑！”
<br/>　　冷峻青年语气虽然平淡，但言语中却充满着无形的霸气，不容反抗的霸气。
<br/>　　“呵呵，你就这么自信能擒住我？”苏莫闻言冷笑了起来，他倒是好奇了起来，此人为何有如此的自信？
<br/>　　不过，他虽然好奇，但也暗暗的戒备了起来。
<br/>　　此人既然如此自信，想必定然是底气十足，或许此人是妖孽天才。
<br/>　　“我名，厉恨天！”冷峻青年淡淡的说道，仿佛这个名字，便代表着自信，代表着实力。
<br/>　　“厉恨天？”
<br/>　　苏莫闻言嗤笑一声，摇头道：“不好意思，无论你恨天还是恨地，我都没有听说过。”
<br/>　　苏莫心中感觉有些好笑，这人如此摆谱，难道真以为仅凭一个名字，就能将他吓倒？
<br/>　　“看来你很不识趣，是要我出手了！”冷峻青年眸光一冷，淡淡的说道。
<br/>　　“出手又怎样？”苏莫冷笑一声，体内玄力运转了起来，随时准备动手。
<br/>　　让他束手就擒，这怎么可能？
<br/>　　一旦去了天虚宗，他绝对是有死无生，所以他不可能束手就擒。
