　　两种特殊体质？
<br/>　　众人闻听玄净尊者之言，顿时惊愕不已，一脸的呆滞。
<br/>　　就连谢彬、尤天韩、巫人王等认识苏莫的人，都是瞪大了眼眸，满脸的不可思议之色。
<br/>　　两种特殊体质？
<br/>　　这不可能吧？
<br/>　　一种特殊体质，就已经是人中龙凤，两种特殊体质，这可从来没有听说过。
<br/>　　所有人都是满脸的惊疑，同样的是不太相信，实在是此事太过夸张，与四生战魂同样的夸张。
<br/>　　纵观古灵星河无尽岁月，好像从未出现过，同时拥有两种特殊体质的天才吧！
<br/>　　特殊体质与战魂不同，动用战魂有战魂之力留露而出，轻易就能感知的到。
<br/>　　而特殊体质也就只能靠显露的异像，才能察觉的出来，甚至很多特殊体质，根本没有任何异像。
<br/>　　那影像之中，苏莫全身爆射出刺目的白光，其中还掺杂着琉璃色的光芒，很难分出是一种特殊的体质，还是两种特殊体质。
<br/>　　因为两种光芒，并不能代表就是两种特殊体质。
<br/>　　“玄净尊者前辈，这是真的吗？那苏莫真的有两种特殊体质？”一名矮壮青年率先回过神来，不确定的问道。
<br/>　　“是啊！此言可要有依据啊！”
<br/>　　“若是他真的有两种特殊体质，我就承认他是真圣榜第一名！”
<br/>　　“两种特殊体质，那是哪两种呢？”
<br/>　　“……”
<br/>　　不少人亦是纷纷开口，个个眸光复杂，有惊疑、又好奇、有难以置信！
<br/>　　玄净尊者深吸了口气，眸中亦是有震撼之色，朗声道：“根据商会两位武圣的观察，这是大日神体和琉璃宝体，绝对不会有错！”
<br/>　　玄净尊者的言语非常肯定，这不是猜测，而是无比的确定。
<br/>　　对此，他亦是深深的震撼。
<br/>　　“大日神体和琉璃宝体？”
<br/>　　“不错，这个特征的确是大日神体和琉璃宝体的特征！”
<br/>　　“是啊，只不过两种体质的特征结合在了一起，现在特征不是那么的明显了，更没有那么纯粹了！”
<br/>　　“太恐怖了，两种特殊体质啊！”
<br/>　　所有人都是面带震撼之色，大多数人都相信了，既然是武圣亲自鉴别而出，还无比的肯定，那就不会有错了。
<br/>　　苏莫身上显现的特征，很多人都能分辨出来，的确是符合大日神体和琉璃宝体。
<br/>　　再者，这苏莫是神武学府的弟子，又不是拍卖会背后商会的人，没有必要故意捧苏莫吧！
<br/>　　这一刻，大多数的人，都认可了将苏莫排在第一名的事实。
<br/>　　虽然很多人心中还是有些不敢置信，还是有些惊疑不定，但也渐渐默认了。
<br/>　　“琉璃宝体！”巫人王口中低声自喃，他的眼睛已经眯成了一条缝，眸中尽是疑惑之色。
<br/>　　对于这种体制，他再了解不过了，因为巫幽兰便是琉璃宝体。
<br/>　　只不过，巫幽兰早已死在了苏莫的手上。
<br/>　　这让巫人王心中疑惑不已，苏莫也拥有琉璃宝体？居然如此的巧合？这可能吗？
<br/>　　苏莫的琉璃宝体，和巫幽兰的琉璃宝体，是不是有某种联系？
<br/>　　巫人王的眼眸渐渐的闪亮了起来，他觉得他好像发现了一个秘密，一个无比惊人的秘密。
<br/>　　自从上次少司选拔，他和巫尘一战之后，他发现自己的天玄武体的血脉之力大减，威力不足以往的三成。
<br/>　　这一段时间，他非常疑惑，不知道是怎么回事？
<br/>　　但现在，他有些明白了，定然是巫尘搞得鬼，而巫尘就是苏莫。
<br/>　　“难道苏莫有什么逆天秘法，能吸取别人的血脉？”
<br/>　　巫人王心中突然升起了这种想法，顿时心中一震，连他自己都被这个想法吓到了。
<br/>　　若是真的有这种逆天秘法的话，他无论如何一定要得到，这是一个无比惊人的机遇。
<br/>　　本来，他还有些恼怒这个真圣榜，因为他并没有上榜，这是对他的侮辱。
<br/>　　他的天赋和战力，完全不输与尤天韩，但对方排名高达一百六十七名，而他却连榜单都没有登上去。
<br/>　　但是现在，发现了苏莫身上蕴含的秘密之后，这个真圣榜的排名，巫人王已经完全不关心了。
<br/>　　另一边，尤天韩脸色阴沉如水，眸中闪烁着阴霾的光芒。
<br/>　　四生战魂？两种特殊体质？
<br/>　　这怎么可能？
<br/>　　苏莫有这种恐怖的天赋，若真是如此，那还得了？
<br/>　　尤天韩心中生出淡淡的杀意，苏莫如此恐怖的天赋，若是成长了起来，对于他来说定然是大敌。
<br/>　　坐席上的苏莫，眉头紧皱，虽然玄净尊者的话，没有将他的天赋全部暴露出来，但也暴露了一部分。
<br/>　　而且，他的行踪将彻底暴露给了巫族。
<br/>　　他现在要考虑的是，接下来该如何应对？
<br/>　　如果即刻逃走，能否安全的逃出古周星？逃出古灵星河？
<br/>　　他现在不清楚巫族是不是查到了古周星，若是查到了古周星，那来到古周星的巫族之人，就绝对不是巫人王一人。
<br/>　　若是如此的话，他很难逃出去。
<br/>　　还有就是，他一旦也逃走的话，不可能独自逃走，肯定要将苍穹神宫也带走。
<br/>　　只是，夕儿他们都出去历练了，也不知道有没有都回到途山镇！
<br/>　　一时间，苏莫头大如斗，感觉危机正在慢慢来临。
<br/>　　他也拿不定注意，是立刻逃走，还是回神武学府，此事必须慎之又慎。
<br/>　　不然，一个不慎就会陷入万劫不复之地。
<br/>　　中央拍卖台上，玄净尊者目光环视四周，少倾之后，朗声道：“众位，综合战魂天赋和特殊体质天赋，苏莫都是逆天妖孽，无人能出其右，所以，虽然其现在修为不高，但我们还是将他排在了第一位！众位应该没有什么异议吧？”
<br/>　　众人闻言沉默了，这种逆天的天赋，排在第一位的确是没有什么不妥。
<br/>　　就算苏莫的战魂天赋，并没有那么夸张，并不是四生战魂，但就算是三生战魂，配合上两种特殊体质，也够资格排在第一位了。
<br/>　　玄净尊者的脸上，露出了和煦的笑容，众人的沉默，便已经表明了答案。
<br/>　　“玄净尊者前辈，苏莫排在第一名，我没有异议了，但还是有个问题不解？”一名黄衣青年站了起来，朗声问道。
<br/>　　这名青年正是之前在拍卖场外，说苏莫是土包子的黄衣青年。
<br/>　　“请说！”玄净尊者点了点头。
<br/>　　“苏莫排在了第一位，那我三神宗的姜神夜师兄呢？姜神夜师兄就算不能排在第一位，但还不至于上不了榜吧？”黄衣青年大声问道。
<br/>　　“对啊！姜神夜为何没有上榜？”
<br/>　　“三神宗的石惊生都上榜了，姜神夜不论是天赋还是战力，可都是比石惊生还要强的多，居然上不了榜？”
<br/>　　“这不符合常理吧！”
<br/>　　黄衣青年开口之后，又有几人开口，问出了心中的疑惑。
<br/>　　“根据我们的消息，姜神夜修炼情况有变，遭了大难，能否度过此劫，还是两说呢！”
<br/>　　玄净尊者解释道，随即他又道：“所以，姜神夜虽然天赋超凡，但暂不排进榜单！”
<br/>　　“什么？”
<br/>　　众人闻言惊讶，千古妖孽姜神夜修炼有变？这什么情况？
<br/>　　就连开口询问的三神宗黄衣青年，以及三神宗石惊生，都是诧异不已，完全不知道有此事。
<br/>　　“此事，我们也不太清楚！”
<br/>　　玄净尊者摇了摇头，朗声道：“现在真圣榜已经公布了，众位也没有大的异议，那此事便告一段落，接下来我们便正式开始拍卖会吧！”
