        听着季思妍这话，童筱颖真觉得很好笑，耐着性子解释道：“季大小姐，这个照片里的女人，真的不是我。你有没有看过电视？上网也行。现在有个女明星名叫苏漫雪，长得跟我一模一样。”
<br/>
<br/>        “你觉得，你这种谎话，我会信吗？”季思妍不以为然。
<br/>
<br/>        童筱颖一脸无奈道：“看样子，你是有很久很久没看过电视也没上过网了。没关系，我帮你查。”
<br/>
<br/>        语毕，童筱颖拿出手机，在浏览器里输入了“苏漫雪”这三个字后，将苏漫雪的个人资料和照片点开，连同手机一起递给了季思妍。
<br/>
<br/>        “你自己看。”
<br/>
<br/>        季思妍狐疑地接过童筱颖的手机，看到手机里的这女人后，仍旧不相信童筱颖所说的话。
<br/>
<br/>        “女明星都喜欢用假名，说不定，你就是这个苏漫雪。”季思妍咬牙切齿地反驳。
<br/>
<br/>        “苏漫雪一直在国内活动，而我之前一直在澳洲生活，难不成，我还有分身术？”
<br/>
<br/>        童筱颖叹了口气，又拿起茶几上的信封，将里面的床照一一拿了出来。
<br/>
<br/>        她找出苏漫雪的一张未穿衣服的后背照，然后脱了自己的外套，里面穿着抹胸，转过身去背对着季思妍，解释道：“我左边的蝴蝶骨上有颗痣。照片里的苏漫雪没有。”
<br/>
<br/>        她左边蝴蝶骨上的那颗痣，还是大哥告诉她的，在大哥看到之前，她自己都不知道。
<br/>
<br/>        苏漫雪是做过整容手术的，一定连带着身体也一起整了，所以苏漫雪浑身上下是一颗痣也没有。
<br/>
<br/>        季思妍这回半信半疑起来。
<br/>
<br/>        童筱颖将外套穿好，很客气地对季思妍下逐客令：“季大小姐，请回吧！”
<br/>
<br/>        而此刻，季思妍的脸色越来越惨白。
<br/>
<br/>        季思妍起身离开，后脚刚抬，童筱颖便毫不客气地将大门给关上了。
<br/>
<br/>        待静下心来时，她又觉得季思妍好像有些不大对劲。
<br/>
<br/>        童筱颖忽然意识到什么，连忙打开家门探出身去，只见季思妍就晕倒在离电梯门口不到两米远的地方。
<br/>
<br/>        “季小姐！季小姐！”童筱颖急忙走过去，蹲下身，将季思妍揽入了怀中，连忙拨打了急救电话。
<br/>
<br/>        童筱颖跟随救护车一起去了医院，途中，她拿着季思妍的手机给傅昼景打了通电话，傅昼景未接，她只好给他语音留言。
<br/>
<br/>        在医院里跑上跑下，办理了各种缴费手续后，童筱颖才得以坐在季思妍的病床边的圆凳上喘口气。
<br/>
<br/>        她真搞不懂，自己上辈子是不是欠了她季思妍什么东西，这辈子才会被这个女人这般折腾。
<br/>
<br/>        明明这个女人是破坏自己前次婚姻的小三，结果她不仅替这个女人照顾过她的孩子，现在还轮到要在这照顾她。
<br/>
<br/>        医生给季思妍做完检查后，初步得出的结论还是抑郁症加孕期贫血，好在季思妍肚子里的宝宝并无大碍。
<br/>
<br/>        童筱颖替季思妍谢过医生后，医生刚走，躺在病床上的季思妍也苏醒了。
<br/>
<br/>        季思妍睁开眼后，有那么一小段时间里没有理清楚头绪，待到她缓过神来后，看到童筱颖就坐在自己的病床边，心里忽然五味陈杂起来。
<br/>
<br/>        “是你救了我……”她气若游丝地说。
<br/>
<br/>        童筱颖抿了抿唇，傲娇道：“我只是想出门去买点东西，刚好碰到了而已。”
<br/>
<br/>        “那也是你救的我。”季思妍苦笑，顿了顿后，接着说道，“你难道不希望我去死吗？我死了，阿景就是你的了。”
<br/>
<br/>        “你死了，昼景也不可能是我的。我跟他已经结束了，不可能了。”童筱颖十分淡定道。
<br/>
<br/>        季思妍缓缓垂下了眼帘，眼角挂着一滴眼泪，龟裂的嘴唇微微颤抖着哽咽道：“我到底该怎么做，阿景才会回到我身边来？”
<br/>
<br/>        从国外留学回来后，一开始，她确实是抱着不纯的目的接近傅昼景的，但是，真正接触后，她发现，她打心底地还是爱着这个男人，所以自己再一次陷进去了。
<br/>
<br/>        只是，令她没料到，她爱的这个男人，已经不再是当年那个校园里的翩翩少年。
<br/>
<br/>        傅昼景去澳洲的这三年时光，不但没令她忘记傅昼景，反而更多的是不甘心与压抑。
<br/>
<br/>        她给他生了个儿子，他怎么可以搂着另一个女人远走高飞呢？
<br/>
<br/>        “童筱颖，我到底哪里不如你？我把我有的，都给了阿景，可是阿景的心仍旧想要选择你！就连现在他出轨的对象，都要找跟你长得一模一样的女人。”季思妍把心底的苦与恨，娓娓道来。
<br/>
<br/>        童筱颖微微垂着眼帘，默不吭声。
<br/>
<br/>        “你知道，当初阿景为什么要跟你离婚吗？”季思妍苦笑。
