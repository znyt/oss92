　　毒天使右手这个用特殊金属打造而成的手套铠甲自然不会仅仅只是防御的铠甲而已，五根手指指尖位置尖锐无比，直接被一爪子抓在身上，那绝对是开膛破肚的下场。
<br/>　　手背位置，有三个用高强度防弹玻璃打造外部套着金属外壳的小玻璃管，在玻璃管中装着绿色的不知名液体。玻璃管一头延伸出纤细的金属软管，将玻璃管中中的绿色液体引流到除大拇指以外，其他四根金属手指上。
<br/>　　在这玻璃管中所装的绿色液体，是毒天使一手调配出来的剧毒，至于其毒性自然是不言而喻，而这也使得毒天使手上这个手套铠甲具有了骇人的杀伤力。
<br/>　　即便没有被铠甲尖锐的手指开膛破肚，只是被抓破一个小伤口，利爪上的剧毒也能致人于死地。
<br/>　　毒天使在右手扣上这手套铠甲后，五根手指轻轻活动了一下；左手从腰带左侧的刀鞘中拔出一把鲨鱼刀握在手里。
<br/>　　鲨鱼刀，是联合刀具厂制造出来的一种格斗刀具。鲨鱼刀在造型上跟普通的格斗刀也是有很大不同，不仅是刀背位置的锯齿被改动成了鲨鱼翅造型的尖三角，整个刀身也是被分成了两半，中间那道缝隙一直延伸到刀柄位置。
<br/>　　因为其构造的关系，很少会有人将鲨鱼刀当成格斗刀具使用；但也同样因为它与众不同的造型，很多人都会选择收藏一把作为自己的收藏品进行把玩。
<br/>　　在实战中并不被人看好的鲨鱼刀却被毒天使当成了格斗武器，当然毒天使手里这把鲨鱼刀自然不是普通的鲨鱼刀，不仅是打造这把鲨鱼刀所用的材料，还有这把鲨鱼刀内部的构造。
<br/>　　既然叫“毒天使”，那么他的攻击手段自然是跟毒有关。毒天使手中这把鲨鱼刀刀柄中暗藏玄机，刀柄是中空的，里面是一个装有剧毒的玻璃管，在刀刃底部有一个小洞，通过刀柄上的开关进行控制，小圆孔被封闭的时候，这就是一把比普通鲨鱼刀更加锋利和坚硬的格斗刀，而一旦小圆孔被打开，刀柄玻璃管中的剧毒就会在短短几秒钟内流满鲨鱼刀，让这把刀在眨眼间变成一把剧毒之刃。
<br/>　　毒天使右手的手套铠甲和左手的鲨鱼刀不仅能在短时间内变成剧毒武器，而且两种武器上所使用的剧毒还不相同。这就是毒天使高明的地方，不同的剧毒针对不同的解药，使得中毒之人不会那么轻易就能解救，不服解药会死，服错了解药也会死，不过毒天使自问没有人能在自己的剧毒之下撑过一分钟，除了那个人！
<br/>　　刚才出现在停车场上，破坏了自己好事的巫师！
<br/>　　毒天使握在左手的鲨鱼刀挽了一个刀花，对David说道：“对面来了个厉害的帮手，我负责解决那个人，剩下的人交给你们自己。”
<br/>　　“没问题。”David回答道。
<br/>　　“别指望我刚才放出去的毒雾能帮助你们多少。”毒天使瞥了眼David说道：“那个人来了，即便是他们刚才中了毒，现在也差不多解了，如果想着他们中了毒，你们就能轻易解决掉他们的话，那就大错特错了。”
<br/>　　别说，David和Bob原来的确是这么想的，可是毒天使的一番话，却是彻底击碎了他们心中的幻想。
<br/>　　“他来了！”
<br/>　　兜帽下，毒天使的眼睛中迸射出慑人的寒光，脚步一错，便是向前冲去，身手敏捷的跳上一辆轿车的引擎盖，紧接着在引擎盖上用力一踏，以此借力向前腾跃而起，轿车引擎盖也被毒天使这一脚踩得凹陷下去。
<br/>　　“巫师！！”
<br/>　　毒天使大吼一声，抬起右手，一爪子朝着巫师挥去。
<br/>　　当！
<br/>　　巫师抬手，用手中弯刀格挡住了毒天使右手的利爪。弯刀和利爪相接，溅出一点黄色的火星，即便是在这火光冲天的停车场，这点火星也是显得非常明显。
<br/>　　“圣女弯刀？”
<br/>　　看到巫师握在手中的弯刀，毒天使第一时间就认出了这把弯刀的来自何处，刹那间，毒天使的神情变得有些狰狞起来，冲着巫师吼道：“这把刀为什么会在你这里！？”
<br/>　　毒天使握在左手的鲨鱼刀朝着巫师挥去，巫师右脚在地上一踏，整个人向后退开，避开了毒天使那把已经染上了剧毒的鲨鱼刀。巫师手中的圣女弯刀一直供奉在黑苗和白苗领地交界的山洞中，除非是得到两族族长的同意，不然谁都无法从山洞中将弯刀拿出来，因为山洞中除了有黑苗族布下的毒陷阱，还有白苗族布下的机关。
<br/>　　毒天使也曾想过将圣女弯刀拿出山洞，但是失败了，这也是他被驱逐出黑苗族的原因之一。
<br/>　　唰！唰！唰！
<br/>　　毒天使右手攻击再起，连攻三招，一招快速一招。第一招攻击留下的残影还没散去，第二招攻击又至……
<br/>　　当！当！当！
<br/>　　巫师挥动着圣女弯刀将毒天使的攻击尽数接下，空气中迸溅起三点火星，巫师向后再退三步，退到了一辆SUV边。
<br/>　　“能拿到圣女弯刀自然是得到了两族族长的同意。”巫师将圣女弯刀横在自己面前，说道：“我回去把你在外面做的‘好事’说了一遍，两族达成暂时的合作，从山洞中取出了圣女弯刀。”
<br/>　　“哼，说得自己好像有多高尚一样。”
<br/>　　毒天使冷哼一声，说道：“你自己也是雇佣兵，也是一条‘战争野狗’！”
<br/>　　“我是雇佣兵不假，但是我从来没有杀过一个无辜的人！”
<br/>　　巫师话音刚落，身形向前一闪，圣女弯刀在空中拉出一道刀芒，斩向毒天使。
<br/>　　毒天使身体向后急退，避开巫师这沉重的一刀，活动着自己的右手，手套铠甲五根手指之间相互触碰，发出金属摩擦声，手背上三根玻璃管中的剧毒通过金属软管流淌到五根锋利的手指上。
<br/>　　嗤啦——！
<br/>　　毒天使脚在地上滑动，鞋底跟水泥地面摩擦，发出“嗤啦”的声响来。
<br/>　　“去死吧！”
<br/>　　毒天使身形一闪，再次发起快攻，“你死了，圣女弯刀就是我的了！”
