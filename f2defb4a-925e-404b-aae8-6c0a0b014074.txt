　　林峰二话不说放出了孙悟空和妲己。
<br/>　　“猴子，妲己，我需要你们的帮忙。”
<br/>　　“是，师傅，交给俺老孙。妖孽，吃俺老孙一棒！”
<br/>　　孙悟空拿着金箍棒一棍子横扫而去，来了一招如意金箍，一个大金箍棒砸下来直接砸爆了几十个丧尸。
<br/>　　“主人别担心，妲己来帮你。”
<br/>　　妲己来了一招女王崇拜，顿时身边的丧尸纷纷被爱心击穿了脑袋软绵绵的倒了下去。
<br/>　　林峰把千年僵尸也放出来了。千年僵尸双手就像是绞肉机一样，抓到一个丧尸撕碎一个丧尸，根本没有任何含糊，非常暴力。而那群丧尸扑倒他身上也压根咬不动啃不动更别谈抓了。
<br/>　　顿时林峰压力骤减，总算视线之中出现了丧尸以外的东西。
<br/>　　黑龙镇说大不大说小不小。
<br/>　　林峰才发现自己已经和寒鸦他们脱离方位了。
<br/>　　不过他可以看到一个干事的。
<br/>　　这个干事被几百只丧尸包围，他身后跟着的一百个正式成员只剩下了三四十个，每个人都一身的鲜血。
<br/>　　“去死吧！”
<br/>　　这个干事是个五阶的实力，据说之前是个刑满释放的人员，不过虽然犯罪过，但人不坏，只是一时冲动犯下了错误才蹲的大牢。
<br/>　　此时此刻颇为的凶猛，但再凶猛也架不住丧尸太多，而且最主要的是还有许多三阶以上丧尸。
<br/>　　林峰想要冲过去帮忙但已经为时已晚，这个干事被扑倒在地，他要挣扎着站起来，但刚摆脱了身上的三个丧尸，却是五个三阶丧尸扑了过来，这就像是非洲大陆的狮群猎杀猎物一样，一个个扑过去，咬喉咙咬屁股，只要猎物腿软了倒在了地上，那基本上就死定了。
<br/>　　“啊，老婆，孩子，我对不起你！三年前我不该冲动犯下过错！让你们等了我三年！让孩子没了他的父亲！！！来世我再弥补！！”
<br/>　　旋即这个人拿出了一个手榴弹砰的一下同归于尽了。
<br/>　　“完蛋了！！！李干事死了！”
<br/>　　“该死！”
<br/>　　“这群可恶的家伙！”
<br/>　　“太多了，这里的丧尸太多，不只是李干事要死，我们都要死！”
<br/>　　剩下的几十个正式成员纷纷面色发白，他们最强的也不过四阶罢了。
<br/>　　大部分人都还只是三阶上下罢了。看到一群丧尸前赴后继杀来，其中好几个都是三阶丧尸，他们双腿都在发软。没了干事在前面牵头，他们能怎么办？？？
<br/>　　狂龙出海！
<br/>　　也就在这时一道刀芒闪烁而起。
<br/>　　瞬间那几头三阶丧尸全部被斩杀。
<br/>　　又是一刀下去，十几头丧尸被斩杀，鲜血狂喷，无头尸体直挺挺倒下。
<br/>　　“林峰干事！”
<br/>　　几十个成员看到冲过来的林峰顿时都欣喜道。
<br/>　　林峰的实力很强，他们都知道，在十二个干事之中也是前几的。
<br/>　　“你们去找寒鸦！”
<br/>　　林峰道。
<br/>　　“好！”
<br/>　　几十个成员点了点头朝回去的方向杀去，因为他们是一路血战杀过来的，原路返回倒也没有多少危险。
<br/>　　吼吼。
<br/>　　然而下一刻。
<br/>　　几只变异狼不知道从哪里钻了出来，直接冲入了人群，一爪子一个，一口一个，那几十个成员还没反应过来，全部被斩杀代价。
<br/>　　“该死！”
<br/>　　林峰面色一沉，这十几头变异狼全都是四阶的。
<br/>　　“这群家伙不去对付丧尸，居然对付我们。”
<br/>　　林峰感到意外，怪物吃丧尸，丧尸吃怪物，不过怪物一般要比丧尸厉害，所以往往丧尸都是怪物的盘中餐，但是这些怪物居然会无视这么多丧尸，反而就盯着他们这些活人。
<br/>　　“难道是因为我们的肉比丧尸要好吃吗？”
<br/>　　林峰心中忍不住这样想。
<br/>　　林峰之前听山里的一些猎人说过，有些动物一旦尝过了吃人肉的滋味，那么他就只想着吃人肉了，而不会去吃其余的动物肉。
<br/>　　吼吼
<br/>　　这十几头变异狼龇牙咧嘴，嘴巴里还低着口水，其中一头身上毛是红的，就是这群狼里面的头狼，实力也是最高的，足足有六阶，其余的变异狼也都是五阶上下，这一股狼群可以说非常厉害的存在。
<br/>　　刷刷下一刻一群狼就飞扑而来。
<br/>　　面对狼群的袭击，林峰悍然不惧，拔出斩灵刀就是顺势一刀斩杀而去。
<br/>　　扑哧扑哧扑哧。
<br/>　　瞬间一群狼来的气势汹汹，但是却死的快，他们虽然厉害，但是林峰却是足足八阶的存在，一刀下去，包括那头狼在内的几头狼全部被一刀两半。
<br/>　　剩下的变异狼见此呜呼一声掉头就是跑了。
<br/>　　林峰挖走头狼和几个变异狼脑袋里的晶石也没去追跑掉的，而是去对付黑龙镇内的丧尸怪物。
<br/>　　林峰把军用无人机也拿出来了，他发现这个镇子里有个中学，里面不下于四五百个丧尸聚集在一起。全都是些初中生，穿着极其具有国内特色的校服，估计事发之时，他们都在操场上做广播体操，然后一下子全部感染了。
<br/>　　林峰直接派军用无人机去轰炸。
<br/>　　刷刷刷。
<br/>　　三枚空对地导弹落下后顿时火光爆射。
<br/>　　然而这也吸引来了一群变异鸟，追着军用无人机。
<br/>　　林峰见此立即把军用无人机给收起来了。
<br/>　　“这群变异鸟还真是麻烦。”
<br/>　　林峰掏出火焰符箓对着这群盘旋在天上的变异鸟就是杀了过去。
<br/>　　刷。
<br/>　　火龙冲天而起直接吞没了其中一小部分，但是变异鸟太灵活了，其余的变异鸟翅膀一拍纷纷躲避了，然后在空中掉了一个头就是朝林峰俯冲而来。
<br/>　　“来得好！就怕你不来！”
<br/>　　林峰再次掏出火焰符箓。
<br/>　　随着这一发的使用，火焰符箓直接成为了废纸，里面的灵力消耗殆尽了。
<br/>　　轰！
<br/>　　但是这一击，也罢这变异鸟群给杀的七七八八，剩下的十几只拍着翅膀逃也似的离开了。
<br/>　　见此林峰才再次拿出全球鹰无人机，挂着剩下的五枚导弹在小镇里飞去，哪里丧尸怪物多，就往哪里轰炸。
<br/>　　哒哒哒哒哒。
<br/>　　不断有枪声响起，这是AK47在怒吼，无数子弹倾斜把那些丧尸打的身上一个个都是血窟窿，但是丧尸必须要打爆脑袋才可以死，所以往往更多的子弹都浪费了。
<br/>　　一群人一梭子子弹下去，才能够彻底击毙二三十头丧尸罢了。效率不大。
<br/>　　到最后子弹都打完了，也不过杀了两三百个罢了。
<br/>　　林峰也拿出了自己开空投得到的AK47，一枪一个，后坐力极大的AK47在他手中就好像没有后坐力一样，枪压的十分的完美，一枪一个。这个时候若是有人看到了，肯定会石锤他开了无后坐力挂。
<br/>　　哒！哒！哒！枪的怒吼，犹如死神的呼啸。
