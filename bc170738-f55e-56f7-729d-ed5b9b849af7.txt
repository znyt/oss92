　　第285章
<br/>　　高飞豹腾地一下站起来，看他的意思是要给樊梨花敲锣打鼓庆祝。
<br/>　　“大当家，输就是输了，男子汉大丈夫，可不能输不起。”
<br/>　　程大雷白他一眼，身子如一根箭腾起，借着山石一点，借着马头一点，身体在空中一折，樊梨花眼睛一眨，程大雷的身影已经在眼前消失。
<br/>　　程大雷落在胭脂马上，就在樊梨花身后。
<br/>　　面对樊梨花，程大雷也拿出了看家功夫，这几下鹰飞雀起，当真是神乎其技。
<br/>　　樊梨花贝齿一咬，挥刀向身后抡去，斩的是程大雷的头颅。
<br/>　　程大雷捏住了刀刃。
<br/>　　灵犀一指。
<br/>　　樊梨花感觉刀入山石，陷在其中拔不出来，若论马上较技，程大雷不如樊梨花。但若比气力，樊梨花不如程大雷。
<br/>　　她几次抽刀，都难以动摇。忽地，她使了磨刀法，该抽为推。
<br/>　　程大雷顺水推走，由着她的力气生长，紧跟着，程大雷坐在马上，一手捏刀柄，一手捏住樊梨花左手手腕。
<br/>　　樊梨花的脸腾地红了，贝齿紧扣，一边挥刀一边愤恨骂着。
<br/>　　“狗贼，死啦！”
<br/>　　樊梨花动，程大雷亦动，她旋转身子，程大雷也随着旋转，锋利的刀刃斩不到程大雷。
<br/>　　高飞豹眼睛发亮：大当家高啊！
<br/>　　看二人一举一动，都如影随形，看上去翩翩起舞，举案齐眉。
<br/>　　当然，这只是高飞豹一人的想法。樊梨花此刻的想法，程大雷就像一块狗皮膏药，死贴着自己不放。
<br/>　　她虽然泼辣些，毕竟是女儿身，怎受得了程大雷如此，一股燥意往上涌，顿时有些口干舌燥。火里还夹杂着怒，势要将程大雷一刀两断。
<br/>　　她突然松开绣绒刀，拔出腰间佩剑，隔着胸口向后扎来。
<br/>　　这是存了和程大雷同归于尽的意思。
<br/>　　程大雷一惊：好烈性的女子。
<br/>　　他屈指将樊梨花手中的剑弹飞，樊梨花卖力挣脱，此时手腕一麻，身体一个踉跄，连带着程大雷一下跌落马去。
<br/>　　马下就是水，二人一同坠入水中。
<br/>　　樊梨花终于挣脱程大雷束缚，她仗着好水性，一脚向程大雷踹来。
<br/>　　程大雷拿住了她的脚腕，往自己身前轻轻一带。
<br/>　　“小心了，有蛇。”
<br/>　　浅水区都有毒蛇筑巢，此时被惊动，冲着二人突来。程大雷直接用手捏死了两只，紧跟着小腹一痛，樊梨花的膝盖撞了上了。
<br/>　　“特么地……”
<br/>　　程大雷咬牙切齿，他在拳脚上没有什么修行，无非就是身快力大而已。水性还是不错，不过还是奈何不了樊梨花，程大雷在水底是好好挨了一顿胖揍。
<br/>　　岸边诸人就看到水面上咕嘟嘟冒泡，不时飘上来几条毒蛇的尸体。时而程大雷浮到水面上，时而樊梨花飘上来，至于水里发生什么事，大家都不知道。
<br/>　　过了良久，程大雷浑身湿漉漉的从水底爬上来，只见他鼻青脸肿，头顶还挂着水草。
<br/>　　水面上一人分波开浪，向远处游去，正是那俏女侠樊梨花。
<br/>　　“大当家，你没事吧？”高飞豹关心问道。
<br/>　　“没事，没事。”程大雷看着水面发呆。
<br/>　　“没事，大当家你被人揍得这么惨还说没事？”高飞豹大声道。
<br/>　　程大雷很是无语，忽地向水面上高声喊道：“樊家姑娘，你的马和刀去哪里还你？”
<br/>　　樊梨花拍水而行，抬手向后挥了挥，也不知听见了没有。
<br/>　　程大雷揉着发肿的脸颊，挥挥手道：“查一查，这姑娘哪里来的，总不会是从石头缝里来的。”
<br/>　　樊梨花真就一去不回头了，再没有她的消息。程大雷恍然发现一个事实，越是等级高的人物，想要收服也就越加困难。
<br/>　　廖甲廖乙是对自己死心塌地，而像赵子龙是只认刘悲，收服林冲的时候是差点将命丢了，如此说来，还不能太冲动的将绝世以上的人物召唤出来，召唤出来也未必听自己的。真出来个吕布之类的猛将，怕自己根本拦不住对方。
<br/>　　只是不知道这樊梨花究竟跑到哪里去了，不过眼前大战在即，程大雷也没办法为对方过分分心，还得抓紧时间为即将来临的战斗做准备。
<br/>　　……
<br/>　　事情又过去两三天，这一日程大雷收到江湖传言，以窦竹仝为首的十八家山寨，领兵一万向自己攻来。
<br/>　　这一次，他们选择在白天进攻。黎明时分，水面泛起大雾，随着太阳升起，晨雾渐渐散去，显出一艘艘船。
<br/>　　或大或小，大的三帆齐挂，小的只能容两人划桨，一时间河面上密密麻麻，都是大小船只。
<br/>　　“这看上去没有一万人呐？”程大雷站在瞭望台上往下望：“至多也就三五千人吧？”
<br/>　　“大当家，就算是三五千咱们也守不住？”徐神机道：“咱们可只有一百人。”
<br/>　　“怕什么，我不是说了自有神兵天降么。”程大雷道：“忙活起来，忙活起来，先把人放近了打。”
<br/>　　徐神机不知程大雷打的什么主意，只能下了瞭望台，按照事先的部署防御。
<br/>　　在船只当中，一艘打着窦字旗的大船最是醒目，窦竹仝立在船头，握着钢刀咬牙切齿。
<br/>　　“姓程的，你今日给我纳命来。”
<br/>　　蛤蟆岛的地形极其平坦，却并不易防御，对方冲任何一处进攻登岸都是可行的。今次，三四千人，数百艘大小船只，已经将蛤蟆岛团团围住，十八家寨主，各自选择了一地进攻。
<br/>　　徐神机看到这一幕，心情砰砰打鼓，但事到临头，已无退路。
<br/>　　“放！”
<br/>　　众人潦草放箭，放了一阵，就赶紧撤入蛤蟆岛腹心。
<br/>　　守岸是守不住的，兵力根本铺不开，程大雷也根本没打算守。但岛上是适合展开一场坚守战的，一百多号弟兄，直接退入坚固的堡垒内。
<br/>　　攻上岛的这些人也有些奇怪，从前岛上一马平川，可如今不过几天时间，这些屋子厚墙是如何冒出来的。
<br/>　　但这些也不重要，只要登上岛，他们就取得了这场战斗的胜利。
<br/>　　程大雷窝在墙后，徐神机如铁锅上的蚂蚁，寄的团团乱转。
<br/>　　“大当家，神兵呢，神兵呢，你说的神兵天降呢。”
<br/>　　“着什么急，不就是神兵么，看本当家请动天兵。”程大雷举手向天，心底轻吐两个字：点将。
<br/>
