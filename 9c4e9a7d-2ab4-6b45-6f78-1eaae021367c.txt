　　楼房玻璃窗也被这连续的爆炸给震碎，玻璃碎片朝着楼下水泥地坠落下去，楼房下原本准备靠近楼房的影子部队士兵不得不闪避躲开。
<br/>　　即便是提起做好了闪避，但罗昊和李然两个人脸颊还是被从楼道中吹出来的热风吹得生疼。
<br/>　　李然从地上爬起来对罗昊问道：“你是想把这栋楼房给炸塌了吗？”
<br/>　　“你刚才的话不就是这个意思吗？”罗昊眨巴着眼睛对李然反问道。
<br/>　　李然愣了愣，随后咧嘴一笑，说道：“好吧，我承认，我的确是这个意思。”
<br/>　　罗昊看了眼那个还在冒着浓烟的楼梯口，对李然说道：“一劳永逸，现在估计他们不会再想着从这个地方冲上来了。”
<br/>　　解开地上那两具影子部队士兵尸体上M4A1自动步枪的枪带，把其中一把枪扔给李然，从他们战术背心中拿出备用弹匣和手雷装入自己的战术背心中。
<br/>　　对于罗昊和李然来说，现在最紧缺的就是弹药。
<br/>　　这次他们是潜入突袭式行动，所以只携带了0.5个基数的弹药，现在进行阵地式防御战，弹药就成了他们最大的问题。
<br/>　　李然接过罗昊扔给自己的M4A1，背在身上，从身上拿出信标，准备放置到楼顶四个角上，为后续抵达的Bravo小队标明位置。可是就在李然放置第二个信标的时候，一道从地面射上楼顶的绳枪引起了李然的注意。
<br/>　　李然侧身探出身体往下看去，看到一个影子部队士兵正抓着绳索网上攀爬。
<br/>　　地面上的影子部队士兵看到李然，一串子弹就朝着李然扫来，将李然逼退回去。
<br/>　　“该死的！”
<br/>　　李然嘴里咒骂一声，矮着身体，走到那根勾在楼顶边缘的绳索走去，拔出军刀，一刀砍在绳索上，将绳索砍断，那个正抓着绳索网上攀爬的影子部队士兵大声惊叫着跌落下楼房，楼房下传来“嘭”的一声闷响声；可是李然刚把这根绳索砍断，就又有绳索从地面射上楼顶，不一会儿，楼顶四周就全都是绳枪上的铁爪。
<br/>　　“草！”
<br/>　　罗昊嘴里也跟着爆出一句粗口，显然也是发现了这么多射绳枪后，明白了楼房下那些影子部队士兵想要干什么了。
<br/>　　利用射绳枪从楼房外墙爬上楼顶，就如同古代打仗时的攻城部队一样，这对于守城部队来说，那些爬着梯子试图往城墙上爬的攻城部队就是活靶子；只是现在的情况却又不一样，因为楼顶上只有罗昊、李然、李毅峰、慕容飞四个人，而军事监狱中的影子部队士兵数量却是他们的几十倍甚至上百倍，再加上地面上有火力掩护，即便是有人会死，但是只要有位置空缺，就立刻会有影子部队士兵补上去，四个人是无法将整个楼顶都防御下来的，总会有人能爬上楼顶。
<br/>　　来回跑了几个位置，阻止了往楼顶攀爬的影子部队士兵后，李然很快就意识到这样下去不是办法，立刻命令楼顶上四个人收缩防线。
<br/>　　罗昊抓着Parker拖到楼梯口，然后跟李然、慕容飞、李然四个人相互背对，防御自己方向爬上来的影子部队士兵。
<br/>　　“噗噗！”，“噗噗！”，“噗噗！”，“噗噗！”
<br/>　　“噗！”，“噗！”，“噗！”，“噗！”
<br/>　　四个人不断对着沿着射绳枪爬上来的影子部队士兵射击，刚开始那些爬上楼顶的影子部队士兵还没来得及翻过栏杆，就被子弹放倒，摔落下楼顶；但是随着爬上楼顶的影子部队士兵越来越多，罗昊、李然、李毅峰、慕容飞四个人火力不足的弱点逐渐显现出来。
<br/>　　罗昊拔下空弹匣，将旁边用胶布固定的备用弹匣塞入弹匣槽，拉动枪栓，将子弹上膛，对身边其他三个人说道：“楼顶守不住了，把‘八爪鱼’带进楼房，我们在那里继续防御。”
<br/>　　李毅峰一把抓住Parker的衣服，把他拖进墙壁上都被炸出裂缝的楼梯，李然、罗昊、慕容飞三个人交替掩护着向楼梯中撤退。
<br/>　　就在这个时候，一颗闪光震撼弹落在罗昊前方不远处！
<br/>　　“闪光弹！！”
<br/>　　罗昊大声叫喊，楼梯的铁门在刚才的连环爆炸中已经被炸毁，此时罗昊、李然、慕容飞三个人就彻底暴露在那颗闪光震撼弹之下。
<br/>　　“哐！”
<br/>　　闪光震撼弹爆炸。
<br/>　　刺眼的白光和刺耳的噪音，令得罗昊、李然、慕容飞跌倒在地上。眼前白茫茫的一片只能看到模糊的影子，耳朵边也是嗡嗡作响，听不到耳麦中的声音，甚至连自己喊得话都听不见。
<br/>　　罗昊左手撑着楼梯，支撑着自己的身体，右手单手握着G36C自动步枪，眼前模糊的人影射击着，但毫无疑问，罗昊此时的射击根本没有任何的精度可言。
<br/>　　爬上楼顶的影子部队士兵朝着蜂拥而去。
<br/>　　通过战场视频系统，观看着军事监狱中行动过程的詹姆斯、宋萱、云熙等人看到屏幕中的画面，全都紧张起来，云熙更是双手紧握，两只手冰凉。
<br/>　　“e-on！e-on！e-on！”
<br/>　　詹姆斯嘴里不断地重复着这句话，现在除了把希望寄托在负责接应支援的Bravo小队身上，没有任何的办法。
<br/>　　军事监狱上空，“黑鹰”直升机快速逼近，驾驶舱中，飞行员看到了李然放置在楼顶上的红外信标，通过无线电对机舱中的Bravo小队说道：“我看到他们了！”
<br/>　　麦克米兰看到了楼顶上密密麻麻的影子部队士兵，说道：“他们遇到麻烦了，伙计们！”
<br/>　　端起HK416A5自动步枪，对着楼顶上两个影子部队士兵打出两个点射。
<br/>　　机舱门口，控制着M134机枪的机组成员，按下了机枪发射键，六根黑色的枪管开始缓缓转动起来，电动马达所发出的声响，就如同死神对楼顶上那些影子部队士兵所发出的召唤。
<br/>　　“哒哒哒哒哒哒哒哒哒哒……”
<br/>　　M134机枪开火了，令人恐怖的射速下，从M134机枪六根枪管中飞出的子弹就像是连接成了一条夺人性命的鞭子一样，甩向楼顶上的影子部队士兵。
