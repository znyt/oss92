　　“你人头猪脑吗？那些死人来看儿孙，都是逢年过节之类，不可能时刻跟他在一起。这三个小鬼在人间没走，长期跟弟弟在一起，而且本身也是灵智未成，不知道轻松，三个去找他一个，他们只顾亲近，哪里知道这样做给孩子带来的损伤，不过他们也略微有所察觉，因此每天晚上都会用他的身体吃奶，让他勉强能活下去。”
<br/>　　说到最后一句的时候，叶少阳想到，那三个小鬼这么做，可能也是想借用弟弟的身体，感受一下没怎么享受过的父母之爱。总之，他坚信那三个小鬼不是故意要害人。
<br/>　　张道长想了半天，郑重地对叶少阳说道：“你是对的。”
<br/>　　叶少阳耸了耸肩膀。
<br/>　　“但是，既然如此，又怎么样呢，为了保护活人，我们还是要灭了这三个小鬼……”
<br/>　　叶少阳摇摇头，“我解释这么多，就让想说，要超度他们，而不是杀了他们。”
<br/>　　张道长和他的弟子们一听都怔住了。张道长沉声说道：“他们是邪修的恶鬼，修为都不错。”
<br/>　　叶少阳道：“那又怎么了，是他们自己想要邪修的吗？还不是他们的父亲巧合地把他们埋在了养尸地里，才有了今天的他们。”
<br/>　　张道长道：“但他们已然成了恶鬼，必须灭了，不然还能超度不成？”
<br/>　　叶少阳道：“答对了，我就是要超度他们。”
<br/>　　张道长冷笑，“而今这种情况，超度他们，比灭杀他们的难度要大的多。”
<br/>　　“我知道。”叶少阳平静的回答。
<br/>　　这种态度让张道长气不打一处来，说道：“他们在人间有牵绊，你怎么超度，就算强行超度，他们若还留恋父母亲人，还是会找机会上来的，到时候可就更麻烦了。”
<br/>　　叶少阳道：“至少有一种办法，能了却他们在人间的牵挂，让他们安心去阴司报道，也不会再找机会回来。”
<br/>　　张道长一听，眉头立刻皱了起来，死死盯着他，用眼神在问：什么办法？
<br/>　　叶少阳道：“斩断魂之羁绊。”
<br/>　　张道长愣住了，他的那些弟子也都愣住了，朝他们的师父看去。叶少阳和张道长谈话的内容，已经是道门中很深入的知识，他们早就是一脸懵逼。
<br/>　　“不可能，我没听说过有这样的法术，魂之羁绊，无形无相，怎么斩断，用什么来斩？”
<br/>　　叶少阳冲他笑了笑，道：“我有办法让魂之羁绊现形，不过解铃还需系铃人，只有也处在同一羁绊之内的人才看得见，才斩得断。”
<br/>　　“什么！我怎么没听说过有这样的法术！”
<br/>　　“因为你出身普通，自身实力也有限……我这么说不是贬低你，因为这种神妙的法术，普天下百分之九十九的法师都不知道。”
<br/>　　张道长冷哼一声：“那你凭什么知道？”
<br/>　　叶少阳盯着他，静静地说道：“凭我是茅山天师。”
<br/>　　万道归宗，茅山正朔。茅山与龙虎山，是道门法术门派中里两大魁首，地位就像是大学中的北大和清华，只要是法师，没有不服气的，这两个门派并不是靠人多势众——茅山一直秉承单传，同代内门弟子最多两人，选其一继承掌教之位。
<br/>　　这一两个茅山道士，在法术界虽然绝对是泰斗级的人物，但因为处在高端，在民间却是没什么名声，普通法师一般都没有听过。
<br/>　　茅山和龙虎山真正最有名的，是各自门派的秘传法术，这些法术都来自宗门千年的传承和发展，各代天师在总结前人法术的基础上，又会自创出一些法术。虽然部分会失传，但也有不少保留了下来，其中大部分都已公开，别的门派也可以修炼。
<br/>　　但是有一些最核心的内门法术，就像武侠小说里的门派秘籍一样，那是绝对不会公开的。叶少阳所说的能斩断魂之羁绊的法术，就是茅山内门法术的一种，张道长这种普通道门出来的弟子，自然不会知道。
<br/>　　张道长定定地望着叶少阳，目光变得愈发复杂起来，半晌喃喃说道：“我怎么相信你？”
<br/>　　叶少阳道：“帮我布阵，斩了魂之羁绊，我让你来超度三个恶鬼，阴德送你。到时候你总信了吧？”
<br/>　　张道长心中一动，道：“你不要阴德？”
<br/>　　“无所谓。”叶少阳耸了耸肩，实际上他这么说，也是故意卖个面子：自己是这个世界的外来者，根本没有办法超度亡魂。这一点之前就是证明了的，干脆就卖个面子给张道长，也赚个人情。不过说起来，超度三个恶鬼，也就是几百年的阴德，就算给自己，也真的是可有可无，但对于一般法师，也算得上是一份很丰富的收获了。
<br/>　　张道长虽然怀疑他的身份，主要是他的实力与茅山天师的身份完全不对等，不过，想到这一切待会作法的时候就会知道了，张道长也没有多问，同意了合作。
<br/>　　在叶少阳安排下，张道长带着他的弟子们在院子里开始布阵，叶少阳在最中间的乾位上，彼此呼应，将天地三才阵布置起来。
<br/>　　叶少阳定位之后，布置了一番，然后来到一直在槐树下呆着的刘四等人身边。
<br/>　　刘四夫妇，还有刘老汉，全程听了之前叶少阳等人的对话，只是不懂那些法术界方面的知识，一知半解，懵懂地看着叶少阳。
<br/>　　见叶少阳走来，刘四媳妇立刻泪眼婆娑地询问他有关那三个孩子的事。
<br/>　　叶少阳如实相告。夫妻二人想到死去的三个孩子，忍不住抱头痛哭。
<br/>　　叶少阳拍着刘四的肩膀，说道：“那三个孩子毕竟是你生下的，虽然现在是鬼，但还留在人间，与你还算有父子关系，你可愿超度他们去阴间，有机会投胎，再世为人？”
<br/>　　这番话说得十分浅白，刘四一个普通人也听懂了，急忙点头不迭，哭着说道：“是我的错，把他们埋在这里，才会变成孤魂野鬼，我当然想让他们能够去投胎啊，求道长成全啊！”
<br/>　　刘四也顾不上尴尬，跪下对着叶少阳磕头起来。
