但是厉老夫人冷声打断了她，“不要叫我妈，如果不是看在嫣然的面子上，我根本就不会让你进我们厉家的大门！”
<br/>
<br/>        “我们厉家是百年簪缨世家，历年来的当家主母都不求门第，而重人品，当年你喜欢君墨就给君墨下药，是为不知廉耻，你借腹上位，嫁进厉家，是为心机叵测，后来你又耐不住深闺寂寞，做下了做事，简直枉为人妻枉为人母！”
<br/>
<br/>        “天下间的好事不可能全部被你占了，君墨不喜欢你，你是一开始就知道的，你用尽心机嫁给君墨，还妄想君墨跟你浓情蜜意，我们厉家对你们夜家已经仁至义尽，这些年间君墨一手将夜家捧成了帝都四大豪门之一，更在风雨飘摇之际做了你们做强有力的后盾，夜滢，你也该知足了。”
<br/>
<br/>        “你口口声声说我和君墨不喜欢嫣然，不重视嫣然，难道嫣然身上顶着的不是厉家的荣耀吗，你给君墨戴了一顶绿帽子，君墨和我难道不是看在嫣然还未长大的份上没有给你一纸离婚书，将你逐出厉家大门吗？”
<br/>
<br/>        “别以为我看不出你的小心机，你还想和君墨重修于好，再给君墨添个儿子，这简直是痴心妄想了！”
<br/>
<br/>        厉老夫人这一段又一段的犀利话语重重的砸了下来，直接让夜滢双腿一软，她所有的幻想都破灭了，她知道自己终其一生都无法拥有她想要的一切了。
<br/>
<br/>        她不甘心。
<br/>
<br/>        她太不甘心了！
<br/>
<br/>        “妈，既然你心里还是有嫣然的，但你为什么还要认夏夕绾当你孙女，她究竟给你下了什么迷魂药？”
<br/>
<br/>        夜滢真的不懂这个夏夕绾身上究竟有什么魅力，不但男人为她倾倒，就连老夫人都被她折服。
<br/>
<br/>        这个问题让厉老夫人脸上一暖，其实厉老夫人也不知道她为什么这么喜欢夏夕绾，在海城她见夏夕绾的第一眼就喜欢上了这个眼眸澄澈聪慧而灵动的女孩，当时她的第一个念头就是想认夏夕绾为孙女。
<br/>
<br/>        现在这个念头越发的强烈，她想要带夏夕绾进厉家，让夏夕绾冠以厉姓。
<br/>
<br/>        厉老夫人觉得自己可能真的被夏夕绾给下了迷魂汤吧。
<br/>
<br/>        “夜滢，我心意已决，你就不要再在绾绾丫头的问题上跟我做纠缠了！我乏了，你出去吧！”
<br/>
<br/>        这时门外的李嬷嬷打开了房间门，做出了“请”的手势，“夫人，你可以出去了。”
<br/>
<br/>        夜滢咬了一下牙，然后心不甘情不愿的离开了这个房间。
<br/>
<br/>        ……
<br/>
<br/>        夜滢走了，李嬷嬷走进来，“老夫人，我当初就说了这个夜滢心术不正，配不上少爷，少爷风华无双，夜滢简直就是少爷身上的一个污点。”
<br/>
<br/>        厉老夫人叹息了一声，“这大概都是命吧。”
<br/>
<br/>        这时“叮铃铃”的电话铃声响起了，来电话了。
<br/>
<br/>        李嬷嬷迅速笑道，“老夫人，应该是少爷打来的电话，少爷虽然今天没回来，但是惦记着你的生日呢。”
<br/>
<br/>        李嬷嬷是随着厉老夫人陪嫁进来的大丫鬟，所以这些年都没有改口，一直称呼厉君墨“少爷”。
