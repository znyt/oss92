　　　　须弥空间的珍惜程度，简直是不亚于一些天级灵器，甚至是比起天级灵器都要珍贵的多。
<br/>　　从远古时代，便是寥寥无几。
<br/>　　须弥空间，只是可以盛放活物这一点，便是确定了他的价值。
<br/>　　像是一些武者，若是得罪了一些大势力，被人追杀的话。
<br/>　　若是躲在这须弥空间之中，便是就可以躲过一劫。
<br/>　　一些巨大的须弥空间，更是可以在其中蒙养妖兽，可以随时召出来的对敌，是绝对的底牌。
<br/>　　不过眼前这须弥空间，看起来倒是没有那般巨大，不过却也可以让几个人进入其中去。
<br/>　　“这须弥空间中，似乎有人？”
<br/>　　楚风眠看了一眼其中，却是发现一个人影，这里面的人，似乎是被夺花尊者下了禁制，封印了起来。
<br/>　　楚风眠大手一抓，便是从这须弥空间中，将这个人抓了出来。
<br/>　　这人影看起来是一名少女，她一出现，便是突然大骂道。
<br/>　　“该死的魔头，想要绑架我来换取我们虚家的赎金，想都别想！今日不放了本小姐，我们虚家会将你们彻底毁灭！”
<br/>　　这声音尖锐到极点，极为蛮横。
<br/>　　听到这声音，楚风眠三人的表情，都是变的极为古怪。
<br/>　　这少女，不是别人，正是曾经跟楚风眠见过，还跟有着冲突的蓝衣女子。
<br/>　　“怎么会是你们？地魔门的人呢？”
<br/>　　那蓝衣女子，睁开眼睛，突然看见楚风眠三人，也是一愣。
<br/>　　随后她又扫视了周围一圈，便是看出了刚刚发生的战斗。
<br/>　　“是你们杀了地魔门的那些魔头？”
<br/>　　蓝衣女子冷哼一声，看着楚风眠，大声喊道。
<br/>　　“还不快解开我的禁制。”
<br/>　　“恩？”
<br/>　　听到这蓝衣女子的话，楚风眠不由眉头一皱。
<br/>　　对着蓝衣女子，楚风眠的确是没有什么好感，不过楚风眠跟她也没有什么大恩怨。
<br/>　　如今倒是机缘巧合下救了她，不过这等蛮横，的确是让人不爽。
<br/>　　不过楚风眠还是心神一动，一股灵力打出，将这蓝衣女子身上的禁制打破。
<br/>　　禁制被破，蓝衣女子身上的气息立刻恢复，脸上露出了几分轻松，看向楚风眠，突然开口道。
<br/>　　“我叫虚堇，你们是谁？怎么会出现在这里。”
<br/>　　那蓝衣女子虚堇，看着楚风眠有些疑惑道。
<br/>　　她虽然跟楚风眠一行人有过一面之缘，却是没有交谈过，自然不清楚楚风眠一行人的身份。
<br/>　　“我们是来自与秦皇国度，准备是前往北邙学院，参加考核。”
<br/>　　楚风眠平静的开口道。
<br/>　　“这次是横渡虚空过来，遇到了那夺花尊者。”
<br/>　　“夺花尊者是你们杀的？”
<br/>　　那虚堇，看着楚风眠，眼神中有些质疑，她本来以为，楚风眠应该是什么好运，捡到了这须弥空间，才将她放出来的。
<br/>　　毕竟夺花尊者，可是足以媲美生死境的人物，她在夺花尊者的手中，都没有什么反抗之力，一下子都败在了夺花尊者的手中。
<br/>　　楚风眠一行人，一个御风境巅峰的武者，两尊神力境七重的圣者，怎么可能跟夺花尊者抗衡。
<br/>　　她自然不可能相信楚风眠一行人有着斩杀了夺花尊者的实力。
<br/>　　“我们遇到夺花尊者的时候，他已经是重伤，所以我们才能杀了他。”
<br/>　　楚风眠也懒得解释，直接开口道。
<br/>　　反正这虚堇，是敌是友也不清楚，楚风眠当然懒得解释什么。
<br/>　　“哼。”
<br/>　　虚堇听到楚风眠的话，眼神更加不屑了许多。
<br/>　　要是能够杀了夺花尊者的话，那她倒是还要高看楚风眠一行人一眼，不过只是重伤，捡了个便宜，她自然毫不在乎。
<br/>　　“对了，你们杀了夺花尊者，那须弥空间是不是在你们手中，那是我的东西，还给我！”
<br/>　　虚堇突然眼神一转，露出几分贪婪，冷声说道。
<br/>　　“这是我们虚家的东西，请你立刻交还回来！”
<br/>　　“什么？这须弥空间，是我们从夺花尊者手中得到的，跟你有什么关系！”
<br/>　　寒月漓在一旁，都是不由的皱了皱眉说道。
<br/>　　须弥空间，这等宝物，怎么可能因为这虚堇一句话就给她。
<br/>　　更何况看起来，这须弥空间，也不可能是这虚堇的。
<br/>　　“我说是我的就是我的！”
<br/>　　“把须弥空间拿过来！”
<br/>　　虚堇冷哼一声，一身灵力猛然爆发出来，想要对着寒月漓压迫过去。

<br/>　　顿时这两股灵力碰撞，虚堇的灵力，被压制的死死的，根本无法反抗。

<br/>　　“我们走吧。”

<br/>　　“不过属下担心，她的话中有诈，毕竟以此人的性格，不可能会乖乖报恩才对，少主还要谨慎行事才行。”
