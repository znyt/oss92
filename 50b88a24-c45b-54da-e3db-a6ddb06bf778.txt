　　贞观十九年的除夕很是别致，百姓，官员，皇族，所有人聚集在一起，大家聊着彼此关心的话题，时不时发出一阵爽朗的笑声。
<br/>　　孩子们在大人们身边撒着欢的奔跑着，发泄着他们永远都用不完的精力，他们是大唐的未来，看到他们李承乾承仿佛看看到了童年的自己。
<br/>　　百姓们慢慢的也都放松了精神，开始有说有笑起来，皇宫中的各式彩灯流光溢彩，让人流连，这是一次难得的机会，他们会永远记住这一天。
<br/>　　那些大唐的官员们虽然并不认为皇帝陛下有这样做的必要，但随着时间的推移，看着那些百姓脸上洋溢的笑容，他们突然发现似乎皇宫开放之后也没有什么大不了的，百姓开心，皇帝开心，一切都是那么和谐。
<br/>　　那就这样继续下去吧，左右不过就是开放皇宫而已，又不是让百姓参与到政事之中，大家还是像以前那样该干什么干什么，何必惹的皇帝不高兴。
<br/>　　事情想开了，也就变的简单了，文臣武将的全都不再纠结于皇宫开放日，转而开始讨论上元节过后的长安杯马球大赛。
<br/>　　得到冠军的话，那可是十万贯奖金，这已经不是什么小钱钱了，对于一个小家族来说，几乎够他们赚一辈子的。
<br/>　　……
<br/>　　不过要说感触最深的还是那些刚刚归附大唐不久的异族，他们的归附大部分还是被形势所迫，打又打过，说也说不通，最后只能投降。
<br/>　　但这些人其实并没有直的对未来死心，一个个还都惦记着什么什么能够复国，脱离大唐的控制，毕竟谁也不喜欢自己的头上有个‘太上皇’在发号司令不是。
<br/>　　可是贞观十九年的除夕却给他们上了很好的一课，撇开那赫赫军威不谈，单说大唐的富庶就不是他们能够反抗的。
<br/>　　数千年以降，这些人从来就不知道灯头是可以朝下的，看着那一盏盏怎么吹都吹不灭的电灯，看着那四周繁花似锦的景色一个个眉头狂皱。
<br/>　　“这就是在炫耀，有什么了不起，如果不是有我们新罗人帮忙，大唐绝不会有今天这么强大。”一个新罗籍的家伙身着身边的同伴，狂傲的说道：“而且你们知不知道？李唐皇室的身上至少有八分之一我们新罗人的血脉，所以严格说来，李唐皇帝也可以说是我们新罗人。”
<br/>　　这牛、、逼吹的，还能再大点不？
<br/>　　来自天竺的黑胖子实在听不下去这家伙在身边唠叨，忍不住对他说道：“你怎么说都好，不过这话最好不要让大唐人听到，否则你的脑袋可就要保不住了。”
<br/>　　“我说的是事实，如果李唐皇帝没有我们新罗人的血统，他们绝对不可能统一这么大一片国土。”
<br/>　　“那你敢不敢大点声，最好是喊出来！”黑胖子往嘴里塞了一块糕点，吱吱唔唔说道。
<br/>　　“我又不是傻子，皇帝不想承认这件事情，如果我给揭发出来立刻就是掉脑袋的大罪，我才不傻呢。”
<br/>　　“不敢就是不敢，原来新罗人都是胆小鬼，这样好了，那边就有禁军，我帮你说一下怎么样？”黑胖子将嘴里的东西咽下去，指了指不远处标枪盘耸立的禁军，然后，身边顿时清静了，也不知道刚刚那货又跑到什么地方去折磨人去了。
<br/>　　如此情况让黑胖子暗暗叹了口气，大唐还是太强大了，强太到让人羡慕的鬼迷心窍，如果什么时候天竺也能如此强大，估计就能复国了吧？
<br/>　　不过还是算了，暂时就这样吧，等到无数年后的后人来实现这个梦想好了！
<br/>　　所有的异族其实都知道，他们眼前的一切其实就是用钱堆起来的，而钱这东西有些时候是可以通神的，只要有钱，大唐就不会畏惧打仗，而只要打仗，大唐又总是能赚到钱。
<br/>　　如此循环下来，似乎只有等苍天开眼的那天，让大唐出一个傻子皇帝，也只有等到那个时候，各国才会有机会复国。
<br/>　　太极宫中那些被邀请来的异族贵族各怀鬼胎的想着心事，强撑笑脸的迎合着大唐皇帝，尽管心中有再大的报负，也不敢在脸上表现出来。
<br/>　　……
<br/>　　除夕夜的宴会对李承乾来说无疑是成功的，至少在大唐百姓的心目中，大唐皇帝的形象变的更加真实，这就让他很满意了。
<br/>　　另外还有一点就是长安杯马球赛已经到了最后的准备时间，各大家族和世家正在组织人手拼命的训练，而皇室的队伍也在抓紧时间训练。
<br/>　　张启元，飞凤军的一个队长，不过现在他的任务是带着二十多个手下兄弟杀出重围，夺取长安杯的桂冠。
<br/>　　训练场上，张启元拼命的催动战马，时而急停，时而急转，时而加速，只有拳头大的马球一直被他和他的队伍控制，向着对面的‘敌人’的球门冲杀过去。
<br/>　　“这小子不错，怎么发现他的？”李承乾站在赛场的边缘，看着场中的正在训练的队伍，对身边的长乐问道。
<br/>　　长乐看了一眼身边的大哥，笑着说道：“武媚推荐的，说这家伙马上功夫不错，现在一看好像那丫头说的有些保守了。”
<br/>　　“别那丫头那丫头的，那可是你嫂子。”李承乾翻了个白眼说道。
<br/>　　“她比我小啊，叫她丫头也没有什么错。”长乐诡辩道。
<br/>　　李承乾真的是拿自己这个妹妹没有办法，咂咂嘴琢磨了一会儿，最终放弃说服她，岔开话题说道：“你是怎么想的？真打算让你的球队拿冠军？”
<br/>　　“这有什么不对么？如果不能得到冠军，我皇室的脸面往哪里放！”长乐对李承乾的问表示十分诧异，看着他的目光满是疑惑.
<br/>　　“我觉得你应该把眼界放宽一点，最好可以再组织一个队伍，一个可以代表大唐的队伍，而非代表某一家的队伍。”李承乾指了指赛场中间，然后对长乐说道：“我并不认为皇家的队伍就可以代表大唐，它只是代表皇室而已，或者说代表了我李家。”
<br/>　　“李家就是大唐啊，这有什么不对么？”长乐几乎不敢相信自己的耳朵，她实在无法相信自己的哥哥竟然会说出这样的话。
