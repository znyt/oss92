        “是不是你最近工作太劳累呢？”夏玄灵一边念叨，一边拿着红糖去了茶水间。
<br/>
<br/>        裴金澈望了一眼夏玄灵的背影，又回眸看向连华生，不冷不热地说道：“好走，不送。”
<br/>
<br/>        “一句谢谢都没有，真没良心。”连华生摆摆手，只身往门口走去，出门前还跟夏玄灵打了声招呼，“玄灵，我先走了。”
<br/>
<br/>        “谢谢你啊！都没来得及给你泡茶的！”夏玄灵扯高了音调喊道。
<br/>
<br/>        连华生笑了笑：“咱们都什么关系了！还泡茶！你好好照顾阿澈吧！”
<br/>
<br/>        说完，连华生就走了。
<br/>
<br/>        裴金澈又软瘫在了沙发上，脑海里回忆着这一个月来在实验室里所发生的事。
<br/>
<br/>        一片纯白的实验室，无论是地面还是实验室台面都是一尘不染，各种试管参差有落地摆放，各种仪器滴滴答答地作响。
<br/>
<br/>        那个女人就躺在无菌的透明玻璃箱里，身上只盖着一条白色的棉布，而这白色棉布下，她的身体插满了各种针管，粘满了各种仪器的探头。
<br/>
<br/>        在他与她的血液交换循环的作用下，她身上的各种伤口、病痛，除了那颗已经移除的肾无法复原以外，其他都已经痊愈，唯独她的大脑仍旧处于昏迷状态。
<br/>
<br/>        大脑是复杂的，他的血液与她的血液融合后，修复她的大脑需要更多的时间。
<br/>
<br/>        其实，这个女人，他小时候就见过。
<br/>
<br/>        只不过，他那个时候第一次见到她时，她还是培养皿中被筛选出来的一颗最优质的胚胎。
<br/>
<br/>        然后，她比他幸运，她被移植到了一个有背景的名门望族的大家庭里，成为了那个家里的少小姐。
<br/>
<br/>        而他却只是移植到实验室里一个姓裴的女医生的腹中，那女医生在生下他后，只不过是赐了他一个姓，却从未把他当亲生儿子看待。
<br/>
<br/>        因为，他在他们的眼里，就只是一个实验品。
<br/>
<br/>        他对“夏玄灵”的情有独钟，就是源于小时候在实验室里的那段记忆，因为羡慕，以至于他长大后，对现在的这个夏玄灵暗生情愫。
<br/>
<br/>        童筱颖苏醒的那一刻，在与他对视时，将他年幼那残忍得不堪入目的童年唤醒。
<br/>
<br/>        刚刚苏醒的她，犹如初生的婴儿，眼神纯净得天真无邪。
<br/>
<br/>        在实验室里，她很乖巧、很听话，就因为她是解药的母体，实验室里的科研人员对她的照顾无微不至。
<br/>
<br/>        但是，在裴金澈看来，她只不过是他们的实验品罢了。
<br/>
<br/>        他不堪自己一个人默默地承受这一切，便将自己和她是实验品的事情偷偷地告知了她，这才引得她暴躁不安，反抗他们在她身上采血。
<br/>
<br/>        结果，他没料到自己会害到她，连华生的父亲决定将她变成植物人，让她永久沉睡。
<br/>
<br/>        这就是事情的真相，一个让他无法逃离的噩梦。
<br/>
<br/>        如果不是他这么多年来一直在装“乖”任由他们摆布，只怕他早就变成了植物人，哪会有今天的成就。
<br/>
<br/>        但是，哪怕是他有了一半的自由，也逃脱不了那个实验室。
<br/>
<br/>        连华生每次替他父亲，将药剂带给他。
<br/>
<br/>        因为，没有他们的药剂，抑制他体内的代谢，否则他会过早衰老。
<br/>
<br/>        说白了，他只不过是他们实验里的一个失败品而已，而童筱颖是一个成功品，他这个失败品最终成为了成功品的辅药。
<br/>
<br/>        夏玄灵将红糖水端了过来，在裴金澈的身旁坐下：“已经不烫了，你快喝了它。”
<br/>
<br/>        她的话，拉回了裴金澈的思绪，裴金澈苦笑了笑，接过夏玄灵手中的杯子，将杯中的红糖水一饮而尽。
<br/>
<br/>        “我去给你把饭菜再热一下。”夏玄灵从裴金澈手中拿走空杯子，起身微笑着说道。
<br/>
<br/>        裴金澈眸光黯然地点了点头。
<br/>
<br/>        离开裴金澈的公寓后，连华生在深更半夜的时候，去了自己父亲的实验室，原来他来这里是进出自由的，但是现在他的指纹失效了，半年前就这样子了。
<br/>
<br/>        只不过他之前没在意，但是听了裴金澈的那些话后，他上心了。
<br/>
<br/>        就算他盗用了他老爸的指纹，还有面部扫描那关，他过不了，要想把童筱颖救出来，还真是难上加难，他得想办法通知傅夜沉才行。
<br/>
<br/>        现在，唯一能救童筱颖的人，也就只有傅夜沉了。
<br/>
<br/>        只是，连华生还是有些小担心，万一傅夜沉把他爸的实验室炸了怎么办？
<br/>
<br/>        连华生站在实验室的大门外，一边思忖着，一边左右徘徊的时候，大门内突然间传来了细微的类似于玻璃破碎的声音。
<br/>
<br/>        会不会是他产生了幻觉？
<br/>
<br/>        连华生狐疑地朝大门凑近了一些。
