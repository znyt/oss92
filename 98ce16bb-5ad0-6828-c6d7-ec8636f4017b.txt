　　可如今一看，楚风眠已经是踏入了八劫古帝境界，短短几个月的时间，实力攀升如此之快，超出了他的想象，就算是见识过无数天才的碧螺老祖，都无比吃惊。
<br/>　　这可是一步八劫古帝啊，一步七劫古帝，便是绝世，一步八劫古帝，都堪称神话了。
<br/>　　这样的天才，已经是不存在于历史了，而是在神话之中。
<br/>　　纵然是号称大罗天星域第一天才的云尊，也是从准帝一步七劫古帝，成就了绝世古帝。
<br/>　　一个云尊都让三大势力感觉到无比棘手了。
<br/>　　他们如今现在敢去算计对付圣龙秘境，对付圣龙宗，也是因为云尊已经是数十年，没有出现在圣龙宗之内了。
<br/>　　一步八劫古帝，那岂不是代表，未来楚风眠可能还要超越云尊？
<br/>　　“杀了此子！”
<br/>　　碧螺老祖盯着，楚风眠猛然出手，一股灵力凝聚在他的掌心之上，身形一闪，向着楚风眠顿时追逐了上去。
<br/>　　楚风眠有着如此可怕的资质，那绝不能让楚风眠活着离开。
<br/>　　现在楚风眠还只是八劫古帝，未成大势，一旦是让楚风眠成就了九劫古帝的的话，那纵然是三大势力，想要对付楚风眠，都是无比困难。
<br/>　　“碧螺竹！”
<br/>　　碧螺老祖心神一动，一掌轰击而出，就在周围的地底之下，一根根翠绿的竹子，突然间的生长了出来，向着楚风眠刺了过来。
<br/>　　这竹子的锋芒，比起天级灵器，还要锋利，稍有不慎，就会被刺穿，天罗地网一般，向着楚风眠围困而来。
<br/>　　“纯阳天火！”
<br/>　　楚风眠眼神一冷，看到这无数竹子的围困，手心之中一团无色火焰，凝聚而出，一指点出。
<br/>　　这一缕纯阳天火，沾染到这一根根翠竹之上，便是燃烧起来，化为了一片火海。
<br/>　　火克木。
<br/>　　九重天火之中最为霸道的纯阳天火，正是这无数翠竹的克星。
<br/>　　这些翠竹，虽然比起天级灵器，还要更加的坚硬，但是沾染到了这纯阳天火，却是没有任何反抗的能力。
<br/>　　这些刚刚还在围困这楚风眠的无数翠竹，在纯阳天火的面前，被燃烧殆尽。
<br/>　　“纯阳天火？先天神体？”
<br/>　　碧螺老祖目光之中闪过几分惊讶，九重天火之中最为霸道的纯阳天火，他自然是听说过。
<br/>　　只是这种火焰，不是武者可以掌控的，除非是天人，才有可能掌控。
<br/>　　现在楚风眠显然没有达到天人境，可却是足以掌控纯阳天火，这必然是先天神体的力量。
<br/>　　“五行刀祖，还不出手？”
<br/>　　碧螺老祖长啸一声。
<br/>　　楚风眠拥有这纯阳天火，便是克制了他的大部分的力量，他一个人想要杀了楚风眠，几乎是没有可能。
<br/>　　“神刀灭世！”
<br/>　　碧螺老祖的话音未落，五行刀祖的身形已经是来到了楚风眠的天空之上，突然一刀向着楚风眠斩杀而来。
<br/>　　这一刀，强横无比，蕴含灭世之力，一刀斩下，天地尽灭，乃是他压箱底的手段。
<br/>　　从天而降，冲着楚风眠斩杀而来。
<br/>　　“你以为我会没有防备？剑出！”
<br/>　　嗖！
<br/>　　在楚风眠背后的天涯剑匣之中，足足数百道剑气同时飞了出来，在空中交织在一起，化为了一道剑阵。
<br/>　　抵挡着那灭世一刀的斩杀。
<br/>　　这灭世一刀，轰击在这剑阵之上，剑阵在节节破碎，不过那一刀的刀芒，也随之破碎。
<br/>　　轰隆！
<br/>　　就在这交锋的瞬间，又是一股强横的天地之力，降临了。
<br/>　　在这铜门之内，又有一尊新的九劫古帝到来了。
<br/>　　这大型宝藏开启的消息，只怕已经是传递了出去，周围的九劫古帝，都在往这里赶。
<br/>　　在拖延下去的话，到来的九劫古帝，可是会越来越多，虽然这一位降临的九劫古帝，并非是三大势力一方的。
<br/>　　但是总会有三大势力一方的九劫古帝，再次到来，以一敌二，楚风眠都无比勉强，以一敌三，楚风眠想要全身而退，都没有机会。
<br/>　　一旦是被拖延在这里，那楚风眠可就完了。
<br/>　　“不跟他们拖延下去了！三阳！”
<br/>　　轰！！！
<br/>　　在楚风眠的剑锋之上，一抹火焰凝聚在上面，同时剑神境的剑意，一齐凝聚在了剑锋之上。
<br/>　　三阳烈炎剑！
<br/>　　第一剑帝的无敌剑术！也是现在楚风眠九域剑术的万千剑术之中，最强的一招。
<br/>　　“碧螺老祖，快退！”
<br/>　　五行刀祖看出了这一剑其中蕴含的力量，急忙退后。
<br/>　　他已经是跟楚风眠交手过许久了，自然是清楚无比，楚风眠的实力，已经是跟九劫古帝，没有任何的区别了。
<br/>　　而楚风眠身上的机缘，灵器，剑术，比起真正的九劫古帝，拥有的，还要更加玄妙，才足以是让楚风眠以一敌二。
<br/>　　这一剑的威力，纵然是他，也不敢硬抗。
<br/>　　这碧螺老祖，也深知这一点，一步退后，凝聚灵力。
<br/>　　轰隆！
<br/>　　剑锋之上的力量，积蓄到了极致。
<br/>　　“烈焰焚天！”
<br/>　　剑锋横扫而过，其中蕴含的，却不是剑气，而是火焰，以纯阳天火，凝聚而成的火海，顿时横扫了整个天际。
<br/>　　三阳烈炎剑，全盛之时，一剑之下，足以毁灭千万荒兽。
<br/>　　楚风眠施展出的三阳烈炎剑，虽没有如此威力，但楚风眠本就是冰火神体，对火焰的控制无比玄妙。
<br/>　　他的这一剑，比起寻常仙术，都要恐怖的多。
<br/>　　那五行刀祖，碧螺老祖，都是将压箱底的底牌，都给施展了出来，化为了一道道的屏障，抵挡着火海的冲击。
<br/>　　“就是现在，走！”
<br/>　　楚风眠施展出这三阳烈炎剑后，便是瞬间转身，头也不会的，向着宫殿的方向飞去。
<br/>　　三阳烈炎剑的威力，虽然霸道无比，媲美仙术，但想要一击之内，轰杀两名九劫古帝，却也是天方夜谭。
<br/>　　九劫古帝，哪一个不是得到了无数奇遇的天才，纵然是没有仙术，但是他们也定然是得到过无数玄妙的保命手段。
<br/>　　这一招，只是拖延，楚风眠的真正目标，还是先走，得到了这独桥尽头的宝藏再说！
<br/>　　楚风眠身形闪烁，一下子步入到了宫殿之内，目光看向了建木树枝，猛然出手，一把向着那建木树枝摄取了过去。
