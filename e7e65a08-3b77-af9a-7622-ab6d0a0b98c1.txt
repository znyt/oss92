　　此时，夜已至深，饭桌上其乐融融，陆雪琪已然融入了这个大家庭，速度快的令他咋舌。
<br/>　　她初来此处，欧阳紫嫣她们自然是照顾的无微不至，但对慕清霄却是毫不理睬，不时投去一个幽怨的眼神。
<br/>　　慕清霄讪讪一笑，饭桌上的妻子，各有各的特色，他眼睛都不知道往何处放，唯独不敢正视她们明亮的双眼。
<br/>　　半个时辰的饭局下来，陆雪琪已经完全融入欧阳紫嫣她们的圈子里。
<br/>　　饭局结束，阿碧与雅儿就进厨房打扫，语嫣，小依和阿朱则坐在沙发上聊天，欧阳紫嫣帮着陆雪琪收拾房间，并且给她拿了几套漂亮的睡衣和裙子。
<br/>　　慕清霄坐在沙发上，嘴角牵起一丝欣慰的笑容，欧阳紫嫣和陆雪琪倒是有相似之处，一个是高高在上的总裁，一个是冰霜女神。
<br/>　　欧阳紫嫣虽不如陆雪琪冷若寒霜，但以前那副生人勿进的模样却是相似极了。
<br/>　　深夜，一道黑影窜入别墅卧室中，随即卧室内就响起一阵，不为外人知晓的妩媚呻吟声。
<br/>　　二楼左侧的客房中，由于陆雪琪今天才住进来，所以其中比较简洁。
<br/>　　偌大的床上，陆雪琪痴痴的趴在慕清霄的怀中，轻声细语，道：“紫嫣姐姐她们真是温柔，真不知道是你几辈子修来的福气。”
<br/>　　“你也不差。”
<br/>　　慕清霄邪异一笑，伸手在她雪白的峰峦上揉捏了一下。
<br/>　　伴随着陆雪琪的惊呼声，他快速翻身压了下去，又是半个时辰的翻云覆雨，待雨歇云收后，别墅内再次回归宁静。
<br/>　　......
<br/>　　......
<br/>　　竖日清晨，慕清霄坐在沙发上，吃着美味的早餐，看着电视上报道的新闻，剑眉皱了皱。
<br/>　　欧阳紫嫣美眸中尽是诧异，道：“最近新闻上老是报道呢，频频有人失踪，不止这一个地区，而且年龄都在二十到四十岁之间，怪事真多。”
<br/>　　慕清霄微不可查的点头，并没有太在意，道：“紫嫣，最近集团怎么样了？”
<br/>　　“亏你还记得集团。”
<br/>　　欧阳紫嫣白了他一眼，抿了口牛奶，露出一个绝美的笑容，道：“你这个甩手掌柜倒是舒服，不过你放心吧，集团有我，还有雅儿和阿朱，没什么大事。”
<br/>　　慕清霄伸手揽住她的细腰，温柔道：“辛苦你了。”
<br/>　　欧阳紫嫣美眸中尽是幸福，柔情似水，道：“有你在，不辛苦。”
<br/>　　温柔的话语触动他心间最柔软的地方，慕清霄直接将她从沙发上抱了起来，不顾她的惊呼声，在阿碧她们面红耳赤，羞涩的目光中，径直向浴室中走去。
<br/>　　“流氓，你快放我下来......唔唔！”
<br/>　　半个时辰后，浴室中传来一阵强烈的气息波动，陆雪琪虽羞涩，但美眸还是盯着浴室方向，这气息与她突破太清境的时候，完全一样。
<br/>　　昨晚在饭桌上，她就发现，欧阳紫嫣她们的气息都极强，比她突破太清境前还要强，最起码也是玉清境八重巅峰，甚至更高。
<br/>　　当时，她心中震惊无比，要知道欧阳紫嫣她们的年龄，比她大不了多少，甚至小依的年龄还比她小，但修为却如此强大，若是在她原本的世界，每一个都是绝世天才啊！
<br/>　　真不明白，自己这位风流的丈夫，是如何让她们死心塌地的。
<br/>　　只是，反过来仔细想想自己，陆雪琪美眸中就有些羞涩起来，她的资质也不差，不是照样被他骗了去。
<br/>　　浴室中，慕清霄身无寸缕的抱着欧阳紫嫣，嘴角勾起一丝邪异的笑容，灵液的效果当真不错，只是刚刚服下，便突破金丹的壁障，而且气息不断的往上升。
<br/>　　片刻后，欧阳紫嫣娇躯一震，一股无形的波动在空中扩散而出，紧接着睁开明亮的美眸，粉唇微微张开，红润绝美的脸蛋上挂满了震惊之色。
<br/>　　“金丹四重，修为居然一下子暴涨了这么多，你刚刚给我喝的透明液体，到底是什么东西？”
<br/>　　慕清霄嘴角抽了抽，这妮子说话真是越来越不矜持，我们两个正坦诚相待，这一番话语中，信息量真是有点大啊！
<br/>　　察觉到慕清霄怪异的神色，欧阳紫嫣绝美的脸颊上更是通红无比，就像是成熟的西红柿，纤纤玉指，捏在他的软肋上，银牙紧咬，道：
<br/>　　“你这家伙又不正经，就知道想龌龊的事情！”
<br/>　　慕清霄嘿嘿一笑，拍了拍她圆润的玉臀，道：“在龌龊，还能龌龊到哪去？”
<br/>　　对上她恶狠狠的目光，慕清霄干咳两声，道：“刚刚的液体，称为灵液，由大量灵气长年累月，积攒而成，蕴含的能量也极其庞大......”
<br/>　　欧阳紫嫣的资质，可是经龙元改造过的，资质方面，陆雪琪都无法比拟，修炼到后面，更能体现出来，也正是因此，她想要突破，所需要的灵气，往往比别的修士要多的多。
<br/>　　所以，七分之一的灵液只让她突破金丹，一步迈至金丹四重，而陆雪琪则是金丹六重，这一点根本不足为奇。
<br/>　　慕清霄解释完毕后，欧阳紫嫣两眼放光，小舌头舔了舔粉唇，小脑袋在他胸口蹭了蹭，撒起娇来，道：“老公，还有没有灵液，我还要喝。”
<br/>　　慕清霄微微一笑，想到欧阳紫嫣当初的经历，来自她家族的压力，十八岁的她独自一人来到静海打拼，直到遇见自己，也只有在自己的面前，她才会毫无防备，露出这幅小女人的模样。
<br/>　　“灵液倒是有不少，可要是继续服用下去，会影响到你以后的修炼，根基不稳，以后在修炼的道路上，可能会止步在某个境界。”
<br/>　　听到慕清霄的话，欧阳紫嫣似懂非懂的点了点头，她修炼的时间已经不短，但懂得却不多。
<br/>　　虽然懂的不是很多，但她知道，慕清霄绝对是为她好，迫切提升修为的欲望也渐渐消散。
<br/>　　慕清霄看她的眼神，欣慰一笑，道：“还有几样好东西要教你，你当初不是说，喜欢御空飞行吗，你现在的修为也够了，我先将御剑决教给你。”
<br/>　　闻言，欧阳紫嫣美眸中闪过一丝精芒，心中大喜，忍不住在他的脸上吻了一口。
<br/>　　随后，慕清霄强行压制逐渐攀升的邪火，耐心的将御剑诀心法路线教给了欧阳紫嫣。
