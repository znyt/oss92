　　一处山林深处，幽暗的山洞之中，盘膝坐着一道青衣身影。
<br/>　　这青衣身影是一位青年，脸颊稍长，面带锐气，给人一种凌厉之感。
<br/>　　这青年不是别人，正是苏莫的分身，掌控苏莫的天梦元神。
<br/>　　“紫霄，我不会让你得逞的！”分身睁开了眼眸，眸中闪过犀利的精光。
<br/>　　分身即是苏莫，他不能坐以待毙，等着紫霄胁迫加身。
<br/>　　紫霄有芊寻月这个筹码，在苏莫看来，其应该不会公开。
<br/>　　因为一旦公开，其将会成为众矢之的，人人都会抢夺芊寻月。
<br/>　　毕竟，抢夺到了芊寻月，就很可能得到终极剑道。
<br/>　　所以，以紫霄这种自私自利的心性，肯定会独自行动，不仅不会公开，估计连终极神王殿高层的其他人，都不会告知。
<br/>　　那么，苏莫就要利用这一点，对付紫霄。
<br/>　　不过，这是一个漫长的过程，目标不在于杀死紫霄，而在于抢回芊寻月。
<br/>　　毕竟，杀死紫霄，基本上没有可能。
<br/>　　至于让司空炎蛰伏在终极神王殿，苏莫也没有具体的安排，算是埋下一个暗线，希望能起到作用。
<br/>　　不过，他接下来的行动，风险很大，尤其是对芊寻月而言，风险非常的大。
<br/>　　不过，他只能一搏，别无它法。
<br/>　　因为，紫霄就是一座大山，和帝一魂一样，无法战胜的大山，况且，紫霄还有终极神王殿这个靠山。
<br/>　　少倾，分身离开了山洞，向终极剑域所在的方向飞去。
<br/>　　分身目前的修为，只是二品，飞行的速度不快，但是，三个月之后，也进入了天池神朝的境内。
<br/>　　此时此刻，天池神朝境内，秦陨之名极为响亮，走到了哪里，都有人在谈论。
<br/>　　终极剑道和终极神剑的风波，远远没有过去，很多大势力，都在打听秦陨的下落。
<br/>　　御天界域，一座中等大小的城池之中，有着一家颇为豪华的酒楼，这酒楼以美酒闻名周边地域。
<br/>　　每日，酒楼中都有大量的酒客，来此品尝神酒，畅谈神域轶事。
<br/>　　“现如今，天池各大势力，几乎都行动了起来，派出大量的弟子，在追查那秦陨的下落！”
<br/>　　“可不是吗？我听说终极神王殿，派出了数十万弟子！”
<br/>　　“神域虽大，但是这秦陨最终肯定会被找到。”
<br/>　　“怀璧其罪啊，这种宝物，不是神王以下的武者，能够拥有的！”
<br/>　　“估计要不了多久，其它神朝都会开始追查秦陨的下落。”
<br/>　　酒楼的宽敞的大堂之中，有上百名酒客，修为低的只有武圣境，修为高的足有真神境五六品。
<br/>　　大堂之中，声音嘈杂，但是有一个共同点，几乎所有的人都在谈论秦陨。
<br/>　　窗边，一袭青衫的苏莫分身，淡然而坐，一边饮酒，一边听着大堂中酒客的谈论。
<br/>　　这酒，名为‘神华酒’，分为各种不同的属性，效力极佳，对武者大有裨益。
<br/>　　“呵呵，诸位，你们根本就不了解内幕！”分身端起一杯神华酒，一饮而尽，环视众人朗声说道。
<br/>　　听闻分身之言，大堂中微微一静，所有人的目光，都落在了分身的身上。
<br/>　　“这位兄台，此事有什么内幕？”一位络腮胡的大汉，好奇的向分身问道。
<br/>　　其他人，同样面带好奇之色，难道此事，还有什么内幕不成。
<br/>　　“当然有内幕，秦陨根本就没有得到终极剑道和终极神剑？”分身微微一笑说道。
<br/>　　“没得到？”
<br/>　　众人闻言一怔，随即立刻有人嗤笑出声：“小子，你怕不是在开玩笑，那秦陨在万道剑冢，登上了剑山之巅，取走了终极神剑，众所周知！”
<br/>　　“是啊！终极神王殿的山门都没了，现在新建的终极神王城！”
<br/>　　“小子，你简直在胡说八道。”
<br/>　　不少人纷纷开口，均是将分身之言，当成是一个笑话。
<br/>　　“你们有所不知！”
<br/>　　分身摇了摇头，然后环视一圈，压低声音，道：“这只是传出来的消息，真实的内幕另有隐情，不然，以那秦陨的修为，得到了终极神剑之后，能安全离开终极神域？”
<br/>　　“哦？”
<br/>　　众人闻言，纷纷惊疑，眼前的这个青年，说的有点道理。
<br/>　　据闻，秦陨的修为，好像只有三品，得到终极神剑之后，不可能逃离终极剑域。
<br/>　　要知道，终极神王殿，可是有好几位神王，殿主终极神王更是高级神王。
<br/>　　“小兄弟，说说看，到底是什么内幕？”络腮胡大汉好奇的问道。
<br/>　　“对啊，这位兄台，你不要吊人胃口啊！”
<br/>　　“具体是什么隐情？”
<br/>　　众人纷纷开口，好奇之人人皆有之，能来此地饮酒之人，多是一些闲来无事之人，自然更加想要了解真实情况。
<br/>　　“我告诉你们，那秦陨只是别人的棋子，终极剑道和终极神剑，早已被别人得到了！”分身面色郑重的说道。
<br/>　　“谁？”
<br/>　　“被谁得到了？”
<br/>　　“真的假的？”
<br/>　　众人纷纷惊讶，先不谈眼前之人说的真假，他们都好奇了起来。
<br/>　　“紫霄神王。”分身低声说道。
<br/>　　众人闻言纷纷心中一震，紫霄神王得到了终极剑道和终极神剑？这可能吗？
<br/>　　“小兄弟，你这话说的也太假了吧，紫霄得到了终极剑道和终极神剑，那终极神王殿为何还在追查秦陨的下落。”
<br/>　　“是啊，你明显是在扯淡！”
<br/>　　“终极剑道和终极神剑，本就是终极神王殿之物，他们还不至于不敢承认自己得到了！”
<br/>　　不少人开口，俱是不信分身之言，此言毫无根据，很难让人信服。
<br/>　　“你们知道什么？”
<br/>　　分身不屑一笑，道：“那秦陨乃是紫霄的密门弟子，当初秦陨杀了终极神王殿许焱等四人，都被紫霄隐瞒下去了！”
<br/>　　“这不太可能吧！”众人依旧不太相信，但是，很多人面上，已经露出了疑惑之色。
<br/>　　“具体什么情况，谁又知道呢！或许终极神王殿不想外传，也或许紫霄想要独吞终极剑道和终极神剑！”分身摇了摇头道。
<br/>　　“此事可有证据？”有人面色惊疑的问道。
<br/>　　“有，据我所知，秦陨的爱人陨落，尸身一直就在紫霄的身边，只有紫霄能救活。”分身沉声说道，言罢，他将杯中之酒，一饮而尽，而后便起身，大步离开了酒楼。
<br/>　　他不会再继续多言了，留下这些信息，已经足够了。
<br/>　　走出酒楼，分身深深的吸了口气，能不能救出芊寻月，他并没有把握，只能走一步看一步了。
<br/>　　“这……！”
<br/>　　众人见分身离去，不由得无语，这话还未说完，怎么就走了。
<br/>　　众人个个惊疑不定，难道此事真有隐情不成？
<br/>　　随后，大堂中有几人，也起身离开，要将这个消息，禀报给各自的宗门。
<br/>　　此事或许不实，但是，调查一番便能知道大概的结果。
<br/>　　接下来的日子之中，分身的身影，在各地晃荡，今日酒楼中的一幕，在其它很多地方不断的上演。
<br/>　　关于秦陨是紫霄密门弟子，终极剑道和终极神剑被紫霄所得一事，也开始大规模的传播。
<br/>　　此消息一出，让得无数势力惊疑，纷纷开始调查了起来。
<br/>　　一查之下，紫霄确实可疑。
<br/>　　首先，秦陨确实是终极神王殿的弟子，当初的确是杀了几名终极神王殿的弟子，还恰巧是紫霄剑宫的弟子。
<br/>　　但是，此事根本没有引起任何风波，居然没有任何的动静，被紫霄暗自压了下去。
