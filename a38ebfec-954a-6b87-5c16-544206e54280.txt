　　“最后一关，我一定要进去！”
<br/>　　距离山巅约莫两千五百丈的地方，古剑山面色凝重，全身源力滚滚。
<br/>　　他心中不停的自语，现在他领先所有人，是距离山巅最近的地方。
<br/>　　自古以来，就算再妖孽的天才，都无法登上山巅，但是，他偏偏要登上去。
<br/>　　这无关于修为，修为越高，在这里承受的压力便越大。

<br/>　　这倒不是说，他就比古剑山厉害多少，而是，他不允许对方蹬上山巅。

<br/>　　当然，这只是一部分原因，最主要的原因，还是剑山之巅，太过难以攀登。

<br/>　　“这个慕容雷利，难道还没有集齐道剑？”韩生叹了口气，他不担心古剑山，担心的是拓跋流明和云沐这两位四品的强者。

<br/>　　而面对拓跋流明和云沐，韩生底气不是很足，尤其是面对拓跋流明，这位四品中级的强者。
<br/>　　所以，到时候，若是拓跋流明和云沐两人，有登上山巅的趋势的话，他还需要慕容雷利的协助。
<br/>　　叹了口气，韩生便继续前进，仅仅的吊在古剑山的后方。
<br/>　　轰！轰！轰！
<br/>　　一声声沉默的声响，不断的响起，拓跋流明也快速的追了上来。
<br/>　　拓跋流明，由于来的较晚，落后于古剑山和韩生，但是，凭借强大的天赋和高深的修为，他在快速的追赶。
<br/>　　“最后一关，你们都没有资格！”
<br/>　　拓跋流明大步前进，强大的气息滚滚涌动，气势如虹。
<br/>　　这一次，拓跋流明也是抱着坚定的信念，想要登上山巅，看看最后一关，有没有终极剑道的秘密。
<br/>　　登上山巅！
<br/>　　曾经阻挡了无数的妖孽和天骄，包括身为轮回之身的拓跋青云。
<br/>　　“二叔失败了，我为他完成心愿！”
<br/>　　拓跋流明心中暗道，拓跋青云是他的二叔，他要为二叔，完成这个目标。
<br/>　　后方，苏莫心静如水，他整个身躯，都被混沌之力包裹，地狱这脚下剑山散发的锋芒。
<br/>　　他的速度极快，已经攀登了三千余丈，脚下的剑山，散发的锋芒，越来越剧烈。
<br/>　　恐怖的锋芒，让他体外裹挟的混沌之力，都被不断撕开一道道小口子。
<br/>　　当然，这个级别的锋锐之气，还不足以彻底破开他的混沌源力。
<br/>　　此时此刻，千幻邪君跟在苏莫的后方，约莫百丈之处。
<br/>　　以千幻邪君的实力，本可以更快，甚至是追上苏莫，但是他并未如此。

<br/>　　但是，他要扼杀，在苏莫的光华，还没有升起之前，就彻底的灭杀。

<br/>　　每个人都没有急躁，缓步前进。

<br/>　　而付元青，天赋不够，行进的太慢了，早已经落在了最后。
