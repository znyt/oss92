　　“走不走，走不走，不走我打死你”
<br/>　　余子童不断的用脚踢黄金瞳，但此猫哪怕被踢出血，也倔强的就是不走，一人一猫僵持了下来。
<br/>　　只见苏羽忽然开口：“余大师，你无需如此，你的猫要是能捉住那只鼠精，我就算你完成了订单，出去不会说您半句不好”
<br/>　　余子童闻言眼中一喜，指着黑袍男子：“那他...”
<br/>　　涂小安这时站出来，冷淡无波的开口：“余大师尽管看戏就好，无需你出马”
<br/>　　看着黄金瞳被一脚一脚的踢，就如梅花傲然直立，不愿屈服，不愿离去。
<br/>　　这只猫有自己的底线跟尊严，而碍于主人的命令，让它现在左右为难。
<br/>　　涂小安看不下去，只能给余子童一个台阶下，免的这只猫不断的被虐待。
<br/>　　多好的猫呀，涂小安都想强制性的给收服。
<br/>　　不过这种品种的生物，他无法强制性收服，就算能，也不可以做，那不成强盗了。
<br/>　　必须是别人的宠物，没有强买强卖的。
<br/>　　另外他看的出来，余子童虽然对这只猫拳打脚踢，但此猫却非常的忠心。
<br/>　　不然已这种性情凶猛宛如小老虎的猫，岂能委曲求全。
<br/>　　那可是暴脾气呀。
<br/>　　余子童脸色缓和了不少，要是不需要他出手就能完成订单，还能名声不受影响，到是一举两得。
<br/>　　“宝贝，去吧，将那只老鼠给抓了”余子童和悦的笑了起来，利益为先。
<br/>　　“嗷...”
<br/>　　黄金瞳如猛虎出山，冲了过去，那只小白鼠化为光影，同样迎了上来。
<br/>　　猫鼠大战第二场开始。
<br/>　　“小安兄弟，接下来就麻烦你了”
<br/>　　苏羽想要解决工地的麻烦，必须对付黑袍男子，余子童他是指望不上，出门全靠一只猫，沽名钓誉之辈。
<br/>　　此时的涂小安踏前一步，看着黑袍男子：“我们来打一场，你若输了，离开工地，以后也不准来找麻烦，可好”
<br/>　　“都说元素修武者是一等一的高手，含着金钥匙出生”林凡眼中闪过一丝傲气：“我到想看看，是不是真的那么厉害，谁说生物掌控者，就一定不如元素修武者”
<br/>　　“你若赢了，我立刻消失”
<br/>　　林凡也是帮人办事，跟苏羽本来就无仇恨。
<br/>　　这事情其实并不是什么死结。
<br/>　　“好，痛快”涂小安风轻云淡的笑了起来。
<br/>　　这林凡的实力，涂小安早就胸有成竹，二十一级，说实话，对上他，有点弱。
<br/>　　就连妖精管理局组长冷霜，那可是二十九级的元素修武者，在涂小安的面前，也完全不够看。
<br/>　　“你出手吧，我若先出手，你就没机会了”
<br/>　　涂小安举手投足间，尽显高手风范。
<br/>　　林凡闻言便怒了，黑袍下，眉宇间似是凝着化不开的冰霜，好大的口气，他可是二阶生物掌控者，在温城都可以横着走了。
<br/>　　“你会为你的口出狂言，付出代价的”
<br/>　　林凡一声话落，嘴巴念念有词。
<br/>　　吱吱吱....
<br/>　　忽然，他脚下的一群大老鼠暴动了起来，只见一只大老鼠如弩弓一样的朝着涂小安骤然射去。
<br/>　　空中，这只老鼠露出一排寒芒般的牙齿，锋利如刀，通灵慑厉。
<br/>　　第一只老鼠射出去后，仅仅只是一个帷幕，紧接着，林凡脚下的老鼠咻咻咻的朝着涂小安飞扑而去。
<br/>　　远远望去，就好像万箭齐发，画面惊心动魄，不可思议。
<br/>　　苏羽跟余子童早早的退到了最后面，一个个嘴巴张的跟一个小箱子那么大。
<br/>　　将一群老鼠玩成了暗器一样，不得不说生物掌控者，天生对动物有着超乎常人的控制手段。
<br/>　　只是，这些手段在常人面前觉得不可思议，但在涂小安眼中，简直不值一提。
<br/>　　他妖治的眼睛微微收缩，浑然一眯，无形中一股强大的意识力骤然投放而出。
<br/>　　这些爆射而来的大老鼠，还没接近涂小安，就被定格在了空中，好像时间被静止了一样。
<br/>　　“给我回去”
<br/>　　涂小安轻喝一声，只见空中的几十只大老鼠调转了方向，无法自控的朝着林凡的位置齐刷刷的射回去。
<br/>　　林凡见此，微微一惊，手一挥，脚下再次射出去几十只老鼠。
<br/>　　轰轰轰...
<br/>　　两股老鼠既然相互的击撞到了一起去，霎时间血肉横飞，一股血腥味顿时在空中弥漫而来。
<br/>　　“别用这种雕虫小技，没意义”
<br/>　　涂小安冷淡无波的看着不远处的黑袍男子。
<br/>　　林凡眼神一凝，表情俨然起来，他眉头一紧，念头转动，狠狠一咬舌尖，闭上眼睛，双手结出一个手印，脸部变得血红。
<br/>　　忽然，空气像是煮沸了一般，他脚下的大老鼠都躁动不安起来。
<br/>　　林凡口中念念有词，身上有一股能量在涌动，只见他双眼猛然睁开，一脚重重踩在地面，整个人飞身而起。
<br/>　　“给我爆...”
<br/>　　猛然间，工地上剧烈震动起来，随后，只听“轰”的一声巨响，涂小安所在的地面既然层层被掀开，轰然爆炸。
<br/>　　吱吱吱...
<br/>　　烟雾弥漫，涂小安笼罩其中，一大群老鼠骤然的前仆后继，像是去争夺什么天下美食。
<br/>　　“不好，小安兄弟有危险”
<br/>　　远处的苏羽看到这一幕，一阵惊悸，毛发着了魔一样地冰冷地直立起来。
<br/>　　一旁的余子童反而眼中闪过激动，这小子被干掉最好，嚣张什么。

<br/>　　在他这种人眼中，觉醒天赋简直可遇不可求。

<br/>　　二流的觉醒者，就是身负异能，说是超能力也可以。
<br/>　　这种有很多，比如生物掌控者，掌控动物，也有掌控植物什么的。
