　　顿时，白羽看向白允儿的目光中，就充满了怨念。
<br/>　　不是说要找那个精神系的新人类么？不是说找到后让其提升等级，让后唤醒楚涵么？
<br/>　　怎么跑来这里跟一个制裁者会面？
<br/>　　还是那个被誉为史上最恐怖的制裁者！
<br/>　　此人是追杀白家族长到天涯海角，甚至让白家的族长一生都不敢走出白家禁地，还灭了一整个肖家的何培源啊！
<br/>　　白羽惊悚的即将晕过去，与正常神秘家族成员见到制裁者时的样子，没有太多区别。
<br/>　　何培源也见怪不怪，干脆没有搭理这个吓到瑟瑟发抖的少年，只是有一搭没一搭的与白允儿聊天。
<br/>　　两人也算是颇有渊源，从末世爆发之初开始就有过一段同路旅程，只是那时候的白允儿并没认出何培源的身份，而何培源却一眼看破了白允儿身上的古怪。
<br/>　　“你最后还是回到了白家？不过也正常，你本身就是白家嫡系之后。”何培源慈祥的说道。
<br/>　　白允儿点头，抬眼中目光深幽：“不过，失忆了，过去二十年的记忆，只有有关何峰和您的记忆，剩下的都是在失忆后才有的常识。”
<br/>　　何培源表情顿了顿，而后绽放了一个笑容：“这么说来，你自己也知道原因了吧？只有制裁家族两人的有关记忆没被抹去，看来是映照了八门相克啊！”
<br/>　　“是。”白允儿点头，道：“一开始并没有猜到，但寻找美阎罗而前来的路上，想到您在这里，便也了然。”
<br/>　　莫名的失忆，却偏偏有关何培源与何峰的记忆没有被抹去，说明在这记忆清除的过程中，这两人是不可跨越的禁忌。
<br/>　　由此看来，也就不能猜中白允儿记忆的消失，是白忧的能力。
<br/>　　听着白允儿的话，何培源眨了眨眼睛：“原来你是来找美阎罗。”
<br/>　　白允儿点头：“您可知道她在哪？”
<br/>　　何培源神秘一笑：“你，还有猎王，你们还真有意思，弯弯绕绕的都不过是在以一个人为中心。”
<br/>　　谈话间，角落里的白羽已经镇定下来，虽然他还是震惊于白允儿竟然与这位可怕的制裁者有渊源。
<br/>　　王之暴君原本想找个机会逃离，但听到两人的谈话后又改变了主意，心头灵机一动。
<br/>　　刺杀何培源原本以为会轻松得手，却没想到半途杀出了一个与楚涵关系好的白允儿，在他打算放弃的时候，却听到了有关这个白允儿的敏感话题。
<br/>　　白家嫡系，那岂不是与白忧为同路人？
<br/>　　王之暴君忽然明悟，难怪白允儿这两年都没在和楚涵接触。
<br/>　　想到这里他乍然开口，插入到了白允儿与何培源的闲聊中：“白家大小姐，我是奉白家大少白忧之命，前来刺杀何培源的，请您配合不要阻止。”
<br/>　　以王之暴君的想法，白允儿既然是白家的嫡系大小姐，怎么都应当与白忧的目的一致，更不可能在家族命令中生出另外想法。
<br/>　　听到王之暴君的话，白允儿微微偏头，用余光看了他一眼。
<br/>　　声音清冷到了极致：“白忧？”
<br/>　　“是。”王之暴君点头，心头不知怎么忽然感觉到一丝不对。
<br/>　　果然下一秒，以冰山美人著称的白允儿忽而展颜一笑，下一句话震惊四座：“既然是白忧要杀的人，那我肯定是要保了。”
<br/>　　“什么？！”王之暴君愕然抬头，暴露出的腥红瞳孔里，满是震惊。
<br/>　　被白羽看了个正着的这一幕，让这个少年立即出声：“小姐姐，这个王之暴君是异种！”
<br/>　　刷！
<br/>　　一道残影闪过，王之暴君的反应很快，当察觉到白允儿态度不对的时候，就立即朝着房门处移动，想要先跑了再说。
<br/>　　同时他更是对神秘家族内部的事情，越发看不明白！
<br/>　　白忧不是白家话语权最大，并且还是未来的族长吗？
<br/>　　身为同族还是同嫡系的白允儿，竟然公然反抗白忧？
<br/>　　在王之暴君的手刚触碰到房门，想要打开的时候，忽的就感受到一股能量凭空传递而来，一把勒住了他的脖子！
<br/>　　“额——”他发出一声痛苦的声音，双手在自己的脖子上胡乱抓着，可那里什么都没有。
<br/>　　噗噗！
<br/>　　异种特有的尖锐指甲将脖颈的皮肤都划破，露出了带有浓烈酸臭味的腥红异种血，不仅什么都没抓到反而将自己抓的血肉模糊。
<br/>　　喉咙处的那股能量，却像是紧勒的绳索，想要将他的脖子都勒断，将他的脑袋整个拆下。
<br/>　　这到底是什么东西？
<br/>　　王之暴君惊悚无比，调动起体内的所有能量，开始与喉咙处的那股凝聚成形的无形能量抗衡。
<br/>　　依旧坐在沙发上的白允儿这才眉头动了动，她轻轻放下茶杯，凭空伸出了右手。
<br/>　　只见那青葱般细长白·嫩的五指，就这么猛然在半空中一握！
<br/>　　紧接着——
<br/>　　咔咔咔！
<br/>　　一阵古怪又恐怖的声音响起，站在门旁的王之暴君，已经浑身逸散而出恐怖的能量刹那间一滞，紧接着便双手无力的垂下，脖子缓慢的歪在了一边。
<br/>　　咔！
<br/>　　又是一声，似乎是内部某根重要的骨头应声而断，低垂在一旁无力的晃荡着。
<br/>　　整个过程中白允儿都没有起身，甚至没有回头看一眼，只有在王之暴君奋力抵抗的时候，伸出了一只手而已。
<br/>　　如此恐怖的战力，一下子让屋内安静的可怕。
<br/>　　白羽惊吓的咽了咽口水，想要走上前确认这个异种身份。
<br/>　　何培源也被白允儿吓了一跳，忍不住开口道：“杀人于无形……你这小丫头是撞上了什么机遇？不会已经满阶了吧？”
<br/>　　白允儿只是轻笑一下没有出声，继续抬起茶杯抿了一口。
<br/>　　就在此时，忽的——
<br/>　　嘭！
<br/>　　一声巨响，这间屋子的房门被人用力撞开，下一刻‘哗啦啦’的一大群人就这么冲了进来，一个个神色紧张，更是人人手握武器，一副要大开杀戒的样子。
<br/>　　还没走出几步的白羽顿时退回角落，目瞪口呆的望着这批连门都没敲便冲进来的人。
<br/>　　带头冲入的正是江凌轩，他面色苍白冒着冷汗，眼神里尽是慌张。
<br/>　　可当他带人狂奔而来，毫不犹豫的撞开这扇门后，却看到了屋内让他完全没想到的场景。
