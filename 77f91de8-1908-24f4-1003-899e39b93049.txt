　　听着夜雪狂歌的声音，刚刚进入房间中的蒋念一行人也都是连忙跑出了自己的房间。
<br/>　　“怎么了夜雪？”来到叶帝的门前，蒋念连忙开口问道。
<br/>　　“我也不知道，刚刚准备休息的时候听到小叶子这屋有声音，现在怎么敲门都没人回应。”夜雪狂歌有些担心的说道。
<br/>　　“小叶子！！！”
<br/>　　“小叶子！！！”
<br/>　　听着夜雪狂歌的话，蒋念一行人也是连忙开口对着叶帝喊道，只不过此刻叶帝的肩膀已经疼的他有些意识模糊了，根本听不清他们的喊话，更别说回应了。
<br/>　　“别等了，这么大声音就算是睡着了也会把叶哥叫醒的，他在屋子里肯定出事了，赶紧把门撞开！”香蕉开口喊道。
<br/>　　而就在几人说话的时候，楼下的一群女生这个时候也是听到声音向着楼上跑了过来。
<br/>　　“怎么了？”看到几人这样的架势，叶颖连忙开口问道。
<br/>　　“小叶子在屋子里好像出什么事了，叫他也没回应。”秦霄开口说道。
<br/>　　“这门能撞开么？”李景禹也是有些焦急的问道。
<br/>　　“撞不开，赶紧去楼下拿钥匙，我记得咱们这好像有备用钥匙。”林雅楠跟着说道。
<br/>　　听着林雅楠的话，飞刀、香蕉几人连忙向着楼下跑去。
<br/>　　等到他们打开叶帝的房间的时候，看到脸色煞白，衣服已经被汗水浸湿的叶帝，一个个顿时都被吓了一跳。
<br/>　　“蒋念快下去把车开出来，送小叶子去医院！”看到这个情况秦霄连忙对着蒋念指挥着。
<br/>　　蒋念也是二话不说，飞快的向着楼下跑去。
<br/>　　“小叶怎么了？你这是怎么了？”叶颖抱着叶帝的脑袋，话语中满是哭腔。
<br/>　　“没事姐，没事！”叶帝声音有些嘶哑的小声说道，这个时候他肩膀处传来的疼痛，让他根本无法大声回应他们，接着叶帝又对着叶颖提醒道：“不要告诉家里！”
<br/>　　“你都这样了，还在那里操心，这到底是怎么回事，白天不还是好好的么！”叶颖有些生气的说道。
<br/>　　“车已经发动好了，赶紧把小叶子背下来。”就在几人说话的时候，楼下已经传来了蒋念的声音。
<br/>　　“姐，我来背小叶子！”看着有些慌了神的叶颖，楚南赶紧开口出声。
<br/>　　这个时候大家都不知道叶帝到底发生了什么，所以此刻最好的打算无疑就是将他送到医院去。
<br/>　　“对对对，快背下去，咱们也开车跟过去，你们谁来开车，一定要注意安全。”有些慌神的叶颖连忙说道。
<br/>　　片刻后荣耀基地内便是有着四辆轿车开了出去。
<br/>　　李玲和周方，两人都是小报的记者，他们负责跟拍叶帝这一行人，只不过叶帝他们平时的作息都非常的严格，而且一个个也都没有什么绯闻。
<br/>　　所以他们这一组基本上从来都没拿出过有用的新闻，并且每次在开会的时候总是被领导斥责一遍，所以每到周一这天的时候，两人都很煎熬。
<br/>　　明天又是周一了，想到自己两人又要被领导给批评，李玲两人也都是有些无奈。
<br/>　　“领导也不是不知道荣耀战队的情况，这无论是换了谁过来，都不可能拿到新闻的啊。”李玲开口抱怨着。
<br/>　　“这你还不懂么，咱们两个都没给领导上炮，其他人都给领导送过东西，要不然怎么可能一直让咱们两个跟着大魔王他们，对于一个没有绯闻的明星来说，咱们这种其实是最不吃香的。”周方跟着说道。
<br/>　　“就是不送，凭什么就要给领导送东西，我这才刚刚毕业，哪里有那么多钱啊。”李玲踢着脚下的石头，有些颓然的说道。
<br/>　　“咦？那不是暗夜的车么，李玲你快看是不是！”就在李玲说完话的时候，一旁的周方看着远处飞驰过来的几辆车，连忙开口喊道。
<br/>　　“没错就是他们的，那四辆车都是荣耀战队的，怎么这么晚了他们还出去。”看着从荣耀基地内行驶出来的四辆车，李玲连忙应道。
<br/>　　“有新闻，肯定有大新闻，赶紧跟上去，我感觉咱们两个的春天要来了。”周方连忙开口说道，说话的同时手中的相机已经是冲着这边的几个车子处飞快的拍了几张照片。
<br/>　　随后便是飞也是的朝着自己两人的车子跑去，等到李玲上来的时候，打火起步，一个漂移直接飞了出去。
<br/>　　在知道叶帝他们的身上已经没有什么新闻可挖的时候，除了李玲他们之外，其他报社的记者早就已经撤走了，在这里也是浪费资源，他们倒不如去别的地方在挖掘其他的明星呢。
<br/>　　当到达医院的时候，当李玲和周方两人看到叶帝被背着送进医院里的时候，两人都知道这次的新闻肯定要惊破天了。
<br/>　　虽然对于叶帝的状况两人也都非常的担心，不过有着职业操守的两人还是飞快的拍好了照片。
<br/>　　“你说大魔王他怎么了，是不是真的出什么事了。”李玲有些担心的问道，两人都是叶帝的粉丝，乍看到叶帝这幅模样，自然是非常的担心。
<br/>　　“不知道，不过看暗夜他们行色匆匆的样子，肯定是出事了，先给领导打电话吧。”周方跟着说道。
<br/>　　随后两人便是拿起了电话，向着领导的电话拨了过去。
<br/>　　“大半夜的还让不让人睡觉了，不想干明天就直接写辞职信。”半夜被弄醒的领导十分的不爽，直接开口骂道。
<br/>　　“领导，我们两个刚刚拍到大魔王住院了。”听着领导的话，如果可以的话李玲真的恨不得一个巴掌扇过去，当然这只能是在心中yy一下而已，话语中还是恭敬的说着。
<br/>　　“我管你谁住院，今天你们要是不给我个说法......大魔王住院了？”领导继续愤怒的说道，不过说道一半当他反应过来的时候，瞬间原本还有些睡意的领导一下子就清醒了过来。
<br/>　　“在哪个医院，一定要盯住，我马上过去！”这一刻领导恨不得抱着两人亲上一口，终于等到大魔王的新闻了，而且这还是一个惊破天的新闻。
<br/>　　所有人都知道叶帝在英雄联盟中的地位，如今他骤然住院，可想而知接下来带来的影响会有多么的大。
