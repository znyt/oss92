        &lt;h3&gt;第三卷第二百八十七章  父女谈心&lt;/h3&gt;
<br/>
<br/>        陈元良要进行手术了，陈家兄弟姐妹无一例外的都到了一附院，这一次还是缺少陈子寒，但是这一次却多了代凉。
<br/>
<br/>        大家的心情略显沉重，这种疾病手术后存活率也不是很高，好的几年，不好的几个月，但是陈元良却像没事人一样，有说有笑。
<br/>
<br/>        “你们不准苦着脸啊，我这还好好的活着呢，就算是活不了了，你们也不用难过，生老病死是人之常情，我自己的身体我清楚。子慧你和红卫、战强都回去，别在医院呆着了，我这手术做完还要住icu病房，得几天呢，几天后你们再来看我。”
<br/>
<br/>        “子霖，你带着你姑姑好好转几天，凉儿，感谢你能回来看我，我这颗心啊满足了，如果你有事情了就回去吧，等我手术后我们兄妹再聚，我会抽时间去你家看看的。”
<br/>
<br/>        “青松，你也回去吧，你工作忙，还要照顾子澜，辛苦你了，就不要天天往我这里跑了，子澜身体好起来比我这老头子重要很多，回去告诉子澜，她是我的骄傲。”
<br/>
<br/>        “子乔，你和萧林也回去，萧林的生意也很忙，没什么事情就不要来了，子乔要照顾好俩个孩子，一定要教育好，成为一个有用的人。”
<br/>
<br/>        “子悦，女人当自强，你现在是一个女强人了，幸福都是在一念之间，你觉得生活是幸福的那它就是幸福的，累了休息会，苦了你还有兄弟姐妹们，你也回去，将俩个孩子照顾好，做生意呢也要安安分分，过几天再来看我。”
<br/>
<br/>        “子建，你留下来照顾我，慧芝，你将你妈妈带回去，让她这几天不用来医院了，好好在家里等着我就是。”
<br/>
<br/>        “子昂留下，你们都出去，我有话想和子昂说说。”陈元良一个一个的叮咛，留下陈子昂。
<br/>
<br/>        大家犹豫了一下，虽然心里不舍，却也知道陈元良说的话是真的，手术后他们在这里都是多余的，不过等到老人家手术做完，看着能安心一些，老人家对生死看得如此透彻让他们心里既难过也轻松。
<br/>
<br/>        父亲有父亲的尊严，母亲有母亲的爱恋，大家听了陈元良的话，都退下了，冉灵卉也准备出去。
<br/>
<br/>        陈元良叫住了冉灵卉：“老婆子，你等会啊，你急急忙忙的跑个什么劲。”
<br/>
<br/>        “嗨，你不是让大家都走嘛，又没有让我留下来。”冉灵卉有点不高兴。
<br/>
<br/>        “嘿，杠精上身，过来，我还没有进手术室了，进了手术室之后你再走。”陈元良招招手，冉灵卉走到陈元良的身边，俩个人手拉着手。
<br/>
<br/>        “子慧，把门带上啊。”陈元良看着大家出门，开口提醒陈子慧带上门。
<br/>
<br/>        “子昂，来，给我讲讲你这些年发生的事情，我想和你妈听你亲口给我们说。”陈元良看着陈子昂。
<br/>
<br/>        冯睿的话还一直回响在陈元良和冉灵卉的耳边，俩个老人家见到了张函也见到了江俞轩，他们是过来人，那俩个孩子对陈子昂的感情都不一般，这让他们对冯睿的话产生了怀疑，毕竟这孩子有几年不着家，谁也不知道她都干了些什么。
<br/>
<br/>        陈子昂回来的这几天，一直在忙忙碌碌，也就是晚上回来陪陪父亲和母亲，俩个老人家也能看得出来陈子昂坚强背后的疲惫，想问的话一直都问不出口，陈元良觉得现在他应该知道，他也怕他的时日不多。
<br/>
<br/>        陈子昂并不知道冯睿回去找了父亲，因为她回来大家都只字未提冯睿回家的事情。
<br/>
<br/>        “爸，你想知道什么？”陈子昂微笑着问：
<br/>
<br/>        “我这个半生如果要讲给您听的话，大概得好久，一两句话是说不完的。我简单的告诉你几句可以吗？”
<br/>
<br/>        “能概括就行，也别隐瞒我和你妈。”
<br/>
<br/>        “我和张函的事情你们都知道，张函无声无息的走了，我未婚先孕怀了孩子，我怕给老陈家丢人，我就只身到了北京，然后在北京的街头碰见了岭弯的落雪，在她的资助下在北京开起了公司，经过几年的打拼才有现在的规模。”
<br/>
<br/>        “你和江家那小子是什么关系？”
<br/>
<br/>        “江俞轩？”陈子昂没有想到父亲会问起江俞轩，她的心里竟然微微一疼，她和江俞轩是什么关系？
<br/>
<br/>        “江俞轩在我开公司的那一年就到了我公司，我现在有这样的成就也和他有一定的关系，他能干，勤奋，我们是同事也是朋友。”
<br/>
<br/>        “就仅仅是同事和朋友的关系？”陈元良再次问。
<br/>
<br/>        “父亲这是担心什么呢？江俞轩的妻子你也看见了，是张函的妹妹，我和他这些年清清白白。您难道会怀疑您女儿的人品？是谁告诉你什么了吗？”
<br/>
<br/>        陈元良和冉灵卉相互看了一眼。
<br/>
<br/>        “你以后打算怎么办？”陈元良有些心疼自己的小女儿，他们难道不应该相信自己的孩子嘛，这样问子昂，子昂心里肯定很难过。
<br/>
<br/>        “爸，我要去找六哥，不瞒您说，我已经谋划了好久，之前有见过六哥一次，但是六哥不敢认我，爸，您也不希望六哥单打独斗吧？”
<br/>
<br/>        “他怎么会是单打独斗呢？他的同事呢？”陈元良是见过陈子寒的同事的。
<br/>
<br/>        “和他唯一有联系的人牺牲了，他不能暴露身份，也无法联系。所以爸，我得去帮他。”陈子昂走过跪在父亲的脚下。“等你手术做完我就要离开这里，爸，你一定要等着我和六哥回来。”
<br/>
<br/>        “子昂。”陈元良伸出手，抚摸着陈子昂的头，陈子昂拉着陈元良的手，将脸埋在陈元良的手中，温热的泪湿了陈元良的手心。
<br/>
<br/>        冉灵卉俯下身抱着陈子昂，她生下了八个孩子，四个小的都是多灾多难，要是知道生下他们，他们过得这么辛苦，她就不应该生他们啊，子寒不知道生活在什么样的情况下，如今又要搭上子昂，她的心疼啊，还有老三，老三得在床上躺一年。
<br/>
<br/>        她就是一个大字不识的农民，陈元良就是她的天她的地，陈元良要她怎么做她就怎么做，可是她想她的儿子啊。
<br/>
<br/>        “爸、妈，您们放心，您看您的子昂已经长大了，我现在也是当妈妈的人了，我会为了孩子像您们一样坚强，您们等着，我一定会将六哥给您们带回来。”陈子昂抬起头，眼里有光也有泪花闪现。
<br/>
<br/>        就算不为六哥，为了她自己和孩子，她也必须要面对很多。
<br/>
<br/>        “子昂啊，兄妹齐心，其利断金，你去吧，去和你六哥一起，去做你六哥的左膀右臂，我们等着你们回来。家里你就不用操心了，我替你盯着呢。”陈元良摸摸陈子昂的头。
<br/>
<br/>        陈子昂能了解陈子寒那么多事情也就证明了陈子昂现在有能力了，几个老大连陈子寒干什么都不知道，就更不用说去帮助陈子寒了。
<br/>
<br/>        他老了已经无能为力，心里有千万份不舍也只能如此，如果能在有生之年看到子寒回来，他也心安。
<br/>
<br/>        他后来经常看电视，看那些谍战片，看得他热泪盈眶，看得他心痛不已，他以为那些都是杜撰出来的，可是想到他自己的儿子，他就会想有可能他的儿子生活得比这还残酷。
<br/>
<br/>        陈元良看着陈子昂，他希望他的孩子都平安，可是想到躺在床上的陈子澜，陈元良的心又揪起来了。
<br/>
<br/>        “子昂，危险啊！”
<br/>
<br/>        “如果大家都害怕危险，那这个世界还有什么正义可言？爸，我没有太大的志向，我就想要六哥能够回来和我们团聚。”陈子昂坚定的说。
<br/>
<br/>        “你去吧！”陈元良拉起陈子昂，老泪纵横。
