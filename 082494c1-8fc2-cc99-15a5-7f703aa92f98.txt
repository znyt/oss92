　　轰隆隆！
<br/>　　高空之上，空间急速塌陷，传出轰隆隆的巨响，震动天地。
<br/>　　所有都是精神大震，真圣界终于要开启了。
<br/>　　那处塌陷的空间，足有方圆数百丈，随着空间的不断塌陷，形成了一个白光弥漫的空间通道。
<br/>　　空间通道非常的虚幻，但随着时间的流逝不断的凝实，约莫过去了盏茶的时间之后，空间通道终于彻底凝实了。
<br/>　　“进真圣界！”
<br/>　　“我们走！”
<br/>　　“哈哈！走！”
<br/>　　霎时之间，呼喝声震天，此地聚集的无数武者，纷纷冲天而起，如同蝗虫一般，密密麻麻，铺天盖地的向空间通道中飞去。
<br/>　　苏莫也是毫不犹豫，和任云都等人一起，冲进了空间通道之中。
<br/>　　他必须进入真圣界，只有进入真圣界，才有可能找到逃生的办法。
<br/>　　嗖嗖嗖！！
<br/>　　各大种族之中，一名名年轻的武者身形闪掠，冲进了空间通道之中。
<br/>　　这个过程没有持续多久，仅仅只是六、七个呼吸的时间之后，数以千万计的年轻武者，便全部进入了真圣界。
<br/>　　而后，通向真圣界的空间通道，便开始缓缓消散。
<br/>　　现场，只留下了大祭司、逍遥剑圣、巨昙尊者、吴晋导师等等各大势力带队的强者。
<br/>　　真圣界的空间通道，严禁任何武帝境以上的武者入内，就算是武圣境强者，都无法进入其中。
<br/>　　不过，众强者虽然无法进入真圣界，却是有办法看到真圣界内的情况。
<br/>　　只见不少人纷纷手臂一挥，身前便出现了一副清晰的画面，正是真圣界内的情况。
<br/>　　他们之所以能如此，是因为进入真圣界内的人，有些人根本不是要接受通圣三关的考验，而是充当外界的‘眼睛’。
<br/>　　通道之中，苏莫急速飞驰，呼吸之间，他便穿过了通道，来到了真圣界。
<br/>　　所谓的真圣界，是一片独立的小世界，但又不是正常的小世界。
<br/>　　这片世界很大，一眼望不到尽头，这里没有实地，只是一片虚无之中的小世界。
<br/>　　抬眼望去，这片世界的中央，虚空之中屹立着一块巨大的石碑，高达千丈，通天彻地，巨大无比。
<br/>　　这块石碑之上，还浮现这一个个名字，粗略一扫，足有百余个之多。
<br/>　　但是，当苏莫等众人来到此地之后，这些名字便渐渐的模糊了起来，转瞬间便消失不见。
<br/>　　“这是上一次在真圣碑上留名的人！”有人惊呼道。
<br/>　　上一次真圣界开启，所留下的名字，一直保存到这一次真圣界开启。
<br/>　　随着真圣碑上那些原有的名字消失，真圣碑上立刻绽放出夺目的光芒，形成了一个无比巨大的气罩，将所有人隔绝在了气罩之外。
<br/>　　这气罩足有方圆三万里，通体透明，光芒流转。
<br/>　　而在气罩的前方，面相众人的方向，有着一个巨大的门户，是唯一可以进入气罩内的入口。
<br/>　　这就是圣门关！
<br/>　　通过圣门，才可以更接近真圣碑，接受第二关的考验。
<br/>　　进入真圣界的人，足有近两千万之多，铺天盖地，密密麻麻。
<br/>　　众人看到圣门出现，毫不犹豫，全部都向圣门冲了过去。
<br/>　　嗖嗖嗖！！
<br/>　　身影闪烁，破空声不绝于耳，飞灵族的人速度最快，冲在了最前方，尤其是飞长云，一马当先。
<br/>　　只见其双翅一扇，整个人如同瞬移一般，速度快的不可思议。
<br/>　　唰！
<br/>　　飞长云第一个便冲进了圣门，进入了气罩之内，其他飞灵族的人仅随其后。
<br/>　　而就在这个时候，神奇的一幕出现了，很多飞灵族的人根本无法跨越圣门，刚一进入圣门，身形便瞬间消失无踪，被传送出了真圣界。
<br/>　　后面的人依旧前赴后继，无论有没有信心闯过圣门关，他们都要一试。
<br/>　　不断的有人进入圣门，通过圣门关的考验，但更多的人是消失不见，能进入圣门的人，连一成都不到。
<br/>　　苏莫跟在众人的后方，他也不急，一边慢悠悠的向前飞行，一边四处打量，看看有没有什么能脱身的地方。
<br/>　　但是，真圣界内除了这个巨大的真圣碑，空无一物，一片虚无，让他倍感无奈。
<br/>　　他的心中，充满了阴郁，巨大的危机感笼罩全身，前方全部都是死路啊。
<br/>　　就在此时，他突然感觉到一缕凌厉的杀机落在了自己的身上，转头一看，只见前方的巫人王一边向圣门飞去，一边回头冷视着他，眸中的杀机毫不掩饰。
<br/>　　“不知死活！”苏莫心中冷笑一声，看来这个家伙已经等不到第三关了，估计进入圣门之后，就要向他动手了。
<br/>　　在苏莫等无数人飞向圣门之时，后方也有少量的人没有行动。
<br/>　　这些人，均是修为不高，手托不知名的玉石，玉石中有影像变换，影像正是真圣界之内的情景。
<br/>　　这些人手中的玉石，会将影像传递到外界，让得等候在外界的强者，看到真圣界内的情况。
<br/>　　一名名武者进入了圣门，姜神夜、巨摩、黑锋、巫殇、巫人王、尤天韩等真圣榜的天才人物，全部都顺利的进入了圣门，没有丝毫波澜的通过了第一关。
<br/>　　大量的武者被淘汰，苏莫亲眼见到，和他一起前来的神武学府弟子，有九成九的人都被淘汰了。
<br/>　　谢彬，也毫无意外的被淘汰了。
<br/>　　任云都，倒是通过了圣门，成功度过了第一层考验。
<br/>　　约莫三、四个呼吸之间，除了少量不参加考核的人之外，几乎所有人都经过了圣门的考验，通过圣门考验的人，只有二十余万人。
<br/>　　而苏莫，是最后一个飞进圣门的人，他只感觉到一股无形的力量扫过他的身体，而后他便毫无阻隔的飞进了圣门之内，轻松的通过了第一关的考验。
<br/>　　所有进入圣门的人，都是停下了身形，目光望着巨大的真圣碑，等待第二关的开始。
<br/>　　没有人敢贸然冲向真圣碑，因为直接冲上去，会被直接淘汰，根本不可能触摸到真圣碑。
<br/>　　“苏莫，你还债的时候到了！”就在此刻，一声暴喝声突然响起。
<br/>　　只见早一步进入真圣界的巫人王，见苏莫进入圣门之后，立刻大手一探，直接向苏莫凌空抓了过去。
