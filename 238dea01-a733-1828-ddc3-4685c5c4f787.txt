        傅夜沉刚走，童筱颖就接到了上次那位雇主夏凌焰的电话。
<br/>
<br/>        “小筱颖，你来滨江城一趟，快来辅助我完成设计啊！交稿时间快到了，我一个人赶不出来！”电话那头，夏凌焰扯着嗓子哀求。
<br/>
<br/>        童筱颖反问：“你怎么也在滨江城？”
<br/>
<br/>        “我在这里接了单子啊！是一栋私人别墅的设计，来吧来吧！小筱颖，江湖救急，薪酬绝对不会少你！”夏凌焰十分认真地说。
<br/>
<br/>        “你那拖延症的毛病什么时候能改改啊？”童筱颖忍不住调侃道。
<br/>
<br/>        她虽然只跟夏凌焰合作过一次，但是在上次项目合作的朝夕相处之间，她发现这个夏凌焰是个患有非常严重的“拖延症”患者。
<br/>
<br/>        一个项目接到手，定了多长时间内完成，对方也预付了酬劳，而他夏凌焰，偏偏要在这工作的时间内先玩够了，等到了临近交稿日期，他才会开始着手设计。
<br/>
<br/>        所以，童筱颖上次不得不陪着他一起住在酒店里，日夜兼程地赶工。
<br/>
<br/>        这次也毫不例外，十有八九又是这种情况！
<br/>
<br/>        “小筱颖，我这毛病改不了啊！拜托你了嘛！我跟你说啊！我这次来滨江城，又有了一个美丽的艳遇！你知道对方是谁吗？对方是个千金大小姐！而且，她在认识我之前，刚拒绝了一个男人的婚约！我跟她是在一个高档餐厅里邂逅的，是我主动去找她的，你说我是不是很……”
<br/>
<br/>        “你把地址给我，我现在就过去给你帮忙，不过这次我要双倍薪酬！”童筱颖汗颜地打断了夏凌焰的滔滔不绝。
<br/>
<br/>        工作才是重点啊！
<br/>
<br/>        上次在云城，他就艳遇了一个，玩腻了，还拿她童筱颖当幌子，甩了那个女人。
<br/>
<br/>        当然，他夏凌焰玩女人和完成工作是两不误，把女人给甩了，工作也完成了。
<br/>
<br/>        夏凌焰一听这话，几乎是喜上眉梢：“没问题，我马上发给你！”
<br/>
<br/>        童筱颖等来了司机后，让司机帮她把买的衣服、鞋子、香包那些东西全部带回私宅去，而她自己单独去赴夏凌焰的约。
<br/>
<br/>        滨江城，星际大酒店。
<br/>
<br/>        与上次一样，夏凌焰仍旧是订的小套房，内有两个卧室一个会议厅。
<br/>
<br/>        童筱颖按响了门铃，夏凌焰一开门就张开双臂，想给她一个大大的拥抱，但是被童筱颖伸手按上了他的额头，毫不留情地拒绝了他这个不怀好意的拥抱。
<br/>
<br/>        夏凌焰识趣地努了努嘴，放下手来，邀请童筱颖进了屋。
<br/>
<br/>        会议室的办公桌上，夏凌焰已经开好了笔记本电脑，摆好了模型纸和工具，就等着她童筱颖一来就开工。
<br/>
<br/>        但是……
<br/>
<br/>        就在童筱颖刚准备在桌前坐下，夏凌焰将童筱颖上上下下打量了一番后，忍不住夸赞道：“噢，筱颖，你今天真漂亮！”
<br/>
<br/>        “谢谢！”童筱颖抿嘴一笑，端坐在椅子上后，拿起模型纸，一脸认真地询问，“我今天的任务是不是帮你做模型？”
<br/>
<br/>        然而，夏凌焰却答非所问：“你这一身累计起码价值五万，还是国际一线高档限量版，怎么？最近你发财了吗？”
<br/>
<br/>        “我这一身衣服最少要五万吗？”童筱颖瞬间有些难以置信地抬头看着一屁股坐在会议桌上的夏凌焰。
<br/>
<br/>        “嗯？”夏凌焰双手抱臂，一只腿悬在半空中，一只腿踮脚着地，目光无意间落在了童筱颖左手的中指上，一副吊儿郎当的样子问，“我去，你又订婚了啊？”
<br/>
<br/>        “……”童筱颖汗颜。
<br/>
<br/>        “你这次的二婚老公看起来还挺有钱的嘛！是做什么的？有没有业务人脉介绍给我啊？”夏凌焰挑了挑眉，不放过自己的任何商机。
<br/>
<br/>        童筱颖咧嘴笑了笑：“他的集团旗下，有自己的设计部门，压根就不需要外聘。”
<br/>
<br/>        “呦！你不错嘛！傍上了富豪！这次应该不是渣男了吧？”夏凌焰从会议桌上跳了下来，拉开了童筱颖身旁的椅子，慵懒地坐下。
<br/>
<br/>        上次和她童筱颖合作的时候，他就跟她八卦过她的私事。
<br/>
<br/>        本来，看她长得挺漂亮，还想追她的，谁知她结过婚，原因是老公婚内出轨有了私生子，他就立马放弃了要追求她的念头。
<br/>
<br/>        “渣不渣，我不知道，昨天才领的证。不过……”童筱颖一边掰着模型纸，一边感慨道，“他有过三个女人，都被那三个女人给甩了，而且，我在他身上闻到了和他助理的女秘书身上一样的薄荷香气。这种香气，不像是香水，而像是用新鲜的薄荷叶经过蒸馏提炼出来的纯露。”
