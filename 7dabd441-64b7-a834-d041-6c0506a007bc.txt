　　却说叶尘四人和天海三少，出了帝豪大酒店之后，
<br/>　　唰唰唰！
<br/>　　一旁的马路上，顿时便窜出十几道身影，将叶尘四人包围在了中央。
<br/>　　而且这些人，个个气势不弱，一看就知道都是练家子，其中不乏高手。
<br/>　　除了叶尘之前，其他三人全都脸色一变，就连身手不弱的唐威，也露出一脸凝重之色，脸色有些发白。
<br/>　　这些人当中，有几个的修为，他根本看不透，肯定远在他之上，说不定是化劲宗师以上的修为！
<br/>　　虽然他一直觉得叶尘不简单，可是毕竟太年轻了，因而唐威并不看好自己一方。
<br/>　　“死胖子！本少就在最后再给你们一次机会，乖乖磕头道歉，本少可以既往不咎，否则的话，嘿嘿！”
<br/>　　李虎脸色一阵变幻，最终一咬牙，
<br/>　　“好！我可以磕头道歉，但是你能不能放过我的室友？”
<br/>　　唐威和陈东顿时急了，
<br/>　　“老四！你说什么呢？”
<br/>　　“就是呀！大不了咱们报警！”
<br/>　　天海三少顿时纷纷大笑，仿佛听到了天底下最好笑的笑话，
<br/>　　“报警？我们又不会亲自动手，你们报警又能如何？”
<br/>　　“少废话！死胖子，你磕还是不磕？不磕的话，你们四个，每人给我留下一条胳膊再走吧！”
<br/>　　李虎伸手拦住唐威和陈东两人，
<br/>　　“你们不要再说了！大丈夫能屈能伸，不就是磕个头吗？也没有什么大不了的！”
<br/>　　李虎说着，当即便要跪下，然而这时，却又被叶尘一把拉住了胳膊。
<br/>　　“尘哥，你——”
<br/>　　叶尘微微一笑，
<br/>　　“老四，你记着，只要有我在，这世上，便没有人可以逼迫你做任何事情！”
<br/>　　说完这话，叶尘缓缓朝天海三少等人望去，眼眸中泛起一抹杀意。
<br/>　　“为什么这世上，总有人喜欢找死呢？”
<br/>　　叶尘自言自语了一句，一旁的李虎三人，全都有些摸不着头脑，不知道他想要干什么。
<br/>　　然而，
<br/>　　就在叶尘准备直接一道神念，将天海三少全部灭杀之时。
<br/>　　忽然，
<br/>　　一道突兀的声音，陡然自众人后方响起，
<br/>　　“是什么人敢在我酒店门前闹事？”
<br/>　　天海三少中的那位江少，听到这话，想也没想，直接回了一句，
<br/>　　“是我们天海三少！不想死的，就少管闲事！”
<br/>　　不料，那道声音冷冷一笑，
<br/>　　“江飞龙，你好大的煞气啊！”
<br/>　　声音落下的同时，那道声音的主人，已经从后方人群中，缓缓走了进来。
<br/>　　这是一名身穿魁梧，气质凌厉的青年男子。
<br/>　　看到此人，原本还想动用神念，直接将天海三少轰杀的叶尘，也顿时微微一愣，旋即嘴角露出一抹笑容，低声喃喃道：
<br/>　　“他怎么来了……”
<br/>　　一旁的李虎，听到叶尘这话，顿时好奇道：
<br/>　　“尘哥，他是谁啊？”
<br/>　　叶尘淡淡一笑，
<br/>　　“很快你就知道了！”
<br/>　　那江飞龙听到有人竟然敢直呼其名，顿时两眼一瞪，满脸煞气的转过身去，
<br/>　　“我特么——”
<br/>　　话才只说一半，江飞龙仿佛看到了什么恐怖的事情，下面的话顿时便说不出口了，原本还一脸的煞气，顷刻之间便挤出了满脸笑容，
<br/>　　“唐唐……唐哥？您怎么来了？”
<br/>　　众人顿时哗然，同时个个一脸茫然，
<br/>　　“堂哥？难道这人是江少的堂哥？可如果真是这样的话，却为何要拆江少的台？”
<br/>　　众人正暗暗纳闷，这时另外两位大少，竟然也纷纷鞠躬行礼，齐声喊了一声，
<br/>　　“唐哥！”
<br/>　　“什么情况？这人到底谁的堂哥啊？”
<br/>　　众人顿时更加懵逼了，不过很快就有人说道了真相，
<br/>　　“什么堂哥？这位是唐少！是云州唐家的太子爷，唐清山！”
<br/>　　“听说这家帝豪大酒店，之前是武家的产业，如今已经归入唐家旗下了！”
<br/>　　“难怪那三位大少对他如此恭敬，没想到竟是这尊大神！”
<br/>　　……
<br/>　　原来，此人不是旁人，赫然正是唐清雅的哥哥唐清山！
<br/>　　云州唐家，如今是天南省第一大家族，可是云州毕竟只是地级市，自然无法和天海市这样的一线城市相比。
<br/>　　因而，唐家有意在将家族事业的重心，转移到天海市，而唐清山如今，全权负责唐家在天海的各项事务。
<br/>　　虽然他还年纪尚轻，可是背后有唐家撑腰，如今在天海市商界，已经是大佬级的人物！
<br/>　　天海三少不过就是三大家族的纨绔大少，自然远远无法和唐清山相提并论。
<br/>　　就连此时，已经追出来的杨之凡，看到这一幕，也不由得露出一抹肃然之色，
<br/>　　云州唐家！
<br/>　　别人或许不清楚，可是他杨家，身为苏叶集团代理商，自然明白这个家族有多牛！
<br/>　　一年之前，云州唐家还只是天南省的二、三流势力，如今已经是公认的天南之首！
<br/>　　便是放眼整个华夏，也已经隐隐跻身一流势力之列！
<br/>　　而眼前的这个唐清山，又是唐家下一任家主的不二人选，据说还是那位云州叶先生钦点的继承人。
<br/>　　在杨之凡看来，或许唐家如今还不是华夏的顶级家族，可是只要唐家背后的那位叶先生还在，成为顶级家族也只是时间问题罢了。
<br/>　　一念及此，杨之凡心头更加火热，已经开始寻思着，待会如何和这位前途无量的唐家太子爷，好好的结交一番了。
<br/>　　……
<br/>　　唐清山在那三人身上扫了一遍，轻哼了一声，道：
<br/>　　“你们三个臭小子，又在欺负人了是不是？要不要我把你们的所作所为，跟江总、王总还有方总，一一汇报一下？”
<br/>　　之前还无比嚣张的三大少，听到这话，顿时一下子全怂了，慌忙一路小跑到唐清山跟前，陪着笑脸道：
<br/>　　“唐哥，您就是我亲哥，可千万别呀！”
<br/>　　“而且我们可没有欺负人，是那四个家伙主动挑衅我们，我们才是被欺负的那一方！”
<br/>　　“是呀！唐哥，您要给我们做主啊！”
<br/>　　三人眼看情况不对，转而装作一副可怜兮兮的样子，居然向唐清山哭诉起来。
<br/>　　唐清山自然不信，不过还是顺着三人指的方向望去，笑道：
<br/>　　“在天海还有人敢欺负你们三位？我倒是想看看是……”
<br/>　　唐清山话还没有说完，猛然看到一道熟悉的身影，声音也戛然而止。
<br/>　　起初，唐清山还以为是自己眼花了，狠狠摇了摇头，这才确认是真的，当即将跟前的三少扒开一旁，然后快速冲了过去。
