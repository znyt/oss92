        “等着她回来跟他提离婚”的这句话，让傅夜沉心头一颤。
<br/>
<br/>        突然间，他第一次尝到在感情上很没安全感是什么滋味。
<br/>
<br/>        傅夜沉猛地放下手里的工作，疾步走了出去。
<br/>
<br/>        他突然一声不吭地离开，让连华生有些丈二的和尚摸不着头脑。
<br/>
<br/>        澳洲，某处农庄里。
<br/>
<br/>        餐厅的木栏外是一片一望无际的青色草原，不远处移动着一群黑白花奶牛，还有几只澳洲牧牛犬在围着牛群跑。
<br/>
<br/>        而此刻，童筱颖正坐在木桌前，手中拿着刀叉切着一份全熟的牛排。
<br/>
<br/>        但是，餐桌上，还摆着“零、一、三、五、七”这五种熟度的五块牛排。
<br/>
<br/>        秦烈坐在童筱颖的对面，面前餐盘里的牛排是五分熟，他一边自己切着牛排，一边还不忘跟童筱颖打趣道：“筱颖，你知道吗？我和烟儿第一次来这里吃牛排的时候，烟儿点的也是全熟的。烟儿跟我说，在国内，你们很多人去西餐厅点牛排的时候，总会学着西方人的样子点七分熟以上的牛排。不然，会觉得自己掉档次。殊不知，牛排的熟度在国外，只是不同人喜欢的不同口感罢了，没有哪种熟度最好吃之说。只是不明白，为什么到了她们那儿就变成了七分熟最好吃。烟儿吃不惯那种夹生的，所以就会点全熟的。不过，你可以试试牛排其他熟度的口味，看看你最喜欢吃哪种。”
<br/>
<br/>        这些天，一日三餐，童筱颖都是跟着秦烈一起吃的，因为只有秦烈知道在澳洲的哪些地方有好吃的特色，不过他们是AA制。
<br/>
<br/>        秦烈也不知道自己是怎么了，他最近总能在童筱颖的身上看到烟儿的影子。
<br/>
<br/>        她的喜好，还有一些对事情的见解，都和烟儿很像。
<br/>
<br/>        就在童筱颖刚想对秦烈说什么的时候，她的目光落在了秦烈的身后，顿时目瞪口呆了。
<br/>
<br/>        秦烈还未反应过来时，余光里突然出现一抹黑影，拉开了他身旁的椅子坐了下来。
<br/>
<br/>        “服务员，我要一份全熟的牛排。”男人向服务员招了招手，说出了一口流利的美式英语。
<br/>
<br/>        秦烈侧目看着这个男人，还未反应过来时，耳畔就听到童筱颖弱弱地唤了声：“大哥。”
<br/>
<br/>        咻，原来是筱颖的哥哥啊！
<br/>
<br/>        秦烈从惊怔中恢复到原状，一脸微笑地向傅夜沉用中文打了声招呼：“大哥，你好。”
<br/>
<br/>        傅夜沉却看都不看秦烈一眼，犀利的黑眸直勾勾地落在童筱颖的身上，冷冷地说道：“解释一下。”
<br/>
<br/>        童筱颖怔愣了一下，瞥了秦烈一眼，咧嘴道：“他是我老公。”
<br/>
<br/>        “嗤”地一声，秦烈连忙捂住自己的嘴巴，然后剧烈“咳”了起来。
<br/>
<br/>        童筱颖连忙给秦烈递上水杯。
<br/>
<br/>        秦烈喝了口水，咽下嘴里的牛排后，自己给自己拍了拍胸脯缓口气。
<br/>
<br/>        “这段时间，你们玩得很开心，嗯？”傅夜沉十分淡定从容地问话。
<br/>
<br/>        童筱颖却听着，总觉得这话有点儿酸。
<br/>
<br/>        大哥该不会是吃醋了吧？
<br/>
<br/>        而秦烈的立场顿时尴尬了。
<br/>
<br/>        筱颖一开始就跟他说过，她有老公的，只是他自己不相信而已。
<br/>
<br/>        更何况，他认识她的这段时间，也没见她老公给她打过电话或者是发过简讯，这更加让他误以为她是在跟他撒谎。
<br/>
<br/>        当这男人真的出现时，他仍旧对童筱颖抱有狐疑的态度。
<br/>
<br/>        筱颖，你该不会是随便请来的一个男人，来假装是你老公骗我的吧？——秦烈。
<br/>
<br/>        秦烈拿起手机，悄悄地给童筱颖发了条短信。
<br/>
<br/>        童筱颖瞥了一眼放在桌上的手机，看到弹幕上的内容后，对秦烈笑着说道：“他真是我老公，傅氏集团的幕后老板。”
<br/>
<br/>        “噢！傅大Boss对吧！”秦烈的脸上露出一个大大的傻笑，立马起身，向傅夜沉友好地伸出手来，“幸会幸会。”
<br/>
<br/>        为了不失风雅，傅夜沉也站起身来，握住了秦烈的手，勾唇一笑：“我也很高兴认识秦先生。”
<br/>
<br/>        她都没做介绍，他就知道他姓“秦”。
<br/>
<br/>        看来，他是真的在监听她的一举一动。
<br/>
<br/>        童筱颖瘪了瘪嘴，顿感失落。
<br/>
<br/>        反正，这餐饭吃得特别扭。
<br/>
<br/>        散席时，秦烈给傅夜沉递上了自己的名片，还希望日后有机会可以跟傅氏集团合作。
<br/>
<br/>        傅夜沉落落大方地答应。
<br/>
<br/>        秦烈也很识趣，找了个理由提前离开了。
<br/>
<br/>        回民宿的路上，和傅夜沉坐在同一辆出租车里，童筱颖就感觉到气氛很压抑。
<br/>
<br/>        她总觉得他像是在生气，但又觉得他没有生气，只是在吃醋。
<br/>
<br/>        两人沉默了许久，童筱颖才开口打破僵局：“我绝对没有做对不起你的事情噢！”
