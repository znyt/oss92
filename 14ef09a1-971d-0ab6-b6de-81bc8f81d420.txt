        &lt;h3&gt;第三卷第一百零七十九章  刘诚之拜访安教授&lt;/h3&gt;
<br/>
<br/>        凌辰的新年过得一塌糊涂，每天都是在买醉中，当初说了，如果有结局了他会放下的，真的是没有见到宁雅是心里的那份痛苦被无限放大，只想买醉。
<br/>
<br/>        家里催着他要结婚，给他介绍对象相亲，新年的几天，凌辰都是喝的醉醺醺的去见相亲对象。
<br/>
<br/>        当凌辰托着疲惫的脚步回到家时，那种空落落的感觉又来了，想忘记一个人真难啊，他也想重新开始一段感情，要不就在家里随便找一个处处试一下，也许还能处处感情了。
<br/>
<br/>        正当凌辰胡思乱想的时候，狸猫的消息来了，看到狸猫的消息凌辰还是很开心，他还是有机会的，可是想到那天的电话，他心里又无端的难过起来。
<br/>
<br/>        宁雅没有男朋友，那么那天晚上又怎么解释？
<br/>
<br/>        他想将这件事找个人分析分析，但是又没有办法说出口，想问问狸猫，但是又感觉不妥。
<br/>
<br/>        最后还是觉得问一下刘诚之。
<br/>
<br/>        刘诚之陪着父亲在年前拜访了自己的导师安林曦教授，几个人坐在一起讨论了时下的一些事情和对目前形态的分析后安林曦对刘诚之说：“诚之，你认识董亚子？”
<br/>
<br/>        刘诚之：“前几天去贵州出差认识的。”
<br/>
<br/>        安林曦：“亚子是一个非常聪明的女孩子，性情热辣，只是他的父亲想让她回去帮着管理工厂，要不然.......”
<br/>
<br/>        刘父笑着说：“你这桃李遍天下，哪一个不是优秀的，你都要大家和你一起做学问？”
<br/>
<br/>        安林曦：“非也，非也，时代在前进，社会在发展，哪一个行业都是在做着贡献，学问不分彼，行行出状元嘛！”
<br/>
<br/>        刘父点点头：“哈哈哈，你说的对，行行出状元，我们的这个社会需要各行各业齐头奋进，共同发展才能达到均衡。”
<br/>
<br/>        安林曦：“看着他们这一代年轻人个个奋发向上，心安矣！”
<br/>
<br/>        刘父：“只是时下奢靡之风盛行，各种不良风气滋生，我这颗心不安啊。”
<br/>
<br/>        安林曦：“社会要发展就必然会滋生一些不良的习气，有些人把握不好尺度，但是物极必反，现在不是已经送各个方面进行整顿了吗？”
<br/>
<br/>        刘父：“哈哈哈，确实，现在的一带一路，是利国利民，不，不但是利了我们自己，而且还利了全世界，彰显了我大国的气度和魄力。”
<br/>
<br/>        安林曦：“我们要做的就是紧跟着我们领导人的步伐，团结奋进，让百花齐放、让万物重生！”
<br/>
<br/>        刘父：“我们要做的就是要让我们的下一代挺起腰杆做人，自强自立、爱国爱民！”
<br/>
<br/>        安林曦：“肃风整纪是你们的事情。”
<br/>
<br/>        刘父：“教书育人是你的事情。”
<br/>
<br/>        刘诚之：“社会教育、家庭教育、学校教育是不能分割的，所以你们俩人所做的事情是整体的，不分彼此！”
<br/>
<br/>        “哈哈哈，还是年轻人理解透彻啊！”刘父和安林曦大笑。
<br/>
<br/>        刘父：“你这女儿快要结婚了吧？”
<br/>
<br/>        安林曦：“证都领了，就是等着举办婚礼。”
<br/>
<br/>        刘父：“我们这个小子还是个憨憨，我这心里还是有点着急的啊。”
<br/>
<br/>        安林曦：“你这个憨憨小子，怕是不憨啊。”
<br/>
<br/>        “诚之啊，亚子初五就会来我家的，既然你们认识，初五你就来我家一趟，陪我招呼一下远来的客人。”安林曦对刘诚之笑着说。
<br/>
<br/>        “啊，初五，我还打算上班去呢？”刘诚之想到董亚子那暴力的模样莫名其妙的脸红了。
<br/>
<br/>        “谁初五就上班？”俩个老人家齐声问。“撒谎都不会真是个憨憨！”
<br/>
<br/>        “安教授，你放心，初五我就给你将他提溜过来，哈哈哈。”刘父开怀大笑。
<br/>
<br/>        .........
<br/>
<br/>        “你去甸城，是不是为了董家？”刘父从安教授的家里出来就问刘诚之。
<br/>
<br/>        “有一半吧，我听说甸城那个代理商以势欺人，强势要求做总代，然后还要娶董亚子，心里有点打抱不平，就想过去看看那是一个什么地方。”
<br/>
<br/>        “甸城这几年倒是风调雨顺，因为那边地处偏远，过去的人都基本回来都能有所作为，但是好像都和陆家走得近，陆家这些年虽然看起来可圈可点，但是总觉得有些奇怪，至于哪里奇怪我也说不上来，就是感觉的问题。”刘父眉头紧皱。
<br/>
<br/>        “你好像和陆伯伯一直不太对付，是不是偏见？”
<br/>
<br/>        “什么叫偏见？你看看他把那个女儿养成什么了？不过这两年去了甸城，据说想带动当地信息化大发展，申请了很多资金，我可警告你啊，你切不可像陆家的那个女儿一样。”
<br/>
<br/>        “我是男孩子好不好？”刘诚之有些无语。
<br/>
<br/>        “无论男孩子女孩子都得好好教育，不过陆家那小子不错，做事情兢兢业业，还很靠谱。”
<br/>
<br/>        “嗯，陆哥哥做事情确实很好，年轻有为，以后肯定是栋梁之材。”刘诚之符合着。
<br/>
<br/>        “别那么敷衍我，你就说这几年你打算怎么办？”
<br/>
<br/>        “你不是疑心甸城的情况吗，我一个名不见经传的小人物过去没有人会注意到了的，刚好明年过去帮你考察考察，想娶董亚子的人据说有个很厉害的背景呢。”
<br/>
<br/>        “你这是为了想讨老婆要拉上你爸爸给你垫背呢？”
<br/>
<br/>        “老爷子，我告诉你，你要这样说我还懒得去的了，不行了我也不上班了，回去给安老师当下手。”
<br/>
<br/>        “不去就不去，谁怕谁？你安老师可是初五让你陪他远道来的客人。”刘父哈哈大笑。
<br/>
<br/>        “我陪客人你有这么开心吗？”
<br/>
<br/>        “当然，你这个憨憨开窍了。不过距离是个问题，娶回来是可以的，去贵州你妈怕是不愿意啊。”
<br/>
<br/>        “你想多了，董亚子喜欢的是凌辰，而我也不喜欢她，只是凌辰有了女朋友，不想和她有来往，将我推过去当挡箭牌而已。”
<br/>
<br/>        “凌家那小子还没有搞定宁家那丫头？”刘诚之和他父亲八卦过凌辰的情况，再加上有些家族刘父未必认识但也知道一些情况。
<br/>
<br/>        “不是没有搞定小雅姐姐，而是没有搞定小雅姐姐的父亲。”
<br/>
<br/>        “哦。宁雅还在北京那家公司上班？”
<br/>
<br/>        “是的，那家公司还不错，她这两年一直在那家公司上班。老爸您这一天还关注的很多啊。”
<br/>
<br/>        “就是这么大的圈子，谁家的家长里短你以为我不知道？”
<br/>
<br/>        “嘻嘻，八卦。”
<br/>
<br/>        “什么叫八卦，这叫与时俱进。”
