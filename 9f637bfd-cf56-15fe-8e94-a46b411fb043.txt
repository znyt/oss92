　　苏莫大步的向山上走去，他本可以直接飞过去找玄羽宗的宗主，但这样太过冒犯了，怕是对方不会给他使用星空传送大阵。
<br/>　　所以，他准备先混进玄羽宗内，看看情况再说。
<br/>　　“以乾坤为本，逆自然之易，夺造化之功，由阴阳和.合而返本归元，复归于虚无之道……！”
<br/>　　苏莫还未走到山上的广场，便听到一阵郎朗之音，是有人在讲解武道的经验。
<br/>　　很快，苏莫便来到了广场边缘，入目之处，广阔的广场占地方圆数千丈，一名老者正坐在高台上讲道。
<br/>　　这名老者瘦如竹篙，一身藏布长衫，面貌古奇，颇有文士之风，身上气息浩大深邃，赫然便是武尊境的武者。
<br/>　　而在下方，大量的弟子盘膝而坐，均是在聚精会神的听讲。
<br/>　　“这应该是玄羽宗的宗主吧！”
<br/>　　苏莫暗道，随即他努力收敛全身气息，催动隐身战魂，小心翼翼的走到了广场上。
<br/>　　望着高台上的老者，他略一沉吟，准备等待片刻，等到讲道结束，再找这名老者商议使用星空传送阵之事。
<br/>　　毕竟，现在人多，他一个外人，直接混进对方宗门中，贸然要求使用传送阵，对方肯定不会给他使用。
<br/>　　因为这种做法，是对于玄羽宗的不敬，对方身为玄羽宗的宗主，当着宗内大量弟子的面，肯定不会同意他。
<br/>　　不然，对方身为玄羽宗第一强者，也是脸面无光。
<br/>　　随即，苏莫小心翼翼的向一旁走去，想走到旁边安静等待。
<br/>　　就在此时，蓦地，那名老者突然停止了口中的话语，向苏莫望了过来。
<br/>　　“被发现了？”苏莫心中一惊，他现在修为达到了武帝境七重的巅峰，本以为动用隐身战魂，足以瞒过武尊境一重武者了，没想到这么快就被对方发现了。
<br/>　　“这位朋友，既然来了，何必鬼鬼祟祟，何不现身一叙！”老者望着苏莫所在的位置，朗声说道。
<br/>　　其实，他并没有发现苏莫，只是心中生出了一丝微弱的感应，感应到苏莫所在之地的空间，有一丝不同寻常。
<br/>　　一众玄羽宗的弟子闻言，均是个个面带惊讶的望着苏莫所在之地，他们完全没有发现有任何人。
<br/>　　苏莫闻言，不由得苦笑一声，对方果然发现了他，隐身战魂可以轻松瞒过武帝境九重的武者，但还是无法瞒过武尊境的武者。
<br/>　　既然被发现了，那也没有什么好隐藏的了！
<br/>　　“在下上官昊，见过前辈！”苏莫收起隐身战魂，身形显露而出，向老者遥遥抱拳。
<br/>　　随着苏莫现身，一瞬间，无数道凌厉的目光落在了他的身上。
<br/>　　对于闯进山门的外人，所有玄羽宗的人，均是抱着强烈的敌意。
<br/>　　“上官昊？”
<br/>　　老者闻言眼眸微眯，随即仔细的打量了苏莫一番，见苏莫只是武帝境巅峰修为，他的脸色顿时一冷。
<br/>　　“小子，敢擅闯我玄羽宗，快说，你有什么目的？”老者厉声喝道，眸中冷芒爆射。
<br/>　　“前辈，在下想借贵宗的星空传送大阵一用！”
<br/>　　苏莫沉声说道，言罢，他又补充道：“当然，在下愿意支付报酬，下品灵晶一万块！”
<br/>　　下品灵晶一万块，这个报酬已经不小了，毕竟，催动一次星空传送大阵，也要不了多少的下品灵晶。
<br/>　　“使用星空传送大阵？一万下品灵晶？”老者闻言双眉一挑，因为玄巍星地处偏僻，以往来借用星空传送大阵的人也经常出现，这个报酬的确不小了。
<br/>　　不过，眼前的青年，擅闯他玄羽宗，已经犯了大罪，仅凭一万下品灵晶，就想要使用星空传送大阵，未免太便宜了。
<br/>　　“小子，你擅闯我玄羽宗，分明就是对我玄羽宗的藐视，你认为你出一万下品灵晶，就能使用本宗的星空传送大阵？”老者冷笑的说道。
<br/>　　“那前辈开个价吧！”苏莫沉声说道，只要对方不是太得寸进尺，他就不会反对，毕竟他是有求于人。
<br/>　　“十万下品灵晶，不然，你想使用本宗的星空传送大阵，简直是痴人说梦！”老者冷漠的说道，他要敲诈眼前的青年一笔。
<br/>　　他本想将眼前的青年直接擒拿，毕竟对方擅自闯入，他不出手的话有损他的威严。
<br/>　　但考虑到眼前的青年，年纪不大能有武帝境七重巅峰的修为，想必是古灵星河某个大势力的弟子。
<br/>　　他虽然是此星的第一强者，但在古灵星河其它势力眼中，根本不值一提，所以，为了不惹祸上身，他忍住没有出手。
<br/>　　但是，即便不出手，也要眼前的青年大出一笔灵晶。
<br/>　　“十万！”苏莫闻言眉头一皱，没想到对方真的狮子大开口了。
<br/>　　催动一次星空传送大阵，只要几百块下品灵晶就够了，就算传送的距离较远，几千块下品灵晶也是顶天了，对方居然开价十万！
<br/>　　“怎么？没有吗？”
<br/>　　老者见苏莫皱眉，冷笑一声，身上气息涌动，庞大的威压立刻压迫在了苏莫的身上，威严的说道：“你若是没有，不仅无法使用星空传送大阵，本座还要治你擅闯本宗之罪！”
<br/>　　听闻对方之言，苏莫的眼眸微眯了下来，冷冷的望着对方，这个老家伙真是不知所谓，开口就是十万下品灵晶，这不是抢吗？他就算有也不会拿出来。
<br/>　　“两万下品灵晶，不能再多了！”苏莫淡淡的说道，在对方的威压之下，他依旧面不改色。
<br/>　　“那就别怪本座不客气了！”老者站了起来，全身浩大的气势冲天而起，如同滚滚浪涛，就欲对苏莫出手。
<br/>　　其实，他并不是要擒杀苏莫，只是要压迫苏莫就范，他知道苏莫身上肯定能拿出十万下品灵晶。
<br/>　　十万下品灵晶虽然不少，但对于武帝境七重武者来说，大多数都能拿出来。
<br/>　　就算拿不出来，七、八万总该能拿出来吧！
<br/>　　总之，他要从苏莫的身上，狠狠的榨出一笔灵晶。
<br/>　　苏莫见此，面色一冷，他已经够客气了，给足了对方面子，但对方不识抬举，那他就只能用强了。
<br/>　　嗖！嗖！嗖！
<br/>　　就在大战一触即发之际，蓦地，天际的尽头传来刺耳的破空之声，只见三道强大的身影划破长空，从远处急速而来。
