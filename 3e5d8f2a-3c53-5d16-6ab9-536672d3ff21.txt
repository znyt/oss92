　　奇异空间内。
<br/>　　两道身影凭空而现。
<br/>　　正是萧逸与万山剑主。
<br/>　　“万山剑主，你什么意思？”萧逸冷冷地盯着万山剑主。
<br/>　　万山剑主摆摆手，道，“诶哟，惭愧惭愧。”
<br/>　　“刚才一不小心，碰到了光球。”
<br/>　　“又一不小心，把北山剑主你给拉进来了。”
<br/>　　“真是抱歉。”
<br/>　　万山剑主嘴上说着抱歉。
<br/>　　但脸上却丝毫没有‘抱歉’的意思，反倒是得意的笑着。
<br/>　　“少给我装模作样。”萧逸冷冷说道。
<br/>　　“哈哈哈哈。”万山剑主大笑几声。
<br/>　　得意的笑容，忽然变为了狞笑。
<br/>　　“北山剑主，我知道你通过试炼的速度很快。”
<br/>　　“仅仅大半天时间，你就通过了上百次。”
<br/>　　“可那又如何，你现在便要止步于此。”
<br/>　　“嗯？”萧逸皱起了眉头。
<br/>　　万山剑主狞笑道，“这可是第二层中最难的试炼。”
<br/>　　“连天元境武者都难以通过。”
<br/>　　“你唯一的选择，只有使用剑主令。”
<br/>　　“强行离开剑阁。”
<br/>　　剑宗大长老，早就吩咐过。
<br/>　　试炼者不得擅自动无把握应付的光球。
<br/>　　否则，只能使用剑主令离开剑阁。
<br/>　　而一旦试炼者这样做，等同于考核失败。
<br/>　　“你这样做，你也要离开剑阁，结束试炼。”萧逸沉声说道。
<br/>　　“我有什么所谓。”万山剑主狞笑道。
<br/>　　“反正，我已无缘最强剑主称号。”
<br/>　　“提前离开，也不过少试炼两天，少得些好处罢了。”
<br/>　　“但你嘛，呵呵…”
<br/>　　“流星剑主、猛虎剑主他们，还有两天时间。”
<br/>　　“足以超过你的试炼次数。”
<br/>　　“分数，也会远比你高。”
<br/>　　“这一轮考核，你连前6都进不了，更别说参加下一轮考核了。”
<br/>　　“也就是说。”
<br/>　　“最强剑主称号，已经与你没有关系了。”
<br/>　　“哈哈哈哈。”
<br/>　　万山剑主得意地大笑着。
<br/>　　萧逸的脸色，此时阴沉如水。
<br/>　　万万没想到，万山剑主会做出如此损人不利己之事。
<br/>　　“啧啧，这脸色，真难看。”万山剑主讥讽道。
<br/>　　一想起上次在森林内，在众剑主面前，败于萧逸之手。
<br/>　　他就满腔怒火。
<br/>　　现在，算是‘大仇得报’了。
<br/>　　萧逸怒极反笑，“万山剑主，你就这么确定，我过不了这个试炼？”
<br/>　　“呵。”万山剑主不屑地嗤笑一声，“还死鸭子嘴硬。”
<br/>　　“北山剑主，你连第一层中的最难试炼，都过得如此狼狈。”
<br/>　　“这第二层的最难试炼，连天元境武者都难以通过。”
<br/>　　“我看你拿什么去应付。”
<br/>　　他想起了之前萧逸从光球里出来时，那血迹斑斑的狼狈模样。
<br/>　　说罢，他拿出了剑主令。
<br/>　　“北山剑主，我先走一步。”
<br/>　　“剑阁之外，我等着你出来。”
<br/>　　最后一丝狞笑落下，剑主令上光芒一闪。
<br/>　　万山剑主消失在空间之内。
<br/>　　萧逸的脸色，难看到极点。
<br/>　　现在他只有两个选择，要么通过这一个试炼。
<br/>　　要么，使用剑主令，离开剑阁。
<br/>　　但，剑阁试炼的规矩，规定了若试炼者离开剑阁。
<br/>　　等同试炼失败。
<br/>　　这是规矩，就连剑宗大长老都无权改变。
<br/>　　而且，事后。
<br/>　　万山剑主只需说只是失手才将他拉入这个试炼。
<br/>　　便可将责任推得一干二净。
<br/>　　毕竟，万山剑主也失去了试炼资格。
<br/>　　这时，奇异空间内。
<br/>　　一层白光油然而起。
<br/>　　一个通体雪白的傀儡，散发着极致寒冷。
<br/>　　正一步步朝萧逸靠近。
<br/>　　很显然，这一个试炼，乃是对战傀儡。
<br/>　　每一个试炼，试炼者都有权选择参加或者离开。
<br/>　　当然了，离开便等同于失败。
<br/>　　现在，萧逸迟迟没有离开奇异空间。
<br/>　　便代表接受了这个试炼。
<br/>　　雪白傀儡，瞬间对萧逸出手。
<br/>　　萧逸祭出血戮剑，一剑刺出。
<br/>　　他自然不会轻易放弃试炼。
<br/>　　先交手一番。
<br/>　　若是真的不敌，再使用剑主令离开也不迟。
<br/>　　“嗯？”萧逸皱了皱眉头。
<br/>　　剑未至，雪白傀儡上那极致寒冷，已经让他浑身不适。
<br/>　　“好恐怖的寒气。”
<br/>　　“此乃寒冰傀儡，难道烙印的，是寒冰堂长老的战斗经验？”
<br/>　　萧逸思索着。
<br/>　　忽然，锵的一声。
<br/>　　血戮剑在寒冰傀儡上，划过数道火花。
<br/>　　“嗯？”萧逸眉头皱得更紧。
<br/>　　连挥数剑，直接将寒冰傀儡劈飞。
<br/>　　寒冰傀儡的实力，似乎比想象中的要弱。
<br/>　　“怎么可能，才地元五重的实力？”
<br/>　　不错，在萧逸的感知中。
<br/>　　这寒冰傀儡，只有地元五重的水平，顶多不过地元六重。
<br/>　　第二层的试炼，最容易的试炼，起步都是地元五重。
<br/>　　怎么可能这最难的光球，仍旧只有地元五重？
<br/>　　萧逸疑惑着，但很快，他便明白了。
<br/>　　当寒冰傀儡再次攻来。
<br/>　　他再次出剑时。
<br/>　　这一次，又是锵的一声。
<br/>　　但，并非寒冰傀儡被刺中。
<br/>　　而是，血戮剑上，猛地结出了一层寒霜。
<br/>　　剑身锋芒与寒霜气息做着交锋。
<br/>　　这还是刚刚靠近寒冰傀儡。
<br/>　　待得萧逸的剑，再次接近几分时。
<br/>　　血戮剑，直接被寒霜冻结，再无锋芒。
<br/>　　同时，寒冰傀儡上那极致的寒冷。
<br/>　　让得他浑身血液冻结，身体僵硬，动作缓慢。
<br/>　　嘭的一声。
<br/>　　寒冰傀儡，直接一剑将他轰飞。
<br/>　　“噗。”萧逸稳下身子后，吐出一口腥血。
<br/>　　脸色惊骇到极点。
<br/>　　“怎么可能。”
<br/>　　“血戮剑已恢复极品灵器范畴，竟被冻结？”
<br/>　　“我媲美地元五重妖兽的强悍肉身。”
<br/>　　“竟在一瞬间血液凝固，陷入无法反抗的地步？”
<br/>　　未等萧逸想明白。
<br/>　　寒冰傀儡，已然再次攻来。
<br/>　　嘭。
<br/>　　又是熟悉的寒霜，以及冻结。
<br/>　　萧逸毫无反抗之力，被一剑轰飞。
<br/>　　再次吐出一口腥血。
<br/>　　脸色一惊变得苍白。
<br/>　　对战傀儡时，傀儡一般不会一开始就实力尽出。
<br/>　　就像之前对战烙印了段云长老战斗经验的那头傀儡。
<br/>　　一开始，萧逸还能虐它。
<br/>　　到后面，那傀儡使出‘游云杀步’后。
<br/>　　萧逸就落于下风了。
<br/>　　现在也一样，这寒冰傀儡使出极致寒冰后。
<br/>　　萧逸完全不是对手。
<br/>　　“修罗变。”萧逸暗喝一声。
<br/>　　萧逸竟被逼得调动体内的‘修罗之力’。
<br/>　　施展修罗变后。
<br/>　　肉体力量，飙升至地元八重。
<br/>　　这时，寒冰傀儡再次攻来。
<br/>　　那极致寒冷气息涌出。
<br/>　　萧逸顿觉冰冷无比，但不至于像之前那般血液凝固。
<br/>　　不过，身体速度和反应灵敏度，同样大大下降。
<br/>　　恐怕，要彻底抵御这极致寒冷，起码要有地元九重以上的实力。
<br/>　　甚至更高。
<br/>　　同时，血戮剑上，再度结出寒霜。
<br/>　　“破。”萧逸大喝一声。
<br/>　　血戮剑上，一股霸道威势降临。
<br/>　　这是剑势。
<br/>　　裂天剑魔的三成剑道知识下，调动的剑势。
<br/>　　堪堪破开了寒霜。
<br/>　　让得血戮剑恢复了锋芒。
<br/>　　这还是寒冰傀儡只有地元五重的水准。
<br/>　　若是寒冰傀儡本身的实力更强呢？
<br/>　　萧逸有些不敢相信。
<br/>　　“破。”萧逸再次大喝一声。
<br/>　　意欲劈飞寒冰傀儡。
<br/>　　只要寒冰傀儡被打中，并被破损。
<br/>　　次数足够多，傀儡就会彻底毁坏。
<br/>　　然而，这时的寒冰傀儡，却持剑一挡。
<br/>　　那把剑，也是通体雪白，乃是一把寒冰之剑。
<br/>　　锵的一声。
<br/>　　血戮剑，轻松被挡下。
<br/>　　不仅没能伤到寒冰傀儡，反倒是萧逸自己被震退。
<br/>　　接着，寒冰之剑，一阵阵剑花舞出。
<br/>　　竟让得萧逸，再次毫无招架之力。
<br/>　　萧逸的脸色，也再次变为震惊。
<br/>　　刚才寒冰傀儡的这一剑，以及那道道剑花。
<br/>　　比之极致寒冰更让他震撼。
<br/>　　他无法形容这一剑。
<br/>　　那如同神来之笔的忽然一挡，简直是惊为天人。
<br/>　　那恰到好处的出手，更是让人说不出的精妙。
<br/>　　那道道剑花，更是玄妙得骇人。
<br/>　　只有一种解释。
<br/>　　那便是，这个层次的剑法，或者说，寒冰傀儡上拥有的剑道知识。
<br/>　　已然超出了萧逸的理解范畴。
<br/>　　嘭的一声。
<br/>　　萧逸直接被寒冰之剑轰飞。
<br/>　　再度受伤。
