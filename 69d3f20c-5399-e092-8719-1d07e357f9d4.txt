　　整个帝玄山脉再次安静了下来，所有的人都瞪大了眼眸，凝视着巫森，凝视着这位苍穹世界唯一的武帝。
<br/>　　整个苍穹世界，见过巫森的人不多，也只有少部分武皇境强者，以前有幸见过。
<br/>　　苏莫目光凌厉，同样在凝视着这突然到来人，他知道此人就是黄泉魔宗宗主巫森。
<br/>　　不过，对方的身上，气息若有若无，仿佛就是一个普通人，让人根本看不出深浅。
<br/>　　“你就是苏莫？”巫森淡然而立，漆黑的眼眸幽冷无比。
<br/>　　“你就是巫森？”苏莫没有正面回答，同样反问了起来。
<br/>　　“看来的确是你，没错了！”
<br/>　　巫森微微颌首，漆黑的眼眸中露出了一丝笑意，道：“区区武皇境四重的蝼蚁，也敢与本帝一战，真是无知者无畏！”
<br/>　　巫森言罢，身上陡然间气息狂涌，一股霸绝天地的气势，以他为中心，冲破云霄，浩荡天地。
<br/>　　这股气势，浩大、无敌、霸道，威不可挡！
<br/>　　磅礴的帝之威压，伴随在气势之中，如同毁天灭地的狂风，向八方扫荡。
<br/>　　轰隆隆！！
<br/>　　气势扫过之处，空间不断塌陷、炸裂、毁灭，如同世界末日一般。
<br/>　　“不好，快退！”
<br/>　　“赶紧走！”
<br/>　　“逃啊！”
<br/>　　无数的武者见此，立刻疯狂后退，急速与巫森和苏莫拉开距离。
<br/>　　但是，有些实力较弱的武者，根本来不及逃走，气势扫过，顿时灰飞烟灭。
<br/>　　几乎是转瞬之间，数之不尽的武者，全部都退到了近万里之外。
<br/>　　就连战空、究楚北、海明江和帝释等准帝强者，都远远的退开了，免得被等会的战斗波及。
<br/>　　所有人都是心中骇然，恐怖，实在是恐怖！
<br/>　　仅是身上的气势，就有如此威能，那若是巫森全力出手的话，该有多么的强大！
<br/>　　这就是武帝强者吗？
<br/>　　众人暗暗惊骇。
<br/>　　“境界不是衡量实力的唯一标准！”苏莫身形站了起来，同样挺拔如松，面对巫森的气势，他面不改色。
<br/>　　对方的气势从他身上扫过，并不能伤其分毫。
<br/>　　“你的意思是，你的实力不能以境界来衡量？”巫森闻言冷冷一笑，眸中充满戏谑之意。
<br/>　　他知道苏莫的战力不弱，武皇境三重之时，就有准帝的实力，这也是他为何应战的愿意。
<br/>　　不然的话，他哪里会理会！
<br/>　　不过，即便苏莫战力再强，在他眼中也是不值一提。
<br/>　　“你会败！”苏莫面色冷漠了下来，目光直视巫森，一字一句的说道。
<br/>　　“是吗？”
<br/>　　巫森闻言轻笑一声，淡漠的道：“你算是苍穹世界万载以来最杰出的天才，能死在本帝的手中，你足以自傲了！”
<br/>　　言罢，巫森的手中黑光流转，出现了一杆黑色的长矛。
<br/>　　长矛如同阴冷的毒蛇，闪烁着森冷的寒光。
<br/>　　“苏莫，你杀我圣宗无数弟子，今日，本帝将用你的鲜血，慰藉他们的亡灵！”
<br/>　　巫森骤然暴喝一声，身上气势再次暴涨，他整个人如妖似魔，身形无限拔高，仿佛是世间唯一的主宰。
<br/>　　“且慢！”
<br/>　　就在巫森即将出手之时，苏莫突然开口，道：“巫森，你我一战之前，我要问你一个问题！”
<br/>　　巫森闻言眸光一凝，暂时没有出手，他并不着急，一切都在他的掌控之中。
<br/>　　“宏青璇在哪里？”苏莫朗声问道，他上次可是从皇甫擎的口中得知，宏青璇被交给了巫森。
<br/>　　宏青璇的安危，一直是苏莫的心病，所以他不得不稳。
<br/>　　“宏青璇？”巫森闻言微怔，随即他身边空间微微波动，一名女子出现在了其身边。
<br/>　　“你是说她吗？”巫森瞥了一眼女子，向苏莫问道。
<br/>　　“青璇！”苏莫瞪大了眼眸，望着这名女子，因为这名突然出现的女子，正是宏青璇。
<br/>　　不过，此时的宏青璇，面色有些木讷，眼眸中没有半点神采，仿若是一根木头一般。
<br/>　　“青璇，你怎么了？”苏莫见此顿时疾呼了一声，他不清楚对方到底怎么了，好在对方的身上，生机浓郁，性命暂时无忧。
<br/>　　“她很好！”巫森轻笑一声，随即，宏青璇的身影突兀的消失不见了。
<br/>　　“找死！”苏莫见此顿时怒了，毫不犹豫，帝皇剑瞬间出现在了他的手中，一剑狠狠的斩向巫森。
<br/>　　刹那之间，数千丈长的恐怖剑气，横贯天地，如同一道浩大的三色闪电，向巫森当头斩去。
<br/>　　这一剑虽然是随意斩出，但威力奇大无比，让得远处观看之人无不心惊。
<br/>　　仅从这一剑来看，苏莫的实力就算不如武帝，也相差不是太远了。
<br/>　　“雕虫小技！”巫森面色平淡，手中长矛随意一刺，犀利无比的劲气，如同疾刺的利箭，爆射而出。
<br/>　　咻！
<br/>　　气劲洞穿空间，裹挟刺耳的厉啸声，轰然的刺在了剑气之上。
<br/>　　轰！
<br/>　　一声爆响，剑气顿时炸裂，一击而碎。
<br/>　　长矛刺出的劲气，也崩溃瓦解，四散而开。
<br/>　　汹涌的气浪扫荡八方，逆卷九天，让得方圆数千里之内，彻底变成了一片虚无之地。
<br/>　　一击无功，苏莫没有再急着出手，他很快便冷静了下来。
<br/>　　看来巫森的身上，也有空间宝器，将宏青璇收进了其内。
<br/>　　只不过，让他疑惑的是，对方到底是什么打算！
<br/>　　宏青璇的体质是纯阴之体，这点苏莫非常清楚，而黄泉魔宗抓取宏青璇，无外乎就是双修，靠宏青璇的纯阴之气增加修为。
<br/>　　但是据他刚才的观察，宏青璇并没有失.身，依旧是处子之身，这点绝不会错！
<br/>　　以苏莫现在的修为，感知力无比的强大，他能清晰的感觉到轰青璇身上的纯阴之气，非常的浓郁。
<br/>　　但是，这就说不通了，对方冲击到武帝之境，都没有和宏青璇双修，那对方一直劫持着宏青璇，到底意欲何为？
<br/>　　难道是想培养起来，以后再给巫山做炉鼎？
<br/>　　但这也说不通啊！
<br/>　　因为就算如此的话，也不用巫森亲自培养吧！
<br/>　　苏莫完全想不明白，但巫森已经不给他考虑的时间了。
<br/>　　“苏莫，你可以死了！”
<br/>　　巫森冷喝一声，陡然出手了，他直接就动用了杀招，准备强势诛杀苏莫。
<br/>(本章完)
