　　方才就是他观想出韦驮天，挡住了清长风那一击。不然他跟叶少阳就是两败俱伤，现在自己伤了，叶少阳却没事。
<br/>　　这是什么路子？
<br/>　　清长风只是惊诧，他不知道的是，在这几天里，叶少阳他们无时不刻不在研究战术，后来发明出了这套奇葩的战术，战术的想法来自于亡者农药，名字就叫肉保输出。
<br/>　　叶少阳是核心输出，四宝的罗汉金身防御力极强，两人一前一后，由四宝来吸收攻击，叶少阳只管输出，清长风本来一打一都未必赢得了他，又完全搞不懂他们的新式战术，只有挨打的份。
<br/>　　更不要说，他还断了一条手臂。
<br/>　　“来啊来啊！”四宝拍着胸脯，朝他怒吼着。
<br/>　　清长风咬了咬牙，单手结印，那七枚铁离子飞到身边，环绕着他，一招手，紫气东来，护住周身，再度冲了上来，手指一弹，便是一道奔雷，打向四宝。
<br/>　　四宝观想出韦驮天，硬抗伤害，感觉有点吃不消，这清长风，显然是拼命了。
<br/>　　好在有叶少阳输出。
<br/>　　只见他身形一晃，叶少阳超过他，急速贴过去，口中念咒，手里阴阳镜射出一道明光，打向清长风双腿。
<br/>　　“道无可道，星光闪耀，七星北斗，索命无常！急急如律令！”
<br/>　　七枚铁离子一瞬间更加明亮，贴住叶少阳，不断变换着形状，每变换一次，便是一道符文形成，却不消失，等到连变七次，阵法形成，七道符文在一起，仿佛叠加了七层BuFF，声势骇人，骤然紧缩，叶少阳只觉得周身一瞬间被困住，急忙运气反击。
<br/>　　清长风却抓住机会一通狂攻，技能都怼上了。
<br/>　　好在有四宝这个超级血牛在前面顶着，化解了若干攻势，叶少阳虽然受伤，但都是皮外伤，并不严重。三人就这样消耗起来。
<br/>　　星光见血，反而爆发出一股可怕的气势，围攻不停。
<br/>　　叶少阳被星光困在中间，仿佛看到了七个清长风，从不同角度攻来，只能运转天罡步急切躲闪，避过重招，心想这清长风能在轩辕山开宗立派，实力的确不是盖的。
<br/>　　其余人的战斗，也都在进行之中。
<br/>　　因为道场太小，人又太多，根本拉不开战斗，于是大伙越打越靠外，大部分都出了道观，在山野间打起来。
<br/>　　岳恒一个人挡不住毕方的攻击，好在有孙映月帮忙，两人对付他，也只能勉强支撑，虽然他嘴硬不要帮忙，但小九还是加入了战局，本来她一个人都能跟毕方打平，以三对一，战况立刻逆转，毕方被三人围住，只有防守的份了。
<br/>　　彩云仙子与杨宫梓这两个操控气息的专家，在天上互殴，两人身边都是气息涌动，一层接着一层，一道对着一道，一记下去，气息溃散，两人只一挥手，便又有气息生成，朝对方打去。
<br/>　　仿佛两人比的不是术法，而是集气的速度。
<br/>　　灵感仙被美华、橙子、小青、小白四人围攻，故意逼着他转过山丘，下到山谷里，灵感仙实力再强，位列妖王，但以一敌四，也最多是平局，占不了上风。
<br/>　　他目光一扫，发现山谷里有一汪清泉，在月光下泛着明光，心头一喜，且战且退，一头扎了进去。
<br/>　　这是泉水深潭，连着山中地下水系，深不见底，灵感仙本就是鲤鱼精，一入水中，立刻浑身舒服，现出真身，尾巴一甩，卷起一波巨浪，朝四人射去，想要把他们卷下来。
<br/>　　结果这四人都没等到巨浪，就主动进入水中了。
<br/>　　五个人接着打，灵感仙在水中实力大增，喜上眉梢，正要大杀四方，然后打了一会，他就笑不出来了，眼前一只鲛人，两只蛇妖，一条水鬼，妈的四个人都跟他一样，全是水战强者。
<br/>　　纵然他入水后提升了两倍实力，对方各自也提升两倍，差距一下子就出来了。
<br/>　　灵感仙这才知道自己上当了。
<br/>　　五个人在地下河中狂追猛打，在地下河里掀起一股有一股的强大水波，撞击着山岩，让整座山峰都震颤起来。
<br/>　　“秃头，我跟你说，你这光脑壳特别适合我的板砖，要不要来一下，来来就一下！”
<br/>　　小马追着太平教主飞出了道观。
<br/>　　他和秦风夫妻三人一起围攻太平教主，太平教主打起来是个超级谨慎的，法术一板一眼十分传统，偏偏小马是个不传统的，偏偏又耐揍，靠着秦风两口子托着太平教主，自己总想着摸到太平教主身后去敲他脑袋。
<br/>　　打斗之中，太平教主的斗笠掉了，露出一个光秃秃的脑门，小马一看到这脑瓜子就流口水，更加跃跃欲试。
<br/>　　三人围着太平教主，转战到另一座山峰上。
<br/>　　“当心！”
<br/>　　从小马身边掠过时，秦风提醒了他一句：“他并没有施展全力！咱们都当心，记住任务！”
<br/>　　“你这剑不错，人也不错。”
<br/>　　星月奴望着建文帝手中的鱼肠剑，说道：“你曾为人皇，奈何从贼？”
<br/>　　建文帝冷笑道：“窃钩者诛窃国者侯，谁是贼谁是君，要看结果。”
<br/>　　“不错。”星月奴点头赞道。
<br/>　　建文帝带着风之谷十二门徒，冲了上去。南宫影紧紧跟随，等待机会。他本来是要对付太平教主的，但被小马他们抢走了对手，于是就来对付这个最厉害的。
<br/>　　包子、瓜瓜等剩下的，也都围住了星月奴。
<br/>　　见他们动手，星月奴并不着急，反而把轩辕剑收起来，张开双臂，有点像老头打太极的姿势，然后在身前轻轻一抓，撕裂了空间，两手之间出现了一道虚空裂缝，产生出一股强大的吸力。
<br/>　　建文帝急忙运功定住，身后的十二门徒因为冲得太猛，一时收不住，有三个人直接被吸了进去。
<br/>　　“那是她创立的幻境！不要被他吸进去了！”
<br/>　　瓜瓜大声喊，他是进去过一回的，那次真是上天保佑，星月奴当时在闭关，他们才有幸逃脱，如今星月奴可是醒着的，在她的世界里，她是规则的制定者，只要进去肯定就凉了。
