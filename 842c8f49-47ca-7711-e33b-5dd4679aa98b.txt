　　打败了幽狼之后，苏莫面色坦然，转头看了一眼云悠悠，他发现对方也在望着他。
<br/>　　微微一笑，苏莫也没有表示什么，便飞身返回了看台。
<br/>　　他可不会像扬天成那般，如此没有下限，此事，还需要循环渐进，不可操之过急。
<br/>　　面对苏莫的微笑，云悠悠微微点了点头，她对于苏莫这种淡然沉稳的性格，倒是莫名的生出了一丝好感，当然，也只是好感而已。
<br/>　　相比于扬天成那种直白，苏莫的为人，明显更为让人舒服。
<br/>　　此时，被击败的幽狼，已经化为了本体，他的脸色一阵青一阵红，他很不想承认自己败了，但事实让他不得不承认。
<br/>　　当着无数人的面，被修为低于自己的人击败，让他感觉到巨大的耻辱，一时间如芒在背，怔怔的站在原地，不知道如何是好。
<br/>　　“幽狼，回来吧！”啸天狼族的森狼半圣轻喊了一声。
<br/>　　幽狼闻言，只能长叹一声，默默的飞回了看台。
<br/>　　接下来，武斗场暂时的没有上台挑战，所有人都在议论苏莫，都认为他有夺得第一的希望。
<br/>　　毕竟，能打败幽狼，便说明了在武帝境之中，基本上已经有了无敌的实力。
<br/>　　其它几个势力的妖孽天才，心中都不禁沉了下来，均是暗暗的将苏莫当成了自己夺得第一的大敌。
<br/>　　直到过去了盏茶的功夫，苏莫与幽狼一战的余热过去了，才再次的有人上台挑战。
<br/>　　而战胜了幽狼的苏莫，回到看台之后，受到了特殊的礼遇，真枯半圣特意让他坐在了其身边。
<br/>　　“苏莫，你是何时加入道宗的？”真枯半圣微笑望着苏莫，开口问道，他也是对于苏莫有些疑惑，按理说道宗有这么逆天的弟子，不至于默默无闻吧！
<br/>　　“刚加入了不久，一两个月吧！”苏莫道。
<br/>　　“原来如此！”真枯半圣闻言恍然，原来是才加入的道宗，难怪默默无闻。
<br/>　　想到苏莫是真贤圣长老特意交代的人，他心中明了，看来苏莫是真贤半圣召进道宗的，估计也是发现了苏莫的天赋，所以才会这么做。
<br/>　　“以你的天赋实力，将来成为真传弟子，轻而易举！”真枯半圣说道。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“你这次若是能得到会武的第一名，不仅能得到太阴族的奖励，回到道宗之后，还能得到宗门内的奖励！”真枯半圣笑着说道。
<br/>　　苏莫闻言，心中沉吟，奖励他倒是不太在乎，关键是有机会能接触到青璇。
<br/>　　“真枯长老，我若是得到了第一名，能经常出入太阴族吗”苏莫沉声问道，他没有奢望仅凭一次会武，太阴族就会招纳他！
<br/>　　“不能！”
<br/>　　真枯半圣摇了摇头，道：“只有成为道宗的真传弟子，你才有机会出入太阴族，而且只是外族！”
<br/>　　“那内族呢？”苏莫皱眉问道，出入外族有什么用，青璇可是神女，肯定平常是在内族之中生活。
<br/>　　真枯半圣瞥了苏莫一眼，他知道苏莫是想要进入太阴族，这倒是没有什么，毕竟太阴族可是明虚星河的霸主，谁不想加入！
<br/>　　“待你以后修为高了之后，自然有机会进入内族！”真枯半圣说道，七大势力的顶尖天才，大多都被吸收进了太阴族，像苏莫这种天才，以后肯定也不例外。
<br/>　　“嗯！”苏莫点头，心中暗叹，看来想进入太阴内族还真不简单。
<br/>　　苏莫和真枯半圣有一句没一句的闲聊，通过对方，得到了不少有用的消息。
<br/>　　在两人闲聊之间，战台上一场有一场的战斗结束了。
<br/>　　各个势力的妖孽天才，也均是纷纷登场，各个一招败敌，让得很多人惊叹。
<br/>　　或许是受到了苏莫的刺激，这些妖孽天才，不想在气势上被苏莫压一头，各个展现了超强的战斗力。
<br/>　　良久之后，第三轮挑战终于结束了。
<br/>　　三轮挑战一过，所以连败三场的人，全部被淘汰了出去。
<br/>　　七成以上的人被淘汰了，只剩下了四百余人。
<br/>　　“第四轮挑战开始！”
<br/>　　随着褐袍老者的话音，第一轮挑战赛开始了。
<br/>　　苏莫站了起来，脚步一踏，率先飞上了战台，拉起了第四轮挑战的序幕。
<br/>　　众人见此惊讶，没想到苏莫这种天才，居然第一个上场了。
<br/>　　身形落在战台上，苏莫转头望向了天虚宗弟子的方向，目光落在了扬天成的身上。
<br/>　　“扬天成，就你了！”苏莫轻笑着说道，他之所以第一个上场，就是为了挑战扬天成。
<br/>　　因为若是对方先上场的话，他就无法再挑战对方了。
<br/>　　所以，他直接第一个上场了。
<br/>　　此人居然敢向青璇表白，还作出如此恶心的诗，不虐其一波不足以平怒气。
<br/>　　苏莫的心是很霸道的，他对青璇也有很深的感情，不可能放任别人追求青璇。
<br/>　　苏莫的话语传出，再次掀起了全场一片波澜，他此举出乎了所有人的预料。
<br/>　　因为作为妖孽天才，不到会武的最后阶段，一般都是不会互相挑战的。
<br/>　　前期还是获得积分为主，到得最后，几大妖孽都是满分，才会互相挑战。
<br/>　　因为，只有击败了满分的人，自己才能获得第一。
<br/>　　但现在，会武还要走好几轮，苏莫居然就要挑战扬天成了！
<br/>　　扬天成闻言一怔，随即眼眸微眯了下来，他转念一想，便明白了苏莫的想法。
<br/>　　苏莫第一轮好像没有上场，所以比他们几大妖孽都要少一分，所以要提前挑战他们，才能登顶第一。
<br/>　　“好大的野心！”扬天成心中冷哼一声，第一名他志在必得，谁都别想从他手中夺去，随即，他身形一闪，便飞落在了战台上。
<br/>　　“神女，这是我第四战！”扬天成向云悠悠抱拳，面带和煦微笑。
<br/>　　云悠悠无奈的点了点头，对方每次比试之前，都要和她打声招呼，让她略显尴尬。
<br/>　　“神女，刚才我特意为你作了一首琴曲，等会便为你弹奏，希望你能喜欢！”扬天成微笑着说道，之前一战他没有表示，一直在暗暗准备琴曲，现在已经准备好了。
<br/>　　苏莫闻言无语，卧槽，不是作诗就是作曲，这家伙真是奇葩，你是来参加会武的吗？你是来参加才艺大赛的吧！
<br/>　　“不好意思，你的琴曲怕是弹不出来了！”苏莫率先开口，淡淡的说道。
