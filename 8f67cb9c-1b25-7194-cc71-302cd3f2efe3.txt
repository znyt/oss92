　　“你问这个做什么？”云渊长老疑惑地看着萧逸。
<br/>　　萧逸不语，皱眉沉思着。
<br/>　　气宗与器宗间的区别，他倒是现在才清楚。
<br/>　　气宗，以体内元力，修炼出剑力。
<br/>　　自然的，剑力可以融入元力之中。
<br/>　　而本身，武者使用武技、剑技，是需要消耗元力的；故剑力，也可以融入武技、剑技中。
<br/>　　另外一点，秘法也同样如此。
<br/>　　至于器宗，则以身体，或四肢、或体内五脏六腑等等，修炼出剑力。
<br/>　　故剑力，可以融入肉体、四肢百骸等等。
<br/>　　当然了，器宗已‘器’为名，自然剑力亦可融入手中之剑。
<br/>　　气剑力与器剑力，截然不同。
<br/>　　一者，可融入元力、剑技。
<br/>　　一者，可融入肉体、手中之剑。
<br/>　　单纯论基础实力，修习气剑力的剑者，在元力爆发下；与修习器剑力的剑者，在剑力融入肉体下；二者差不多。
<br/>　　故唯一存在高低差距的，便只剩下剑技与手中之剑的高低之差。
<br/>　　说到底，气宗与器宗之争；或许便是剑技对剑者的增幅强些，还是一把剑对剑者的增幅强些。
<br/>　　两种剑力，层次一样，但效果却截然不同。
<br/>　　但，还有着第三种特殊剑力，便是魂剑力。
<br/>　　无论是融入剑中，还是融入剑技中、元力中、肉体中，皆可增幅。
<br/>　　“魂剑力吗？”萧逸皱了皱眉，“剑帝？”
<br/>　　说起来，他才来剑域数天时间，算上他之前修炼剑力的三天时间，真正行走于剑域的时间也就一两天。
<br/>　　即便他在风刹殿细查过这边的大致情况。
<br/>　　但要说到样样知晓，却不可能。
<br/>　　“剑帝是？”萧逸疑惑地看着云渊长老。
<br/>　　云渊长老笑笑，“自然是炎龙大陆历史上最惊才绝艳，也是最强的剑者。”
<br/>　　“具体的，等盛事开始你便知晓。”
<br/>　　萧逸点了点头，继续问道，“为何剑力会这般特殊？”
<br/>　　“我曾经感知过，剑力，其武道意味在天地武道中，根本不正常。”
<br/>　　云渊长老，可是当世巅峰剑修，又曾在以前便来过剑域，参加过剑域盛事。
<br/>　　自然的，他知晓的东西或者说秘辛更多。
<br/>　　萧逸自然也追问起来。
<br/>　　“感知？”孰料，云渊长老眉头一皱，“小子，你能感知剑域之内的剑力？”
<br/>　　“据我所知，在剑者掌控剑力之前，是无法感知剑域之内的剑力意味的…”
<br/>　　讳执事适时笑笑，打断道，“云渊，他既感兴趣，你为他解疑便是。”
<br/>　　云渊长老点点头，“确实，如你感知的那般，剑力并不寻常。”
<br/>　　“它是天地武道规则中的力量，但又并不充斥天地武道规则中。”
<br/>　　“也就是说，他是独立的一种力量。”
<br/>　　所谓充斥于天地武道规则中的力量，便是如元力、魂力、肉体力量，阵法之力，等等。
<br/>　　这些力量，自成体系，亦构成了炎龙大陆的一门门职业。
<br/>　　但剑力，却仅仅是存在于剑域的力量。
<br/>　　云渊长老继续道，“因其是独立的力量，故它能融入到元力中、肉体力量中、或者外物之中。”
<br/>　　“你可曾见过元力和肉体力量能融合？或者元力和魂力能融合？”
<br/>　　“没有。”萧逸摇了摇头，“元力、肉体力量、魂力，本就不是一个体系，怎可能融合，叠加倒是行。”
<br/>　　这一点比较好理解。
<br/>　　萧逸自己就身兼多个职业。
<br/>　　拿他使用肉体力量时来说，可无法把元力也融合到肉体力量中，或者把阵法之力融合到肉体力量中。
<br/>　　顶多就是布下大阵，增幅实力；或者边借助肉体力量的爆发，边使用秘法增幅；这就是单纯的叠加。
<br/>　　当然了，这种叠加，也让得他手段颇多，战力惊人。
<br/>　　但现在魂剑力这种独特的力量，却能融合各种力量。
<br/>　　这也是前些天萧逸惊讶于剑力的强大的原因，也让得他特地留了三天时间去修炼剑力。
<br/>　　“剑帝血脉又是什么？”萧逸追问道。
<br/>　　“自然是一种血脉。”云渊长老平时显然不喜言谈，渐渐有些不耐烦。
<br/>　　“林家族人，便是拥有这种血脉的剑者。”
<br/>　　“话说，你问个不停，你还没回答我的问题，你问魂剑力之事做什么？”
<br/>　　“额。”萧逸尴尬地摸了摸鼻子。
<br/>　　所谓林家族人，自然便是万剑城城主的家族，剑域第一家族。
<br/>　　嘶…
<br/>　　萧逸手中微微凝聚出一抹微弱的白色光芒。
<br/>　　光芒很淡，并不惹人注目，且只是一闪而过，只有云渊长老与讳执事二人看到。
<br/>　　“剑力？”云渊长老脸色一惊，他明显看到，刚才那摸白色光芒不仅涌现在萧逸肉体上，还伴随一丝元力凝聚而出。
<br/>　　能同时出现于肉体以及元力的剑力，只有一种，便是魂剑力。
<br/>　　“魂剑力？”云渊长老压低了声音。
<br/>　　“你别告诉我，你是林家族长的私生子。”
<br/>　　萧逸撇撇嘴，翻了个白眼。
<br/>　　一旁讳执事没好气道，“要是这小子真是林家族长的私生子，这些年他能被欺负成这个样子？”
<br/>　　讳执事说得没错，若真是剑域第一家族林家的人，放眼中域，谁敢轻侮之。
<br/>　　“那你这魂剑力怎么炼出来的？”云渊长老瞥了眼周遭，确定周遭无人注意这边，方才问道。
<br/>　　“鬼知道，练啊练的就练出来了。”萧逸撇撇嘴。
<br/>　　“练啊练的，我练了这么多年，怎不见我练出来，你小子骗鬼呢。”云渊长老瞪了萧逸一眼。
<br/>　　“不信我也没办法。”萧逸双手一摊。
<br/>　　魂剑力的事，萧逸没打算瞒这两位前辈。
<br/>　　不过，这魂剑力为何能修炼出来，萧逸自己也说不出原因，也只能胡说八道了。
<br/>　　这时，比武台上，丁秋月与那气宗弟子的比拼，渐渐分出。
<br/>　　“他们争些什么？”萧逸问道。
<br/>　　“若是单纯比拼，似乎没什么意义。”
<br/>　　云渊长老看了眼比武台，道，“气宗一脉，和器宗一脉，争斗了无数年。”
<br/>　　“一来，是拼年轻一辈的实力。”
<br/>　　“二来嘛，你该知道，若无好处，两大势力间怎可能争斗不休。”
<br/>　　“什么好处？”萧逸问道。
<br/>　　“我不告诉你。”云渊长老笑笑，负手而立。
<br/>　　萧逸闻言，撇撇嘴，并未追问。
