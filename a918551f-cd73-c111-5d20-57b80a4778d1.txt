　　仍旧是那一片流光溢彩的巨大冰宫广场中。
<br/>　　冰宫弟子做好登记与记录，冰宫长老现身，讲解第三轮的规则。
<br/>　　唯一不同的，是现今萧逸一行人身后，还有一种剑宗强者。
<br/>　　参加冰缘大会的各方武者，除了少量独行武者外。
<br/>　　其余人等，皆有自己势力强者陪同而来。
<br/>　　不得不说，剑宗强者，与其他势力的强者站在一起，丝毫不显逊色，反倒威势十足。
<br/>　　除却不夜宫主与雪刀门主两位极境强者的话。
<br/>　　剑宗强者，完全可以放到与一宫一门二教同一层次的行列中。
<br/>　　言归正传。
<br/>　　冰宫大长老高声道，“诸位可有疑惑？”
<br/>　　“若无，冰缘大会第三轮，圣果之缘，即将开始。”
<br/>　　无人出言。
<br/>　　冰宫各长老点了点头，随即大喝一声，“起。”
<br/>　　话音刚落，地面一阵颤抖。
<br/>　　各方强者，快速后退。
<br/>　　不多时，一块巨大的寒冰，拔地而起。
<br/>　　寒冰之巨，超越了冰宫广场，大概占了整个不夜冰宫一半以上的面积。
<br/>　　寒冰，成正方形，晶莹剔透，光滑无比。
<br/>　　这，其实是一由寒冰而成的比武场。
<br/>　　同时，寒冰比武场四周，风雪涌动。
<br/>　　待得风雪吹拂而过，一个个观赛席凭空而现。
<br/>　　观赛席，自然也是由寒冰而成。
<br/>　　放眼望去，这简直是一位冰雕大师的惊人杰作。
<br/>　　第三轮，名唤圣果之缘，自然与各种珍贵无比的圣果有关。
<br/>　　第三轮，又是冰缘大会的最后一轮。
<br/>　　这，将决定着‘冰心圣果’的归属，也将是所有参加者收获最大的一轮。
<br/>　　这一轮的机缘，将由武者本身实力而定。
<br/>　　一千五百位参加者，将在这寒冰比武场上分出胜负，决出排名。
<br/>　　亦是这次冰缘大会的最终名次。
<br/>　　……
<br/>　　此时，寒冰比武场上，站立着一众参加者。
<br/>　　周遭观赛席上，也坐满了各方势力的强者。
<br/>　　冰宫大长老，凌空立于比武场半空，高声道，“可有疑惑，或对第三轮规则不明之处？”
<br/>　　没人发问。
<br/>　　冰宫大长老点了点头，道，“既无，诸位稍作准备，一盏茶时间后，正式开始。”
<br/>　　第三轮，乃是武斗，规则相当粗暴而简单。
<br/>　　一千五百人，在比武场上混战。
<br/>　　一直战至比武台上只剩一人，方为结束。
<br/>　　整个比武场，占地方圆数十里，足够巨大。
<br/>　　排名，视失败顺序而定。
<br/>　　死亡、认输、离开比武台范围，皆为失败。
<br/>　　最先失败者，便是倒数第一名，依次类推。
<br/>　　在这比武台上，坚持到最后之人，便是第一名。
<br/>　　亦是这次冰缘大会的第一名。
<br/>　　此时，比武台上，萧逸一行人，闲聊着。
<br/>　　对自身实力的足够自信下，并不需要准备什么。
<br/>　　韩苍、风花月等人，忽然走来。
<br/>　　“哟，几位很自信嘛。”韩苍阴测测地笑道。
<br/>　　“有实力，自然自信。”钟无忧冷笑一声。
<br/>　　随即，伸出手指，点了点身旁之人。
<br/>　　“萧逸、白冰雪、凌羽、林劲、柳嫣然，嗯，叶铭你也算一个吧。”
<br/>　　钟无忧在点到叶铭时，手指顿了顿。
<br/>　　而后，狂傲道，“合共七人，第三轮的前七排名，便归我们了。”
<br/>　　“你…”韩苍等人，脸色一冷，随即满脸戏谑笑意。
<br/>　　“得意吧，姑且让你们再嚣张一会儿。”
<br/>　　“排名前七？哼，你信不信，你们会是倒数前七？”
<br/>　　“哈哈哈哈。”钟无忧狂笑几声，“孙子，也不怕风大闪了舌头。”
<br/>　　“前面两轮的账，老子待会儿慢慢和你们算。”
<br/>　　……
<br/>　　一盏茶时间，转瞬便过。
<br/>　　“武道，开始！”冰宫大长老，大喝一声。
<br/>　　声音落下。
<br/>　　战斗瞬间打响。
<br/>　　“萧逸，敢否一战？”韩苍与风花月，冷哼一声。
<br/>　　韩苍、沐冰、风花月，还有三位极寒之地这边的天骄，合共六人，联手朝萧逸攻来。
<br/>　　萧逸早有所料，身影一闪，御空而起。
<br/>　　虽不惧这六人，但萧逸并不想自己的战斗，影响到钟无忧等人。
<br/>　　他们有着属于自己的战斗。
<br/>　　嗖…嗖…嗖…
<br/>　　几乎是萧逸立于高空时。
<br/>　　六人已立刻包围了他。
<br/>　　冰炎、刀气、寒冰掌风…一道道攻击，快速打向他。
<br/>　　萧逸丝毫不放在眼中，随手打出剑气。
<br/>　　以他的实力，秒杀这六人，轻而易举。
<br/>　　但，令人意外之事，忽然发生。
<br/>　　萧逸的剑气，竟瞬间被这六人的攻击轰碎。
<br/>　　随后，冰炎、刀气等攻击，毫无阻拦地打到了萧逸身上。
<br/>　　嘭…嘭…嘭…连续六声巨响。
<br/>　　萧逸淹没于一片能量爆炸当中。
<br/>　　“哈哈哈哈。”韩苍大笑几声，“萧逸，今日本公子要你死。”
<br/>　　爆炸过后，露出了里头的萧逸。
<br/>　　此刻的萧逸，身上丝毫无损，一身公子装，在风雪中飘袂作响，潇洒至极。
<br/>　　“巅峰战力？”萧逸皱了皱眉头。
<br/>　　不错，此刻韩苍等六人，竟是巅峰战力。
<br/>　　刚才自己随手而发出的剑气，也是因此，被他们的攻击所打破。
<br/>　　不过，这里可并非极寒冰海。
<br/>　　这六人的秘法，绝不可能有如此大的增幅。
<br/>　　“寒冰之髓。”萧逸双眼一眯，反应了过来。
<br/>　　他明显在这六人身上，感受到寒冰之髓那种纯净的气息。
<br/>　　同时，还附带着惊人的极寒力量。
<br/>　　“原来如此。”萧逸瞬间明悟。
<br/>　　这六人，吸收了大量寒冰之髓。
<br/>　　不过，他们并没有用于转化为元力，增加修为。
<br/>　　而是将寒冰之髓的精纯力量，直接用于施展‘秘法’的力量来源。
<br/>　　寒冰之髓的精纯力量，一瞬间的爆发，让得他们的‘秘法’，发挥了最大效果。
<br/>　　便也让他们短暂有了巅峰战力。
<br/>　　只是，这种方式，太过奢侈。
<br/>　　一旦他们的‘秘法’将寒冰之髓消耗光，他们的战力就会恢复原状。
<br/>　　且修为不会有任何增加。
<br/>　　“看来，五百斤寒冰之髓，我还是要少了。”萧逸双眸一冷。
<br/>　　“呵呵。”韩苍冷笑一声，“萧逸，我说过，这一轮，倒数前七，便归你们了。”
<br/>　　“嗯？”萧逸皱了皱眉头。
<br/>　　目光，看向下方。
<br/>　　那里，一千五百名武者，互相间，根本没有战斗。
<br/>　　而是在围攻钟无忧等六人。
<br/>　　“你们联手了？”萧逸脸色冰冷。
<br/>　　“现在才发现，迟了。”韩苍眼中怨毒之色，一闪而过。
<br/>　　六人，六位巅峰战力，瞬间出手。
<br/>　　远处，观赛席周遭。
<br/>　　各方势力强者，似是对比武台上的情况，早有所料。
<br/>　　一个个，玩味地笑了起来。
<br/>　　一道道讥讽的目光，聚集在裂天剑宗强者身上。
<br/>　　很显然，所有人都在等着裂天剑宗出丑。
<br/>　　等着看这个武道圣地，在这一届冰缘大会排名垫底。
<br/>　　剑宗强者的观赛席这边。
<br/>　　宗主以及各位长老，脸色冷漠，不语。
<br/>　　大长老不屑地看了眼周遭讥笑之人，冷笑一声。
<br/>　　“一群傻瓜，别说台上这区区一千五百位武者。”
<br/>　　“就是你们这些成名强者加起来一并上，都不够萧逸小子一顿杀的。”
