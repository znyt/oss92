　　道风点头默认，他想来想去，能在地位上与司命星君对抗的（前提是能真来帮忙的，否则阴司倒有几个大佬地位极高，但人家现在自顾不暇），只有青牛，镇元大仙算半个，所谓半个，是说他名气大，地位崇高，号称地仙之祖，他倒是也在修罗界，但他对道门的影响，主要是开创了一个门派，改进了很多法术，流传至今，严格说并不算所有道士的祖先。
<br/>　　道门祖先，必须是对道教产生起到巨大推动或改革作用的祖圣人，如老子、庄子、张道陵、葛洪、陈抟。
<br/>　　老子不用说，道门千古第一大佬，太上老君。
<br/>　　亚圣庄子，作《南华真经》，道门万古不变的入门教科书;
<br/>　　张道陵，正式开创道教，古今第一天师。
<br/>　　葛洪是古今炼丹第一人（第二人是淮南王刘安），中医大拿，对后世的道门和中医方面影响都极大，如今道门的炼丹术，仍然是在葛洪那一套上改良的。
<br/>　　陈抟是道门正史上最后一个飞升成仙的，对道门经卷十分有研究，后代道士所修炼的心法和功法，有一半都是他的功劳。
<br/>　　这五个大佬，对道门贡献极大，一脉传承，少了谁道门历史都要改变。不过这五个老板早就不在三界之内了（去哪儿了、在不在世上，都没人知道）。
<br/>　　司命星君是庄子的正牌师弟，在道门还有仙籍，祖先清白，根正苗红，往如今的人间一站，天下道士都得叫爸爸。
<br/>　　唯一能与之对比的，就是青牛祖师李浩然，老子的坐骑加嫡传弟子，牛逼轰天，无出其右。
<br/>　　这么看来，道风找李浩然帮忙，实在是太正确了。
<br/>　　“你找到他怎么说？”
<br/>　　“他只能做奇兵，不可能帮我们冲锋陷阵，不过这就够了。”
<br/>　　叶少阳又问他人在哪里，道风说等到关键时刻，他会出现。
<br/>　　叶少阳“哦”了一声，道：“你去见李浩然，应该叫我的，我还想去见下碧清。”
<br/>　　碧清走的时候，说了是去找李浩然，叶少阳一直惦记这事。
<br/>　　“她没去找青牛。”道风淡淡说了一句。
<br/>　　“她不在那里？那她去哪儿了？”
<br/>　　道风瞥了他一眼，“我怎么会知道！”
<br/>　　两人顺着楼梯下去，叶少阳想到接引道人，于是问道风，是否知道接引道人到底要他们做什么。
<br/>　　道风在楼梯拐角站住，道：“我答应他，等这一切结束，如果我还活着，我就去修罗界，帮他守阵百年。”
<br/>　　叶少阳怔在当场。
<br/>　　“为什么找你？天下又不只你一个强者。”
<br/>　　“强者不多，有资格进修罗界的更少，最重要的是，他要我去主阵，一百零八座岛屿，最中间的那一座，便是主阵之人，别人可以暂时离开岛屿，可以去串门，但主阵之人不行，得时刻提防着，一旦有情况，要立刻发动法阵，一坐就是一百年，明白了吧。”
<br/>　　叶少阳倒吸冷气，一百年待在岛屿上，只能修行，虽然收获肯定也是巨大的，但一个人要忍耐百年寂寞，这简直就是酷刑。
<br/>　　“现在主阵之人，是张正常，他坐了一百多年，坐够了，等着换人。”
<br/>　　张正常……元明时期的著名天师，也是最后一代御赐天师，元朝皇帝封的，到了明朝，进京的时候被朱元璋撤掉天师封号，改为真人，理由是“天师”这称呼太过霸气——天子之师，天子九五至尊，是最大的，能有老师吗？“为什么找你呢，不是有一百零八个呢？为什么不找李浩然，不找师父，不找镇元大仙？”
<br/>　　道风道：“大家都各守自己的位置，怎么能替换，自然要找新人，也不光是看中实力，受不受得住寂寞，也是要考虑的。”
<br/>　　天下强者虽多，但都在别的空间好好的，人家也不会愿意去修罗界苦修，还要担这么大的干系。接引道人能找到道风，也是不容易。
<br/>　　“那你到时候去吗？”
<br/>　　“我答应了。”
<br/>　　“那……说句不该说的，你万一没挺到道那时候就挂了呢？”
<br/>　　道风看了他一眼，淡淡说道：“他的意思，如果我死了，就由你来补缺。”
<br/>　　“开玩笑！”叶少阳反应巨大，“我可没答应。”
<br/>　　“我替你答应了。”不等叶少阳开口，他接着道：“没有这条件，你觉得他凭什么帮我们？”
<br/>　　怪不得接引道人跟自己说了那么多关于冥河和修罗界的事……
<br/>　　叶少阳无奈苦笑道：“这么苦的差事，还是你去吧，遇到要死的时候，我替你死吧。”
<br/>　　“还是你去吧。”道风露出蜜汁微笑，“寂寞也是一种磨炼，我是师兄，当然让给你。”
<br/>　　“靠！”
<br/>　　叶少阳拍掌说道，“干脆别争了，咱俩都别死了，一起去呗，一起做个伴！你看咋样？”
<br/>　　“好主意。”
<br/>　　从藏经阁下来，见到大伙，叶少阳没说太多，让瓜瓜穿越到空界去，看看那边的战争进行得怎样了。他有点不放心林三生。
<br/>　　第五把剑，究竟是什么？
<br/>　　从修罗界回来之后，道风一个人下到山谷里，顺着溪流一直走到源头处，有一座半截山，刀劈一般竖立在山谷一侧，山下有一汪泉水汇聚而成的清潭，泉水凛冽甘甜，青云子当年就是在这里观看鱼龟争食，突然悟道。后来长年让叶少阳给他打泉水上山，给他泡茶喝。
<br/>　　道风自己也经常到这里来，一个人想问题。
<br/>　　第五把剑……虽然现在考虑这个还早了点，但却是非有它不可。
<br/>　　他调查了十年，四大灵兽都找到了，五把仙剑，也只找到了四把——其中轩辕剑还在星月奴手上。还是想办法先把这把剑弄到手吧。
<br/>　　“轩辕剑……”
<br/>　　龙虎山上，偏殿的书房里，龙虎山几个长老围坐在一起，正在开会。
<br/>　　慕寒手托下巴，想了半天，用坚毅的语气说道：“如今叶少阳有三把剑了，其中有一把剑还是我们龙虎山镇山之宝……第五把剑，不知道是什么，但这把轩辕剑，我们一定要抢到，然后我再想办法把松纹古定剑要回来，你们以为如何？”
<br/>　　（下午写到少阳和道风从修罗界归来时，觉得很有感觉，突然想听一下雪山飞狐的主题曲，听完不久，就得知了金老爷子去世的消息，冥冥中自由注定吧。一代武侠宗师。心痛。）
