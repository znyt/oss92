田黄石现在都是可遇不可求的东西，实在是我本人也比较喜欢，所以……”陈默有些赫然的说道，心中却是想着，这个东西这么好，是不是自己以后可以去田黄石的故乡看看，能不能通过神识找找田黄石，如果能找到，那岂不是发财了。<br/>　　<br/>　　对于自己收藏的这一块，他是真的没有转让的心思，因为这个可是他第一桶金所得，比较有纪念意义。<br/>　　<br/>　　再说了他现在写毛笔字也不错，说不定以后也会得瑟一下，弄个书法显摆一下，那么如果没有印章的话，就会感觉缺少什么，所以这个是他以后得瑟的不可缺少的东西。<br/>　　<br/>　　丘阳华要是知道陈默，是这个原因，才想留下那个极品田黄石的章料，绝对会跳起来KO陈默，然后将田黄石抢夺过来。<br/>　　<br/>　　“好吧！如果你以后想转让，一定记得转让给我！价钱一定让你满意！”丘阳华有些失望的说道。<br/>　　<br/>　　“好！我记住了！”陈默也点点头回答道。<br/>　　<br/>　　众人又说了些话，陈默扫过这些人，就看到并没有胡月在场。<br/>　　<br/>　　“何老，胡先生呢？”虽然心中鄙视胡月，但是在场面上还是要说的过去的，所以叫声先生也就那样了。<br/>　　<br/>　　“哦！胡月先在在房里陪着老胡呢！”何老说道：“老胡在里面躺着呢！由于他的病情比较重，想走走都是不能的。这几天可能病情有些波动，你过来正好，要不然我今天还要过去催你。实在是老胡的病不能再等了！”<br/>　　<br/>　　“噢！病情很重？”<br/>　　<br/>　　“是很重，一天大部分时间都是昏迷的。胡月他不好意思去催你，这两天尽催问我呢！”何老说道。<br/>　　<br/>　　“好吧！我先去看看胡老吧！”陈默说道。<br/>　　<br/>　　“成！你先去，剩下的这些人都还好，慢慢来不着急的！”吴老也是在旁边听着，此刻也插话的说道。毕竟老胡他也是认识的，虽然关系没有何老他们亲近，但是看着一个病人躺在床上，那个景象，看着总是心中有些不得劲！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
