　　第357章
<br/>　　徐灵儿很早就来到落玉寨，帮助樊梨花养马。她很早就跟着程大雷，经过时间的历练，养马技能也获得提升。
<br/>　　从原先的『养马』已经升级到『驭马』，本身实力也到顶级。
<br/>　　一个顶级马夫，谁知她升级到绝世时，会不会带来其他变化。
<br/>　　面对徐灵儿的揶揄，程大雷有些无语，赔着笑脸问：
<br/>　　“好妹子，你别诓我，人究竟怎样了？”
<br/>　　徐灵儿白她一眼：“祸还不是你惹出来的，终究要给个说法，你等得起，人家大姑娘可等不起。”
<br/>　　光阴不饶人，程大雷才到行冠礼的时候，算起来还年轻得很。可樊梨花却是一日日大了，虽然还不到三十，可在这个时代，三十已和后世的四十岁差不多，该想想老去是怎么会事。
<br/>　　樊梨花既已倾心，程大雷却也不介意屋里多个人。只是如何处置，程大雷一时有些难以抉择。
<br/>　　徐灵儿道：“你若不好说，我便与你去说，事情不管怎样，总要有个说法。”
<br/>　　程大雷眼前一亮，是呐，这种事那有自己亲自去说的，总要有人从中牵条线，事情才好往下走么。
<br/>　　“好妹子，一切就拜托你了。”
<br/>　　“聊什么呢，拜托什么？”门口响起樊梨花的声音。
<br/>　　“没什么，没什么。”程大雷站起身来，道：“正好有些事要与你商量。”
<br/>　　许久未见，樊梨花像是消瘦了几分，眼眼也有些暗淡。只不过在程大雷面前，仍旧强打着精神。
<br/>　　“聊什么呀，什么事你还不能自己做主。”樊梨花正眼也不看程大雷，把程大雷挤到一旁，在正中那张太师椅上坐了。
<br/>　　程大雷讨了个没趣，自己拣了张椅子坐下，便向樊梨花说起此次的来意。
<br/>　　樊梨花自然没有不同意的道理，不过话里话外仍挤兑着程大雷。
<br/>　　程大雷当天就差人去琴川关，将改造的山寨的工人领过来。
<br/>　　三级山寨需要满足两个条件：一是山寨人口超过一千，二就是根据山寨的地形做一次彻底的改造，使其易于防守。
<br/>　　改造图纸是系统提供的，程大雷获得一份三级山寨的建设图纸后，就能根据各山寨的地形做出不同改变。
<br/>　　只要工人过来，事情就可以着手进行，倒也不需要耗费太多时间。
<br/>　　这是一件水到渠成的事情，程大雷却也并不在意，他心里担心的是另外一件事情能否水到渠成。
<br/>　　当晚，程大雷和银眸就住在落玉寨，银眸睡在外榻，程大雷自己在卧房内来回踱步。
<br/>　　门突然被打开，探出一个小脑袋。
<br/>　　“还没睡呢？”
<br/>　　程大雷一见徐灵儿便着急问道：“怎么样，她点头了么？”
<br/>　　“你觉得呢？”
<br/>　　“应该不会有什么意外吧。”程大雷先让徐灵儿去探一探樊梨花的口风，自己再求苏樱一个点头，按这个时代女人三从四德的道德观，事情应该很顺利。
<br/>　　“你想得太天真了。”徐灵儿耸耸肩：“她给否了。”
<br/>　　“呃……”程大雷一怔：“你该不会诓我吧？”
<br/>　　“你爱信不信，不信你去问她。”
<br/>　　“好妹子，若我能去问她，何必求你帮我。她究竟怎么说的，你好好告诉我？”
<br/>　　“给你做媒人当真不好当，两面不讨好。”徐灵儿叹口气，有些少女的忧愁烦恼：“她其实也没有说什么，反正左右意思是不同意。”
<br/>　　程大雷心情也郁郁起来，女人心海底针，果然是摸不透呐。
<br/>　　次日天明，又见到了樊梨花，程大雷毕竟有些尴尬。可樊梨花却像是无事人一般，似乎昨天徐灵儿并未曾对她说那些话。
<br/>　　到今天，琴川关的人也到了，程大雷只好将这烦心事暂且压在心头，着手将落玉寨升到三级。
<br/>　　已经有过数次经验，这对程大雷来说可以算是轻车熟路，而如今施工的工人，也不是之前的草台班子，都是受过训练的瓦匠、木匠、石匠等生活职业。这次既然是程大雷亲自监工，派出了的也自然是训练。
<br/>　　本来程大雷重视战斗职业，而轻视生活职业，但到现在他才发现，能打的好找，可高等级的生活职业却极其稀罕。这属于技术工种，真把作用完全发挥出来，能顶十个优秀战士。
<br/>　　其实倒也不用程大雷忙碌什么，他不过是走走看看，给大家鼓鼓劲，自然有专门的工头带着大家监工。
<br/>　　以目前的施工进度，大概有十天就可以竣工，比以前任何一次都要快。
<br/>　　程大雷心里主要惦念的，还是与樊梨花的事，始终悬着也不是个办法。这次他来落玉寨，便有解决这件事的用意。
<br/>　　当天夜里，他又求到徐灵儿身上，去探一探樊梨花的口风，看樊梨花心里究竟怎么想的。
<br/>　　徐灵儿走后，程大雷在卧房里来回踱步，竟有几分患得患失。忽然他心中一动，这女儿家谈事，肯定有诸多不能与自己言的。自己在这里等徐灵儿的转述，哪有自己亲耳听到真切。
<br/>　　而以自己的本事，想要偷听二人说话，当然也不算什么难事。
<br/>　　想到便付诸行动，程大雷直接跳窗出去，连外榻的银眸都没有惊动。然后他跃上屋顶，几个纵跃便落在樊梨花的房顶之上，脚踩着瓦片，轻飘飘的没有发出任何声音。
<br/>　　他将身子倒吊在屋檐上，隔着窗户纸隐隐有光线投过来。
<br/>　　程大雷拔了一根头发，拧在一起，轻轻扎透窗户纸，露出一个针眼大小的小洞，把眼睛凑上去，观察着屋内的动静。
<br/>　　这种套路还是蛤蟆寨的几个当初在长安城飞天蛤蟆白元飞说给程大雷听的，在程大雷手里用来，自然要强上白元飞数倍，整个过程中没有发出一丁点声音。
<br/>　　这还是因为程大雷入了绝世等级，如果以前的他，断然做不到这种程度。
<br/>　　屋里徐灵儿和樊梨花正在聊天，看也知道，二人的交流不算愉快。
<br/>　　徐灵儿完美的继承了其父的鸡贼，有些鬼心眼，小聪明。然而樊梨花却生性洒脱，天真烂漫，二人就算相处这么久，关系却也谈不上融洽。
<br/>　　程大雷暗暗在想，是否自己有些所托非人。
<br/>　　
<br/>(本章完)
