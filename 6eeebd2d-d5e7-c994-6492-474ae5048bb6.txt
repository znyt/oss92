　　道风告诉他，他昨天已经在山上转过，没发现附近有单独的寺庙。
<br/>　　也就是说，这天台宗的山门，就在风景区核心的这座国清寺。
<br/>　　这一点还是让叶少阳挺意外的，一般来说，修道的人都是喜欢清静的，会把原来的寺庙道观之类的让出来，或者另外开辟一个风景区，总之门派是一定会迁到附近的，这样能避免跟游客接触，潜心修炼，如果景区里道观寺庙，一般也就是分配几个弟子前去打理，收纳香火。
<br/>　　像国清寺这样，把宗门放在景区里的，还真是少见。
<br/>　　在国清寺转了几圈，观察了一下路线，两人就下山到了景区的街上，叶少阳去吃东西，道风也跟着，一起呆到了天黑，两人摸到山上，国清寺大门已经关上了，寺里炊烟升起，僧人们在吃饭。
<br/>　　“来太早了，这种事得等深夜。”叶少阳提议。
<br/>　　“深夜太安静，动静反而大，而且免不了有守夜的，再等一会。”
<br/>　　两人绕着寺庙围墙前进，到了展览馆的后面，静静等待着，好在是冬天，山里没有蚊虫，就是有点冷，叶少阳穿了大衣，缩在了一棵大树下面，用大衣把自己裹起来，看了一眼身边站着只穿着蓝色长衫的道风，眼中尽是羡慕。
<br/>　　“果然不是人有不是人的好处，不怕冷热，一年四季都可以穿得这么酷。”
<br/>　　“我怎么感觉听着像骂我。”道风懒得理他。
<br/>　　叶少阳拿出之前挟带的那本有关修行分类的笔记，招呼道风过来一起看，道风也很有兴趣，两人一边讨论着，不觉之中，寺庙里敲响了钟声，过了不多一会，里面传来集体诵经的声音。
<br/>　　是僧人们在上晚课了。
<br/>　　“行动！”
<br/>　　道风招呼叶少阳来到寺庙外墙下，自己先进去，观察展览馆内外的情况——以他的法力，想要隐身的话，地仙以下的法师都不可能看得见他。
<br/>　　叶少阳在外面等着，很快，雪花玛瑙戒亮了一下，于是他翻墙跳进去，落在展览厅的外面，展览厅有铁门拦着，但从院墙可以轻易地翻进去。
<br/>　　虽然里面不少价值连城的文物，但寺庙和景区管委会应该想不到会有人进来偷东西，再说佛像之类的也不好偷走，因此防范并不严，也没有保安什么的，进入展厅的大门只有一把铁链锁上，边上的窗户装有铁栅栏，看上去就年久失修，锈迹斑斑的。
<br/>　　叶少阳伸手掰了几下，几乎能掰断，但差了点力气，于是拿出准备好的石块，为了消音，裹了一块布在上面，转头对道风道：“我可是豁出去了，万一要是被抓，你记得给我送饭！”
<br/>　　用力砸了几下，砸弯了两根刚钢筋，试了一下，勉强能让自己过去，于是钻了进去。
<br/>　　打开手机自带的手电，叶少阳跨过隔离红线，来到药师佛下面。药师佛被一个玻璃罩罩住了。
<br/>　　对道风来说，人间一切障碍都可以无视，直接穿过玻璃，来到药师佛后面，双手在药师佛后背上摸索起来。
<br/>　　“需要用火，把树胶烤化。”道风说道，他虽然能幻化肉身，但毕竟不是人类，没法凭空变出人间的火焰。
<br/>　　“我还是想想怎么进去吧。”
<br/>　　叶少阳绕着玻璃走了一圈，找到一个玻璃门，上面有锁，倒不是很复杂的那种，只有一道插销暗锁。
<br/>　　试着推拉了一下，稍微有点晃动。
<br/>　　“没办法，只能撬锁了。”道风也凑上来围观，说道。
<br/>　　叶少阳白了他一眼道：“敢情不是你动手！”
<br/>　　法师不是超级英雄，道风这一身本领，只对生灵起作用，对实物能造不成什么伤害，这也是天地大道的的体现。叶少阳估摸自己倒是能一脚把门踹开，不过那样动静太大，把人引来就不好了。
<br/>　　“用灭灵钉，把门锁撬开。”
<br/>　　在道风指挥下，叶少阳在背包里翻找起来，半晌才想起来，灭灵钉送给了谢雨晴……落寞地发了一会呆，找出来一把铜剪刀，努力插进玻璃门的缝隙里，用力撬动。
<br/>　　“咔嚓！”
<br/>　　几分钟后，在不破坏玻璃门的情况下，叶少阳总算把门撬开了，用力呼吸了一口气，钻了进去，点燃了一只蜡烛，在药师佛的后背烘烤起来。
<br/>　　这可是价值数千万的文物……叶少阳有点心惊肉跳的，但为了太阿剑，也只能这么干了。
<br/>　　树胶烤化，流淌下来，颜色也掉了，这才露出了里面的麻布，一圈圈的。两人找到绳头，一个人用火烤化金漆，一个人用力撕开。
<br/>　　药师佛的腰越来越细。叶少阳的心情也越发激动。
<br/>　　终于，太阿剑古铜色的剑鞘出现在了手电的光照下，叶少阳心中一块石头落地，近千年了，不知道多少人见过这尊药师佛，这个藏在药师佛中的惊天秘密，居然一直都没被发现。
<br/>　　叶少阳一边想办法把它弄出来，一边说道：“道风，你之前怎么说来着，这太阿剑是要有缘人才能拿到的，咱们这算是有缘人吗？”
<br/>　　“为什么不算？”
<br/>　　“我怎么感觉像是作弊呢，如果没有山海印，咱们可能永远也找不到它。”
<br/>　　道风想了一下，说道：“为什么是你得到山海印，不是别人？这本来也是机缘。”
<br/>　　总算把太阿剑取了出来。
<br/>　　沉甸甸的宝剑拿在手中，叶少阳心情不免也激动起来。
<br/>　　代表人间终极元素的第五把剑，总算拿到了。
<br/>　　叶少阳深吸一口气，迫不及待都拔出宝剑。
<br/>　　一道青光射出，强大的灵力刺的两人都微微一颤，相视看了一眼，露出微笑。
<br/>　　“什么人，我们已经报警了！”
<br/>　　门外突然响起一声呵斥，接着数道手电筒的光打在了叶少阳身上。
<br/>　　被发现了！
<br/>　　这一刻叶少阳内心是紧张的，不管怎么样，毕竟是做贼，还被抓了现形。
<br/>　　叶少阳急忙站起来，高举双手，试图解释，但自己明明干的就是偷盗的事，怎么解释？
<br/>　　大概是为了防止他们逃跑，外面的人呼啦一下涌到玻璃门外，挡住了出口。叶少阳眼睛也适应了强光，大致看了眼，进来七八个都是和尚，手里都提着棍。
