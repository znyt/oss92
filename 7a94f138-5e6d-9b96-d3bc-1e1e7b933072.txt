　　比赛室内，看着这一场比赛终于是告一段落，叶帝五人都是松了一口气，不管以后的比赛怎么说，起码他们今天已经来了一个很好的开局了。
<br/>　　“今天大家的发挥都很好，继续努力！”叶帝对着众人勉励了一番后，便是带领着队伍向着ST的比赛室走去。
<br/>　　等到他们来到对方比赛室的时候，金相赫一行人已经是调整好了情绪，连忙站了起来。
<br/>　　大家互相握手拥抱，说了几句话后，叶帝几人便是向着比赛室外走去，毕竟现在还是在现场，而且他们之前已经叙过旧了，没必要在这上面耽误一些时间。
<br/>　　至于对方会不会因为这一场比赛而丧失信心，这一点就不是叶帝他们考虑的了，如果连这都坚持不下去，那对方就枉费世界第一战队这个称号了。
<br/>　　当叶帝他们一行人走到舞台上的时候，场下的观众再次嘶吼了起来。
<br/>　　刚开始大家喊的还有些乱，不过几秒钟过后，这些口号便是渐渐的汇聚成了两个字：荣耀！
<br/>　　听着这震耳欲聋的欢呼声，叶帝几人也都是心潮澎湃，能够让现场20万观众同时呐喊，这想必应该是每个职业选手都想要追逐的吧，这让他们感觉到了自己存在的价值。
<br/>　　“今天这场比赛的mvp你猜会是谁？”看着叶帝他们打过招呼退场，米粒才开始对着林雅楠问道。
<br/>　　“我感觉夜雪和香蕉都有机会，这一场比赛他们两个发挥的实在是太好了，如果非要我给一个mvp的话，我想应该是香蕉。”林雅楠跟着开口说道。
<br/>　　“反正这两人都有一半的机会，香蕉的节奏也是被夜雪带起来的，如果真的让我给的话，两人我都给一次mvp。”米粒笑着说道。
<br/>　　片刻之后场上的大屏幕上出现了本场比赛的mvp，最终的mvp是给到了香蕉的流浪的身上，这一场比赛无论是战绩还是参团率，亦或者是打出的伤害，香蕉的流浪在队伍里都是当之无愧的第一名。
<br/>　　接下来的事情就要变的简单了很多，香蕉去接受采访，叶帝他们在后台休息室等待着香蕉，因为今天是首日比赛，所以仅仅只有5场bo1的比赛，5场比赛都是A组的第一组战队。
<br/>　　接下来的比赛则是每天会进行10场bo1，明天是第二组，后天是第三组，以此类推，直到比赛结束，决出小组的一二名为止。
<br/>　　等到香蕉这边接受完采访后，叶帝一行人便是乘车向着基地内返去，而场下那些观战的队伍，这个时候也是陆续离开。
<br/>　　大家今天想要观战的就是ST和荣耀白队的对决，接下来的几支战队对于他们来说都没有什么威胁，所以这个时候这些人也不想在这继续耽误时间，虽然这样有些不尊重人，但是在时间如此紧张的环节下，大家为了争取更好的名次，也只能是分秒必争了。
<br/>　　不过场下那些买过票的观众倒是没有立场，依旧在那里观看着比赛，现在已经是一票难求的阶段，他们能够买到入场票已经是非常的好运了，所以自然是要看完所有的比赛了。
<br/>　　而在叶帝他们离开比赛场的时候，关于荣耀战队和ST战队比赛的新闻便是已经被各家媒体争相推送了。
<br/>　　《荣耀白队上野发力，轻松结束ST！》
<br/>　　《荣耀战队已无人可挡，火力全开拿下揭幕战首胜》
<br/>　　《荣耀时代来临，蓝白有望决赛相遇》
<br/>　　.......
<br/>　　国内几乎所有的媒体都在这样的说着，这个时候大家都已经知道了叶帝的背景，所以自然是不会有媒体再去抹黑他了，至于那些在网上喷叶帝的，那也是因为网络的自由性，大家无所顾忌而已。
<br/>　　如果真要是放到了现实，他们还真不一定敢这样明目张胆的骂人。
<br/>　　叶帝和蒋念他们是分两个车向着基地内返去的，开车的依旧是陈献忠，现在局势已经渐渐稳定下来了，所以基地内也就留下陈献忠他们几人而已。
<br/>　　回到基地内，众人难免又庆祝了一番，因为A租是前三组，B组是后三组，所以距离蓝队的比赛还有几天的时间，他们倒是也不急。
<br/>　　“陆伟他们在第四组了吧，那个组没有什么强队，估计他们能闯到8强。”吃饭的时候，几人在那里闲聊着。
<br/>　　“前两天陆伟那小子还跟我聊呢，说他们注定只是陪衬了，能闯到8强就已经不错了，遇到咱们蓝白两队，他是一点信心都没有。”林萧笑着开口说道。
<br/>　　因为以前他就是雷霆战队的和陆伟他们关系都很好，所以平常的时候聊天的时间自然是要多上了一些。
<br/>　　“8强已经不错了，咱们华夏能够在8强前面包揽三个位置，这已经是一个非常喜人的成绩了。”茄子开口说道。
<br/>　　“其实挺心疼ST的，他们的运气实在是太背了，居然和你们抽到了一个小组，要不然的话他们还能得个第一的名次。”飞刀有些替金相赫他们感到遗憾。
<br/>　　“也不能这么说，接下来还有一个循环呢，到时候很可能ST会过来找回场子的，到时候我们两支战队就是并列第一了，小组赛可是没有加赛的，只要打完巡回，决出一二名就结束了，所以一切还是个未知数呢。”叶帝摆了摆手说道。
<br/>　　“没错，ST战队确实很强，今天你们有很大的运气成分在那里了，如果ST下路那一波不是那么急的话，不让matin的冰女传送下来，也许结果会变的有些不同，那一波彻底将matin弄炸了。”秦霄表示同意。
<br/>　　“明天还有KA战队的比赛，这个战队也要值得针对以下，估计第二组首名的位置就是他们的了，早晚我们都会遇到的，现在只是热身，小组赛之后的突围赛才是真正的开始，所以大家不要掉以轻心。”李景禹开口为大家打气。
<br/>　　这一刻大家的心都非常的齐，并没有藏私什么的，现在大家的目标就一个，那就是在决赛中相遇，现在他们已经被分配到了两个小组中，先决条件已经有了，剩下的就看他们自己的表现了。
