　　楚云天并不知晓如何离开这里，下潜的那块巨石将会把人带去哪，也超出了楚云天的所知范畴，周围到处都是灌木丛，隐藏着数不清的危险。
<br/>　　他不是新人类，没有能力独自走出这快地方，此时他唯一能做的便是尽可能的将所见到的一切记录下来，也尽可能的多活一些日子。
<br/>　　已经受到皮包骨头的手在纸上写写画画，正是整个阳谷入口地图，楚云天又停下翻开前一页所画，顿时他在阴谷的路线便就此呈现。
<br/>　　所有他走过的地方，都被他画了下来，旁边有着详细标注。
<br/>　　他不知道自己还能坚持多久，但他知道先进去的那批人要对付他的儿子！
<br/>　　楚云天与世无争，文明时代就愿意放弃大好前程守着老婆孩子，末世纪元与亲人天各一方，便投身于科研之中，他在这个世上只心系两个人，一个是妻子袁熙叶，一个就是儿子楚涵。
<br/>　　白家和罗家竟然想要杀楚涵，这就等于是碰了楚云天的逆鳞！
<br/>　　他的确是个没什么用处的老头，但他脑子里的那些研究成果及猜想，可不代表不危险……
<br/>　　就在楚云天目光深邃之际，阴谷的出口处忽然传来一阵动静，紧接着一大批人便从远处走来，步步踏上了这一平台之上。
<br/>　　出来的这批人自然就是高家人，只是高漫秋并不在，众人你一句我一言的说着，闲逛在四处等待高漫秋出来。
<br/>　　“这里就是阳谷入口？”
<br/>　　“大小姐在跟火族人说什么？”
<br/>　　“谁知道呢，不过这次小少爷没来，我们还能一路顺利真是多亏了大小姐。”
<br/>　　“但小少爷不在，我们只能分出一批人单独寻找阴谷内的高家旧址，这样一来两拨人分开，还是有点伤。”
<br/>　　“小少爷不仅熟悉阴谷和高家的旧址所在，更去过阳谷，这回没小少爷带队……”
<br/>　　“那可是墓园啊……”
<br/>　　高家的这批精英都有些畏惧阳谷，这里毕竟是所有神秘家族都不敢靠近的地方。
<br/>　　高家此行由于高少辉的缺席，也不得不将精英队分成两批，一小部分人约莫二十留在阴谷，逐步寻找高家旧址的所在地，剩下的八十人则随高漫秋直奔阳谷。
<br/>　　只是在阴谷的出口地带，高家同样受到了火族的阻拦，只是相对于前两个家族的大动干戈甚至下了杀手，火族对高家的态度就有些诡异了。
<br/>　　火族的少主只是炎良只是在原地站了一会儿，便直接冲着高漫秋说了一句‘不得了’。
<br/>　　紧接着便又是其余人先行离开，高漫秋留下与火族少主坐席长谈……
<br/>　　来到阳谷入口出的高家众人谈论间，楚云天也将所有听到的信息记下，他不清楚眼前的这批人什么身份，只知道所有的一切都需要分析记录，因为极有可能与楚涵有关。
<br/>　　楚云天知道自己已经穷途末路，甚至活不了多久，但他想要为自己的儿子做点什么。
<br/>　　只是让他没想到的是，最后又出来的这批人似乎有点不同，没有第一批白家和罗家之人的戾气，也没有先前江凌轩的焦急感，处处透露着古怪和神秘。
<br/>　　楚云天知道神秘家族的所在，也从一路以来听到的消息中步步了解到一些信息，比如狼牙，比如自己的儿子，竟然与这么多的神秘家族有联系。
<br/>　　他将自己的存在感降到最低，一动不动只维持着呼吸，此时的高家众人也将话题聊到了正规。
<br/>　　“你们说小少爷的缺席，到底是为什么？他忽然与我们分开，去去哪？”有人开口提到这个话题。
<br/>　　“最关键的是，大小姐竟然没有管。”有人补充。
<br/>　　“是跟狼牙的那位有关吧？”忽然一个人出声，让四周都一静。
<br/>　　灌木丛中隐藏的楚云天呼吸猛然急促起来，心脏也开始激烈跳动。
<br/>　　狼牙！
<br/>　　“谁？！”一名高家精英猛然朝楚云天所在的地方望去，目光犀利。
<br/>　　忽然变化的呼吸频率很容易被神秘家族的成员发现，他们一个个都战力超绝感知敏锐，对比之下楚云天努力到极致的隐藏，只需要稍微注意一点就能发现。
<br/>　　第一批的白家和罗家人没能发现楚云天，是因为受惊过度后的情绪不稳，又着急进入阳谷且彼此防备。
<br/>　　第二个到达的江凌轩没能发现楚云天，也是着急赶路，而根本没有注意周围动向。
<br/>　　高家的这批人则不一样，他们在这处地方等着高漫秋，严谨注意周围动静太正常。
<br/>　　楚云天深吸一口气后闭上眼睛，他知道，他被发现了。
<br/>　　很快，眼前的茂密灌木丛被人拨开，几个高家精英走近，他们都有些莫名的看着坐在满是植被深处的这个老头，不禁神色古怪。
<br/>　　“一个普通人，连新人类都不是。”高家的一名成员开口道。
<br/>　　“怎么说？”旁边的高家成员都拿不定主意，有些不知所措。
<br/>　　普通人，非新人类，一个老头？
<br/>　　他们全体都是高家纯种血统的成员，哪怕稍微伤到这个老头的一根手指头，都有可能收到制裁者的惩罚。
<br/>　　想到这里，几名高家成员都退后了几步，束手无措。
<br/>　　此时在阴谷内，高漫秋与炎良面对面站在阴谷出口处，四周的火族人都退居到十几米外守着，安静的一片地带都充满肃杀感。
<br/>　　与水族的柔婉不同，火族人直来直往，并没有要请高漫秋去火族旧址内坐一坐的打算，沏茶什么的更是笑话，他们火族平常也不在意这些东西。
<br/>　　只是在高漫秋与炎良谈话间，不少火族人都私下窃语。
<br/>　　“这个女人……不是高家嫡系么？”
<br/>　　“少主对她的态度有点特别，不会是看上她了吧？”
<br/>　　“别瞎说，少主什么身份，怎么会看上一个区区四偏门成员？”
<br/>　　“四偏门的嫡系，身份也不低。”
<br/>　　“但比起四正门，还是差远了！更何况当下我们已经回归阴阳谷，是真正的火族，能配上我们少主的除非是神女！”
<br/>　　“说起来，你们不觉得这个高漫秋气息有点古怪么？”
<br/>　　“她身上有水族气息，还是很强大的那种。”
<br/>　　不少火族人都皱起眉，显然高漫秋见过了水族人，但究竟是哪位水族人，为何与炎良一样，见了这位四偏门成员？
