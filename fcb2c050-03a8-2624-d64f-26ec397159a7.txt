　　“掉在我们院子里了？”
<br/>　　那人愣了一下旋即嗤笑道。
<br/>　　“小子，掉到我们院子里了那就是我们的了。你还想要回去？”
<br/>　　“老铁，那东西你们要了也没用啊。”
<br/>　　林峰道。
<br/>　　别说没用，你们看都看不见。就在那搁着呢，你们都看不到。
<br/>　　“没用那也是我们的。”此人极其霸道。
<br/>　　林峰皱起眉头，看来正面是进不去了，只能偷偷的翻墙进去了。
<br/>　　“墨迹什么了老三，赶快把这个尸体丢出去，免得发臭了。”
<br/>　　里面一人道。
<br/>　　“知道了。小子速度滚远点，再不滚，别怪爷爷欺负未成年了！！！”
<br/>　　这汉子没搭理林峰，随手把门一关，但没关上。
<br/>　　林峰还要说什么但下一刻他就面色变了一下。
<br/>　　因为透过门缝，他看到了这群人从内屋里拖出来了一个尸体，是个女学生。
<br/>　　不是别人！
<br/>　　正是他们育才高中的！而且还就是林峰的同学！
<br/>　　林峰瞬间瞳孔微微眯起，此时此刻，这个女同学裤子没了，就上面还挂着一个胸罩，浑身都是牙齿印，甚至私处已经是糜烂了。
<br/>　　林峰眼中杀机毕露，这个女学生她好像记得叫什么刘媛媛，算是一个很文静的女孩，之前也是要回去找自己父母，结果没想到死在了这里，而且明显死之前还被这群人给强女干过。
<br/>　　“这群人渣！垃圾！！该死！”
<br/>　　林峰拔出了武士刀眼中露出了一抹愤怒。
<br/>　　砰的一下。
<br/>　　林峰一脚踹开了铁门。他也不翻墙了，直接破门而入。
<br/>　　“你特么找死啊，居然敢踹我们狼窝的门？”
<br/>　　之前开门的那壮汉看到是林峰顿时暴怒道。
<br/>　　“三阶的？”
<br/>　　林峰二话不说直接是手一动一枚飞镖飞了过去。
<br/>　　噗嗤。
<br/>　　瞬间一枚飞镖直接爆头，这壮汉都没有回过神整个人就倒在了地上。
<br/>　　“草，老三！”
<br/>　　“妈的，找死！砍了他！”
<br/>　　“这小王八犊子，还真是找死来了！”
<br/>　　其余几人都回过神纷纷拿起了武器都是些斧头水管之类的。
<br/>　　个个都怒吼着朝林峰劈砍而来。
<br/>　　狂龙出海！
<br/>　　武士刀一刀斩杀而去。
<br/>　　噗嗤噗嗤。
<br/>　　顿时三个人头腾空而起。
<br/>　　唯独一个比陈勾还要胖的大胖子是辛免于难。
<br/>　　头虽然没有砍下来，但是肚子上却出现了一个刀痕，肠子都可以看得见，这一刀差点就把他的腹部给砍穿了。
<br/>　　“你，你是五阶的？”
<br/>　　这个大块头实力很高，足足有四阶。
<br/>　　即使如此，他也十分的震惊。
<br/>　　他作为这个据点的队长才好不容易混到这个等级，这个高中生居然比他还高。
<br/>　　“没错，恭喜你答对了，可惜你要死。”
<br/>　　林峰冷冰冰道。
<br/>　　“死？你到底是谁，为什么要对我们动手？”
<br/>　　大块头面色难看道。
<br/>　　“你知不知道我们是狼窝的人？”
<br/>　　大块头眼神落在桌子上的一个对讲机慢慢的凑过去。
<br/>　　“狼窝？”
<br/>　　林峰冷笑。
<br/>　　“没错，我们狼窝是盘踞这一代的大组织，这附近都是我们的地盘，你小子敢动我们，你死顶了！”
<br/>　　大块头色厉内荏道。
<br/>　　说着他就一把拿过对讲机。
<br/>　　“求……”
<br/>　　撕拉拉。
<br/>　　但是下一刻。
<br/>　　他整个人就开始颤抖，然后直挺挺的倒在了地上翻着白眼。
<br/>　　正是林峰的麻痹飞镖。
<br/>　　“还想求救？”
<br/>　　林峰走了过去，这个大块头脂肪很多，普通飞镖估计很难秒杀，所以林峰用麻痹飞镖。
<br/>　　捡起对方的对讲机。
<br/>　　这是个好东西。
<br/>　　林峰直接收入物品栏。
<br/>　　然后目光落在空投上。
<br/>　　这是林峰遇到的第二个C级空投。
<br/>　　“不知道能够得到什么。”
<br/>　　……
<br/>　　与此同时。
<br/>　　在江城的郊区。
<br/>　　这里建造了一个五十米墙壁围起来的幸存者基地。
<br/>　　“梅姐。”
<br/>　　一个别墅内的豪华办公室。这里是王家的地盘。
<br/>　　王梅正皱着眉头思忖着什么。
<br/>　　在他的面前放着这个幸存者基地的基本资料。
<br/>　　根据她的推测，这个幸存者基地，早在一年前就已经开始秘密的构造了。不说别的，就是这五十米的墙壁，那也不可能是这几天时间就可以造的出来的。
<br/>　　“难道家族和军方高层早就有所准备？这一次灾难到底是什么缘故导致的？生化脏弹？还是说是核弹？亦或者是空间裂缝？”
<br/>　　王梅百思不得其解，作为江城三大家族之一的王家长女，她都无法知道这件事情背后的真相，拨开一层迷雾看到的却又是一层新的迷雾。
<br/>　　“进。”
<br/>　　门口进来一个女人。穿着一身紧身的黑色皮衣，把曼妙身材都衬托出来。
<br/>　　她是王梅的贴身女秘书，也是女保镖之一，代号寡妇。
<br/>　　“梅姐，我今天就准备去彻查曹龙身死的事情。”
<br/>　　“嗯，多带一些人马，另外江城那边的一个千年僵尸墓你也去探查一二，据说里面有不少好东西，据我所知，张家和古家都派人去了，我们王家不能落后他们。”王梅道。
<br/>　　“既然如此，那梅姐，我现在就出发了。”
<br/>　　寡妇面无表情的道。
<br/>　　“去吧。记住，杀害龙哥的人，我要活口。”
<br/>　　“是。”
<br/>　　寡妇立即领命来到了别墅外。
<br/>　　这里已经有三个人等候多时。
<br/>　　“寡妇，梅姐怎么说。”
<br/>　　穿着一袭白色西装，口袋上还放了朵红色玫瑰，被叫做嗜血绅士的年轻人开口道。
<br/>　　“活捉。”寡妇淡然道。
<br/>　　“有意思，沙漠之鹰，我已经三年没玩这种枪了。最后一次玩还是在非洲和国际反盗猎组织那群人PK时玩了两把。”穿着黑色汗衫，人称坦克的大块头说道。
<br/>　　“活捉？啧啧，这个人可惨了，梅姐可是被称之为毒蛇的存在，落在他手中，那可是有死无生啊。”一个被叫做地鼠的侏儒，舔着自己锋利的长指甲冷笑道。
<br/>　　“这一次，曹龙死，不是小事，虽然王家看不起曹龙这个人，但是梅姐心中，是有他的地位，所以这一次任务对于我们小队来说是非常重要的，无论如何，也要拿下，不然只会让其余小队的人笑话我们，让梅姐质疑我们的能力。”寡妇道。
<br/>　　“另外，千年僵尸墓的事情，梅姐也一并交代了，到时候，我们分开行动，现在回去清点你们的人手！半小时后，出发！”
<br/>　　话说这个院子内。
