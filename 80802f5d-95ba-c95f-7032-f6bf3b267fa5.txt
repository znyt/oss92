　　接下来两日的比赛，也是国内观众非常关注的，因为两天都有国内战队的比赛。
<br/>　　在第三日的比赛中，荣耀蓝队不出意外的轻松战胜了og战队，至此国内已经是有着两支战队跻身进了4强的队伍中，是各大赛区唯一有两支队伍走到这里的赛区。
<br/>　　看到这一幕，众多国内的粉丝自然是换新鼓舞，而在欢欣鼓舞的同时，大家对于这最后一天雷霆和TM战队的比赛也是越发的关注了起来。
<br/>　　既然已经走到了这一步，那么他们自然是想要看到更加传奇的一幕，也许4强队伍真的可能会有3支国内的队伍呢，这样的话，他们决赛的名额已经是确保了。
<br/>　　因为如果雷霆战队晋级到半决赛，那么下半区无论谁胜谁负，他们都已经稳稳的拿下了一个国内晋级决赛的资格了。
<br/>　　在无数观众期待的目光中，8进4最后一天的比赛终于到来，这一天叶帝还有蒋念，包括楚白衣等众多国内的选手都是来到了现场，为雷霆战队加油打气。
<br/>　　“这一场比赛你怎么看？”不知道什么时候凑到叶帝身边的楚白衣开口问道。
<br/>　　“TM战队那边有god，雷霆战队这边有小伟，两人都是十王者选手，而且都是中单，所以这条路估计是旗鼓相当，TM战队这边下路略强一些，而雷霆这边打野要比对面强一些，综合起来55开吧，还是要看双方的临场发挥。”叶帝思索了一番后，开口说道。
<br/>　　“没想到雷霆战队居然闯到了这一步，本来他们闯到8强，粉丝们就很知足了，不过现在这个情况，恐怕不仅4强，大家心里都会有遗憾的。”秦霄跟着开口说道。
<br/>　　“是啊，现在希望就在眼前了，4支战队有3支国内战队晋级，这个说出去多好听，所以小伟他们现在肯定也是卯足了力气。”林萧笑着说道。
<br/>　　就在众人闲聊的时候，场上双方已经是开始了ban人，不出意外的双方都是ban的目前版本很强势的英雄。、
<br/>　　只不过让叶帝他们有些失望的是，在第一局的比赛中，雷霆战队这边并没有打出中野或者上野的联动，小刀和陆伟两人居然被对面给压制的死死的，完全不是他们平时发挥的水准。
<br/>　　“怎么个情况，这一局小伟他们的发挥也太不正常了吧。”看到第一局在35分钟的时候雷霆战队落败，楚白衣有些诧异的说道。
<br/>　　“这一局他们确实是被TM战队这边压着打，不过不是他们发挥不好，而是TM战队那边已经是明白了他们的套路了，对面的打野始终在上半野区待着，目的就是不想让小刀发挥出威力来。”叶帝开口说道。
<br/>　　“是啊，如果雷霆战队不改变策略的话，那么今天这场对决，估计雷霆战队就有些难了。”秦霄跟着开口说道。
<br/>　　场下众多雷霆战队的粉丝，这个时候也都是在那里开口议论着，一个个脸上都是充满了担心，直播间里更是如此。
<br/>　　10分钟后，在无数观众紧张的目光中，双方的第二场比赛再次开始，这一局小刀的打野风格做了一些改变，不再使用暴力的打野英雄，而是选择了偏保护型的英雄。
<br/>　　只不过这下子雷霆战队这边又着了对方的套了，他们这边选择偏发育的打野，而对面却是选择出了十分强势的蜘蛛。
<br/>　　在比赛开始3分钟的时候，蜘蛛便是来到中路，强行逼掉陆伟的闪现，紧接着5分钟再来，强势帮助god拿到一血，至此陆伟中路这一条线已经是没法和对方对决。
<br/>　　而随着蜘蛛的起势，雷霆战队这边上下两路也是打的异常的艰难，小龙这些资源通通都被对方给吃掉了，最终这一场比赛在时间到达28分钟的时候，TM战队再下一城。
<br/>　　看到这里，众多粉丝终于是有些坐不住了，就是连带着叶帝等人也是如此，两场比赛下来雷霆战队这边都被对方给压住了，弄的他们根本无法发挥出自己的正常水平。
<br/>　　“卧槽，怎么个情况啊，我还想看半决赛咱们打内战呢，这雷霆战队发挥的有些不对啊。”
<br/>　　“战术上被压制住了，现在雷霆战队必须转换思路了，不过现在TM战队已经打到赛点了，现在是真的难了。”
<br/>　　“真的难办啊，以TM战队这个态势，如果他们3:0取胜的话，那么蓝队和他们半决赛的对决谁输谁赢还真的很难说啊。”
<br/>　　“没错，不得不说TM战队这个教练组真的是可以的，已经将雷霆战队给研究透彻了。”
<br/>　　直播间里的众多观众开口议论着，而就在他们议论的时候，叶帝这边却是接到了一个出人意料的电话，来电话的正是雷霆战队的陆伟。
<br/>　　“怎么了小伟？”叶帝拿着手机向前走去，同时叫上了秦霄和李景禹两人，这个时候他已经有预感对方找他是来做什么了。
<br/>　　听到陆伟的名字后，秦霄和李景禹两人心里便是一动，跟着向后台走去，他们这边的动静同样也是引起了现场众多观众的关注。
<br/>　　而电话那边陆伟所说的果然和叶帝想的差不多，这个时候他们已经是陷入到了瓶颈，所以想要让叶帝他们帮忙研究一下。
<br/>　　听着这话，叶帝也没犹豫，同属国内战队，这个时候是为华夏赛区拿下无上荣誉的时候，他们自然是不会藏私。
<br/>　　当然了他们也是知道后果的，在国内还好说，毕竟大家都是向着自己人的，不过如果被国外的网友知道了这个消息，估计他们肯定是要喷雷霆战队的，毕竟这样确实是有些不地道，在这种关键时刻‘借人’。
<br/>　　只不过这本来就是一个注重结果的比赛，而且叶帝他们也就是给个建议，最终要上场的还是陆伟他们五人，所以即使那些人说，他们也说不出什么来。
<br/>　　就这样在后台叶帝三人和对方的教练组讨论了七八分钟，将自己心中的想法说出来后，双方最终才决定好接下来的战术。
<br/>　　很快双方的第三场比赛，正式开始。
