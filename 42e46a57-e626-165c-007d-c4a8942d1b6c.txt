　　　　
<br/>　　由于深处已经没有蝙蝠的缘故，离开乱斗地点后，慕清霄就直接搂着陆雪琪，飞速前进，路中所遇到的幻境全部轻松破除。
<br/>　　陆雪琪对自己男人的实力又有了新的认知，慕清霄不说，她也不问。
<br/>　　片刻后，慕清霄停下脚步，身下已是深不见底的深渊，漆黑一片看不到尽头，瞥了眼路口处那块巨石，上面刻着‘死灵渊’三个大字。
<br/>　　没有在死灵渊多做停留，慕清霄带着陆雪琪直接穿过死灵渊，来到无情海。
<br/>　　漆黑的大海，冷冽的海风，无边无际的海域，隐隐传来潮水冲刷岸边的声音，当真巧夺天工，谁又能够想到，万蝠古窟下有这样一处神奇之地。
<br/>　　“这就是传说中的无情海，气息真令人感觉到此处孤寂与压抑，快点办完事离开此处吧，我不想在这里多待。”
<br/>　　“好。”
<br/>　　“轰隆隆......!”
<br/>　　就在这时，远处忽然传来一阵巨大的轰鸣声，无情海中泛起滔天巨浪，海浪席卷而来，浪涛声震耳欲聋。
<br/>　　漆黑的海面上，陡然亮起两盏巨大的绿色明灯，仔细看去，明灯下面拖着一条巨大的身形。
<br/>　　待身形距离岸边越来越近，能够清楚的看到，身影是一头巨蛇，体黑，腹白，绿眼，蛇身粗逾四丈，长逾百丈。
<br/>　　陆雪琪脸色惨白，左手紧攥着天琊，慕清霄双眼眯了眯，嘴角勾起一丝笑意，道：“元婴二重，若非这头畜生食过仙药，寿命绝无可能活过万年。”
<br/>　　此时，陆雪琪脑海中一片空白，显然是吓懵了，声音都有些颤抖，凄然道：“这，这难道是传说中的黑水玄蛇！”
<br/>　　“吼吼！”
<br/>　　黑水玄蛇已经到了岸边，咆哮两声，明显就是冲着两人来的。
<br/>　　恐怖的气息扑面而来，陆雪琪不过筑基修为，自然不可能抵御，仅仅是气息就吓得她浑身颤抖，连战斗的勇气都没有。
<br/>　　黑水玄蛇毕竟是上古异兽，而且活了上万年，修为自然是本位面最顶级的存在，向青云门弟子遇上，绝对是有死无生。
<br/>　　慕清霄轻轻将她拥入怀中，陆雪琪趴在他胸膛上瑟瑟发抖，凄然道：“我们的运气真是差劲，没想到会在此处遇到上古异兽，能够与你死在一起，我不后悔。”
<br/>　　“什么死不死的，多不吉利，不就是一条小蛇就将你吓成这样。”
<br/>　　“噗哧...”
<br/>　　慕清霄的话彻底将陆雪琪给逗乐了，心中的恐惧也渐渐散去，忍不住白了慕清霄一眼，她可是头一次听到，将上古异兽说成小蛇的。
<br/>　　“吼吼！”
<br/>　　黑水玄蛇怒吼一声，居然有两只小虫子当着它的面谈情说爱，简直不将它放在眼里啊，于是张开血盆大口，携带者浓郁的腥气呼啸而来。
<br/>　　陆雪琪眼神有些凄凉，盯着慕清霄看了一眼，缓缓闭上美眸，趴在他的胸膛，静候死亡的来临。
<br/>　　“蓬！”
<br/>　　华丽的黑色羽翼自慕清霄背后弹射而出，眨眼间已经抱着陆雪琪来到空中，使得黑水玄蛇扑了个空，吃了一嘴的泥沙。
<br/>　　晃了晃硕大的头颅，黑水玄蛇瞳孔中满是愤怒，没想到让小虫子摆了一道，拖着百丈身躯，仰天怒吼。
<br/>　　听到黑水玄蛇的嘶吼声，陆雪琪逐渐睁开双眼，视线清晰，发现自己已经处于半空中，大脑一时间转不过弯来。
<br/>　　紧接着，入眼的一双黑色的羽翼，任何美丽的语言都不足以形容它的华贵，抬头臻首，是一张神骏的面容，眉宇中央有一颗黑色的菱形晶石，散发着神圣的气息。
<br/>　　黑水玄蛇的恐怖气息消失，望着面前男子，俾睨众生的眼神，陆雪琪神情有些恍惚，一时间有些呆了，这真的是自己的恋人吗？
<br/>　　“嗡嗡！”
<br/>　　贯彻九霄的剑鸣声响起，魂剑已出现在眼前，慕清霄手握魂剑，气息已经攀升到了顶点，元婴三重的气息毫无保留的扩散而出。
<br/>　　魂剑高高抬起，虚空中一柄由剑气凝聚而成的百丈巨剑乍然而现，慕清霄低喝一声：“斩！”
<br/>　　只见刷的一声，剑刃与空气摩擦，势若奔雷，剑芒四射，无上剑意席卷而下，噗嗤一声，剑光闪过，时间宛若在此时定格。
<br/>　　剑刃消失于天地见，浪涛拍岸声再次响起，黑水玄蛇血盆大口张开，绿油油的蛇瞳中尽是绝望，身躯无力的扭动几下，紧接着响起一道凄厉的嘶吼声。
<br/>　　紧接着，百丈身躯分为两半，向崩断的绳索掉落在地，鲜血与内脏流了一地，站在高空都能闻到底下浓郁的腥气。
<br/>　　陆雪琪呆愣愣的望着底下被斩成两半的黑水玄蛇，美眸中满是不敢置信，上古异兽居然死了，还是一剑斩杀，她师尊绝对没有这种实力，而且还差得远！
<br/>　　“怎么了，我的小美人？”
<br/>　　耳边响起慕清霄的声音，陆雪琪惊醒过来，脸颊微红，抬起臻首，美眸中异彩涟涟，刚才如同剑神一般，一剑斩杀黑水玄蛇的青年，是他的恋人。
<br/>　　此时，她的心中有很多问题，慕清霄收起羽翼，眉宇间的黑色晶体消散，在她玉额上轻吻一下，温柔道：“等离开万蝠古窟，你想知道什么，我都会告诉你。”
<br/>　　“嗯，我都听你的。”
<br/>　　飘然落地，慕清霄来到黑水玄蛇身旁，黑水玄蛇是上古异种，经历万年体内绝对有妖丹，还有蛇胆，绝对是大补品啊！
<br/>　　不出所料，片刻后，慕清霄就找到了一颗头颅大小的绿色蛇胆，其中包含着极其浓郁的力量，还有一颗拳头大小的黑色妖丹。
<br/>　　眼中闪过一丝喜悦，仅仅是黑水玄蛇的蛇胆，估计就能让自己修为提升一些，就是不知道妖丹能不能炼化。
<br/>　　拿出储存空间中的容器，慕清霄将蛇胆与妖丹全部封印了进去，指尖九霄琉璃焱跳动，弹射在黑水玄蛇上，百丈身躯以肉眼可见的速度化作灰烬。
<br/>　　不到一分钟的时间，黑水玄蛇的尸体就消失在视线中，慕清霄笑道：“滴血洞的入口就在附近，我们四处找找。”
