<br/>?    回到燕山，陈明一如既往，新来的记名弟子不少，是时候继续讲道了。

<br/>        陈明再次开始讲道，他并没有讲太深刻的，只是将闻道境的修行说了个通透。

<br/>        后来的弟子，有些不满，但是想到陈明待会儿就到了，现在这个关头，也没人犯傻去抢前方的位置。

<br/>        天色刚明，银翼从床上爬起来，穿上一身骚包的银甲，睡眼朦胧的银翼坐在自己的高台上，晃眼一看，身前的广场上，三千六百个蒲团已经坐满了记名弟子。

<br/>        嗯，今天山主要讲道吗？

<br/>        不说别的，这个阵势，就很爽了！

<br/>    
<br/>        “所谓闻道境，便是刚开始接触道的一个境界，只是接触，并不深入，因为这个时候的你们，还没有资格去接触道......”
<br/>    
<br/>        “想要接触道，首先要熬得住大道临身，所谓修行，天地万物为海，肉身为舟，灵魂为桨，一舟渡海，千舟竞渡，沉舟侧畔千帆过，一步便是朗朗乾坤.......”
<br/>    
<br/>        “肉身为万事之先，闻道境有十脉，便是十层，开脉便可壮大气血，气血环绕周身，气血强盛，则身躯强盛.......”
<br/>    
<br/>        陈明侃侃而谈，将自己看到的闻道境教给这些刚刚接触到道的记名弟子，银翼刚开始听着便已经入神，原来闻道境，竟然这么重要，而且其中蕴含着这么多的秘密！

<br/>        要说这世间谁最了解闻道境，除了陈明还有谁人，闻道境，他可是修行了十遍了，而且九九玄功，更是只修闻道境的极致功法，他闻道境可是已经修行到了三十六层，谁能比他更清楚闻道境？
