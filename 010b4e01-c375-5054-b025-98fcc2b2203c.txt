　　清木往前走了几步，朝慕寒摊开一只手，道：“请吧。”
<br/>　　慕寒朝他身后看了看，道：“你刚说的跟你实力差不多的几位，是你身后这几人吗？”
<br/>　　“差不多，怎么，慕寒师弟不敢跟我斗法么？”清木语气之中带了点轻蔑，随即朝身后招了招手，把几个嫡传弟子都叫到自己身后，冲慕寒冷笑道：“我这几位师兄弟，实力都差不多，不知道慕寒师弟想挑战哪一位，还是要我找个入门最晚、年纪最小的？”
<br/>　　慕寒的目光从他们脸上一扫而过，道：“一起吧。”
<br/>　　大伙怔了一下，清木道：“我没太听懂啊，慕寒师弟莫不是自信到轮流挑战我们？”
<br/>　　“不。”
<br/>　　慕寒静静地看着他，一字一顿：“我是说，你们一起上。”
<br/>　　一起上！
<br/>　　在场众弟子面面相觑，甚至觉得好笑。
<br/>　　清木怔了半晌，哑然失笑：“慕寒师弟，可不要自信过头，说了大话，就算是茅山叶少阳，也不敢如此狂妄吧！”
<br/>　　“叶少阳又怎样？”
<br/>　　慕寒说完，朝他走过去，单手结印，慢慢地朝他拍过去。
<br/>　　清木明白，这是故意给自己准备的时间，免得被人觉得占了自己的便宜。
<br/>　　在他看来，这简直就是轻蔑的侮辱，当下从后腰拔下拂尘，凌空舞动，结了一个不定式的法印，迎着推了上去。
<br/>　　慕寒的右手，贴在了法印上，接触的一瞬间，清木立刻将体内罡气尽数喷吐，希望能杀慕寒一个下马威，结果……罡气刚过度到法印上，突然就泥牛入海一般，完全失去了感应，清木大惊之下，反应也是很快，立刻就要重新组织进攻，却震惊地发现，自己的手缩不回来了。
<br/>　　怎么会这样！
<br/>　　清木浑身战栗着，抬头朝慕寒看去，慕寒脸上绽开了一抹微笑，手臂微微一抖。
<br/>　　一股恐怖的反噬之力，在他掌心爆发，瞬间传遍全身，他人像炮弹一样弹射出去，砸在人群中。
<br/>　　慕寒却不再看他，一个纵身，朝边上那几个目瞪口呆的道士冲过去，双手作法，在手中凝聚起了阴阳二气，急速蔓延，将这六七个人全包裹在中间，这些人到这才反应过来，他们也不是普通法师，立刻布阵站位，一起作法，在极短的时间内，凝聚众人之力，布置出了一道强大的结界，抵挡住阴阳二气的冲击。
<br/>　　“滋滋……”
<br/>　　阴阳二气与结界撞击之后，仿佛一股水源，沿着结界的周围缓缓蔓延开来，不一会便布满了整个表面，然后缓缓渗透进去。
<br/>　　这股力量初始让人感觉绵软，但却势不可挡，七个人使出了全力，依然无法抵挡阴阳二气的渗透，一点点侵蚀着结界。
<br/>　　七人苦苦支撑，也不过十来秒钟的工夫，结界完全被融化，阴阳二气瞬间弥漫，一层层地压下来，七人面色惨白，支撑了几秒钟，相继都瘫在了地上。
<br/>　　阴阳二气继续压下，让他们越发感觉到窒息，一股绝望之情在心头蔓延开来。就在这时，慕寒挥挥手，驱散了阴阳二气。
<br/>　　七个人，连同最早被击败的清木，八个人一起坐在地上，傻傻地望着慕寒，其余人也都傻傻地看着他，连同那几位德高望重的执事，一个个都惊得说不出话来。
<br/>　　他们不敢相信，慕寒真的实现了之前的承诺，以一对八，还赢的那么轻松写意……
<br/>　　清木是天师，那七个人也都是准天师的实力，想要弹指之间击败他们……众人在心头估算了一下，那至少得是地仙高阶，甚至更高……
<br/>　　一个看上去都不到二十五岁的年轻人啊。
<br/>　　这样的天才，别说是龙虎山，就算在整个法术界，恐怕也是百年难遇的吧？
<br/>　　他们总算有点明白，为什么张无生要一反常规地将掌教之位传给他了。
<br/>　　慕寒背着双手，走到台阶的最高处，对龙阳真人伸出手，龙阳真人愣了一下，将掌门信物九天元辰尺交给他。
<br/>　　慕寒一只手拿着，环视众人，朗声说道：“我知道你们很好奇我的来历，为什么我就住在龙虎山后山，但你们从来没见过我，因为我从来没参加过法术界任何活动，没跟人交过手，法术界没人知道我的名字。
<br/>　　龙虎山千百年来，掌教之位一直都传给张家后人，这个规则到我这里，仍然没变——我，就是张家后人，张无生，是我的亲叔叔。”
<br/>　　此言一出，众人哗然，连龙阳真人和曲波也震惊不已，这一点，他们找到慕寒的这几天，慕寒可是提都没提过啊。
<br/>　　慕寒抬起一只手，轻轻作法，掌心金光泄露，凝聚成了一道灵符的形状，在空中摇摆着，不断吸收着周围空气中的灵力。
<br/>　　“天师真符！！”
<br/>　　有人叫起来。
<br/>　　“不错，这是我张家嫡传天师真符，这个，能证明我的身份了吧。”
<br/>　　众人默然。
<br/>　　天师真符，是当年道祖张道陵所创的一种不定式神符，算是张家绝学，从来都是只穿张家后人，而且一代只传一人，传男不传女，传长不传幼。
<br/>　　“我隐修多年，原因……连我也不甚清楚，我也不知道自己居然有一天会当上龙虎山掌教，但我既然是张家后人，那便是当仁不让。我龙虎山两代真人都死在法术公会的手上，这个仇，我一定要报！
<br/>　　茅山叶少阳号称天选之子、一路狂杀，成为人间道神，如今法术界只认茅山，却不知我龙虎山，今后，这些要改改了。
<br/>　　我听说当日叶少阳大闹龙虎山，诸位居然无一人能够抵挡，这不怪你们，但是……这对我龙虎山来说，却是天大的耻辱！我发誓，从今往后，不管任何人，任何邪物，绝不能在我未经允许之下踏入山门一步！”
<br/>　　这一席掷地有声的话语，敲打在众人的心头，燃起了他们心中的热血，
<br/>　　以慕寒方才展示出的实力，没人觉得他说这番话是在吹牛。
<br/>　　张家后人，又是罕见的修道天才，还有决心重整旗鼓、为前两代天师报仇雪恨，那还有什么好说的呢？
