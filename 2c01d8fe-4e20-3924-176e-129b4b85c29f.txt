　　密室之中，苏莫双眸紧闭，心中杂念尽去，在努力的修炼。
<br/>　　他的体内，混沌源丹之中，用处浩荡的混沌源气，苏莫不断的吸收炼化。
<br/>　　修为到了他这个地步，即便资源充足，也不可能快速提升了，只能靠时间来打磨。
<br/>　　当然，他的修炼速度，远不是一般的同阶武者所能比拟，比别人快了不知道多少倍。
<br/>　　外界日落日升，年复一年，苏莫毫不理会。
<br/>　　很快，四年便过去了，这一日，苏莫的身上，气息猛然一震，好似突破了某种桎梏，进入了新的层次。
<br/>　　呼！
<br/>　　苏莫睁开眼眸，张口突出一口浊气，浊气如剑，刺在了密室的墙壁上。
<br/>　　“二品低级！”苏莫眼眸闪亮，经过四年的闭关，他终于再进一步，达到了二品的修为。
<br/>　　虽然有混沌源丹，但修炼的速度，依旧是非常的慢。
<br/>　　而且，以后也会越来越慢，这不仅是因为修为的跨度，境界的难以逾越，也是因为天赋的桎梏。
<br/>　　普通的一品高级武者，想要晋升到二品，动辄需要数十年、数百年、甚至数千年以上。
<br/>　　“以后，还是要吞些血脉和元神！”苏莫低声自语，再吞噬一些血脉和元神，他的潜力和实力才能更强。
<br/>　　当然，一般的血脉和元神，对于他而言，已经没有了多大的用处。
<br/>　　必须是顶级的血脉和元神，才能被他所用。
<br/>　　少倾，苏莫叹了口气，便开始继续修炼了起来。
<br/>　　混沌源丹之中的源气，还有很多很多，足够他再修炼十几年了。
<br/>　　苏莫继续炼化混沌源丹之中的源力，物我两忘，并且一边修炼，一边参悟吞噬法则。
<br/>　　他的吞噬法则，已经达到了一阶后期，要尽快达到一阶圆满。
<br/>　　法则的参悟，他也不会落下，因为，如果法则之力太弱，修为的竞争非常缓慢，甚至是驻足不前。
<br/>　　总之，法则的强弱，也会影响到武道的进展。
<br/>　　现在对于苏莫来说，资源充足，时间也还算充裕，他准备利用这个时间，创造出自己的绝学。
<br/>　　以剑为基，以吞噬之力为辅，创造出属于他的绝技。
<br/>　　时间流逝，苏莫心无杂念，一心修炼。
<br/>　　就这样，又过去了三年，苏莫再次睁开了眼眸，结束了修炼。
<br/>　　他修为，依旧是二品低级，但是距离二品中级，已经非常非常的近了。
<br/>　　不过，他的吞噬法则，则是达到了一阶圆满。
<br/>　　“太慢了！”
<br/>　　苏莫叹了口气，虽然他的修炼速度，相比于别人而言非常的快，但是，对于他而言，却是很慢很慢。
<br/>　　这种速度，随着修为越来越高，还会越来越慢，修炼到神王之境，怕是需要上千年都不止。
<br/>　　这种速度，他接受不了，因为，他不能冒险。
<br/>　　首先，帝一魂和拓跋青云两人，不知道什么时候就能成就高级神王。
<br/>　　再者，神域更换大阵不知道在什么时候，有可能近在眼前了。
<br/>　　到时候，以帝一魂的实力，可以轻易下界。
<br/>　　若是帝一魂对他的亲人朋友动手，那就彻底的糟糕了。
<br/>　　所以，他必须尽快将实力提升上来，提升到能对抗帝一魂的程度。
<br/>　　想要快速提升实力，唯有再去时空界域了。
<br/>　　到时候，借助时空之城的时间流速，他才能大大的节省时间。
<br/>　　“这么多年了，不知道龙腾和司空炎他们，有没有来到神域？”
<br/>　　苏莫低声自喃，以司空炎等人的天赋，如今修为应该都能达到虚神境四变了吧！
<br/>　　以司空炎等人的战力，不知道能不能强行破开界障，提前进入神域。
<br/>　　这一点他不确定，毕竟，界障太过强大，司空炎等人虽然战力很强，但是远远不到逆天的地步。
<br/>　　手臂一挥，一具透明的棺椁，出现在了苏莫的身前。
<br/>　　棺椁之中，躺着一具绝美的身影，安静而祥和，充满了凄美之感。
<br/>　　望着棺椁之中的芊寻月，苏莫面色沉重，这么多年了，也只有这个女人，一直陪伴着他。
<br/>　　说来也是幸运，对方可以说已经死了，所有才能被他带来神域。
<br/>　　“希望人德宫主，能将你复活！”苏莫面上充满了期望，他的心绪，不由得回到了当年。
<br/>　　当年在苍穹世界，和这个女子发生的一幕幕，都在脑海之中浮现。
<br/>　　少倾，苏莫不再多想，收起棺椁，起身便离开了密室。
<br/>　　走出宫殿之后，他来到了宫主的宫殿之前。
<br/>　　“宫主！”
<br/>　　苏莫面对殿门，抱拳轻喊一声，他不知道宫主在不在殿中，但他离开之前，必须见一次宫主。
<br/>　　“进来！”
<br/>　　少倾之后，宫殿的大门自行打开了，其内传出人德宫主的声音。
<br/>　　大殿之中，人德神王坐于正厅，面容肃穆，凝望着殿外的苏莫。
<br/>　　“拜见宫主！”
<br/>　　走进殿中，苏莫向对方抱拳行礼。
<br/>　　“苏莫，你的修为进展不错！”人德宫主打量了苏莫一番，点头说道。
<br/>　　苏莫闻言一怔，因为对方不是称呼他段惊天，而是本名。
<br/>　　“宫主，帝一魂来过了？”苏莫沉声问道。
<br/>　　“帝一魂并未到来，天尘宇来过两次！”人德宫主道，这几年，天尘宇共来了三次。
<br/>　　第一次倒没有说什么，第二次来，企图说服他，并给于巨大的报酬，甚至是帝一魂的许诺。
<br/>　　当然，人德宫主自然没有动摇，不然苏莫也不会站在这里了。
<br/>　　两次无果之后，天尘宇来了第三次，给予了人王宫郑重的警告。
<br/>　　对此，人德宫主毫不在意，依旧是以苏莫已经离开为由。
<br/>　　反正，至始至终，他的态度就是苏莫已经离开，查无可查了。
<br/>　　为此，人王宫还故意派出人手，出去搜寻苏莫的下落，结果自然是无功而返。
<br/>　　“天尘宇？”苏莫闻言，略一思量，他听说过此人，好像是天王宫的无上神王。
<br/>　　“这些你都无需过问了，你的安全无忧！”人德宫主道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，既然对方说无忧，那么他也不用过多询问，随即，苏莫又道：“宫主，我想要离开人王宫一段时间！”
<br/>　　“离开？”
<br/>　　人德宫主闻言，略一沉吟，道：“你若是走出人王宫，会有一定的风险。”
<br/>　　人王宫之外，不知道有多少帝一魂的探子，苏莫一旦离开，就有风险存在。
<br/>　　“宫主，离开人王宫，我才能成长得更快，当然，前提是要有足够的资源。”苏莫道。
<br/>　　人德宫主闻言，沉默不语，他自然希望苏莫快速成长，但是，却不能让风险存在。
<br/>　　“既然你要离开，那老夫为你安排，至于资源，你要多少？”人德宫主道。
<br/>　　“五行源石，各十亿！”苏莫沉声道，他担心人王宫没有多少混沌源石，所以退而求其次，索要五行源石。
<br/>　　“十亿？下品源石？”人德宫主反问道。
<br/>　　十亿五行属性的下品源石，倒是不算多，但若是中品，那这个数目可是不小，虽然人王宫家大业大，但资源也不少天上掉下来的。
<br/>　　况且，是五行属性源石中每一种属性都十亿，那加起来就是五十亿。
<br/>　　“中品源石！”苏莫沉声道，本来他想要的的确是下品源石，但是想到离开人王宫之后，短时间可能不会回来，所以便改成了中品。
<br/>　　“中品的话就太多了，这样吧！老夫给你三亿，不能再多了，你明日来取，但你以后需要，可以再来索取。”人德宫主沉吟着说道。
<br/>　　苏莫闻言，心中暗惊，虽然对方没有通知十亿，但三亿也很惊人了，毕竟是中品源石。
<br/>　　 “多谢宫主！”苏莫立刻向宫主道谢行礼，有了这笔资源，他数十年，甚至上百年都不用愁资源了。
<br/>　　“嗯，你先回去吧！明日给你资源，再安排你离开！”人德宫主挥了挥手道。
<br/>　　虽然他给予了苏莫天幻神丹，但是，为了安全起见，还是要安排一翻。
<br/>　　“宫主，晚辈还有一事相求！”苏莫再次抱拳，随即他一挥手，芊寻月的棺椁出现在了身前。
