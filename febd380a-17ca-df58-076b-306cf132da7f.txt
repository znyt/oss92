　　第462章
<br/>　　“不一样就不一样么，你结巴什么？”程大雷问：“哪里不一样？”
<br/>　　『唯暴制暴，唯恶止恶，暴恶会影响你从点将台获得的英雄』
<br/>　　“也就是说我以后只能招募到反派？”经过和珅的事，程大雷已经对这件事有所准备。
<br/>　　『可以这么理解，但也不仅仅如此』
<br/>　　“还有什么？”
<br/>　　『每当你通过点将台或其他手段招募到一位英雄，同时，其他势力便会随即获得一名同等级英雄』
<br/>　　“呃……”程大雷陷入深深的静默中。
<br/>　　『……』
<br/>　　“喂，你与我好好解释一下，这是什么意思？”
<br/>　　『——』
<br/>　　“喂！你给我出来。”
<br/>　　『——』
<br/>　　……
<br/>　　无论程大雷怎么上窜下跳，大呼小叫，系统都不再给出任何回应。仿佛，它也是自觉理亏，不敢再见程大雷。
<br/>　　程大雷立在哪里，陷入深深的崩溃中。简单来说，自己获得暴恶属性后，战场厮杀能力的确获得提升。可同时，自己以后再召唤英雄，就只能召唤到一些奸臣，贪官，恶贼，于此同时，其他势力就能获得如岳飞、马超这样的绝世武将。
<br/>　　这种隐藏属性，有还不如没有呐。程大雷之前就有预感，蛤蟆寨将成为一个反派大本营，现在果然预感成真了。更糟糕的是，自己还会创造一个专门为消灭自己而存在的正义联盟。
<br/>　　正在休整的起义军看到程大雷在哪里大呼小叫，咧嘴跺脚。众人一头雾水，都不知发生了怎样的事。心中感慨：大当家该不会压力太大，患了失心疯吧。
<br/>　　嘟，获得抽奖次数1，是否现在抽取？
<br/>　　正在程大雷捶胸愤恨，咬牙切齿之时，忽然收到一条系统提示。瞧瞧，这明显是系统做贼心虚，故意补偿讨好自己，否则无端端怎么会获得一次抽奖次数。
<br/>　　“抽奖。”
<br/>　　程大雷恨恨不休，告诉你，别来这一套，完全没用，抽不到一台豹式坦克，武装直升机之类的，我绝对不会原谅你。
<br/>　　扭蛋机在程大雷脑海中出现，骨碌碌旋转。程大雷满怀期待，忽然心中一动，想起一件不好的事。
<br/>　　万一自己抽到一员武将，那么岂不是说别人也可以获得一员武将。系统不是在补偿自己，该不会是故意恶心自己吧。
<br/>　　程大雷想想系统以前的尿性，忽然觉得，这很有可能呐。
<br/>　　“取消，取消……”
<br/>　　程大雷口中说着，可抽奖过程是不可逆的，程大雷只能眼睁睁看着一物从扭蛋机中调出来，随之收到一条系统提示。
<br/>　　嘟，恭喜获得顶级武将单雄信。
<br/>　　单雄信，绰号赤发灵官，用一杆金顶枣阳槊，大隋九省绿林匪道总瓢把子，既然是贼头，自然属于恶贼一类。
<br/>　　程大雷丝毫都不关心这一点，现在他更关心，是哪位心怀正义，立志要杀自己的忠义之士来到这个世界呢。
<br/>　　嘟，顶级武将薛丁山出世。
<br/>　　程大雷整张脸已经麻木了。薛丁山，唐征西元帅，使一杆乱雪银戟，战场之上有万夫不敌之勇。系统将他评为顶级，程大雷觉得评级低了，这薛丁山的潜质至少该是传说级的。
<br/>　　这还不是最关键的，最关键的一点是，这薛丁山似乎和樊梨花有一腿，虽然樊梨花现在已被自己收入房中，半程截了胡。可人家两个明显是官配，万一有朝一日碰见，天雷勾地火，一发不可收拾，自己该如何是好。
<br/>　　程大雷咬牙切齿，心中发誓，一旦碰见这薛丁山，二话不说，先杀了再说。将罪恶的小树苗，扼杀在摇篮之中。
<br/>　　正在此时，一匹快马奔来，马上一员大将，穿布衣，提大槊，赤发蓝脸。一见他出现，正休整的义字军立刻摆出战斗阵势，随时准备进攻。
<br/>　　“我要找程当家，我要找程当家……”赤发大汉吼道。
<br/>　　程大雷分开众人走了出来，口中道：“敢问是单通单兄弟么？”
<br/>　　“呃……程当家认得我？”赤发大汉一楞。
<br/>　　“耳朵里都灌满了。”程大雷挥挥手，散去左右：“单兄弟如何来了？”
<br/>　　“单某在京州落草，也早已听闻程当家义薄云天的名声，近日知道程当家来了并州，所以特来投靠。”
<br/>　　“好说，好说。”
<br/>　　程大雷如今也算是见过世面的人，一个单雄信还不值得他大惊小怪，毕竟，刘关张都在山寨内。
<br/>　　“大当家，下一步准备往何处去？”单雄信问。
<br/>　　程大雷也正寻思这件事。如今三百义字军已经休整完毕，可往何处去，程大雷还未想好。
<br/>　　追兵随时会到，必须想办法自保。而同时并州草原上，自然布满要杀自己的人，凭手下这三百人根本也打不了仗。
<br/>　　还有一件令程大雷揪心的事情，就是徐神机的下落。不知他是生是死，假若还活着，现在又在哪里？是被人抓走了，还是在某地窝着。
<br/>　　希望徐神机的机灵劲能发挥点用场，千万别落到帝国军手中。
<br/>　　先找个安身之处，接着，再寻徐神机的下落吧。
<br/>　　程大雷目光转了一圈，忽然问：“白平，准备得怎样了？”
<br/>　　“禀报大当家，我们已休整完毕，随时可以出发。”白平大声道，果然吃饱饭以后，说话的嗓门也大了很多。
<br/>　　“走，我们去——！”程大雷手指向一个方向：“青牛山。”
<br/>　　青牛山是程大雷起家的地方，对那里的地形熟悉，只要进入青牛山范围，程大雷有底气和帝国军打游击。打来打去，帝国军未必会玩了命的杀自己，之后，再从长计议。
<br/>　　队伍开始向青牛山进发，沿途防范着随时有可能出现的帝国军。三百人的队伍，在草原上不是一个小目标，一旦被发现，免不了就是一场大战，程大雷不能放松警惕。
<br/>　　骑在黑牛背上，程大雷还在寻思自己的隐藏属性。按道理来说，这吕布吕奉先应当也是反派角色吧，杀父弑主，见色忘义，名声也不是很好。如果能把这恶将召唤出来，这暴恶属性也不能说一点用处没有。
<br/>　　薛丁山……想到这个名字，程大雷就感觉头皮一阵阵发麻。这混蛋，现在在哪里呢？
<br/>(本章完)
