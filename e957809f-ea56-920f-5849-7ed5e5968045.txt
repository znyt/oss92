　　苏莫和秦不死站在古殿之外，前面是宽广的街道，他们目前在一座庞大的城池之中。
<br/>　　星空传送阵，传送到古周星的目的地，便是他们身后的古殿。
<br/>　　“苏莫，我想我们是时候分开了！”秦不死望着前方一座座拔天而起的雄伟建筑，轻声说道。
<br/>　　“你准备去哪里？”苏莫问道。
<br/>　　“呵呵！这古周星可是古灵星河主星之一，奇大无比，自有我容身之地！”秦不死笑道。
<br/>　　“以后若是有事，可以去神武学府找我！”苏莫点头说道，在这赶路的几个月中，他也打听了一些神武学府的消息。
<br/>　　这神武学府，是古灵星河最古老的学府之一，乃是数百万年前，由十数个超级大势力联合组建而成。
<br/>　　其中的导师，也有很多都是这十几个大势力之人。
<br/>　　而这座学府中，汇聚了大量古灵星河中的人族天才，天骄如云，妖孽如雨，名震整个古灵星河。
<br/>　　这是古灵星河很多年轻武者向往的圣地，每年都有无数的天才加入神武学府。
<br/>　　当然，神武学府也不是无偿培养人才，那些从神武学府修炼有成的武者，离开学府之后，必须加入这十几个大势力之一。
<br/>　　如若不然，杀无赦！
<br/>　　所以，这十几个大势力，才能因此而长盛不衰。
<br/>　　试问，不断的有修炼有成的天才武者加入，这些势力想衰弱都难！
<br/>　　而柴管事所在的苍云宗，正是这十几个势力之一，由此可见，柴管事所在宗门之强大。
<br/>　　了解清楚了情况之后，苏莫也是对这个神武学府，充满了向往。
<br/>　　所以，他已经打定了注意，前往神武学府。
<br/>　　“我会的！”
<br/>　　秦不死点头，道：“你去追求你的武道，而我也要去追求我的道！”
<br/>　　“希望以后还能有机会再见面！”苏莫叹了口气，他本来是想让秦不死加入苍穹神宫的，但对方的心很大，肯定不会留在苍穹神宫。
<br/>　　“一定会的！”
<br/>　　秦不死面色坚定，目光灼灼的望着苏莫，道：“希望下次再见面之时，你已经成为名震古周星的强者！”
<br/>　　“希望吧！”苏莫点了点头。
<br/>　　秦不死和苏莫聊了很久，最后，大踏步而去，很快便消失在了苏莫的视线中。
<br/>　　若干年后，当两人再次见面之时，早已物是人非！
<br/>　　……
<br/>　　苏莫漫步在雄伟壮阔的城池之中，随便询问了一下，便打听清楚了神武学府的方向。
<br/>　　而后，他也没有耽搁时间，便身形冲天而起，向神武学府的方向飞去。
<br/>　　飞驰在天穹之上，苏莫一直在思索，如何安排苍穹神宫。
<br/>　　苍穹神宫这么多人，都带进神武学府肯定不现实，必须找个地方安顿下来。
<br/>　　可是，这里的城池，每一个都巨大无比，强者无数，若是将苍穹神宫留下来，他实在是不放心。
<br/>　　毕竟，苍穹神宫人数太多，一旦得罪了某个强者，或者是强大的势力，都有可能遭到灭门之祸。
<br/>　　思来想去，他决定找个小地方安排苍穹神宫的人。
<br/>　　随即，苏莫放慢了飞行的速度，一边飞行，一边查看。
<br/>　　就这样，苏莫飞行了两日之后，终于找到了合适的地方。
<br/>　　他的前方大地之上，坐落着一个小镇。
<br/>　　此镇不大，方圆两百余里，三面环山，位置非常的偏僻。
<br/>　　虽然这只是一座小镇，但也比苏莫的家乡林阳城要大上几倍。
<br/>　　苏莫灵识扫荡而出，发现此镇人口也有百万以上，镇中只有一名武帝境的武者，还是很适合安顿苍穹神宫的。
<br/>　　最起码，这镇上的武者，无法威胁到苍穹神宫的安全。
<br/>　　唰！
<br/>　　苏莫身形如电，直接飞到了镇上。
<br/>　　走在镇中的道路上，苏莫仔细的关察了一番，此镇虽然不大，但武道水平是真的高。
<br/>　　武王境武者奇多，武皇境武者也是比比皆是，一个小镇，便足以碾压整个苍穹世界了。
<br/>　　在镇中逛了片刻，苏莫来到了小镇的东部，这里有一片很大的区域，可以作为苍穹神宫暂时的山门。
<br/>　　他本想直接抢夺别人的府邸，但之后还是放弃了这个打算。
<br/>　　其一，直接抢夺别人的府邸，实在是太嚣张了，不是他的风格。
<br/>　　其二，这镇上也没有哪家的府邸，能容纳得了整个苍穹神宫。
<br/>　　所以，他只能让苍穹神宫之人，自己建造府邸了。
<br/>　　随即，苏莫意念一动，空间一阵波动，大量的身影出现在了他的四周。
<br/>　　出现的人，足有一千余人，均是苍穹神宫的高层，包括九宫之主。
<br/>　　“苏莫！”
<br/>　　“宫主！”
<br/>　　“宫主，这是哪里？”
<br/>　　众人乍一出来，便立刻吵杂声四起，个个好奇的打量着四周。
<br/>　　“好浓郁的灵气！”众人深吸一口气，便纷纷震惊了，此地的灵气之浓郁，简直不是苍穹世界可以比拟，相差十几倍都不止。
<br/>　　“诸位，这里是古周星，接下来苍穹神宫的山门，便建在此镇上。”苏莫环视众人朗声说道。
<br/>　　“苏莫哥哥，你逃过巫族尊者的追杀了？”夕儿开口问道
<br/>　　“对啊！宫主，你现在安全了吗？”
<br/>　　“巫族尊者，还会不会追来？”
<br/>　　其他人亦是纷纷开口询问，毕竟，之前苏莫可是告诉众人，他正在遭受巫族尊者的追杀。
<br/>　　“大家放心吧！我们已经彻底安全了！”苏莫笑着说道。
<br/>　　众人闻言大喜，终于彻底逃离了苍穹世界了。
<br/>　　“诸位，我让你们出来，就是商议重建宗门之事！”
<br/>　　苏莫看着前方一片空旷的地域，朗声道：“山门便建在此地，你们亲自出手建造，多久可以建成？”
<br/>　　“最少也要十日吧！”
<br/>　　“估计十日左右！”
<br/>　　“无需十日时间，七、八日足矣！”
<br/>　　众人纷纷开口道，他们都是强者，建造山门对于他们来说，实在是小菜一碟。
<br/>　　“好吧！一切从简，给你们三天时间，建成山门！”苏莫点了点头，吩咐道，将时间直接缩短在了三日。
<br/>　　随即，他又和众人聊了片刻，详细的交代了一番，众人便各自行动了起来。
<br/>　　附近有山，山上有木，建造简单的山门轻而易举。
<br/>　　一名名苍穹神宫的强者，一挥手便能卷起数百颗大树，一刀劈出，山峰便被斩断，巨型的山石直接就搬了过来。
<br/>　　没过多久，山门外围的院墙，便建造完毕，全部是用山石建成。
<br/>　　而且，这山石建造的院墙，平滑无比，往往每一面墙都是一个整体。
<br/>　　这是用利刃斩击而出，直接从山峰上开垦出的院墙。
<br/>　　在一众苍穹神宫的强者忙碌之时，苏莫却是悠闲无比，牵着夕儿的手，坐在一块大石上闲聊。
<br/>　　苏莫没有让夕儿去忙碌，而是在给夕儿介绍这古灵星河的见闻，每每都让夕儿惊奇不已。
<br/>　　时间流逝，三个时辰之后，山门便建成了简易的轮廓。
<br/>　　苍穹神宫千名强者，如此大张旗鼓的建造山门，早已引起了全镇武者的轰动。
<br/>　　远处，大量的武者在围观，议论纷纷。
<br/>　　“哪来的这么多武者？”
<br/>　　“这些人全是高手，准帝都有不少！”
<br/>　　“他们是要在此地新建府邸啊！”
<br/>　　嘈杂的议论声此起彼伏，足足数十万人，伫立在不远处围观。
<br/>　　“都让开！”
<br/>　　就在此时，一声厉喝猛然响起，只见围观的人群被分开，有一大群不速之客，向苍穹神宫正在建造的山门走了过来。
