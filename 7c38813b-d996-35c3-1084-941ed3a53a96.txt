　　元始自从上次决定走出昆仑后，便一直在这天地之间游走，他也没有什么目标，就是随着自己的心意，漫无目的地游走，随遇而安。
<br/>　　转瞬间，便已经过了百年光阴。
<br/>　　百年时间，说长不长，说短不短，对于元始来说，这样的时间只是眨眼般的时间。这些年他游历四方，到过人族大部落，也到过妖族领地，去了海底深处，也入过深山密林。有时候走走停停，日子过得很潇洒。
<br/>　　这段时间，虽然没能让他修为更进一步，但却也让他那一直紧绷的心情舒畅了下来，这也算是一个意外之喜吧。
<br/>　　这天，他躺在北方的一处山林之中的大青石上假寐，腰间木剑横放在头顶，酒壶乱放，颇有酒鬼醉酒的模样。
<br/>　　而与此同时，远处山林之间，走进一个少年。少年眉清目秀，一声青衣，大约十五六岁的样子，脸上虽然青涩，眼中却透露着一种坚定的目光，而且他身上也有一种出尘的气质。
<br/>　　“我一定要寻仙问道，求得长生！”
<br/>　　少年语气之中带着坚定不移的信念，一步步地向着深山走出，完全不顾深山之中的危险，或者说，他并不惧怕那些危险，在他心中，就算再大的劫难，也挡不住他求仙问道的决心。
<br/>　　少年手持一把锈剑，在深林之中游走，希望能够找到传说中的仙师。
<br/>　　这少年乃是北方人族小国净乐国王子，他从小便显露出聪颖的天资，十岁时，便可读各种书籍而且过目不忘。国王和王后视其为掌上明珠，然而，他听说了那些仙人的故事后，便不由心生向往。因为他们国家经常受周边妖魔祸害，他发誓一定要学习道术，扫尽妖魔。
<br/>　　一年前，净乐国举行上元佳节，太玄玩心大盛，在侍从的陪伴下出宫玩赏花灯。上元之夜，街道热闹非凡，形形色色的人出现在他视野之中，各种人将时间百态展现得淋漓尽致。尤其是那些官宦富贵之人，其纵情声色的丑态，让他顿觉红尘万丈之感。然而，世间众人却仍然怡然自得，不知其醉。他目睹这一切之后，便生出了出世之心。
<br/>　　之后，闹市上出现了妖魔，妖魔那强大的力量，让人提不起反抗之心的绝望，看着他的国家子民的死去，更是让他对世间道术升起强烈无比的渴望之情。
<br/>　　于是，他便毫不犹豫地踏上了求仙问道之途。
<br/>　　几日前，他听说此山之中有仙人踪迹，于是，便立刻步入这片深山，希望能够如愿以偿，求得真仙。
<br/>　　一路上，他见过老虎，但还好有点武力，不至于被老虎吃掉，之后，他再次遇到了野豹、毒蛇、食人花、嗜血藤等要命之物，还好他运气极佳，没有将命送掉。但一路被野兽追杀，也让他狼狈不堪了。
<br/>　　几番周折之后，他满身伤痕的走进深山，看到了睡在大青石上的元始。他顿觉得奇怪，这里怎么会有人呢？而且还在此处睡觉，却没有被妖兽吃掉，真是奇怪。
<br/>　　不过，他也没有在意，而是靠近这青石，靠在这石头边休息，他被妖兽追杀了一天，已经筋疲力尽了，让他奇怪的是，一靠近青石，他那烦躁的心情瞬间便安宁下来了。不知为何，他也慢慢睡着了。
<br/>　　不知过了多久，少年从懵懵懂懂中醒来，他一睁开眼睛，便看到了青石上的元始。在他的视线中，那个奇怪的人手中拿着一柄木剑，正在缓缓舞剑。
<br/>　　那剑法缓慢至极，而且毫无章法，像是小孩子在随意挥剑，而且小孩都做得比他好。实在是这人的剑太慢了，能不能抵挡住小孩的剑都难说。
<br/>　　但就是这样随意舞动的慢剑，看起来却蕴含着一种莫名的玄妙力量，能够让他的心静下来。他的目光随着那人的剑在动，渐渐地，他仿佛明白了什么，又仿佛什么都不明白。
<br/>　　一个时辰后，元始慢慢停下手中的木剑，走了过来，看着眼前这个正在闭目深思的青年。他一样便看出来，这个少年正在悟道。
<br/>　　“哦，还有些资质。”
<br/>　　元始心中惊讶道。能够以没有修炼过任何道法的身体，从他的舞剑之中悟出点道理，真是难得，用元始的评价就是，有些资质。
<br/>　　“小家伙，你看到了什么？”良久，待少年醒了过来，元始开口问道。
<br/>　　“我看到了圆，又看到了点。”少年不假思索便回答道。
<br/>　　“哦。”
<br/>　　元始这次倒是惊讶了，这少年能看到圆，自然是资质极佳，但能看到点，便不是一般有些悟性的天才能够做到的，就算是仙人，也难以做到这一步，而眼前这个少年却能看到，真是……不简单。
<br/>　　“小家伙，你叫什么？”微微沉默，元始开口问道。
<br/>　　“我名为太玄！”少年认真答道。
<br/>　　“太玄？”元始略微沉吟，不知想些什么，又突然继续道：“好名字。”
<br/>　　玄者，天玄也，地玄也，人玄也。天浑行不可见也，地不可行也，人心不可测也。故玄，深渊广大矣。又有玄者，幽搞万物而不见形者也。太初始者，玄者幽深，为至理难言也。太玄亦为北斗之机，绕北旋转，时空统一，可定星球时空，万物在时空中起灭。可以说，太玄，便意味着道。这人叫太玄，是不是意味着什么。
<br/>　　“你来此地干什么？”元始明知故问道。
<br/>　　“我来此寻访仙道，拜仙师，求得长生不老之术。你是仙人吗？”少年开口说道，接着又是满怀期待地对着元始问道，他本就聪慧，之前没有多想，如今仔细一想，这人能在深山中睡觉而不怕妖魔侵扰，不是仙人恐怕也不远了。
<br/>　　“仙人？”元始不屑一笑，接着缓缓说道：“我？或许也算是仙人吧。”
<br/>　　少年一听此话，顿时大喜，接着便问道：“你能收我为徒吗？”
<br/>　　“我？”元始摇了摇头，说道：“我已经多年没有收徒了。”
<br/>　　少年闻言，有些沮丧，但接着又听到元始的话，顿时大喜过望。
<br/>　　“不过，我却可以收你为徒。”
<br/>　　“太好了，拜见老师！”太玄大喜，连忙拜见元始，深怕他反悔似的。
<br/>　　元始摇头一笑，开口道：“为师乃是紫虚元君，既然你有如此天资，我便传你《无极上道》，无极生太极，太极逆无极，希望你能不忘初心，早日成道。”
<br/>　　说着，他伸出一根手指，点在太玄额头之上，将心法传于他。元始并没有说出自己的真名，而是另起一个名字，反正他的名字有很多，谁知道谁啊。
<br/>　　太玄深深沉迷于脑海之中的仙法道术之中，不可自拔。良久，他才醒悟过来，连忙拜谢道：“多谢老师。”
<br/>　　“不必多礼，为师会在此处停留三年，三年之内，你有任何问题，都可来想我询问。”元始道。
<br/>　　“是，老师。”太玄本就聪明，聪明之人不会多问为何元始只留三年的原因。
<br/>　　“嗯。”元始点头道。
<br/>　　其实，在太玄说出他的名字之时，元始便算到了他的未来，眼前这人便是未来鼎鼎大名的北极镇天真武玄天上帝，亦称荡魔天尊，玄武大帝。知道他是谁后，元始才收了他做徒弟，若不然，他才懒得收徒弟，最多只是看对方顺眼，顺便扔给他点仙法罢了。
<br/>　　就这样，未来的荡魔天尊，便成了元始的徒弟，开始了他的修仙之途。
