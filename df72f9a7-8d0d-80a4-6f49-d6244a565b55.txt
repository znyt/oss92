　　听到这里，大伙都震惊不已，叶少阳皱眉道：“你撒谎，纵然木子愿意这么做，他是邪灵，又没有魂魄，怎么能去轮回？”
<br/>　　通玄道人道：“说起来是不行，但我先带他去了无量界法诀寺，去听了地藏菩萨讲经，顺便求他发落，地藏菩萨便将他发落到轮回司，轮回人道……”
<br/>　　“天方夜谭！”
<br/>　　叶少阳斥道，“先不说你是怎么认识地藏菩萨的，就算找到他，他能管这小事？”
<br/>　　通玄道人笑道：“我前生张果，乃三界天师，与地藏菩萨也有往来，我也时常去法诀寺听讲，至于超度……要么怎么说是菩萨心肠，只要是一心解脱、想要轮回的，都会被他安排去轮回，来世便有佛缘。连那些地狱恶鬼，地藏菩萨都不怕自降身份，去超度他们，何况是木子这样的？”
<br/>　　叶少阳一群人互相望去，都觉得这事有点天方夜谭，不过通玄道人这么一说，倒也符合逻辑，而且轮回司本来也是佛门控制，地藏菩萨有时候去地狱传道，就算有十恶不赦的地狱饿鬼，只要是一心向佛，也会被他超度轮回，当然前提是真心向佛，并且化尽了戾气。
<br/>　　假如是真的，那么，叶少阳内心也是为木子感到高兴的，毕竟他可以投胎做一个真正的人了。只是……他是为了自己才献出肉身，而且将来再也见不到了。
<br/>　　想到这里，叶少阳内心颇为感慨，其余人也都很难过。
<br/>　　“你做了这么多坏事，地藏菩萨没收了你？”叶少阳抬起头，看着通玄道人，冷冷说道。
<br/>　　通玄道人干笑起来，“老祖虽然不是好人，但所做一切都是为了证道，并非有心为恶，我与佛门的道不同，地藏菩萨又不是喜欢管闲事的，捉我干什么。”
<br/>　　这中间的道理太过深奥，叶少阳隐约明白，但也没有仔细去想，这时瓜瓜凑过来，望着通玄道人，恶狠狠地说道：“老大，你能不能把他交给我，我要暴打他一顿，以报当日之仇……”
<br/>　　当初他曾带人抓过瓜瓜，暴打过他，瓜瓜到现在还记得。
<br/>　　通玄道人慌忙拱手求饶，叶少阳想了一下，让瓜瓜暂时别动，望着通玄道人，感慨说道：“通玄，我也是佩服你的，你这一年来可是够辛苦的，今天投奔这个，明天投奔那个，你也是能忍辱负重的，而且真能做到没脸没皮。”
<br/>　　通玄道人不以为然地拱了拱手，“过奖。”
<br/>　　“你养息地怎么样了？”
<br/>　　“魂魄稍微稳固了些，只是还不能出去，否则要魂飞魄散。”
<br/>　　“万一有人钻进阴阳镜中，我可不管你。”
<br/>　　“那无妨，你这里阴阳调和，魂魄再散，只要元神不灭，都能恢复。”说完还不忘对叶少阳拱了拱手，“多谢关心啊。”
<br/>　　“嗯嗯，应该的。”叶少阳给瓜瓜使了个眼色，“只要不灭了他的元神，随便你怎么玩。”
<br/>　　瓜瓜愣了一下，脸上绽开了一个在通玄道人看来无比邪恶的笑容，丢下一句“多谢老大！”，窜入了阴阳镜中。
<br/>　　“什么意思，叶少阳，啊……”杀猪般的惨叫声响起来。叶少阳保持微笑，贴了一张灵符在阴阳镜上，隔绝了内外气息，眼不见心不烦。
<br/>　　“老大，你现在感觉如何？”美华问道。
<br/>　　叶少阳轻轻活动了一下右胳膊，“问题不大，几天就好了吧。”随即想到什么，“那个，你们在这呆着也没啥意思，都回去吧。”
<br/>　　林三生也道：“是了，今天出了这么大的事，难保尸族不会趁势有所行动，你们先回宫，我再跟少阳说一会就过去。”
<br/>　　小青小白一干人虽然担心叶少阳，但现在他们在空界，随时都能来，于是一个个跟他道别，表示过几天再来看他，然后用通灵石开辟虚空，回到了空界。
<br/>　　“少阳，我也走了……”小九依依不舍说道。
<br/>　　“你……”叶少阳看到小九眼神瞟了一眼，顺着她的目光看到了芮冷玉，立刻明白了小九为啥立刻要走，一时间不知道怎么挽留。
<br/>　　小九善解人意地冲他笑了笑，让他保重，然后对芮冷玉道了个万福，“主母，我先告退了。”
<br/>　　叶少阳看到这一幕，心中一疼，小九身为万妖之王，除了自己，什么时候对一个人这么客气过？
<br/>　　芮冷玉上前拉住她，亲热地说道：“你这么叫我真的不敢当，小九，还是你留下照顾少阳吧，道风最近炼神返虚，正在关头上，他没人照顾，我本不该来的，但我实在担心，既然事情结了，我也该走了。”
<br/>　　说完不等小九开口，让林三生打开山河社稷图，然后拉着叶少阳的手说了几句关心的话，立刻就要走。
<br/>　　叶少阳有点莫名其妙，不过还是让她等一下，然后问林三生：“你有什么话，连他们都要背着？现在说吧。”叶少阳看着林三生，说道。方才就是他一个劲给自己使眼色，林三生犹豫着，没等他开口，一片碧绿色的叶子从窗外飞来，化成人形，正是秋莹，对叶少阳行了一礼。
<br/>　　“你去哪了，怎么才回来？”叶少阳醒过来，发现她人不在，就在打听她的下落，因为她从头到尾隐形，没人知道她在哪。叶少阳激活她的魂印，得到了她报平安的反馈，这才放心。
<br/>　　“我去追右君和女魃，然后……又去看了碧清姑娘，这才回来。”
<br/>　　“碧清！她怎么样了？”
<br/>　　叶少阳猛然想起，碧清一开始就追着苏沫跑了，后面整个战斗过程都没遇到她。
<br/>　　秋莹咬了咬嘴唇，道：“碧清姑娘吞了苏沫，我让她回来，她说她暂时不回来。”
<br/>　　叶少阳嘴巴一下子长得老大，不知道说什么好。
<br/>　　他也不知道为什么，虽然跟苏沫一直是仇人，但听说了她的死讯，心中还是有一种说不出的失落感，就好像她是自己一个多么亲近的人。
<br/>　　“东岳火精呢？被谁夺走了？”林三生问道。
<br/>　　“我……我不知道。”秋莹微微低头，沉默了几秒钟，说道：我虽然追过去，但并没能进入虚空裂缝，于是在附近徘徊许久，这才遇到了碧清姑娘……”
