　　“什么？！”
<br/>　　“怎么可能？穹天道友与乾坤道友都陨落了？！”
<br/>　　穹天老祖与乾坤老祖的陨落，鸿钧与阴阳老祖立刻就有所感应，心中不由大惊，没想到这魔祖罗睺如此厉害，这么快速地就将两人杀死，实在厉害。
<br/>　　此刻，少了穹天老祖与乾坤老祖，形式对他们就更不利了！
<br/>　　“出！”
<br/>　　当下，鸿钧老祖不敢怠慢，没有任何犹豫，连忙放出自己的善尸与恶尸前去镇压诛仙剑阵中的戮仙阵门与绝仙阵门。
<br/>　　两个身影从鸿钧老祖身体之中走出，一道身穿白衣，一道身穿黑衣，面貌与鸿钧老祖有七八分相似。刚开始出来时，两道身影还只是虚幻的，但，当两人完全走出鸿钧老祖身体之后，两道身影就完全化为实质，两人各持一面小旗，那是五行旗中的南方离地焰光旗与西方素色云界旗。善尸与恶尸分别持有一面小旗，向着戮仙与绝仙阵门奔去。
<br/>　　“什么？这是…分身？不！这不可能是分身！分身不可能都有准圣巅峰的实力。
<br/>　　鸿钧！没想到你隐藏的如此之深，竟还有此等底牌。看来本尊小看你了。”
<br/>　　罗睺身为诛仙剑阵之主，当两道身影出现的瞬间，他便察觉到了，顿时，他看向鸿钧老祖的眼神充满了惊讶与不可置信。
<br/>　　他怎么也想不到，鸿钧还有这等底牌。
<br/>　　“哼！就算你有如此底牌，那又如何？看我如何来杀你！”
<br/>　　经过初始的惊讶之后，魔祖罗睺又再次恢复冷漠，挥起手中弑神长枪，向着两尸杀去。
<br/>　　因为两尸没有防御至宝，而且攻击至宝也只是上品先天灵宝，对上罗睺那防御无双的灭世黑莲与攻击无双的弑神枪，不到一时三刻，就被打得节节败退，形势顿时变得岌岌可危。
<br/>　　“罗睺，安敢伤我三尸，吾与你誓不罢休！”
<br/>　　鸿钧见此，面上也是一阵焦急，大声叫道。
<br/>　　虽说三尸是自己身上斩出来的，只要自己不被杀死，就算三尸被灭，自己也还可以重新斩出来。但，那重新斩出来的三尸对比第一次斩出的三尸条件要苛刻许多，而且威力也有些影响。
<br/>　　不到万不得已，鸿钧不想损失自己的三尸。
<br/>　　“轰隆隆！”
<br/>　　奈何，鸿钧与阴阳老祖想上去相助，却被诛仙剑阵困住，力有不及。
<br/>　　两尸此刻已经节节败退，眼看就快要被罗睺打入剑门之中灭杀了，却无人能救。
<br/>　　陡然，天外有一道人踏空作歌，一步一步，看似很慢，实则很快，眨眼间，便踏出阵中。
<br/>　　“鸿蒙初判混沌出，魔神三千吾居先。混沌物质本源道，自号混沌称道君！”
<br/>　　歌声刚落，这道人变刚好落入阵中，与魔祖罗睺对峙。
<br/>　　只见这道人身穿青色道袍，一头黑发随意飘散，他看似二十来岁，英俊无比。
<br/>　　他一出现，就散发着极其强大的气势，这气势带有强烈的混沌气息。
<br/>　　气息苍茫，久远，沧桑，仿佛亘古长存，天地同在！
<br/>　　“两位道友！不必惊慌，本道君前来助尔等灭杀此魔！”
<br/>　　这人来到魔祖罗睺近前，淡漠地回了一声，便举手对着罗睺攻去。
<br/>　　此人正是元始在混沌之中修炼的分身，混沌树分身，混沌道君！
<br/>　　这些年，混沌道君一直在混沌之中参悟混沌树本源，混沌大道。
<br/>　　混沌灵根资质本就逆天，若论根基，比起鸿钧与阴阳老祖、罗睺等人要高多了，更是有着混沌树本源参悟，自是进步极快，现如今刚刚突破混元，踏入证道第三步。
<br/>　　混沌法则作为三大本源法则，自是强大无比，若不是元始本尊需要罗睺的灭世黑莲，本尊又不便出手，他这分身还不会来呢。
<br/>　　“哪来的小子，多管闲事！”
<br/>　　魔祖罗睺见此道人一闯进自己阵中，就向着自己杀来，顿时怒吼一声，手持弑神枪便冲了上去。
<br/>　　“万道森罗！”
<br/>　　枪乃拳的延伸，魔祖以枪衍拳，化出万道森罗，大道轮转，演化出一道巨大的黑色道门，道门之中，漆黑一片，一股惊人的吞噬之力向着混沌道君袭来。
<br/>　　“万道森罗，有意思。”
<br/>　　看到这万道森罗，混沌道君顿时就想到了昔日大战盘古时的混沌魔神魔祖。
<br/>　　“混沌幻灭掌！”
<br/>　　混沌道君无视诛仙剑阵之中的锐利杀气，对着万道森罗门就是一道惊天巨掌拍下，混沌色巨掌无视一切阻拦，一掌拍在万道森罗门上。
<br/>　　“咔擦！”
<br/>　　随着一道声响，万道森罗顿时化为无数碎片，破碎一地。
<br/>　　“可恶，看我神通！”
<br/>　　罗睺大喊一声，杀红了眼，头顶灭世黑莲，手提长枪，对着混沌道君不断刺去，势要将他灭杀。
<br/>　　混沌道君怡然不惧，单手对敌，不管罗睺怎么攻击，他都巧妙化解，不会多耗费一分力气。
<br/>　　罗睺头顶灭世黑莲，混沌道君也不想将他怎么样，他只是要拖延罗睺的时间，让鸿钧与阴阳老祖破解诛仙剑阵，之后让他们自己来解决。
<br/>　　“可恶！”
<br/>　　随着时间的延长，罗睺明显感受到眼前此人的深不可测。但，这道人就是不杀他，想要为鸿钧等人拖延时间。
<br/>　　突然，罗睺放弃了攻击混沌道君，连忙跳出戮仙阵门，向着陷仙阵门杀去，他要先将阴阳老祖灭杀。
<br/>　　混沌道君也没有多管，而是任由他离开，他也乐见于此，阴阳老祖与罗睺，不管谁死了，都对他没有坏处，他无所谓。
<br/>　　“轰隆隆！”
<br/>　　罗睺与阴阳老祖大战在一起，阴阳老祖的先天至宝太极图被陷仙剑气围困住，只能抬手拿出一柄先天灵剑与罗睺厮杀。灵剑虽然威力不凡，但还是攻不破灭世黑莲的防御，况且，黑莲之中还带有毁灭黑光攻击，再加上罗睺的弑神枪攻击，顿时，阴阳老祖落入了下风。
<br/>　　不一会儿，阴阳老祖就身中数枪，伤势严重。而且，就连元神，都被弑神枪破坏严重。
<br/>　　“哈哈哈！罗睺，想要杀我！没门！要死，我们就一起死吧！”
<br/>　　好似知道自己逃生无望，绝望之下，阴阳老祖面目狰狞，大声嘶吼。蓦然之间，元神与身体同时自爆。势要与罗睺同归于尽。
<br/>　　“轰！”
<br/>　　一声巨响，天地震动，来自堪比准圣巅峰强者的自爆，可不是那么好抵挡的。阴阳老祖所在的陷仙阵门，直接就被炸毁。
<br/>　　陷仙剑门被破，其余三大阵门也就不攻自破！
<br/>　　自此，号称洪荒第一杀伐大阵的诛仙剑阵被攻破了！
