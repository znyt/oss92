　　以叶尘如今的实力和状态，想要施展逆生百转第六重，强行提升实力，终究还是有些太勉强了，甚至还有可能，造成不可逆转的伤害！
<br/>　　然而，
<br/>　　一想到，倘若他今日败于对方之手，不但他性命不保，曦月、萧若曦、李虎、唐威、陈东、易山河，乃至神龙卫和数十万军队，只怕都要遭受一场浩劫！
<br/>　　“拼了！”
<br/>　　叶尘一咬牙，直接大吼了一声，体内气血强行运转！
<br/>　　噗噗噗噗噗噗！
<br/>　　下一刻，
<br/>　　不但叶尘口中再次喷出一口鲜血，竟然连同眼睛、鼻子、耳朵，七窍之中，乃至全身经脉，全部鲜血飙飞！
<br/>　　一时间，叶尘化成了一具血人，简直犹如从无尽地狱中爬出的恶魔！
<br/>　　“什么东西？”
<br/>　　无涯老人看到这一幕，也是暗暗心惊，心底一阵发寒，升起了一股极为不妙的预感。
<br/>　　“不管他耍的是什么花招，绝对不能让他顺利完成！”
<br/>　　一念及此，无涯老人不敢再有任何保留，直接施展出了自己最强一击。
<br/>　　轰隆！
<br/>　　眼看那，充斥着毁灭气息的青色剑芒，就要斩落下来，一直双眸紧闭的叶尘，陡然睁开双眼。
<br/>　　呼！
<br/>　　自叶尘眼眸之中，竟然射出两道血芒！
<br/>　　“杀！”
<br/>　　一道仿佛不属于人类的沙哑声音，自叶尘喉咙深处中发出！
<br/>　　刹那间，一股强大的杀气，伴随着令人心悸的恐怖气息，自叶尘的身上冲天而起！
<br/>　　“不！！”
<br/>　　无涯老人大吼了一声，想要拼命阻挡那股强大的气势，可惜那股力量实在太强悍了，即便他如今已经踏入化神境，依旧有些力不从心。
<br/>　　轰隆！
<br/>　　一声恐怖的巨响，
<br/>　　在无涯老人无比不甘的目光中，整个人狠狠砸落在地面上，竟然形成了一个蘑菇云。
<br/>　　而待那蘑菇云消散之后，地面上出现了一个深达上百米的巨坑！
<br/>　　嘶！
<br/>　　一时间，天地一片死寂，周围围观的其他强者，更是全都愣在当场。
<br/>　　谁能想到！
<br/>　　已经突破到化神境的无涯老人，实力暴涨数倍之后，竟然依旧被叶尘一招击败！
<br/>　　“这家伙的实力，究竟强大到何等地步？这到底是个什么怪物啊！”
<br/>　　望着九天之上，满身浴血，犹如魔神一般的少年身影，所有人全都感到发自内心的颤抖、胆寒！
<br/>　　要知道，这前后也不过才短短半个小时不到，叶尘先杀神王路西法，再败无涯老人，斩杀神界、仙界的元婴强者近百人！
<br/>　　难道他是神魔下凡不成？
<br/>　　众强者自信满满的杀入世俗界，本以为只是对付区区一个世俗小子，必定可以手到擒来，没想到却败的如此凄惨！
<br/>　　“死！”
<br/>　　叶尘一招击败无涯老人，没有给他任何喘息的机会，已经再次冲杀至跟前，然后狠狠踏落下去！
<br/>　　“不要！”
<br/>　　无涯老人顿时大吼了一声，可惜他此时身受重伤，一口气也提不上了，只能眼睁睁看着叶尘这一脚踩下。
<br/>　　彭！
<br/>　　没有任何悬念，叶尘一脚踩落，无涯老人的肉身瞬间被叶尘直接踩爆！
<br/>　　轰隆！
<br/>　　原本就已经达到上百米的深坑，再次重重陷落！
<br/>　　而方圆百里之内，大地都跟着一阵摇晃，犹如发生了一场大型地震！
<br/>　　“不要！不要杀我！我现在只剩下一道元神，只要你放过我，我愿意将整个仙界献给您！”
<br/>　　肉身被叶尘灭掉的一瞬间，无涯老人的元神从肉身中逃了出来，可惜叶尘早就有所防备，早已经一把将其擒在手中。
<br/>　　元神被擒，无涯老人只能低声下气的哀求起来。
<br/>　　越是这种活了近千岁的修真之人，其实就越是怕是，尤其是害怕元神被人擒住，如若对方歹毒的话，元神将会承受难以想象的折磨！
<br/>　　只不过，叶尘自然没有心思，
<br/>　　“区区一个蓬莱仙界，我弹指便能灭之，又何须你献？”
<br/>　　说完这话，叶尘心念一动，一道三昧真火，自他的指尖发出。
<br/>　　“啊！！！！”
<br/>　　伴随着一道凄厉的惨叫声，无涯老人的元神，顿时被焚烧的干干净净。
<br/>　　“桀桀！真是没有想到啊，连无涯子这个老家伙，都死在了你的手上！死得好！本祖还真要谢谢你，帮我除掉了两个对头！”
<br/>　　才刚刚灭了无涯老人，一道阴测测的声音，陡然在九天之上响起。
<br/>　　叶尘听到这个声音，根本不需要抬头去看，就已经认出了这道声音的主人，
<br/>　　“魔祖！你也来了！”
<br/>　　叶尘脚下猛然一顿，直接冲天而起，抡起手中的紫琼剑，便要一鼓作气将魔界之主灭掉。
<br/>　　然而下一刻，叶尘仿佛看到了什么可怕的事情，整个人顿时一滞，说什么也无法再动手了。
<br/>　　只见，
<br/>　　那一身黑袍的魔祖身后，萧若曦、李虎、唐威、陈东、易山河五人，被魔族之人死死按住，显然已经被他们控制。
<br/>　　“老大！救命！”
<br/>　　李虎看到叶尘出现，顿时大喊了起来。
<br/>　　叶尘顿时又急又怒，
<br/>　　“魔祖！你堂堂魔界之主，竟然用这么下三滥的手段，就不怕被人耻笑吗？”
<br/>　　魔祖桀桀一笑，
<br/>　　“小子！你不必用话激我，我魔族行事，向来随心所欲，又岂会在意别人的眼光？”
<br/>　　叶尘深吸了一口气，冷然道：
<br/>　　“放了他们，我饶可你们不死！”
<br/>　　魔祖听到这话，仿佛听到了什么天大的笑话一般，桀桀狂笑一声之后，脸色猛然一寒，冷冷道：
<br/>　　“小子！你似乎还没有搞清楚状况啊！”
<br/>　　“你之前施展的秘术的确厉害，可惜如今已经是强弩之末了吧？就算本祖现在和你打，也未必一定会输！”
<br/>　　叶尘默然不语，
<br/>　　的确，他强行施展出逆生百转的第六转，实力透支过于严重，如今体内已经是千疮百孔，就算勉力一战，或许可以灭掉魔祖，只怕自己的经脉，也会永久性损伤，实力大大下滑！
<br/>　　更何况，这家伙手里还有人质，叶尘投鼠忌器，根本不敢轻举妄动。
<br/>　　魔祖见叶尘默然不语，顿时越发得意，
<br/>　　“小子！只要你自废四肢经脉，然后答应入魔界为奴百年，本祖便饶了你和你的这些朋友，如何？”
