        &lt;h3&gt;第三卷第一百一十五章  好闺蜜  （为书友5436加更）&lt;/h3&gt;
<br/>
<br/>        闫欢一直等着陈子昂一起去甸城，张函曾经找过她，似乎是来了解陈子寒的。
<br/>
<br/>        她心里有些纠结。
<br/>
<br/>        她想提前过去，但是遭到了陈子悦的反对
<br/>
<br/>        “子昂不过去，你就不准过去。”
<br/>
<br/>        “为什么啊？”
<br/>
<br/>        “当初同意你去甸城就是因为你说了陪子昂一起过去的，子昂没有过去你过去干嘛。等子昂一起过去。”陈子悦霸道之极。
<br/>
<br/>        “我过去打前站不行吗？”
<br/>
<br/>        “不行，你们俩个人一起过去我才放心。”陈子悦毫不留情的拒绝了。
<br/>
<br/>        她时常从陈子悦的口中探听陈子昂的消息。
<br/>
<br/>        终于知道陈子昂要到三月份过去时，她的心里竟然莫名的激动。
<br/>
<br/>        她们已经把钱准备好了，可以说万事俱备只欠东风，就等着和陈子昂一起走。
<br/>
<br/>        闫欢站在穿衣镜前，看着镜子里穿着浅黄的v领毛衣、月白色的九分裤、细跟短靴、窈窕有致、披着一头卷发的自己，微不可察的叹了口气。
<br/>
<br/>        人，这一生说长也长，说短也很短，所有的一切好像就在昨天。
<br/>
<br/>        这几年的历练早已经将她改变成气势逼人的女神级别的人了。
<br/>
<br/>        谁能想到她过了那么多年不堪的生活呢，不过这一切都是深藏在她的心里，时常蹦出来刺激一下她。
<br/>
<br/>        曾经苦难的生活让她更热爱和珍惜现在的生活，和陈子悦在一起打拼的这些年是她最幸福的日子。
<br/>
<br/>        她准备将她的股份抽出来拿到甸城去投资，她知道此次去甸城一定是凶险万分的，无论所谓的谷强是何种身份，但是她还是像飞蛾扑火一样。
<br/>
<br/>        她想和谷强一起战斗，原来的她是没有能力，现在的她已经有了足够的能力。
<br/>
<br/>        她敲响了陈子悦的家门，他们俩个就住在隔壁，当初买房子的时候就特意买在一起。陈子悦一个人带着俩个孩子在西城，家里雇了一个保姆，她们基本每天都要陪着俩个孩子写作业，蔷悦已经长成大姑娘了。
<br/>
<br/>        开门的是王浩楠。
<br/>
<br/>        她怎么不知道王浩楠来了？
<br/>
<br/>        “王老师，你回来了？”王浩楠很少来西城。
<br/>
<br/>        “嗯，进来坐一会？”
<br/>
<br/>        “不了，我接子悦上班去。”
<br/>
<br/>        “陈子悦，闫欢来接你了。”王浩楠转身就回去了。
<br/>
<br/>        看样子俩个人闹矛盾了。
<br/>
<br/>        “子悦。”闫欢看着陈子悦似乎是刚哭过，俩个孩子已经被保姆送到学校了。
<br/>
<br/>        “走吧。”
<br/>
<br/>        俩个人相对无言，闫欢打开车门，陈子悦低着头一直不说话。
<br/>
<br/>        “你怎么了？哑巴了？”闫欢有些生气。
<br/>
<br/>        “闫欢，你知道吗？这么几年我和王浩楠过的是什么日子？”
<br/>
<br/>        “不知道，你没告诉我。”闫欢气道。
<br/>
<br/>        “那一年，那个女人给我发短信，说他们怎么怎么了，他说他没有，我将信息给他看，他说他爱我，都是那个女人勾引的，发誓不再来往了。
<br/>
<br/>        我还是一如既往地的对待他和他的家人，后来他趁我们在外面拓展业务的时候竟然将那个女人带回去给他妈妈看了，那个女人就说了我很多坏话。
<br/>
<br/>        我提出离婚，他不愿意，他说他从来都没有想到要我和离婚，还说我现在有钱了就想甩了他，而且他是永远都不会在和那个女人来往的了，他说那个女人就是看不得我们夫妻感情好。
<br/>
<br/>        有一次新年刚过，他竟然告诉我，那个女人怀孕了，他要带那个女人去人流，而且还没有钱，我心一软，给了他几千块钱。
<br/>
<br/>        再后来，寒暑假都开车出去旅游。孩子从来都不管。本来我也没有想到要他管。”陈子悦已经泣不成声了。
<br/>
<br/>        “不行就离了吧！”闫欢说。
<br/>
<br/>        “离了又怎么办呢？我不可能让俩个孩子都没有父亲吧。”
<br/>
<br/>        “那你说怎么办？”闫欢一生气踩了一下刹车，陈子悦的头差一点撞到玻璃上了。
<br/>
<br/>        “你别生气嘛，你好好开车。”陈子悦软声说。
<br/>
<br/>        “我生什么气啊，又不是我自己。”
<br/>
<br/>        “虽然之前生孩子的时候他不太高兴，但是孩子现在大了，毕竟他还是孩子的爸爸，那一天我生气骂了他两句，俩个孩子都哭得不行，都心疼他爸爸。”陈子悦的声音透着无力感。
<br/>
<br/>        “那就是个渣男，披着羊皮的渣男。”闫欢恨恨骂着。
<br/>
<br/>        “你也是不成器，你干嘛要一而再再而三的忍让啊？”
<br/>
<br/>        “我嫌丢人。”
<br/>
<br/>        “我，靠，面子值几个钱？”闫欢真是怒其不争。
<br/>
<br/>        “闫欢，你不明白，我和王浩楠结婚了，有孩子了，离婚会牵扯很多事情。”
<br/>
<br/>        “那你怎么办啊。”
<br/>
<br/>        “就这样吧，我们现在就是同在一个屋檐下的陌生人，维持着表面的和谐。”
<br/>
<br/>        “就这样的过一辈子？”
<br/>
<br/>        “是。我想了，我要将我自己的股份全部转到俩个孩子的名下。”
<br/>
<br/>        “随你吧。”闫欢叹了口气，她想让陈子悦离婚，可是像陈子悦说的，离婚也是要付出代价的。
<br/>
<br/>        对于婚姻，闫欢没有发言的权利，目前她还是单身，早之前她提醒过陈子悦，但是怎么生活那是陈子悦的选择，她能做的就是陪在她的身边。
<br/>
<br/>        俩个人都沉默下来了，车里笼罩着一层淡淡的忧伤，伴随着李清照的《声声慢》。
<br/>
<br/>        陈子悦很是生气的将车里的广播给关了，本来心情都不好，还在哪里：冷冷清清呢。
<br/>
<br/>        闫欢被陈子悦的举动给逗笑了：“算了，你也别纠结了，毕竟王浩楠还活着。无论他怎么渣，他还是一个活着的爸爸。”
<br/>
<br/>        “我也没有太难过，就是早上吵了几句，他说他都改了，我还不原谅他。”
<br/>
<br/>        “哦，不过好像最近每周末都回家，真的是散了？”
<br/>
<br/>        “不知道，我也不再过问了，他来就是娃他爸，不来我就当没有这样的一个人。我也没有时间去过问。”
<br/>
<br/>        陈子悦这几年生意做得愈来愈大，王浩楠有吃有喝有钱花，开着豪车，人前摆足了架子，他凭什么会和陈子悦离婚？
<br/>
<br/>        陈子悦对他的不闻不问让他又气又开心，生气的是陈子悦不在乎他，开心的是陈子悦给他自由。
<br/>
<br/>        当然，王浩楠的心思陈子悦压根就没有时间去猜测的，孩子和事业占据了她的全部时间。
<br/>
<br/>        “子昂已经说了，她准备三月份去甸城。”
<br/>
<br/>        “去甸城有风险的，我只拿我的那部分去投资吧？”闫欢再一次提出这样的要求。
<br/>
<br/>        “你想撇开我？没门。”陈子悦怎么可能让闫欢一个人承担风险的。
<br/>
<br/>        这一生她活得卑微，难得有闫欢这样的好闺蜜，好朋友，她又如何轻易舍弃？无论生活如何艰难，无论情感如何一地鸡毛，她们都坚强的踏着自己步子，一步一个脚印的努力着，她们相互扶持一路走到现在，彼此都是生命里最重要的人了。
