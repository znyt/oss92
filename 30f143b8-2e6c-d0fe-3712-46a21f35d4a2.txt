<br/>				    足足又过了半年的时间，空间之中盘膝修炼的凤九这一天浑身的灵力气息和玄力气息都在涌动着，身体里灵力爆涌，丹田之处的那枚青莲的气息也随着溢了出来，弥漫在整片空间之中。<br/>　　<br/>　　    那熟悉的感觉让她知道，自己这是要进阶了！修炼了大半年，终于要突破仙尊级别进入仙帝实力了！<br/>　　<br/>　　    她当即取出为突破仙尊级别进入仙帝而准备的丹药，这一次突破仙帝，她一定要一举突破至巅峰级别！<br/>　　<br/>　　    只有这样，实力才足以震撼那四位仙帝。更何况，那四位仙帝是很多年前就已经是仙帝级别，如今的实力，只怕比仙帝还要再高，她只有突破到仙帝级别，才能入得了那四人的眼。<br/>　　<br/>　　    伴随着这空间里面气息的变化，这空间里的天地也跟着起了变化，而当她在这里面进阶时，外面的人，还都不知道，用不了多久，他们就都可以离开这里了……<br/>　　<br/>　　    凤城之中，古老等人有条不紊的忙碌着，分配着的有的去捕杀凶兽，有的则镇守城中，有的则去外面一些地方巡视着。<br/>　　<br/>　　    自打半年前凤九跟他们说可以带他们离开这地方时，这里面的血腥味似乎少了很多，已经很少有人为了一点东西而交战抢夺，那些修为高的人也没再轻易的便对修为低的人下杀手。<br/>　　<br/>　　    对他们来说，眼下什么事都不重要，能离开这个地方才是最重要的，所以，他们一直在等，等着那一天的到来。<br/>　　<br/>　　    不少人经常聚在凤城的城门外，等在这里，如果真有什么消息传出时，他们也能第一时间知道，只不过，凤城的大门却是经常紧闭着的，除非有人进出才会打开，他们就算是想知道消息，若无凤城里面管事的人允许，消息也是传不到他们耳中来的。<br/>　　<br/>　　    直到，凤城的天空中接连放出了十二道信号弹，那些在外面的凤城中人在看到信号弹后纷纷从各处赶了回来。<br/>　　<br/>　　    接下来的几天，他们只看到一个个都急匆匆的回到凤城，他们想要打听出什么大事了？只是，一个个都没有理会他们。<br/>　　<br/>　　    “你们说，到底是出什么事了？怎么凤城里的人一连发了十二道信号将外面的人都召集了回来？”<br/>　　<br/>　　    “我在想，会不会是凤主要打开这里的结界大阵了？所以将人都召集回来？”<br/>　　<br/>　　    “会吗？这离她当时说出那话到现在，也就半年的时间，不过半年的时间，她就已经知道怎么破那结界大阵了？”<br/>　　<br/>　　    “要不然还能有什么事？如今这里面四方都相处融合，这半年也都很少有人再打打杀杀了，除了破除结界大阵的事情，我觉得应该也没其他事情能让她一下用十二道信号弹将外面的人都召集回来了。”<br/>　　<br/>　　    “说来也是，也许真的是在破开结界大阵了！”<br/>　　<br/>　　    “太好了！我们终于可以离开这地方了！”<br/>　　<br/>　　    凤城外面的人脸上都溢出了兴奋期待的笑意，一个个按压不住激动的交头接耳热聊起来……<br/>
