        傅夜沉从欲望中抽离，当他拉回自己的理智时，她已经退后了一步跟他保持着一定的安全距离。
<br/>
<br/>        童筱颖冷眼看着他迷蒙眼眸里流露的情愫。
<br/>
<br/>        明明他对她满是爱慕与憧憬，但是她却完全看不到他的心意。
<br/>
<br/>        对她来说，她所看到的是，他傅夜沉其实也跟傅昼景一样。
<br/>
<br/>        男人，那种口头中的“爱”，心头里的“爱”，与身体上的“爱”，都是分开的。
<br/>
<br/>        “大哥，我不想成为你发泄生理需求的工具。”童筱颖极其淡定地说道。
<br/>
<br/>        她一句话，怼得他哑口无言，甚至堵得他一个能让他去反驳的理由也想不出来。
<br/>
<br/>        他承认自己是对她充满了欲望，但是他只对她有那种欲望，怎么在她心里就成为了他只是把她当做发泄生理需求的工具呢？
<br/>
<br/>        “大哥如果你有生理需求，你可以去找苏漫雪，或者你的女秘书也可以，再不济，去‘夜惑’里找出台的公主。反正，大哥，你不缺女人的。”童筱颖冷静而慢条斯理地说道。
<br/>
<br/>        听着她的话，他的心口突兀地疼得格外厉害，捉住她肩头的双手随着这份心痛而加重了力度，硬生生地掐到了她的肩胛骨上。
<br/>
<br/>        痛……
<br/>
<br/>        她不悦地皱起了柳眉。
<br/>
<br/>        此刻，傅夜沉英俊的面容有些轻微的扭曲，点漆般深邃的瞳仁渐渐收缩，目光沉沉地凝视着童筱颖的眼睛。
<br/>
<br/>        她的眼神，黯淡无光，瞳孔里完全找不到他的身影。
<br/>
<br/>        白天给他喂蜜糖，燃起他的一丝希望，到了晚上索性直接给他喂毒药了吗？让他像死了一样地去痛。
<br/>
<br/>        傅夜沉的眼神骤然转冷，透着磁性的声音，带着一丝怒火，疾言厉色地质问：“你还在想着，要为阿景守身如玉吗？”
<br/>
<br/>        “是。”童筱颖不假思索地回答，抬起双手，挪开了傅夜沉的桎梏。
<br/>
<br/>        这个“是”字，她是没经大脑就说出来的。
<br/>
<br/>        反正她不爱他，他怎么想，于她而言都无所谓。
<br/>
<br/>        “我累了，先回去了。”童筱颖淡然地转过身去，一步一步地往台阶下走去，留下傅夜沉一个人愣在原地。
<br/>
<br/>        她刚下山，身后的山顶突然传来“嗖——嗖——嘭——嘭——”地烟花绽放的声音。
<br/>
<br/>        漆黑的夜空，瞬间亮如白昼。
<br/>
<br/>        筱颖，我爱你。
<br/>
<br/>        这句由烟花释放的情话，在夜幕上灿烂地绽放。
<br/>
<br/>        他为她准备的惊喜，准时准点地放出。
<br/>
<br/>        然而，她却头也不回地离开了。
<br/>
<br/>        山顶的烟花声，并未吸引童筱颖的驻足。
<br/>
<br/>        她一门心思地往前走。
<br/>
<br/>        不转身，不回头，她永远也遇不到真爱。
<br/>
<br/>        傅夜沉苦笑着坐在石板台阶上，仰望着夜幕上不断出现的“筱颖，我爱你。”这句烟花字幕。
<br/>
<br/>        他以为，今晚他可以就此顺利地向她表白，向她袒露自己对她的一往情深，告诉她从始至终他一直爱的人是她，以及这几年来对她的相思之苦。
<br/>
<br/>        并向她允诺，他傅夜沉这一辈子只爱她童筱颖一人，愿意为她付出一切，包括他的生命。
<br/>
<br/>        可是……
<br/>
<br/>        她不给他机会，一丁点，甚至一丝一毫，都不愿意给他。
<br/>
<br/>        因为，她一直把所有的机会，全留给了他的弟弟傅昼景。
<br/>
<br/>        童筱颖进屋后，将肩上的外套拿了下来，刚好遇见正抱着洗衣篓的刘婶。
<br/>
<br/>        刘婶见童筱颖的手中还拿着一件西装外套，顺便问道：“是大少爷今天穿过的吗？”
<br/>
<br/>        “嗯。”童筱颖点了点头。
<br/>
<br/>        刘婶接着说道：“那给我吧！我一起拿去洗了！对了，帮我翻翻衣兜，把里面的东西拿出来。”
<br/>
<br/>        “好！”童筱颖随之把傅夜沉的西装外套的衣兜翻了个遍，结果只翻出一张寸照出来，然后把外套递给了刘婶。
<br/>
<br/>        刘婶拿着衣服去了洗衣房后，童筱颖才下意识地去看手里的这张寸照。
<br/>
<br/>        照片里的人，不是别人，正是落以柔。
<br/>
<br/>        她素颜丽质，面目清秀，扎着清爽的马尾，穿着白衬衫，俨然一副大学生模样。
<br/>
<br/>        这张寸照，她刚刚好像是从傅夜沉的西装外套的内兜里掏出来的。
<br/>
<br/>        西装外套的内兜，刚好贴近心口。
<br/>
<br/>        大哥把这张照片放在里心口最近的地方，是代表了什么？
<br/>
<br/>        呵——
<br/>
<br/>        果然！
<br/>
<br/>        童筱颖心如止水地拿着这张寸照，只身上了楼。
<br/>
<br/>        她去了傅夜沉的房间，进了衣帽间，将这张寸照随便塞到了一件西装外套的内兜里，然后拿了自己的衣服，去了隔壁的房间。
<br/>
<br/>        这一次，她落了门锁，只为了防他傅夜沉。
<br/>
<br/>        她不想再让自己第二天早上一睁开眼睛，又回到了他傅夜沉的床上的事情发生。
