        &lt;h3&gt;第三卷第一百八十九章  团团圆圆的新年&lt;/h3&gt;
<br/>
<br/>        陈子昂和张函俩个人坐在客厅说了一会话，算是解开了彼此的心结，做不成夫妻，他们也没有必要成仇人，况且，俩个人心中还有着彼此。
<br/>
<br/>        陈子昂看看时间不早了，也该要到吃饭的时间，就算是大人不吃，孩子也应该要吃的。
<br/>
<br/>        于是她对张函说：“我得去厨房做饭了，今天是除夕。”
<br/>
<br/>        张函说：“我来帮你吧，看样子他们一时半会儿也不会走的。”
<br/>
<br/>        “你帮我？”陈子昂有些不确定。
<br/>
<br/>        “子昂是怕我吗？”张函笑着问。
<br/>
<br/>        “我怕你做什么？”陈子昂觉得话说开了，她坦坦荡荡的，她怕张函什么？
<br/>
<br/>        “不怕我，怎么就不敢让我帮忙了？”张函典型的就是给点阳光就灿烂的人。
<br/>
<br/>        “我哪是不敢让你帮忙，是怕张总家里还有人等得着急了，今天可是除夕，你们一家三口都到我家里来了，可是不太好。”陈子昂心里说，你不回家去陪你的太太，你帮我什么忙？
<br/>
<br/>        “是啊，我们一家三口都来了，家里哪里还有人等着呢？所以我就更要去帮忙了，顺便晚上蹭顿年夜饭。”张函说着就站起来了。
<br/>
<br/>        “你当真？”陈子昂的眼睛差点就掉地上了。
<br/>
<br/>        “子昂，我们家里年年都是我们一家三口，每年都冷冷清清的，今年就让我在你家过个热闹的新年吧？”张函说的是真话，每年陆玉都回娘家过年，他们除了去陪爷爷吃一顿饭都各回各家，确实是冷冷清清的。
<br/>
<br/>        今年陆玉更是不回家了，说是甸城工作很忙，过一段时间再回来。
<br/>
<br/>        原本，他们还想着会不会吃闭门羹，没有想到来的时候陈子昂正在贴对联，而且他父母亲和孩子们竟然聊得这么久。
<br/>
<br/>        看样子蹭陈子昂一顿饭吃一下也不是不可以，陈子昂的心底柔软，只要是不触及她的底线她是不会反对的。
<br/>
<br/>        陈子昂其实心里还是爱着他的，只是迫于他们现在的身份，假以时日，他还是有机会的，他们一家四口会团圆的。
<br/>
<br/>        果然陈子昂看了一会他，又看了看书房那边没有什么动静，叹了口气说：“好吧，只是你知道我厨艺不行。”
<br/>
<br/>        “没事，有我呢，你打下手就行了。”张函心花怒放，他的子昂还是以前的那个子昂，真好，不过如此好哄她会不会也经常在外面被人哄？
<br/>
<br/>        张函做饭的手艺还是和陈子昂在一起学的，自从和陈子昂分开后，他很是想念陈子昂做的饭菜，他也会经常下厨给自己做饭，只是再也没有陈子昂做的那个味道了，为了能做出陈子昂的那个味道，他经常在厨房鼓捣，时间久了，他成了一个大厨了。
<br/>
<br/>        他系上围裙，打开陈子昂的冰箱，看了看，冰箱里的食材还是很丰富的，于是拿起刀，霹雳吧啦一阵跺、削、切、看得陈子昂眼花缭乱。
<br/>
<br/>        “你是什么时候这么能干的？”陈子昂问张函。
<br/>
<br/>        “和你分开之后，因为想吃你做的饭菜，只好自己下厨，每次都不是你的味道，又很想吃那个味道，于是就锻炼出来了。”张函边切菜边说。
<br/>
<br/>        陈子昂静静的看着张函熟练的切着菜，想起了当初俩个人在一起的情景，眼眶发酸。
<br/>
<br/>        她真的就当了一个下手，洗菜递菜。
<br/>
<br/>        而团团他们出来的时候就看到了俩个人在厨房的情景。
<br/>
<br/>        团团叹了口气：“渣男上线，陈子昂完蛋了。”
<br/>
<br/>        “说什么呢？没看见那美好的就像是一副画？”嘟嘟狠狠的瞪了哥哥一眼。
<br/>
<br/>        “是啊，美好的就像是一幅画。”弥青云感叹着。
<br/>
<br/>        团团和嘟嘟看了一眼张沛尧和弥青云：“你们还真打算在我家蹭饭啊？”
<br/>
<br/>        “是啊，没看见你爸爸和你妈妈在厨房里给我们做饭吃吗？今天可是除夕，我们一家人是要在一起守岁的。”张沛尧的玩心上来了，虽然团团很是排斥他们，但是看着陈子昂和张函在厨房的情景真的就像是嘟嘟说的那样，美好的就像是一幅画。
<br/>
<br/>        “你们要不要去外面转转，看看我们家的院子，等会饭好了叫你们？”团团问张沛尧夫妇。
<br/>
<br/>        “嗯，我们家院子环境还是不错的，可以去看看的。走吧，我带你们去。”嘟嘟拍了拍小手。
<br/>
<br/>        “好啊，我们去转转。”张沛尧以为孩子想给他们的爸爸妈妈创造机会，便同意了孩子的提议。
<br/>
<br/>        嘟嘟带着张沛尧和弥青云出了门，走到院子中间，对张沛尧和弥青云说：“哎呀，爷爷奶奶，我好冷，我回去加件衣服，你们先自己转着，我等会就来了啊。”
<br/>
<br/>        “去吧，我们在那边等你，穿暖和点。”他们因为是开车过来的，并没有穿羽绒服。
<br/>
<br/>        嘟嘟跑回家，转身关上门，哥哥站在门口笑眯眯的看着她：“做得不错。”然后转身拔掉了门铃的电源。
<br/>
<br/>        俩个孩子从厨房的门缝里看着厨房里忙碌的俩个人，都微微了摇了摇头，叹息了一声：“虽然美好，可惜短暂。”
<br/>
<br/>        “哥哥，你说妈妈和爸爸以后会不会在一起？”嘟嘟问团团。
<br/>
<br/>        “难啊，有外面的那俩个人，他们是没有机会的。”团团摇了摇头。
<br/>
<br/>        “其实，我看他们并没有什么恶意。”嘟嘟又对团团说。
<br/>
<br/>        “坏人的脸上会写着他是坏人？对你不好的人会对你说他会对你不好？”团团摇了摇头：“你们女人啊，就是太天真，我说，落妍，你以后将眼睛擦亮一点，别碰上渣男。”
<br/>
<br/>        “你讨厌，怎么就开始教训我了？不和你玩了。”嘟嘟生气了，坐在餐桌的凳子上，撑着小手看着厨房里的爸爸妈妈。
<br/>
<br/>        “爸爸真的是哥哥口中的渣男吗？好像不太像啊。”嘟嘟心里默默的想着。
<br/>
<br/>        “爸爸也不算太渣吧，就是他们家的家风太渣了，趋炎附势、仗势欺人、这一点要是改了就很完美了。”团团和嘟嘟是双胞胎，俩个人心里的想法基本相似。
<br/>
<br/>        “哥哥，不知道妈妈会把我们藏哪里？”嘟嘟忧郁的问。
<br/>
<br/>        “我也不知道，最好是能到落雪小姨哪里去，很久都没有看到落雪小姨了。”团团会经常想起落雪来。
<br/>
<br/>        “哥哥，我都快忘了落雪小姨是什么样子了。落雪小姨这么多年都不来看我们，是不是都忘记我们了？”嘟嘟努力的去回忆落雪的样子。
<br/>
<br/>        一个小时之后，张函和陈子昂将饭菜端到饭桌上。
<br/>
<br/>        发现张沛尧夫妇不见人了，便问团团：“爷爷奶奶呢？”
<br/>
<br/>        “他们应该是回去了。出去就没有回来。”团团不在意的说。
<br/>
<br/>        “你看？”陈子昂看着张函。
<br/>
<br/>        “不可能回去，我打个电话问问情况。”张函也不确定他爸妈是怎么回事。
<br/>
<br/>        打电话时，电话在客厅里响着。
<br/>
<br/>        陈子昂看着团团和嘟嘟，盯得俩个人低下头：“爷爷和奶奶到院子里转去了。”
<br/>
<br/>        张函说：“我出去找找。”
<br/>
<br/>        “你别出去找了，我去，他们会迷路。”陈子昂打开门。
<br/>
<br/>        张沛尧和弥青云站在门口，冻得只打哆嗦。
<br/>
<br/>        虽然如此，张家一家三口还是在陈子昂家蹭了一顿年夜饭，俩个孩子知道犯错了，极力的表现着自己，惹得大家哈哈大笑，新年在欢声笑语中落幕。
