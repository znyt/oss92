　　天墉城数百里外的小镇，花香镇，此镇以种植花草闻名。
<br/>　　作为天墉城弟子临时落脚点，有天墉城弟子庇护，普通妖魔鬼怪也不敢再此作祟，数十年来，花香镇内都其乐融融。
<br/>　　天空一片蔚蓝，碧空如洗，陡然空间一阵扭曲，一道白袍青年临空而立。
<br/>　　青年正是离开天墉城的慕清霄。
<br/>　　望着脚下的小镇，小镇周围色彩缤纷的花田，喃喃自语道：“这里就是花香镇吗？看来是没错了。”
<br/>　　说罢，慕清霄视线在四周扫视一圈，最后落在小镇千米外的一座山峰上，不是很高，但山顶处烟雾缭绕，显然是一座活火山。
<br/>　　旋即，空间扭曲间，他身影就出现在了千米外的火山上空。
<br/>　　望着脚下赤红色的岩浆，释放着高温的火山口，慕清霄脸上露出一丝笑意，在这里突破无疑最好不过。
<br/>　　旋即，慕清霄身躯就被灵气包裹，化作一道黑影，以迅雷不及掩耳之势冲入向着火山内部冲去。
<br/>　　“嘭...!”
<br/>　　下一秒，伴随着四射的岩浆与一声闷响，慕清霄就感觉到自身被温暖的岩浆所包裹。
<br/>　　感受到周围汇聚过来的精纯火属性能量，划破岩浆，径直向着底部掠去。
<br/>　　向下潜了百米左右，已经到了岩浆底部，就在这时，岩浆中一道赤红色踪影向他飞射过来。
<br/>　　慕清霄身形一闪，便是巧妙的躲了开来，抬头望去，只见悬浮在岩浆中的是一条胳膊粗细，通体赤红的灵蛇。
<br/>　　此时，灵蛇正盘在岩浆底部，吐着芯子警惕的望着他，似乎在告诉慕清霄，这里是它的地盘。
<br/>　　同时，蛇瞳中尽是恐惧，它能从慕清霄的身躯上，感受到无与伦比的恐怖威压。
<br/>　　“金丹期的灵蛇，已经诞生出一丝灵智了吗？”
<br/>　　慕清霄完全不理会它的警告，随手一挥，恐怖的力量就搅动岩浆，直接将灵蛇袭去。
<br/>　　灵蛇措防不及下，直接被恐怖的力量砸中，数米长的身躯倒射而出，砰一声砸在数百米外崎岖的火山壁面上。
<br/>　　被砸的七晕八素，灵蛇心底满是恐惧，龟缩在火山墙角瑟瑟发抖，再也不敢在上前招惹慕清霄。
<br/>　　见它如此识趣，慕清霄也没有在下杀手，区区一条灵蛇，修炼至金丹境，诞生出灵智，也是不容易。
<br/>　　震慑过灵蛇，慕清霄就地盘坐而下，微微一愣。
<br/>　　“火山与地面的交界处，能量最精纯的地方，难怪这头畜生会待在这里，在过数十年，修炼至元婴期也未尝不可。”
<br/>　　微微抬头，望了一眼上空，粘稠的岩浆遮掩了他的视线，千篇一律的赤红，在这样的环境下，令人感到压抑。
<br/>　　当然，这是对于普通人而言，对于慕清霄来说，这种环境更有助于修炼。
<br/>　　如果他只是凤凰，并没有保存人性，说不定会找个火属性能量精纯的地方，修炼个几十上百年。
<br/>　　比如，斗破位面的练气塔底。
<br/>　　将复杂的情绪抛之脑后，慕清霄双手结印，掌心向天，眼眸渐渐闭上，不突破练虚，绝不出火山。
<br/>　　随着慕清霄进入修炼状态，岩浆底部，变得寂寥无声。
<br/>　　唯一不同的就是，整座火山犹如无底洞一般，源源不断的吸收着四周的能量。
<br/>　　...
<br/>　　...
<br/>　　转眼，整整一周的时间就悄悄溜走。
<br/>　　伴随着时间的流逝，慕清霄体内储存的元阴也一缕缕的被炼化，灵气充斥着四肢百骸，气息不断攀升。
<br/>　　此时，火山上空，狂风肆虐，周围诡异的形成了一个灵气旋窝，方圆数十里的灵气都朝着此地汇聚而来。
<br/>　　如此异象，自然也引起了花香镇居民们的恐慌。
<br/>　　数十年来，花香镇中都其乐融融，虽然数千米外有着一座活火山，但居民们也未曾担忧过。
<br/>　　哪能想到，在最近一周内，火山中似乎出现异象，让他们都惶恐不安起来。
<br/>　　驻扎在附近的天墉城弟子们，自然也注意到了火山中的异象，确认灵气汇聚的方向后，数十名弟子都汇聚了过来。
<br/>　　其中四名金丹期的弟子，其余都是筑基期，他们哪里见过这等异象，纷纷抱着凝重的心情前来查探。
<br/>　　“灵气确实是再此汇聚，难道其中有妖魔正在突破？”其中一名金丹期弟子，望着火山下的赤红色岩浆，不敢确定的说道。
<br/>　　“不清楚，几十年下来，此地都相安无事，也并未听说有妖魔的踪迹......”
<br/>　　说着，这名金丹期弟子苦笑道：“不过此事不寻常，已经有人回天墉城禀报，我们只要守着就行，下去查探太危险了。”
<br/>　　其余两名金丹弟子皆是沉重的点了点头。
<br/>　　他们虽是金丹期的修士，在普通人眼中，是万人敬仰的仙人，但只有自己知道，自身是多么的弱小。
<br/>　　哪怕是碰上稍微强大点的妖魔，他们恐怕都是死无葬身之地。
<br/>　　“轰！”
<br/>　　就在这时，火山口周围的天地间，凭空爆发出一阵轰鸣声，旋即这片天地间的灵气，都剧烈波动起来。
<br/>　　突如其来的动静，将周围弟子吓得魂飞魄散，回过神来，弟子们都面面相觑，脸色赤红，尴尬不已。
<br/>　　而伴随着轰鸣声的响起，只见天地间的灵气犹如受到某种牵引一般，开始疯狂的灌入火山口内。
<br/>　　“怎......怎么回事？”
<br/>　　其中一名金丹弟子身躯哆嗦着问道，刚才乍然而现的气势，令他整个人的灵魂都感到余悸。
<br/>　　另外三名弟子也是暗自吞咽着唾沫，神色惊恐的望着火山深处。
<br/>　　此刻，在岩浆表面中心位置，已经出现一颗赤红色的旋窝，而周围涌来的灵气，全都顺着漩涡灌入火山深处。
<br/>　　“火山深处确实有什么东西在疯狂的吸收灵气，这种恐怖的气势，我们最好离这里远点。”
<br/>　　说着，众人相视一眼，都能看到对方眼中的惊恐，于是迅速离开火山，站在数千米外，警惕的望着火山。
<br/>　　如此异象，令他们好奇的同时，内心深处也有着浓浓的不安。
<br/>　　“不知道回去禀报的弟子多久能赶回来，若真是妖魔突破，周围恐怕就要生灵涂炭了，希望他能尽快回来吧。”
