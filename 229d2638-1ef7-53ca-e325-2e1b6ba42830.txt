　　看着晶莹剔透剑气，苏重突发奇想：“如果把这枚剑气吃了会怎么样？”
<br/>　　他当然不会真的吃，而是想着能不能把剑气重新收回体内。
<br/>　　以前他是把剑气当子弹，突突往外打。像现在只是放出来演示一番，没必要扔出去浪费吧？
<br/>　　小心震动道种核心，黄豆剑气顿时化作丝丝缕缕亮光，钻入手指消失不见。
<br/>　　“还真行？”苏重大喜：“还是小看了本源力量！”
<br/>　　道种核心可是经过本源催发而来。即便是作为创造者，也没搞明白道种的全部奥秘。他只能一点点儿去试验发掘！
<br/>　　……
<br/>　　时间飞逝，苏重每天生活都很规律。给大竹峰众人做饭，然后进入藏书阁翻书籍一个时辰，不断增加积累。剩下的时间，他就老老实实的蹲在自己竹屋内修炼。
<br/>　　随着他阵法造诣增强，他又把竹屋周围阵法改了改。竹屋之外还是一片祥和。布满草木精气和天地灵气，但朱屋内，苏重平日所坐的蒲团之下，却凝聚着浓郁的凶戾气。
<br/>　　他能用阵法帮大竹峰众人修炼，自然也不会亏待自己。
<br/>　　就好似太极图中鱼眼，外围浓郁天地灵气，让阵法中心处凝聚出的凶戾气压缩起来，如果有人能看到天地间能量。就会发现苏重蒲团地下一片氤氲黑气，宛若浓墨一般。
<br/>　　他也不怕别人发现。苏重在戾气之外覆盖了一层天机印，完美的封锁掩盖住凶戾气。能把天机印融入道种。他对天机印的理解，冠绝整个青云，直追创始者！
<br/>　　有阵法相助，苏重修炼一日千里，等他回过神，已经到七脉会武的日子。
<br/>　　“可惜，阵法改造时间太短，在给我一段时间，我的修为必会再提升一个台阶。”
<br/>　　阵法中心凶戾气被苏重吸食一空。而且还要去从参加七脉会武，不然那苏重真想一直坐在这里修炼。
<br/>　　丹田内，七百二十枚剑气符汇聚成一条条飘带，绕着道种核心不断旋转。这就是他日夜修炼的成果。
<br/>　　……
<br/>　　大竹峰祠堂，跟着田不易祭拜过大竹峰历代前辈后。田不易开始交代七脉会武注意事项。
<br/>　　“大仁，小凡和尔旦是第一次参加七脉会武，你照顾一下两人，别让他们吃了亏。”
<br/>　　“是！”宋大仁恭敬道。
<br/>　　“还有，他们两个不会御剑飞行，你和大义一人带一个。”正说着，眼光一扫就看到躲在苏重和张小凡两人身后的杜必书。
<br/>　　田不易看着他就气不打一下来，冷哼一声：“你要是敢给我丢脸，以后你就在黑竹林里面安家吧！”
<br/>　　杜必书脸色发苦，却讷讷不敢言。
<br/>　　苏重回头笑着打量杜必书，不怪田不易生气，这家伙前段时间达到御物境界，理所应当下山寻找天材地宝炼制法宝。
<br/>　　可这位六师兄竟带着三个骰子回了师门。
<br/>　　田不易开始还是很高，毕竟大竹峰又带回来的玩意，差点儿没活劈了他！
<br/>　　“师兄不必担心，黑竹林不会死人的。”苏重调侃道。
<br/>　　杜必书不敢反驳，师傅就在眼前，他真要敢闹幺蛾子，师父师娘现在就敢把它扔进黑竹林。
<br/>　　看着杜必书吃瘪模样，众弟子满脸笑意。
<br/>　　田不易冷哼一声，抱起在他脚边打转的大黄狗化作一道红光冲天而起。
<br/>　　苏茹瞪了一眼杜必书无奈摇头，也拉着田灵儿腾空而飞。
<br/>　　几个师兄也相继起飞，大师兄宋大仁走到苏重给身边：“师弟，咱们走吧。”
<br/>　　苏重点头答应，任由宋大仁拉着御剑飞行。和想象中不同，剑气破开空气，周围形成一个无形保护罩，人在其中并没有什么难受感觉。
<br/>　　“师弟还习惯吧。”宋大仁关系道：“第一次御剑飞行可能会恐高，以后你学会御剑飞行，多飞几次就好了。”
<br/>　　苏重看着脚下，心不在焉的点了点头。他倒是不恐高，毕竟不是第一次飞天。他现在想的是如何让自己飞行。
<br/>　　“庚金剑气可没有飞遁法门，还得我自己想办法。”苏重懊恼。
<br/>　　之前一直憋在山上修炼，还没觉得差别。现在一下山，立刻就察觉出。御剑飞行简直就是修行者标配。别说斗法，就是单纯的赶路，也是必不可缺。
<br/>　　世界太大，想用两条腿去闯荡，等赶到目的地，早就不知道是猴年马月了。
<br/>　　“这次七脉会武后，就要考虑飞遁法门了。”苏重又给自己安排了一项研究任务。
<br/>　　正想着呢，旁边传来一股劲风。宋大仁飞剑偏转，划了个弧形重新回到正轨。苏重不可避免的被晃了一个趔趄。
<br/>　　转头就看到杜必书驾驭着三颗硕大筛子，在他们身边晃荡。
<br/>　　“二蛋，好玩吗？可别吓的尿裤子！哈哈哈……”伴随着一阵笑声，杜必书嗖的一下就飞远了。
<br/>　　宋大仁无奈摇头：“师弟别害怕，你六师兄只是开玩笑，咱们掉不下去的。”
<br/>　　苏重摇头：“我没感觉害怕，只是觉得黑竹林阵法威力还不够强。等稍后我回去改进一下，专门给六师兄使用。相比他心里是欢喜的，你看他现在就很高兴不是吗？”
<br/>　　宋大仁打了个哆嗦，开始给自家六师弟默哀。你说你惹谁不好，偏偏要招惹自家小师弟。
<br/>　　那黑竹林就连他进去都感觉吃不消，实在是针扎痛苦太难受，要是再加强威力……宋大仁默默打了个寒颤，下定决心，一定要和六师弟划清界限，不能遭了鱼池之殃啊！
<br/>　　……
<br/>　　青云门占地广阔，七脉主峰分布四面八方，将主峰通天峰围绕在中间，庞大诛仙剑阵笼罩其上，守护者青云安危。
<br/>　　可即便再大，众人一个个御剑飞行，很快就到了主峰通天。
<br/>　　相比于其他七峰，主峰才真正有仙家气象。飘渺浮云、悬空横桥、通天阶梯，直插天际的主峰，无形中便散发着一种震撼人心的气魄。即便苏重看到第一眼，也被深深震撼。
<br/>　　青云门作为天下第一门，雄霸天下千年，底蕴深厚至极。
<br/>　　宋大仁等人落下遁光，田不易和师娘苏茹已经不见人影，他们早一步先去了主峰大殿。
<br/>　　“师弟，到了这里就不能御剑飞行了，需要我们一步步走上去。这是为了表示对青云以及历代捍卫正道前辈的尊敬。”宋大仁一边解释，一边带着众人开始顺着阶梯登峰。
<br/>　　“师弟，看到哪里那个水潭了吗？里面住着我们青云门镇山灵兽灵尊大人。”宋大仁解释。
<br/>　　“灵尊是什么？”身后张小凡好奇问。
<br/>　　“灵尊是水……”轰的一声，刚才还平静无波的水潭，顿时炸开一道水花。一个庞然大物骤然出现在众人面前。
