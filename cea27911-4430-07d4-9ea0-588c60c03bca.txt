        &lt;h3&gt;第三卷第一百九十三章  江氏新年&lt;/h3&gt;
<br/>
<br/>        江俞轩随着父亲回到了江家，这个新年对他来说还是有点糟心的，尤其是和张倩楠的婚礼，原本想着给张倩楠一个风风光光的婚礼，却没有想搞成那个样子。
<br/>
<br/>        他们家是龙头企业，在地产界数一数二，旗下有很多子公司，年会的参与是要联络各公司之间的感情，今年的年会是统一进行，总部在广州，全国各地各子公司之间互动，江俞轩搞信息化多年，江家企业怎么可能落后，现在的江家完全是一整套信息化办公，总会场在广州，分会场在全国各地，什么叫互联互通？
<br/>
<br/>        江家容纳两千多人的会场里，灯壁辉煌，房顶旋转着的琉璃灯将会场照耀的五彩斑斓，忽明忽暗，墙壁上是悬挂着的多屏拼接起来的巨大互动屏幕系统，这些屏幕通过互联网将各分部和总部相连，实行实时互动，这也是陈子昂他们公司get的新技能，远程办公会议系统，这个会议系统是他们公司自己用，江家发展迅速，为了能跟上时代的步伐，江俞轩就给他们江氏集团也安装了一套这样的系统，如果用的好的话，是可以推向市场的。
<br/>
<br/>        江俞轩站在28层的楼上，附身看着楼下来来往往的人们，感叹着江氏集团的庞大，心里油然而生出自豪感和责任感，之前在陈子昂的公司摸爬滚打，俩个人一路跌跌撞撞的走来，历尽千辛万苦，有了现在的成就，可是放在庞大的江家，真的是不可相比。
<br/>
<br/>        父亲让自己甩手那么多年，在陈子昂的公司里埋头苦干，虽然说是锻炼了自己，可是和在庞大的江家怎么能同日而语，想一想，父亲无非就是想让自己多一份任性，给自己一份自由，想到父亲在路上俩个人的谈话，江俞轩才明白，这个世界上最懂自己的也只有父亲了吧。
<br/>
<br/>        和陈子昂在一起工作那么多年，任劳任怨，甘当“卧底”也无非是想陈子昂过得幸福而已。人的缘分是多么的奇妙啊，就一眼让他在陈子昂的公司干了六七年，不舍得离去，他原本不知道陈子昂的一切，就想那样死心塌地的和陈子昂一起奋斗，一起努力，曾经他还想，如果有机会他愿意和她一起奋斗一辈子，只是后来遇到了张倩楠。
<br/>
<br/>        张倩楠将陈子昂的一切都讲给他听，他看着陈子昂和陆宇霆一起聊天，吃饭，看着张函偷偷的跟踪陈子昂，他都迷糊了，到底谁才是陈子昂的良人？他将陈子昂的一切行踪告诉了张倩楠和张函，希望陈子昂和她相爱的人能在一起幸福，希望她余生平安，可是陆宇霆结婚了，结婚的人不是陈子昂，张函也没有和陆玉离婚，直到他和张倩楠将结婚证都领了，陈子昂还是独身一人，那天看着寒伯安向安男求婚的画面，心里抑制不住的悲伤：“陈子昂，我结婚了，你怎么办？”
<br/>
<br/>        陈子昂并没有回答他，但是他听到了她的抽泣声，那个女人表面看着坚强，其实内心脆弱的很，只是她的脆弱都只表现在他的面前，他的心里有一根细细的线，缠缠绕绕的缠在陈子昂的身边，但是他也知道，那个女人哭过之后不到三分钟又是一副：我是爷我怕谁的模样。
<br/>
<br/>        此时，他的脑海中都是陈子昂的身影，为什么这一会要想起她？
<br/>
<br/>        “想什么呢？”姐姐江俞琪看着弟弟站在那里一动不动的看着楼下大厅，也往下看着，只是人头攒动，可是弟弟却看得那么专注，乃至她走到他身边很久都没有感觉到。
<br/>
<br/>        “想陈子昂。”江俞轩脱口而出。
<br/>
<br/>        “嘶，你都已经是结婚的男人了，还想别的女人。”江俞琪觉得不可思议弟弟不是一直爱着张倩楠嘛。
<br/>
<br/>        “你想哪里去了，我在想最近工作上的事情，你看，楼下的大屏幕都是我们研发出来的系统。陈子昂那个女人真是人才呢。”江俞轩的声音里有一丝丝的慌乱也有一丝丝的自豪，还有一丝丝说不出话来的失落。
<br/>
<br/>        他现在连想她的资格都没有了，原来在她的身边还是一个总裁助理的时候陪着她哭，陪着她闹，陪着她笑，现在也只能是一个朋友了，那个助理他当得甘之如饴啊，可是陈子昂那个女人，心里眼里都是张函，还能容下谁？和陆宇霆分得干脆利索，不带一点点留恋，趴在他的肩膀哭过之后就像是什么事情都没有发生过。
<br/>
<br/>        “看你那小眼睛都在冒小星星，陈子昂是很优秀，不过现在和你没有关系了，充其量你们就是朋友，哦，对了，你还是她公司的股东。”姐姐笑着敲打了一下他，拍拍肩膀说：“走了，下楼，爸爸都已经在楼下了。”
<br/>
<br/>        江俞轩随着姐姐一起到大厅，大家都已经是各司其职，各坐各位，父亲冲他们姐弟俩点点头，他们俩个人走到父亲的身边坐下。
<br/>
<br/>        “这个是你们公司今年新搞的事情，不会出什么问题吧？”江爸爸问江俞轩。
<br/>
<br/>        “出什么问题？这个我们已经用了很久了，要不是看在你是我老爸的面子上，我才懒得给你用的。”
<br/>
<br/>        “你这臭小子，说什么呢，难道现在这个公司不是你的？就陈子昂那个小公司才是你的？”
<br/>
<br/>        “哈哈哈哈，稳重稳重，陈子昂那个公司可是我和她一起打拼下来的，你这个公司嘛，是现成的，不可同日而语，你不知道吗，只有自己付出心血的东西才更加珍惜。”江俞轩笑着打击着自己的老父亲。
<br/>
<br/>        “臭小子，哼，以后不准再去陈子昂公司任职了，江家的事情还不够你忙的，既然你没有付出心血，那我就让你多付出心血。”江爸爸似乎是生气了。
<br/>
<br/>        “你确定？说不定我一不高兴就跑路了，毕竟我还有地方可去。”江俞轩看着自己爸爸那孩子气的样子，心里充满了感动。
<br/>
<br/>        “哎，人家是养儿防老，我是养儿跟着别人跑，罢了，罢了，累觉不爱了。”江爸爸满脸委屈。
<br/>
<br/>        “哈哈哈哈，你说不爱就不爱了，我爱你就好，爸，作为你的儿子我真的是很荣幸，我爱你！”江俞轩看着老爸认真的说。
<br/>
<br/>        “哎，你小子受刺激了，这么煽情的？”江爸爸有些吃惊。
<br/>
<br/>        “噗嗤。”江俞琪忍不住捂着嘴笑起来。
<br/>
<br/>        这对父子啊，一个个平时都傲娇的不行，在一起说几句认真话，反而不习惯了，但是她看到了弟弟的认真。
<br/>
<br/>        “爸，我和弟弟都爱你，很爱很爱的那种爱！”
