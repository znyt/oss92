<br/>　　屠雄猛然站了起来，连连大笑起来，这笑容，尽是嘲笑的笑容。
<br/>　　“怎么回事？”

<br/>　　“这小子死定了！”
<br/>　　这突然出现的变故，让所有人都措手不及，没人想到，楚风眠居然会真的放开了这屠雄。

<br/>　　在这样近的距离中，就算是让生死境强者前来，都不一定可以挡得住屠雄的这一次偷袭。
<br/>　　“完了，这小子真是傻了，居然放开了屠雄。”
<br/>　　“哎，看来这一次还是屠雄最后胜利了。”
<br/>　　一些武者议论的言语中，都不由的有些惋惜。
<br/>　　楚风眠如今，像是刚刚崭露头角的黑马，可现在被屠雄阴了，直接陨落。

<br/>　　就算是利用手段，也没有任何可以指责的。
<br/>　　屠雄看着那一道长矛即将要贯穿楚风眠的胸口，便是冷笑一声道。
<br/>　　“小子，等你死了，下辈子记得要当一个聪明人！”

<br/>　　突然之间的一道声音，令屠雄的脸色瞬间僵硬了，脸色的冷笑，也仿佛直接停滞住了。

<br/>　　随着一掌灵力，屠雄的身躯，直接便是炸裂开来，化为了一团血雾，被楚风眠直接收取。
