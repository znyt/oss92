<br/>				    星云仙宗因有那些修士的帮忙而反败为胜，情况已经稳定控制住。然，凤九与混元子那里，纵是两人联手也占不到半点上风，尤其是，那魔主仿佛在耍着他们玩一般，明明可以下杀手，却又不下杀手，看得人不知他究竟想要干什么？<br/>　　<br/>　　    苍擎在看到凤九他们那里不敌魔主时，飞身掠上前想来帮忙，然，他还没靠近那魔主，就见那原本盯着凤九的魔主目光一眯，回头朝那苍擎扫去，手掌一转，一股黑色的气流瞬间化成一头黑色的烟雾狼头，带着凌厉凶残的气势猛然扑向那苍擎。<br/>　　<br/>　　    同时，铺天杀气以及属于他身上的强大威压覆盖而出，生生将那掠上前来的苍擎镇住。<br/>　　<br/>　　    “小心！”<br/>　　<br/>　　    凤九大喝着，看着魔主的那一击蕴含着铺天杀意，她知道，若是这一击落在苍擎的身上，他必死无疑！<br/>　　<br/>　　    然，她在这边想要上前去救他已经是不可能的了，一是速度上来不及，二是就算冲上去了只怕了的当不住那一击。<br/>　　<br/>　　    于是，她当即厉喝一声：“火凤！”<br/>　　<br/>　　    蕴含着上古威压的清冽声音在这一刻传出，伴随着她的声音而出的是上古神兽火凤。从她的空间中出来的火凤鸣叫一声，翅膀一伸一展猛然飞起，它没有上前，而是在半空中翅膀一拍，一股风力呼啸着袭出，生生阻挡住了那魔往的那一攻击。<br/>　　<br/>　　    同一时，原本被震摄住的苍擎也在这一刻被它翅膀拍出的强大风劲吹拂出几十米外，避开了魔主的威压震摄。<br/>　　<br/>　　    “嗯！”<br/>　　<br/>　　    苍擎闷哼一声，只感觉到喉咙中一咸，一丝血腥味在口中弥漫而开。他稳住脚步后一手按在胸口处，抬头看着那天空中的火凤一眼，又脸色凝重的看向那通身杀气的魔主，心下如同骇浪翻腾着，狠狠的拍打着他的心房。<br/>　　<br/>　　    若不是刚才那上古神兽火凤将他推开，只怕，那魔主袭出的那头黑色狼头击落在他的身上，他是必死无疑。<br/>　　<br/>　　    想到以他的实力竟在那魔主的威压之下无法动弹半分，心下便是一阵骇然。难怪是魔族的魔主，他的实力已经远远的超越在场的所有人。<br/>　　<br/>　　    就算是如今有众名修士护住这星云仙宗，但，只要他存了毁灭之心，那么，以他的实力修为，就算是再来一批他们这样的修士也挡不住他想灭宗的脚步。<br/>　　<br/>　　    看到苍擎被火凤救下，避开了那必死的一击，凤九暗松了口气。清幽的眸子落在苍擎的身上，她沉着声音道：“你不要靠近，其他人都不要靠近，你们都不是他的对手。”<br/>　　<br/>　　    她很清楚，谁来了也一样，她手底下的这些修士并没有足以与魔主交手的实力，若是上前帮忙，估计也就只会落得个身死的下场。<br/>　　<br/>　　    “不过区区一个修士而已，你就这么关心他们的死活？”魔主听到凤九的话，也看到了她的担心，很是讶异的看了她一眼，血色的眸子中一抹暗光划过，神色莫名也不知在想着什么。<br/>　　<br/>
