　　“段惊天，你真是太让人震惊了！”
<br/>　　苏莫刚一落下，沁儿便睁着闪亮的大眼睛，满脸兴奋的说道。
<br/>　　只此一役，段惊天足以名震轮回神朝。
<br/>　　“你父亲应该会满意吧？”苏莫笑着说道。
<br/>　　“这还用说吗？除非他眼瞎！”沁儿笃定的说道，完全不在乎玄天长老的威严。
<br/>　　“也是！”苏莫笑了笑，随即，便大步向古城之外走去。
<br/>　　沁儿见此，也没有再继续逗留，急忙大步跟上。
<br/>　　褚彦和骆妤对视了一眼，均是看出了对方眸中的复杂之色。
<br/>　　啊！
<br/>　　就在此刻，惨叫声突然响起，两人抬头一看，顿时面色一呆。
<br/>　　只见上方平台之上，姬月松败给了天王宫的高瘦青年，被打出了平台，口中鲜血喷涌。
<br/>　　褚彦和骆妤两人，心中暗叹，姬月松这一次，不仅败给了高瘦青年，还败给了段惊天和天尘山。
<br/>　　连连受伤之下，估计等会，连前五名都不一定能得到了。
<br/>　　十个名额，每一个名次的福泽都不同，名次月底，福泽越差。
<br/>　　姬月松的战绩，让两人大失所望，心中感慨连连。
<br/>　　“真是不错！”
<br/>　　苏莫自然也看到了情况，心中暗赞，涂赢的实力还是这么强大。
<br/>　　古神不愧是古神，天赋过人不是一般的妖孽所能比拟。
<br/>　　苏莫也没有过多的关注，和沁儿一起，大步的走出了古城，飞上了高空，来到了玄天长老的身前。
<br/>　　“见过玄天长老！”苏莫向对方抱拳行礼。
<br/>　　“段惊天，你很不错！”
<br/>　　玄天长老面带笑意，继续道：“你先静等片刻，待所有人结束，在一起回人王宫。”
<br/>　　玄天长老没有和苏莫多言，毕竟，此地人多眼杂，不是交谈之地。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，便走到了对方的身侧，静静的等候了起来。
<br/>　　其它五大王宫的长老，均是仔细的打量了苏莫一番，面色各异。
<br/>　　对于苏莫这种妖孽，他们均是有惊疑之心。
<br/>　　尤其是天王宫天有为长老，他是清楚涂赢的身份的，苏莫的天赋战力，居然超越了古神，这就有点匪夷所思了。
<br/>　　他暗暗决定，这个段惊天的身份，日后必须好好的调查一番。
<br/>　　苏莫伫立在玄天长老的身边，一边和沁儿闲聊，一边观看古城之中的战斗。
<br/>　　剩下的这九名妖孽，战斗力都相差不大，每一战斗很是激烈。
<br/>　　不过，天尘山明显最强，占据很大的优势。
<br/>　　时间不断的流逝，名次逐渐的明朗了起来。
<br/>　　约莫小半个时辰之后，战斗彻底的结束了。
<br/>　　天尘山得了第二名，涂赢稍差一筹，得了第三名。
<br/>　　至于姬月松，只排在第七名，与他自己的期望，相差甚远。
<br/>　　其实，这倒不是说姬月松很弱，只是姬月松在苏莫和天尘山、涂赢三人的手下，都受了伤，导致战力有所下降。
<br/>　　所以，在后来的战斗之中，无法发挥到巅峰，所以名次稍低。
<br/>　　随后，巨大的天尊幻象，眼眸之中，再次爆发出金光，金光裹挟着九人，再次的消失无踪。
<br/>　　九人，将得到轮回天尊更大的福泽。
<br/>　　九人消失之后，古城中的六宫天才，纷纷走出古城。
<br/>　　古城外的众长老，和众多弟子，也纷纷落到了城门之前。
<br/>　　众多的弟子，声音吵杂，议论之声不绝于耳，均是直讨论之前的战斗。
<br/>　　不过，很多人的人，均是将目光落在苏莫的身上。
<br/>　　对于苏莫，六宫天才，无不心带崇敬之情。
<br/>　　没有等待太久，仅仅是半刻钟的时间，姬月松九人再次出现了，并纷纷来到城外。
<br/>　　“二叔，让您失望了！”姬月松来到玄天长老身前，面色有些惨白，神情更显落寞。
<br/>　　本来，这一次他志在第一，结果却是得了个第七名，简直是一败涂地。
<br/>　　苏莫闻言讶然，原来姬月松和玄天长老，还有这层关系。
<br/>　　也对，同是姬姓，都是一个家族。
<br/>　　“月松，你的表现已经很不错了，只能说天外有天人外有人。”玄天长老道。
<br/>　　“的确是天外有天人外有人！”姬月松深深的吸了口气，并看了苏莫一眼。
<br/>　　“所有弟子，返回人王宫！”
<br/>　　玄天长老招呼一声，随即一刻也不愿意多留，身形冲天而起。
<br/>　　而后，众人纷纷跟随，向人王宫返回。
<br/>　　其它五宫之人，亦是纷纷离开，并没有过多的寒嘘。
<br/>　　一路上，一众人王宫的弟子，纷纷向苏莫道喜，并且和他攀谈。
<br/>　　苏莫也没有孤傲，和众人一路闲聊，气氛融洽。
<br/>　　不过，姬月松、褚彦和骆妤三人，却是不在此列。
<br/>　　三人一路上都沉默不语，仿佛是很不开心。
<br/>　　这一次赶路，或许是玄天长老心急，庞大的气机裹挟着众人，穿梭于虚空深处，速度快到极致。
<br/>　　去时需要半个月的路程，在玄天长老的全速之下，只用了五个时辰。
<br/>　　回到人王宫之后，玄天长老遣散众人，独自带着苏莫，走向人王宫深处。

<br/>　　院落占地方圆千余丈，情景而悠然，附近没有半个人影。
<br/>　　院落之中，除了几座宫殿之外，便是遍地的神草灵药。
<br/>　　各种颜色艳丽的花朵，碧绿的小草，粉红色的果实，布满了整个院落。
<br/>　　这院落之中，居然是一个药园，除了玉石小道之外，种满了花花草草，发出浓郁的清香。
<br/>　　“好多神草！”
<br/>　　走进院落，苏莫轻吸一口气，便感到心旷神怡，精神大震。
<br/>　　这些药草，均是无比珍惜的神草，香味浓郁，但是，却不会传到院落之外。

<br/>　　“丹神！”苏莫闻言惊讶，人王宫宫主的称号，乃是人德神王，没想到还有丹神之名。
<br/>　　“所以，你只要能得到父亲的帮助，好处无穷！”玄天长老道。

<br/>　　“当初，帝一魂和拓跋青云，都曾来见过父亲，想要得到父亲的支持，不过，他们均是没有成功，父亲不愿意掺和！”玄天长老继续道。
<br/>　　苏莫闻言，心中沉吟，希望一切顺利，宫主会全力支持他。
<br/>　　交谈之间，两人来到了一座最大的宫殿之前。
