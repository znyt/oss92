　　“夕儿，你的修为达到什么境界了？”
<br/>　　山巅上，苏莫牵着夕儿的手，望着山巅下滚滚云雾，随意的闲聊着。
<br/>　　“前几日刚突破到武王境八重境界！”
<br/>　　夕儿轻声道，言罢，她将螓首轻轻的靠在了苏莫的肩上，享受着难得的宁静。
<br/>　　“不错！你的青凤武魂很不简单，虽然不是真正的凤凰武魂，但也是凤凰后裔！”
<br/>　　苏莫笑着说道，夕儿的情况他早已都知晓了，对方的天赋，绝对是非常高的。
<br/>　　“对了，夕儿，你修炼的是什么级别的功法？”似乎想到了什么，苏莫立刻问道。
<br/>　　“我修炼的功法，名为《无极玄经》，乃是皇级上品功法！”夕儿道，皇级上品功法，在帝玄宫已经算是最顶级的功法了。
<br/>　　帝玄宫只有一门帝级功法，乃是帝级下品功法，而且这门帝级下品功法不仅有所缺失，还不适合她修炼。
<br/>　　“这《无极玄经》属于什么属性的功法？”苏莫问道，她准备送给夕儿一门适合她的帝级功法。
<br/>　　他身上帝级功法众多，对于夕儿，他自然不会吝啬。
<br/>　　“无极玄经并没有什么特殊的属性，各方面比较中和。”夕儿道。
<br/>　　苏莫闻言沉吟了起来，他本准备找一门帝级中品、乃至帝级上品的功法给夕儿修炼。
<br/>　　但现在，他心中又冒出来一个想法，不知道夕儿的各种属性天赋如何。
<br/>　　若是夕儿某一种属性天赋好的话，倒是可以继承九大宫的传承。
<br/>　　不说多好，只要不是太差，能说的过去就可以了。
<br/>　　所谓肥水不流外人田，九大宫这种超级传承，若是能给夕儿，那真是最理想的结果。
<br/>　　“夕儿，你的属性天赋如何？”苏莫极忙问道。
<br/>　　“属性天赋？”
<br/>　　夕儿闻言，脑袋从苏莫的肩膀上抬了起来，道：“我的木属性天赋最高，水属性天赋其次，其它的几种属性也都还可以。”
<br/>　　夕儿身在帝玄宫，自然也测试过自己的属性天赋，所以她很清楚自己的属性天赋。
<br/>　　“木属性天赋最高？”
<br/>　　苏莫闻言眼眸一亮，不过，他眉头又微微皱了起来，木属性对应的是长青宫，虽然宏青璇现在生死未仆，但长青宫的传承，他早已经决定给宏青璇了。
<br/>　　不过，夕儿的水属性天赋虽然不如木属性，但只要不是太差就够了。
<br/>　　“夕儿，过两****带你去苍穹神宫吧！”苏莫微笑道，他已经决定，让夕儿继承水月宫的传承。
<br/>　　夕儿虽然不是水月宫最理想的传人，但也不算差。
<br/>　　而且，夕儿本身的武魂天赋就很高，足以胜任水月宫宫主之位。
<br/>　　“嗯！”夕儿闻言轻点螓首，她不会拒绝苏莫的要求。
<br/>　　随即，两人便没有在说话了，苏莫在思考九宫传人之事。
<br/>　　如果让夕儿继承水月宫的传承，那就只剩下金煌宫、厚土宫、雷霆宫三大宫没有传人了。
<br/>　　厚土宫和雷霆宫，战乱和小八倒是可以作为候选人，但金煌宫目前还没有适合的人选。
<br/>　　夕儿又将螓首靠在了苏莫的肩上，她见苏莫久久不再开口，贝.齿轻咬红.唇，柔声问道：“苏莫哥哥，你这次来帝玄宫，还有别的事情吗？”
<br/>　　其实，夕儿是想问苏莫，这次来是不是要提亲？但出于女儿家的矜持，她却是羞于问出口。
<br/>　　上次十三长老从苍穹神宫回来之后，可是告诉了夕儿，苏莫很快就会来提亲。
<br/>　　这让夕儿即欣喜，又暗暗期待，这段时间，她一直在等待苏莫的到来。
<br/>　　“没了！主要就是来看看你！”苏莫摇头笑道。
<br/>　　苏莫根本没想过提亲的事，毕竟现在他不仅要发展苍穹神宫，还有黄泉魔宗这个大敌当前，更有宏青璇生死未仆，哪有心情成亲啊！
<br/>　　“这……！”夕儿闻听苏莫之言，顿时心中一沉，苏莫竟然不是来提亲的？
<br/>　　“怎么了夕儿？”苏莫见夕儿神情有异，不解的问道。
<br/>　　“没什么！”夕儿轻轻笑了笑，她很想问问苏莫，但还是忍住了，没有询问。
<br/>　　她也不奢求必须和苏莫成亲，只要她们能在一起，像现在这般，便已经知足了。
<br/>　　就这样，两人伫立在山巅之上，吹着山巅的寒风，享受着这难得的宁静。
<br/>　　嗖！
<br/>　　不多时，破空声响起，一道消瘦的身影飞了过来，落在了苏莫两人的身后。
<br/>　　“师尊！”
<br/>　　苏莫转头，见是十三长老来了，立刻向对方抱拳行礼。
<br/>　　“十三长老！”夕儿同样向十三长老行礼。
<br/>　　“苏莫，圣女，你们不必多礼！”
<br/>　　十三长老满面笑容，笑道：“呵呵，苏莫，难得你还认我这个师尊！真是让老夫汗颜！”
<br/>　　十三长老心中唏嘘不已，他和苏莫只做了一日的师徒，他也没有教授苏莫什么实质性的东西，但苏莫依旧认他这个师尊。
<br/>　　要知道，以苏莫现在的修为实力，和身份地位，完全不必如此。
<br/>　　而且，不论是在私下里，还是在公众场合，苏莫对他都是客气有加，这让十三长老即欣慰又感慨。
<br/>　　同时，十三长老心中也有些后悔，早知道苏莫天赋和心志都如此绝佳，当年在林阳城之时，他就应该收苏莫为徒。
<br/>　　“俗话说一日为师终身为父，您老永远是我的师尊！”苏莫郑重的说道。
<br/>　　“好！好！好！”十三长老闻言，老怀大慰，连说了三个‘好’字。
<br/>　　少倾，十三长老笑道：“苏莫、圣女，宗主请你们去议事大殿！”
<br/>　　“嗯！”
<br/>　　苏莫和夕儿俱是点了点头。
<br/>　　随即，三人一起，联袂向议事大殿飞去。
<br/>　　帝玄宫，议事大殿。
<br/>　　当苏莫到来之时，发现大殿中已经座无虚席，宗主帝释、大长老华太山，以及其他数十名长老，俱是在大殿之中。
<br/>　　“见过父亲！”夕儿向帝释行礼。
<br/>　　“见过宗主！”苏莫同样向帝释行礼，他虽然现在是苍穹神宫宫主，但毕竟还有个帝玄宫弟子的身份，所以也没有托大，依旧向对方行了一礼。
<br/>　　“免礼！”
<br/>　　主座上，一声锦袍的帝释面带威严，随意的摆了摆手，道：“就坐吧！”
<br/>　　“是！”
<br/>　　随即，苏莫和夕儿便找了两个位置坐了下来。
