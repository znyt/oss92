　　分身沉默不语，伫立在玄天神殿大厅的中央。
<br/>　　他心中急思，按照宫主透露的意思，是不会多管轮回之身之间的争斗。
<br/>　　但是，只要他有足够的天赋和实力，对方会庇护于他。
<br/>　　当然，具体要达到什么地步，是一个未知数。
<br/>　　还有就是，他不能完全信任这个宫主，毕竟，如果自己是本体在此，说不定对方是另一种态度。
<br/>　　“段惊天，你的时间不多了，帝一魂和拓跋青云的修为，都已经达到了中级神王，只需几万年的时间，他们便能达到高级神王；而你，落后太多太多了！”宫主摇头说道。
<br/>　　一旦帝一魂和拓跋青云，有任何一人修为达到高级神王，那么大局便定下了。
<br/>　　因为，中级神王的这两人，在轮回神朝，还有人能制衡。
<br/>　　但是，一旦达到高级神王，以这两人的战斗力，在轮回神朝将无人能挡。
<br/>　　当时候，一切都尘埃落定了。
<br/>　　而帝一魂和拓跋青云，也是在竞赛，就看谁能先达到高级神王了。
<br/>　　“宫主，我有信心追上他们！”分身沉声说道，面上是强大的自信。
<br/>　　“只有信心是不够的！”
<br/>　　宫主闻言，微微的摇了摇头，随即，他略一沉吟，道：“这样吧！我给你一个机会，向我证明你的潜力！”
<br/>　　“证明？”
<br/>　　分身闻言，面色沉重，道：“帝一魂一直在找我，怕是我没有办法向您证明！”
<br/>　　他心中清楚，自己一旦名声在外，帝一魂肯定会找到他。
<br/>　　“这你不必担心，当你证明出了自己的潜力，老夫可以庇护你，当然，只是暂时庇护你！”宫主摇了摇头。
<br/>　　对于他乃至整个人王宫而言，给予段惊天庇护，没有任何的困难。
<br/>　　只不过，要让他们看到希望，不然徒增烦恼。
<br/>　　“怎么证明？”分身郑重的问道。
<br/>　　“你如今什么年纪了？”宫主淡淡的问道，因为眼前的段惊天，只是分身，他并不能看出本体的年纪。
<br/>　　“八十余岁！”分身道。
<br/>　　“天赋的确不错！”
<br/>　　宫主闻言，微微颌首，继续道：“两年之后，六大王宫，所有百岁以内，并且已经成神的年轻子弟，都会前去轮回古城，祭拜天尊，到时候你若是能独领风骚，盖压六宫天才、碾压举朝妖孽，老夫和人王宫便会庇护于你！”
<br/>　　“比试切磋吗？”分身凝声问道，若是比试切磋的话，区区百岁之内的天才，哪里能和他争锋？
<br/>　　“并不是这么简单！”
<br/>　　宫主摇了摇头，继续道：“但，实力越强，你的成功率略大！”
<br/>　　“好，希望宫主一言九鼎！”分身点了点头，他感觉这位宫主，应该不是在诓骗他。
<br/>　　以对方的修为，若是对他有歹心，应该不必这么麻烦吧！
<br/>　　“老夫言出法随！”宫主面色郑重的说道。
<br/>　　“好！”分身深深的吸了口气，郑重的点了点头。
<br/>　　既然如此，这轮回古城，他去一次又何妨。
<br/>　　“宫主，我缺资源！”分身凝视着宫主，面带希翼之色。
<br/>　　这么好的机会，他怎么可能错过。
<br/>　　既然对方给他机会证明，那总要给点资源吧！
<br/>　　“你倒是毫不客气！”
<br/>　　宫主面无表情，略一沉吟，他向下首的玄天长老点头示意。
<br/>　　随即，玄天长老手掌一番，掌中取出了一枚储物戒。
<br/>　　“这里有些零碎资源，本来是要给沁儿的，你拿去吧！”玄天长老手掌一抛，储物戒便向分身飞了过去。
<br/>　　分身接住储物戒，稍一查看，顿时心中一震。
<br/>　　储物戒之中，资源非常多，不仅有数百万源石，还有一些神草、神药。
<br/>　　那些源石之中，五行源石只有一小部分，但是，其中居然有不少中品源石。
<br/>　　分身心中暗惊，这也叫零碎资源，简直是一笔很大的财富了。
<br/>　　他不由的心中暗叹，有背景就是好。
<br/>　　他若是向拓跋青云一样，出生在六宫之中，什么都不缺，现在早已站在巅峰了。
<br/>　　不敢说，神域无敌，最起码在神域也没有人能伤到他了。
<br/>　　“多谢宫主，多谢玄天长老！”
<br/>　　分身向两人郑重抱拳，道：“既然如此，那么弟子告退了！”
<br/>　　宫主挥了挥手，并未多言。
<br/>　　分身见此，转身走出了玄天神殿，向惊天府而去。
<br/>　　玄天神殿之中，只剩下了宫主和玄天长老两人。
<br/>　　大殿的殿门，再次关闭了。
<br/>　　“父亲，真不准备帮他？”
<br/>　　玄天长老看向宫主，沉声询问。
<br/>　　要知道，每一具轮回之身越强大，最终融合之后，天尊才会更强大。
<br/>　　所以，他们应该帮助段惊天，不说能让段惊天抗衡帝一魂和拓跋青云，但是，最起码不至于这么弱小。
<br/>　　帝一魂和拓跋青云，都希望自己能融合对方，从而成为轮回天尊。
<br/>　　所以，这两人不会在乎融合的轮回之身是否弱小，但是，他们却说在乎。
<br/>　　因为，轮回天尊的强弱，决定了神域乃至这个宇宙的未来。
<br/>　　“帮与不帮，全看他的本事了！”宫主声音悠悠。
<br/>　　他知道这个段惊天，天赋肯定很高，不然也不会达到如今的地步。
<br/>　　但是，这还远远不够，因为，他们要帮助他段惊天，就要和帝一魂、拓跋青云对抗。
<br/>　　这是存在很大的风险的，毕竟，无论是帝一魂还是拓跋青云，都只是轮回之身，并不是轮回天尊。
<br/>　　如果为了段惊天，让得人王宫遭难，那就得不偿失了。
<br/>　　天尊不会伤害人王宫，但是，帝一魂和拓跋青云会。
<br/>　　这两人的实力，已经极度强大，若是达到高级神王，就无人能压制了。
<br/>　　所以，目前，六宫都在观望，观望帝一魂和拓跋青云，谁能先一步达到高级神王。
<br/>　　不仅是六宫，九大神朝，很多强者，都在等待。
<br/>　　只不过，神王的境界，并不是那么好提升的，就算天赋再出众，没有几万年，乃是十几万年的时间，都不可能突破。
<br/>　　“此分身的元神，极不简单，虽然深藏在丹田深处，但是，如果我所料不差的话，应该是不败战气！”玄天长老道。
<br/>　　“的确是不败战气，如此之外，没有类似的强大元神！”
<br/>　　宫主微微颌首，道：“正因为连一具分身都如此不凡，所以，为父才给予他一个机会，只不过，他实在是太弱小了！”
<br/>　　“失去了先天优势，不过，他很聪明，还有机会！”
<br/>　　玄天长老沉声道，这个段惊天虽然弱小，但只要天赋足够妖孽，也不是没有机会。
<br/>　　“希望吧！”宫主面带感慨，长长的叹了口气。
<br/>　　他想到了轮回天尊，那个顶天立地的男人。
<br/>　　那个男人，需要的不是重生归来，而是在无尽的轮回之中，给这个世界寻找到希望。
