将虫子将病人的体内逼出后，有什么措手不及的事情发生，那么不就是给自己找麻烦么。<br/>　　<br/>　　“都检查过了，并且已经初步检测过，没有任何问题。”李普河说道。刚才搭建完成后，他和常文斌也是利用手段都检查了一遍。<br/>　　<br/>　　“那就好！”陈默也开始穿起防护服，算是一种隔离手段吧。<br/>　　<br/>　　对于李普河、常文斌、袁若珊三个人，都不让进去，都是在隔离外边观看就成。<br/>　　<br/>　　陈默进入后，再次检查了一下伤者。发现季乐佳的中枢神经部位的小虫子，已经吃掉了一小点的白质。在脊髓腔室内，形成了小小的中控。<br/>　　<br/>　　而且现在这个小虫子活动的也开始变快，根据它现在的活动定律，可能也就三天以后吧，就可能将伤者的中枢神经部位所有的白质啃食干净。<br/>　　<br/>　　陈默拿出针灸，开始重复早上的事情，也就是将伤者全身的通道全部封闭，去了脑部以外。<br/>　　<br/>　　最后，在做了一些防备手段后，这才从药箱中取出一个盒子。<br/>　　<br/>　　盒子里面是他准备好的引豸丹，这个丹药被一打开后，就有种味道，虽然人闻不到，但是虫子和昆虫是可以的。<br/>　　<br/>　　将这个地方采用隔离设备，其实也有防备外面的小虫子和昆虫，被吸引进房间。所以才需要全部封闭，这样才能专心实验。<br/>　　<br/>　　当引豸丹拿出来以后，在陈默神识下的小虫子，就开始变得活跃起来，并且似乎加大了吞噬的力度。也造成病人的一阵疼痛。<br/>　　<br/>　　虽然季乐佳昏迷不醒，但是疼痛依然让他表情显现，但是由于陈默封闭了脑袋下的一切身体反应，所以除了表情痛苦以外，也就没有其他动作了。<br/>　　<br/>　　将引豸丹放到伤者的鼻息间，在神识的查探下，小虫子已经开始剧烈的活动。此刻的小虫子已经不再啃食伤者的白质，但是却没有移动的意愿，就是在原地扭动不已。<br/>　　<br/>　　陈默稍稍思索了一下，然后就开始用真元催动引豸丹的味道。这下子，似乎是引诱力大了，虫子终于开始爬动起来。慢慢蠕动间，就开始沿着伤者的鼻腔爬出。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
