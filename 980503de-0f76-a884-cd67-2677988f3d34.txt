　　嘭…嘭…嘭…
<br/>　　一连六股强悍火焰，瞬间凝聚。
<br/>　　紫晶灵炎、地脉金火、十界灭生火、星辰之火、沸腾妖火、冰冥幽火。
<br/>　　六股火焰，在一双兽手的恐怖控火增幅下，威势滔天，自成六道火海。
<br/>　　“融。”萧逸低喝一声。
<br/>　　双手齐出，强行将六股火焰融合。
<br/>　　不过片刻，一个流光溢彩的巨大火球陡然而生。
<br/>　　火球，火焰腾腾，温度惊人。
<br/>　　任凭周遭狂风再猛，却吹不起丝毫涟漪。
<br/>　　任凭金光武器再锋锐，亦损不了火球分毫。
<br/>　　“爆。”萧逸暴喝一声。
<br/>　　话音落下。
<br/>　　萧逸一把拉过雾妖，左脚重重一瞪。
<br/>　　借着兽脚之威，瞬间冲天而起。
<br/>　　下方，一声惊天爆响。
<br/>　　一股火光，自火球中爆发，轰击而下。
<br/>　　火光，出现得快，消散得也快。
<br/>　　这是萧逸第一次将六股火焰融合，并凝聚爆发。
<br/>　　此时威力，却让他倒吸一口凉气。
<br/>　　下方，坑洞之内周遭山壁，已然炸得粉碎。
<br/>　　万米山壁，粉碎大半，断壁残垣，倾洒大片。
<br/>　　这些受金光力量沾染无数岁月，坚硬得连下品圣器都只能仅仅插入的山壁，此刻却是脆弱如斯。
<br/>　　万米山壁，崩塌大半，霎时掩盖了下方吸力。
<br/>　　周遭狂风吹袭与吸力霎时消失。
<br/>　　萧逸的身影，在左脚不断猛瞪下，不过数个呼吸间，便已回到了坑洞之上。
<br/>　　嗖…
<br/>　　萧逸再次一个闪身，带着雾妖退离数十里地，这才心有余悸地松了口气。
<br/>　　堪堪在他稳下身影的瞬间，坑洞之内，再次吸力爆发。
<br/>　　金光险地之内，无数稀薄金光力量被吸摄而回。
<br/>　　数分钟后，吸力消失。
<br/>　　飘荡在金光险地各处的金光力量，同样消失无踪。
<br/>　　金光险地，已然是清晨时分。
<br/>　　“呼。”萧逸再次松了口气。
<br/>　　金光险地内的金光异状，出现得快，消失得也快，往往只在数分钟之内。
<br/>　　刚才在坑洞之内，看似时间很长，实则，不过是短短数十秒时间。
<br/>　　数十秒内，萧逸手段尽出，这才安然回到外头。
<br/>　　即便是刚才六股强悍火焰的融合爆发，炸毁了山壁，暂时阻了吸力一阵子。
<br/>　　可若刚才他慢个数秒时间离去，吸力就会再次产生，届时，他怕是再无机会逃脱。
<br/>　　“金光禁地之下，竟是危险如斯。”萧逸摇了摇头。
<br/>　　北隐宗大长老之言，果真没错。
<br/>　　金光禁地之下，确有重宝，而且是极大的好处。
<br/>　　单单是那十八把剑类上品圣器，就已经是让人无比眼热的重宝。
<br/>　　更别说不断溢出金光，且受十八把上品圣器守护的洞府之内了。
<br/>　　虽说不知晓洞府之内到底具体有些什么。
<br/>　　但很肯定，绝非凡物。
<br/>　　“额。”这时，萧逸忽然吃疼一声。
<br/>　　脚步一软，差些瘫坐在地。
<br/>　　一旁，雾妖空洞的眼眸和恐怖的面庞上，发出一声担忧的呼声，“主人。”
<br/>　　“我没事。”萧逸摇了摇头，看向雾妖。
<br/>　　“嗯？你的灵智似乎高了些许？”
<br/>　　雾妖向来笨拙，连说话都不利索。
<br/>　　可现在，空洞的面庞表情上，却露出了一丝担忧。
<br/>　　萧逸想起来，刚才在坑洞里时，雾妖包裹着他，身体不断受金光武器所伤。
<br/>　　那时雾妖也露出了‘痛苦’的表情，但空洞的眼眸下，却也露出了悍不畏死之色。
<br/>　　能大概表现出表情，证明雾妖的灵智比以前高了一些。
<br/>　　灵智，是会增长的，只不过因其产生本体强弱，而导致增长快慢罢了。
<br/>　　雾妖，本体只是一团白雾，比不得那些强悍的火焰，故灵智增长得很慢。
<br/>　　但现在看来，常年在风圣壶内温养，有助于它的灵智增长。
<br/>　　“你也受了伤势吧，先回风圣壶内修养吧。”萧逸说道。
<br/>　　“是，主人。”雾妖应答一声，随后化作一团白雾，回到风圣壶内。
<br/>　　萧逸看了眼风圣壶，淡淡一笑，“日后若有机会，定要让你灵智增长快些。”
<br/>　　萧逸可不愿意他的这个忠诚的‘伙伴’，只会笨拙地喊几句‘主人’，还有‘是…不是…有…没有’这样的简单话语。
<br/>　　收敛下心思，萧逸看向了自己的双手与左脚。
<br/>　　刚才忽然吃疼一声，并险些瘫坐在地，便是因为双手和左脚上忽然传来一阵无力的虚弱感。
<br/>　　这一双兽手，以及兽脚，显然极强，且来自于威力强悍的不知名妖兽。
<br/>　　自己虽然借助八龙焚火炉，将其幻化到自己身上，让得自己能借用其中力量。
<br/>　　但本身自己的肉体承载这些妖兽部位，必然存在着负担。
<br/>　　自己的肉体力量，远远比不上这些妖兽部位。
<br/>　　经历过与北隐宗大长老激战，还有在金光禁地之下的应付18把利剑，以及刚才抵挡那些金光武器风暴。
<br/>　　自然的，自己的肉体力量负荷极大，一时承受不过来。
<br/>　　当然，这些妖兽部位给予的，只是肉体力量上的疲惫，倒也算不得什么伤势。
<br/>　　休息一阵就好。
<br/>　　萧逸散去了一双兽手与兽脚，休息了十数分钟后，再次站起。
<br/>　　嗖…身影一闪，来到了金光坑洞之旁。
<br/>　　这时，金光坑洞内的异状已然消失。
<br/>　　原本被八龙焚火炉吸收殆尽的金光力量，也因为其它坑洞的金光力量倒涌而恢复。
<br/>　　金光坑洞之内，仍旧金光熠熠。
<br/>　　不过，若看真切些，万米之下，坑洞山壁被炸得粉碎了数百米范围。
<br/>　　萧逸摇了摇头。
<br/>　　下面有重宝不假，但无数岁月来，却无人能拿里头重宝，更别说进入洞府之内。
<br/>　　萧逸也不会再下去了。
<br/>　　那18把上品圣器的剑阵，他并不放在眼里。
<br/>　　但洞府前的那股禁制，单单气息，就足以让他颤抖。
<br/>　　那股禁制，绝对不是现在的他能够抗衡的，再下去，也不过是浪费时间。
<br/>　　嗖…嗖…
<br/>　　萧逸的身影，忽然快速闪烁起来。
<br/>　　离开了这个坑洞，一连闪烁了百里范围。
<br/>　　半晌，萧逸停下身影，面露恍然之色，“果然如此。”
<br/>　　在他面前，再次出现了一个金光坑洞。
<br/>　　这个金光坑洞，同样在数十米长宽大小。
<br/>　　不过，萧逸靠前看下去，坑洞之下，却并非是笔直的深不见底，而是弯弯曲曲，一路蜿蜒。
<br/>　　如果没有猜错的话，这个坑洞之下，一路蜿蜒到尽头，同样回到达两万米之下的另一个洞口。
<br/>　　进入洞口，便是洞府。
<br/>　　金光禁地之下，总共有18个洞口。
<br/>　　也就是说，整个金光险地，总共有18个金光坑洞，分别坐落于各处。
<br/>　　萧逸摇了摇头，金光禁地，他已经不打算去闯了。
<br/>　　“金光险地的历练，便到这里结束吧。”萧逸自语一声。
