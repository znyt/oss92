　　“什么？”
<br/>　　“这怎么可能？”
<br/>　　眼看着古白居然是被楚风眠直接逼退。

<br/>　　古白刚刚出手，可是已经毫无保留，动用的都是古月峡谷的绝技。
<br/>　　可这样，居然是被楚风眠几招击退。
<br/>　　古白的名声，在天羽神域中虽然是在荒圣君之下，可也是天羽神域中巅峰的人物，现在被一个不知名的小子逼退，绝对是无人想象过的。

<br/>　　楚风眠看着古白被逼退，突然再度出手，又是一掌，从天而降。
<br/>　　这战龙之爪从天而降，简直便是遮天辟日一般，好像是任何力量，都无法跟此抗衡。
<br/>　　在楚风眠的背后，群龙咆哮，令楚风眠如同化身成为一代龙帝一般。
<br/>　　“龙族武技？小子，休得嚣张！古月降临，加持我身！”
<br/>　　从古白的背后，一尊黑色弯月的虚影，突然出现，这黑色弯月出现的一刻，古白身上的力量气势，暴涨数倍不止。
<br/>　　“古月降临！这是古月峡谷的绝技，相传天地之中有着一尊古月，蕴含无穷无尽的力量，唯有古月峡谷的秘术，可以动用其中的力量！”
<br/>　　“本以为这只是一个江湖传闻，没想到居然是真的！”
<br/>　　一些武者看到这古月出现，笼罩在古白背后的时候，都是极为吃惊，有些出乎意料。
<br/>　　毕竟古月之事，虚无缥缈。
<br/>　　可现在古白身上暴涨的力量，却是不可能是假的，这便是证明古月存在的最好证明。
<br/>　　古月笼罩，将古白身上的最强力量，都给爆发了出来，这才是古白的真正实力。
<br/>　　比起鹿战神，剑祖荒等人，都毫不逊色。
<br/>　　“小子，能够逼出我的古月之力，已经是你的荣幸了，给我死吧！”
<br/>　　古白冷哼一声，脸色通红，他已经是动了真怒。
<br/>　　不然他也不会为了对付楚风眠，便是将这底牌都给尽出，他唤出古月之力，便是要将楚风眠，彻底轰杀。
<br/>　　“古月之力？”
<br/>　　看到古白身上力量的变化，他站在原地，负手而立，似乎是脸色都毫无变化。
<br/>　　就算是这古白的实力足以媲美剑祖荒，鹿战神又如何？
<br/>　　剑祖荒，鹿战神可都是死在了楚风眠的手中了。
<br/>　　这古白，也唯有一样的下场。
<br/>　　“天羽神域中的七大宗门的弟子，都得死，在这彩虹阶梯上，我就先杀了你再说！”
<br/>　　楚风眠冷笑一声，他的身躯再度动了，他身上的战龙血脉，在这一刻，彻底展现出来，龙吟连绵不断，咆哮而起。
<br/>　　在楚风眠身边的灵力，都是化为了一条条真龙来。
<br/>　　在他的手中，祖龙至尊剑猛然出窍，一剑之下，群龙飞舞，每一条真龙，都是一道锋利无比的剑气。
<br/>　　全部冲着古白的身躯，斩杀而去。
<br/>　　“什么？这是什么剑术？如此大话，你就是杀了剑祖荒跟鹿战神的人？”
<br/>　　古白听到楚风眠的话，突然一下子明白过来，楚风眠的身份。
<br/>　　之前在传闻之中，陆战神，剑祖荒便是被一个武者所斩杀的，那个武者扬言要将七大宗门的所有弟子，都给斩杀。
<br/>　　现在听到这话，古白一下子知晓了楚风眠的身份，那个传闻中的武者，就是楚风眠。

<br/>　　古白一边说着，突然声音变大了无数大声喊道。
<br/>　　“荒圣君，星筠，药海，你们难道打算坐视不理了？此人是我们七大宗门的大敌，先杀了此人，我们在来争夺这云羽天宫的宝藏如何？”
<br/>　　古白看到楚风眠的剑气袭来，心中也是深深的明白楚风眠的实力，决不好对付。
<br/>　　他一人想要斩杀楚风眠，绝无可能，如今自然是要联合其他人来，一起围攻楚风眠。
<br/>　　“好！”
<br/>　　荒圣君第一个答应，毫不迟疑，他本来便是就将楚风眠视为心腹大患，现在正是围杀楚风眠的最好机会，他自然不会拒绝。

<br/>　　荒圣君站在这荒古大道之上，便是深处了另外一处空间一般，玄妙无比，身上蕴含玄妙的荒古之力。

<br/>　　荒圣君出手，便是源源不断的荒古之力，冲刷而下，如同一道洪流，要将楚风眠都给死死的冲刷，镇压。
