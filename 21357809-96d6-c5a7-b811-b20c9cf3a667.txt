　　睁眼看去，原来已经到了轩辕之门附近，道风却站住了，在他周围，有黑白两色的气流，组成了一股强大的旋风，飞速旋转之中，不断缩小着范围，道风站立不动，单手结印，用一身法力做着抵抗。
<br/>　　骤然间，从这风暴之中，黑白二气汇聚成了人形，赫然正是清长风，双手一摊，将风暴尽数凝聚在掌心，朝道风打来。
<br/>　　道风一只手搂着芮冷玉，一只手拿出打神鞭，奋力抵挡。
<br/>　　这两大强者之战，又都出了全力，哪怕是法力碰撞产生的余波，从芮冷玉身上掠过，都让她感到窒息。
<br/>　　“道风，你放我下来，我帮你！”芮冷玉还是咬牙说道。
<br/>　　道风不理，独自战斗。
<br/>　　芮冷玉挣扎起来，突然，神识之中响起道风的声音：
<br/>　　“你走之后，别找任何人，去死寂迷林中找菩提神木，将龙晶埋在树下，在那里等着。”
<br/>　　“等着？等着什么？”
<br/>　　道风没再说话，骤然暴起，手持打神鞭，对着清长风冲过去……
<br/>　　芮冷玉只觉得一阵天旋地转，强大的压力，几乎将她的魂魄撕碎，虽然她紧紧抱着道风的脖子，但被这股恐怖的力量一下子推开了，独自在风暴中飘摇，但是很快——最多只有几秒钟的工夫，在她感觉上像是从风暴中穿过，一切都结束了。
<br/>　　芮冷玉跌坐在了草地上，定神看去，道风就站在自己面前，在他斜后方不远处，清长风垂手而立。
<br/>　　“道风……”
<br/>　　“走！”
<br/>　　芮冷玉回身看去，这才发现，身后就是轩辕之门，像一面被雾气笼罩的镜子，将无尽虚空摄于其中。
<br/>　　芮冷玉刚想开口，只听见咔嚓一声，道风的额头上，突然出现了一道裂缝。
<br/>　　裂缝不断分叉，朝身体蔓延下去，使道风看上去，如同一块即将碎裂的瓷器。他静静地看着芮冷玉，再次命令道：“走！”
<br/>　　清长风已经纵身飞来。
<br/>　　芮冷玉一咬牙，转身钻进了轩辕之门。
<br/>　　她不是想要逃走。
<br/>　　她也不知道，身外化身的毁灭，对道风的本尊有什么影响，她只知道，道风拼着身外化身的毁灭，也要让她带着那枚龙晶离开，自然有他的缘故。
<br/>　　绝不能让他的化身白白牺牲！
<br/>　　清长风看她离去，将要去追，道风突然开口说道：“她没有山海印。”
<br/>　　清长风一滞，停下来望着他说道：“如何信你。”
<br/>　　“她若有山海印，便能穿越时空，你如何找得到她。”
<br/>　　清长风猛然醒悟，盯着他，道：“在你身上？”
<br/>　　“在我身上，我便早就带他们走了，何须这么费事。”
<br/>　　清长风沉吟起来。
<br/>　　“去找我师弟吧，他若死，你们永远得不到山海印。”
<br/>　　“便是得不到山海印，又能如何！”
<br/>　　清长风上下打量他。
<br/>　　“我师侄呢？”
<br/>　　道风知道他说的是假叶少阳，微微一笑，“他若活着，我如何出洞！”
<br/>　　“你杀了他！”
<br/>　　清长风厉声喝道。
<br/>　　道风不语，身上裂缝继续产生，已经逼近了临界点。
<br/>　　清长风深吸一口气，强自压住愤怒，冲道风点头，咬牙道：“好，道风，我乃神念幻化，你是身外化身，这一战并不过瘾，来吧！”
<br/>　　说完，人影消失，化作一道长风，从道风身上掠过。
<br/>　　轰……
<br/>　　道风的肉身四分五裂，崩毁在地上。
<br/>　　清长风待要离去，突然觉察到异样，低头看了一眼，道风肉身崩毁之后，却并未立刻融化，地上是一滩鲜红的血肉和白森森的骨骼。
<br/>　　“什么，这是真的肉身！”
<br/>　　清长风怔住，随后一想，他倒是明白了，的确啊，若非是真的肉身，怎能斩杀入魔后的叶少阳，又跟自己打了这么久……
<br/>　　至于肉身来历，他到是清楚：道风斩三尸第一尸就是肉身，肉身被斩，人当然是死了，但不代表肉身就不能再用——可以修炼成身外化身，平时灵魂可以附着在上面，这样到了人间，依然有肉身可以用，看上去有实质的身体，但因为与魂魄分离，根本不用拘泥人间的规则：
<br/>　　能飞，能遁形，肉身不会衰老，也没有寿命（所以道风看上去一直很年轻），这便是斩肉身的好处，也是斩三尸的第一步。
<br/>　　肉身斩却之后，需要长期祭炼，祭炼的时间越久，通灵性越是强，只是……这肉身再好，也只有一个，对主人来说，好好远超过一切法器，正因此如此，清长风根本不敢相信，道风居然舍弃了这具肉身！
<br/>　　这对他来说，等于自断一臂，今后，就算他再怎么修炼身外化身，也得不到更好的了，毕竟是自己的肉身……
<br/>　　清长风都替道风感到肉疼。
<br/>　　“这家伙，可真是舍得啊！”
<br/>　　清长风摇了摇头，心中却也感到些许安慰：道风肉身之死，自然是被假叶少阳重伤在前，然后才为自己所灭，这么一想，那个假叶少阳用自己的性命换掉了道风的肉身，也算值得了，将来等星月奴醒了，自己跟她也算有了交代。
<br/>　　站在山冈上的清长风睁开了眼睛。朝道风看了过去。
<br/>　　道风也在看着他。
<br/>　　“道风，你是够狠的，只是，你付出肉身，也要送鬼童出去，却是为何，你若不想她死，又为何要带她进来？”
<br/>　　道风轻轻摇头，“真相，根本不在她身上。”
<br/>　　“什么？”
<br/>　　“你稍后便知。”
<br/>　　道风转头扫了一眼，上古邪神跟山山禅师，都还在入定之中，显然仍然在幻境中缠斗着，山山禅师头顶有金波流动，上古邪神头顶流淌的是青色气息。
<br/>　　两股气息都朝着对方飞过去，在中间汇合在一起，交叉旋转，青气霸气凌然，扶摇直上，金波不屈不挠，死死纠缠，两者谁也压不住谁。
<br/>　　“怪了！”清长风摸着自己的下巴，心中疑惑不定，这上古邪神，再怎么厉害也是道风的属下而已，就算是道风，都不敢说跟山山禅师能打成平手，他一个邪神，怎么这般厉害？
<br/>　　“清长风，你等速战速决，莫要打扰我这彩云峰的清静！”
<br/>(本章完)
