　　整个途山镇的武者都被吓傻了，被苏莫的实力吓傻了。
<br/>　　一个武皇境九重的武者，居然瞬息间斩杀武帝，此等实力，绝对是那种超级天才。
<br/>　　而且，众人都看到了，途家家主的战魂，居然被眼前这个恐怖的青年吞噬了。
<br/>　　吞噬别人的战魂，这是什么能力？
<br/>　　而那些途家的人，见到家主死亡，个个吓的面色惨白，而后急速离去。
<br/>　　“大家继续建造吧！”苏莫身形落地之后，没有理会那些逃走的途家之人，向苍穹神宫的众人吩咐道。
<br/>　　众人点了点头，随即纷纷返回，继续建造府邸，夕儿也前去帮忙了。
<br/>　　而苏莫飞到了不远处的青山上，盘膝打坐了起来。
<br/>　　他一边打坐，一边心中思索，思索自己的武魂。
<br/>　　因为武者晋升武帝之间，武魂要进化为战魂，别人冲击武帝之境，自然是努力让武魂进化。
<br/>　　但他却是不同，因为他的武魂可以晋级，与别人的完全不同。
<br/>　　先不说他的武魂能否晋升为战魂，就算能晋升为战魂，是不是要等到武魂等级提升到极致之后，再进行进化？
<br/>　　武魂一旦进化为战魂之后，已经没有了等级之分，所以苏莫在思索。
<br/>　　他将武魂等级提升到极致进化的战魂，肯定比现在天级六阶武魂晋升的战魂强大吧！
<br/>　　对于此，苏莫暗暗猜测，不过，目前他对于这些了解的极少。
<br/>　　思索了良久之后，苏莫便摇了摇头，抛却了心中的杂念。
<br/>　　此事，待到达神武学府之后，多了解一些战魂的情况，再行考虑吧！
<br/>　　随即，苏莫静心修炼了起来。
<br/>　　三天时间，匆匆而过。
<br/>　　经过千名苍穹神宫强者，三天的劳作，一个浩大的府邸，便建造了起来。
<br/>　　这个府邸，差不多有林阳城那么大，颇为辽阔，占地方圆百里。
<br/>　　此府邸虽然不算很大，但也能勉强容纳百万苍穹神宫的弟子了，后期再扩建便是。
<br/>　　只不过，府邸虽然建成，府邸内却是非常的简陋，粗糙不堪，用于弟子居住的房屋，更是远远不够。
<br/>　　当然，这些苏莫也无需操心了，让门下弟子自行建造便可。
<br/>　　百万苍穹神宫的弟子，全部被苏莫放了出来。
<br/>　　一时间，整个府邸之中，热闹非凡，无数的弟子在建造自己的住所，将不远处小山上的大树都砍光了。
<br/>　　一间新建的石殿之中，苏莫和一众苍穹神宫的高层，正在议事。
<br/>　　“苏莫，你这就要离开吗？不如在此多住一段时间？”帝释向苏莫问道，因为苏莫已经表示，离开此地，单独前往神武学府。
<br/>　　“不留了，明日一早便离开！”
<br/>　　苏莫摇了摇头，叹道：“以后苍穹神宫，便靠你们了。”
<br/>　　他也想在此多住一段时间，但是他却不能浪费时间，他有太多的事情要做。
<br/>　　其一，巫族迟早会杀来，他必须尽快提升实力。
<br/>　　其二，那就是可怜的青璇，仿佛成了他心中的一个结，他眼睁睁的看着对方落难，却是无力相救，这种无力感，让他时刻想要变强。
<br/>　　其三，那就是芊寻月，他只有突破到武帝之境，才有希望救活对方。
<br/>　　“苏莫，你放心吧！有我们在你大可不必担心！”龙腾说道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，环视了一眼龙腾、司空炎等九宫之主，道：“我不在的日子里，你们九人为尊，全权处理整个宗门的事宜！”
<br/>　　苏莫对于司空炎等九人还是比较放心的，九人已经完全成长了起来，均有接近武帝的实力了。
<br/>　　而其中战乱和龙腾，都已经有超越普通武帝一重武者的实力了。
<br/>　　以几人的实力，苍穹神宫在这小镇上，就是绝对的霸主
<br/>　　“是！”众人闻言点了点头。
<br/>　　随即，姜枫然仿似想到了什么，问道：“苏莫，这个途家怎么处理？昨日他们派人来，想与我们和解！”
<br/>　　“这个你们自己看着办吧！”苏莫随意的道，这种小事他完全不想管。
<br/>　　随即，苏莫又仔细的安排了一番，苍穹神宫日后的发展方向。
<br/>　　他将身上的不少好东西，都留给了众人。
<br/>　　他修炼的五门帝级上品功法，也全部括印了下来，留给需要的人修炼。
<br/>　　帝皇剑，送给了钱副阁主。
<br/>　　甚至，他将有些残破的虚界神图，也给予了夕儿。
<br/>　　此图，已经不需要了，留下来说不定苍穹神宫就能用的到。
<br/>　　苏莫的愿望，是希望将来的苍穹神宫，在古灵星河也能成为一方强大的势力。
<br/>　　这样的话，无论他将来如何，苍穹神宫都能屹立不倒。
<br/>　　最后，苏莫挥退众人，一众苍穹神宫的高层，纷纷离开了石殿。
<br/>　　片刻之后，石殿中除了苏莫之外，便只剩下了两人。
<br/>　　这两人便是夕儿和李峰。
<br/>　　夕儿作为苏莫的妻子，夫君即将远行，她当然不想分开，想要多陪伴苏莫。
<br/>　　而李锋，却不知为何留了下来。
<br/>　　“李峰，你的修为怎么样？”苏莫望向李峰问道，因为他早已发现，李峰夺回肉身之后，修为没有丝毫减退。
<br/>　　换言之，李峰继承了巫山的一身修为。
<br/>　　李峰面色稍显古板，闻言向苏莫躬身一拜，道：“我现在是武皇境五重修为，伯父为我挑选了一门帝级中品功法，不过，与体内玄力属性不契合，现在还处在转换之中。”
<br/>　　李峰口中的伯父，指的便是苏洪。
<br/>　　苏莫闻言微微颌首，巫山修炼的功法，与他们苍穹神宫的功法，完全不同，自然不会契合。
<br/>　　不过，要不了多久，玄力就会改变过来，只是需要一个过程而已。
<br/>　　李峰言罢，略一沉吟，道：“苏莫，我也想离开这里，出去闯荡！”
<br/>　　苏莫闻言眉头微皱，道：“李峰，你现在修为还低，还是待将来达到武帝之境，再外出闯荡吧！”
<br/>　　古周星强者如云，以李峰目前的修为出去闯荡，实在危险，所以苏莫并不想让对方出去。
<br/>　　“苏莫，我留在这里，一辈子都不可能成为武帝。”李峰苦笑着说道。
<br/>　　苏莫闻言一怔，随即心中恍然，对方继承了巫山的修为，却并不能继承巫山的天赋。
<br/>　　而李峰的武魂，不过是人级七阶武魂，天赋实在是太低了。
<br/>　　“不过，以你现在的修为出去闯荡，实在太危险了！”苏莫还是有些顾忌。
<br/>　　李峰闻言叹了口气，他知道苏莫是为他考虑，但他心意已决。
<br/>　　“苏莫，只有在逆境中，我才能冲破束缚，希望你让我走吧！”李峰一脸决绝之色。
<br/>　　苏莫见此心中暗叹，他就是现在说服了对方，待自己走后，估计对方还是回离开。
<br/>　　“好吧！你自己多加小心！”少倾之后，苏莫无奈的点了点头。
<br/>　　“苏莫，后悔有期！”
<br/>　　李峰向苏莫一抱拳，郑重的说道：“若是我不死，它日必定会回来。”
<br/>　　“保重！”苏莫起身，同样向对方抱拳。
<br/>　　随后，李峰便离开了，转身大步而去。
<br/>　　望着李峰离开的背影，苏莫突然觉得李峰变了。
<br/>　　但具体是哪里变了？他也说不上来，或许是连番的打击，让对方成长了吧！
<br/>　　随即，苏莫便陪伴着夕儿，在府邸中漫步，两人静静的享受着离别前最后的时光。
<br/>　　一夜无话，翌日清楚，苏莫便离开了途山镇，化为一道流光，消失在了天际之中。
