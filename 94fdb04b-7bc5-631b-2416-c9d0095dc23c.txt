        &lt;h3&gt;第三卷第六十一章  团团接电话&lt;/h3&gt;
<br/>
<br/>        江俞轩接到了郝景文的电话。
<br/>
<br/>        江大少现在是一人兼多职，时间安排的满满的，家族的事情他也在慢慢的接手，好的是现在社会发展很快，互联网时代已经将人与人之间的距离拉近了，邮件，qq、视频等等软件给人们办公也带来了好处。
<br/>
<br/>        他正在收拾行李，准备去洛城。洛城的那个项目如果能做下来，他们也是很有成就感的，不让陈子昂露面其实是真的对双方都好。
<br/>
<br/>        傅海生对陈子昂的情感，是个人都会看出来，再能隐藏的人，当看到自己喜欢的人时，那种情感是不可抑止的，是自然流露的。越是想刻意的隐瞒，越是隐瞒不住。
<br/>
<br/>        西城分公司现在也壮大了，没有必要事事都让陈子昂露面。他这一次陪着陈子霖过去，之后也就不会过去了。
<br/>
<br/>        陈子昂如果一旦去了甸城，公司的重担基本都会落到他的身上，而且他还要照顾照顾爸爸的情绪，还得适当的管理一下家族的事情。
<br/>
<br/>        和睿琪大哥他们合伙的公司他基本都是个甩手掌柜，都是大哥和文仔在管理，他也就是偶尔出差的时候拓展一下新业务，上次到洛城的时候，陈子昂对她姐姐陈子乔说了这个事情，陈子乔又给她丈夫长常啸林说了，常啸林倒是个热心人，直接就拿下了这个事情，不几天就在洛城成立了一个分部，而且业务非常好。
<br/>
<br/>        “文仔，好久不见。”他们确实是好久不见，大家都在忙碌着，他们是领导不错，但是他们都在前进的路上各自奔波。
<br/>
<br/>        他们三个人都是双重身份的人，都有自己的家族企业，他们丢掉自己目前的身份，从头做起，也想通过自己的努力来开展一片新的世界。
<br/>
<br/>        在时代改变的浪潮下，每个人、每个企业都是一粒沙，潮来潮往，不经意见就会被卷走，但是一定要相信，美好的花一定会开，美好的事物一定也会接踵而来。
<br/>
<br/>        “轩轩，我们最近又准备搞一个大事情了。”
<br/>
<br/>        “哦，什么大事情啊？”
<br/>
<br/>        “见面说，到时候叫上陈子昂，你什么时候有时间？”
<br/>
<br/>        “我明天一大早要去洛城，从落城回来我们见？”
<br/>
<br/>        “好，不过那边的业务也不错呢。”郝景文这个公司的实际负责人知道公司的运转情况，各个地方对接的业务也是蒸蒸日上。
<br/>
<br/>        “大哥最近好吗？”
<br/>
<br/>        “好呢，大哥天天观察着去寻找商机，我也是很少见到他，忙得不行。”
<br/>
<br/>        “抽时间我们三个好好聚聚。”
<br/>
<br/>        “行，等你从洛城回来吧。”
<br/>
<br/>        郝景文并没有给江俞轩说是什么事情，目前他们的计划还是在萌芽状态，只有等几个人在一起好好商讨一下该怎么实行，陈子昂对信息口子上的事情比较了解，如果全面铺开的话，必须得像他们现在的物流系统一样有一个适用的操作软件。
<br/>
<br/>        江俞轩现在知道了大哥还会有项目要做，让叫上陈子昂可能是会有要和陈子昂合作的范畴，如果纯碎是个人聚会的话大哥是不会叫上陈子昂的。
<br/>
<br/>        江俞轩又拨通了陈子昂的电话。
<br/>
<br/>        陈子昂正在看着书，陪着俩个孩子写作业，俩个孩子已经养成了每天将作业写完再玩的习惯，现在是暑假期间，老师有布置的作业，他们俩个比拼着将老师布置的作业已经写完了，现在写的都是自己在外面买的习题，两个小脑袋面对面的坐着写作业，都是专注而认真的神情，让陈子昂突然就想起了张函，张函每次看书的时候就是那么专注，不过偶尔会转过来看看陈子昂，对陈子昂笑一笑，后来俩个人在一起了，他就会乘陈子昂不注意的时候香一个。
<br/>
<br/>        陈子昂的思绪已经飘回到几年前和张函在一起的情景了，孩子们都已经大了，张函过得也很幸福吧？
<br/>
<br/>        这么多年来，他都不曾找过她，除了上一次在信息会上见了一面，也不曾和她有过交集，他可能已经都忘记他们在一起的时光了吧？
<br/>
<br/>        电话铃声响了好几遍，陈子昂都没有听见，团团拿起妈妈的电话：“妈妈，江叔叔的电话。”
<br/>
<br/>        虽然大家都不知道陈子昂有了孩子，但是陈子昂会回来告诉孩子们她公司的情况、告诉孩子公司有哪些人，都是做什么工作的。
<br/>
<br/>        俩个孩子也能知道陈子昂公司的大致情况。
<br/>
<br/>        “妈妈，江叔叔的电话。”团团又冲着妈妈喊了两遍，陈子昂才回过神来，自从上次见了张函之后，她经常会想起张函。
<br/>
<br/>        “妈妈，你怎么了？”小帅哥已经发现了妈妈的反常。
<br/>
<br/>        “没怎么。”陈子昂都不知道她的泪水已经顺着脸颊流下来了。
<br/>
<br/>        “妈妈，有伤心的事情要说出来给团团听，团团可以为妈妈分担一点了。”
<br/>
<br/>        “妈妈，嘟嘟也能为妈妈分担，一个人的痛苦说出来大家分担一些，然后就不算是痛苦了，一个人的快乐分担出来，大家都会快乐的。妈妈，我爱你！”嘟嘟接过哥哥手中的纸巾替妈妈擦去脸上的泪水。
<br/>
<br/>        “江叔叔，您好，你有什么事情吗？”团团看着妹妹在安慰着妈妈，便接起了电话。
<br/>
<br/>        江俞轩在电话里听到稚嫩的童音，便知道是陈子昂的孩子接电话，他沉默了一下，陈子昂从来都没有告诉过她有孩子的事情，娃接电话，陈子昂知道吗？
<br/>
<br/>        “你好啊，小朋友，我找一下陈总，如果她方便接电话了，麻烦你帮我转述一下好吗？”
<br/>
<br/>        “好啊！”
<br/>
<br/>        陈子昂已经从回忆中回过神来，见到贴心的孩子心里又是一阵愧疚，她不能将他们藏一辈子吧？
<br/>
<br/>        她也没有要过电话，就让团团和江俞轩继续说话，她也没有必要瞒着江俞轩什么了。
<br/>
<br/>        “你告诉陈总，就说过几天我们会和寒总见面，时间等我从洛城回来再定。”
<br/>
<br/>        “好的，江叔叔，我一定转述到，您还有什么事情吗？”
<br/>
<br/>        “我没有什么事情了，可以告诉叔叔你的名字吗？”
<br/>
<br/>        “我叫陈虞，小名又叫团团。”
<br/>
<br/>        “是沉鱼落雁吗？”江俞轩当然知道俩个孩子的名字，张函经常给他讲。
<br/>
<br/>        “嗯，叔叔真聪明，妹妹的名字就叫落妍，小名嘟嘟。叔叔还有别的事情吗？”
<br/>
<br/>        “叔叔没有别的事情了，谢谢团团，带我向妹妹嘟嘟问好啊。”
<br/>
<br/>        “好的，叔叔，谢谢您平时照顾妈妈！再见！”
<br/>
<br/>        陈子昂看着儿子彬彬有礼的和俞轩聊着电话，心里莫名的感动，她的儿子好懂事的有没有？
<br/>
<br/>        而江俞轩看着手中的电话微微发愣，陈子昂的儿子也太懂事而了吧？还谢谢他平时照顾他妈妈，是陈子昂平时会对孩子提前他吗？
<br/>
<br/>        陈子昂会怎么将他介绍给孩子的？
