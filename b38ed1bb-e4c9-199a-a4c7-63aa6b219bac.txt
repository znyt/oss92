　　　　双头蛇，双头蛇。
<br/>　　涂小安古怪的看着地板上那条两颗蛇头摇曳，凶煞可怖的双头蛇，不知作何感情。
<br/>　　他也算见过世面，曾经也是白镇有名的少年捕蛇人，什么样的毒蛇没见过，但说实话，他还是第一次见到两颗蛇头的蛇。
<br/>　　当然，双头蛇并不是不复存在，世界上早有出现，属于一卵双胎，在发育过程中由于受环境等因素的影响，形成了畸形胎，然后颈部分叉的两头蛇，是一种非常罕见的蛇种。
<br/>　　曾经在佘山，就有捕蛇人发现一条很小的双头蛇，可那蛇，蛇身首尾各长一个脑袋，并不是由颈部分叉，两个脑袋都能控制它们的身体，十分奇特。
<br/>　　这种叫被称为钝尾两头蛇，已经被鉴别，是蛇的基因受到了污染或在染色体复制或配对过程中产生了错误，使蛇长出了两个头。
<br/>　　另外印度，那个国家的人是非常非常崇拜蛇的，视蛇为“神”的化身，比白镇的人民更加的疯狂，当地人极度崇拜蛇文化，对印度人来说，有意或无意杀死一条蛇都是犯下弥天大罪。
<br/>　　在他们的神庙中，就出现了一条五个蛇头的大蛇，五个蛇头就宛如手的五个手指，远远看去就一只巨大的手掌。
<br/>　　所以，仅仅双头蛇，其实并不算什么，然而，眼前这条双头蛇就要另当别论，被系统定位超强进化体蛇种。
<br/>　　那是硬生生被雷电劈的变异从而进化出来。
<br/>　　一跃成为觉醒生物，颇有一跃龙门的姿态。
<br/>　　光它身上的气息，就无比狂暴阴冷，比当初变异吸血蛇还要可怕。
<br/>　　它从人的身体孕育，十二条融为一条，雷电加身变异进化，匪夷所思的诞生。
<br/>　　诸多的因素才铸就这一条超强变异双头蛇。
<br/>　　可以说是天地间唯一的存在。
<br/>　　以后也不太可能诞生这种蛇种。
<br/>　　“嘶嘶...嘶嘶...”
<br/>　　变异双头蛇此时通灵摄厉，模样乖张，两颗蛇头摇曳的无法无天，有视一切生物为蝼蚁的睥睨之姿。
<br/>　　它发现了悬浮在空中的金银蛇，两双蛇眸冒着凶光，蛇口一张，便是锋利毒牙。
<br/>　　“哼，不知所谓”
<br/>　　涂小安察觉到双头蛇对自己的敌意跟挑衅，顿时冷淡无波的看着它，任你起点在高，血统再强，哪怕是觉醒生物，终归只有十级。
<br/>　　也敢在他面前放肆？
<br/>　　只见悬浮的金银蛇骤然出现在了双头蛇的眼前。
<br/>　　“吼...”
<br/>　　双头蛇两双眼睛射出冷冷的光，瞬间将这条金银蛇视为自己的食物。
<br/>　　涂小安顿时蛇眸一凝，周身王者之气激荡出来。
<br/>　　他统治能力一开，王者之气笼罩在变异双头蛇身上，霎时间，双头蛇睥睨之姿骤消，一下子露出了难以置信。
<br/>　　这金银蛇比它还强？
<br/>　　“给我跪下”
<br/>　　在他面前，什么蛇敢称尊，别说这条才刚刚出世的双头蛇了。
<br/>　　它或许潜力很大，将来可以成为无敌的存在，但是，那不过是后话，能成长起来，才是强者。
<br/>　　双头蛇开始往后面，两个蛇头向后仰，金银蛇给它一股无比强烈的危机感，甚至都不能反抗。
<br/>　　眼前这条金银蛇，仿佛是它不可战胜的存在。
<br/>　　“还不跪下，臣服与我”
<br/>　　涂小安一声大喝，体型暴涨五米，扶摇直上，身上的王者之气如一座大山一样排山倒海的压倒过去，辗压一切。
<br/>　　双头蛇倏然地的诚惶诚恐起来，看着一下子变大的金银蛇，不敢在做反抗，匍匐在地，两颗蛇头贴在地面，就跟迎接帝王一样。
<br/>　　它刚霸道的诞生，却瞬间被打脸。
<br/>　　“你若等级高，我便吞了你再次进化，可惜现在的你，还不配”
<br/>　　涂小安意念一动，强大的意识力骤然笼罩在变异双头蛇身上，这条蛇的血统很高，由于等级不够，吞了对他也没多少好处，还不如收了当宠物，好生培养。
<br/>　　这种蛇，既然能长出两颗头，那么将来在长出第三颗头，也不是不可能。
<br/>　　“给我收”
<br/>　　区区十级，在加上已经被涂小安的气息压倒，根本不能反抗。
<br/>　　噌...
<br/>　　双头蛇瞬间原地凭空消失。
<br/>　　下一秒，涂小安直接召唤出来。
<br/>　　“主人”
<br/>　　一道冰冷的声音出现在涂小安的脑海中，这超强进化的变异双头蛇，才不过十级，还不具备口吐人言的资格。
<br/>　　“好，你就是本蛇王第一条蛇类宠物”
<br/>　　双头蛇也是命不好，刚超强变异进化出来，就一下子沦为宠物。
<br/>　　它都还没有开始装逼呢。
<br/>　　涂小安以退为进的将双头蛇收为宠物，他万万没想到，打一个蛇胎，还打出了一个宠物出来。
<br/>　　不过这算意外的收获。
<br/>　　同时，涂小安心中的谜团越发的解不开，显然，舒瑶怀的不是一般的蛇胎。
<br/>　　就算没有雷电加身，十二条黑蛇最后存活下来的胜者，也是不可小觑。
<br/>　　起码也是变异蛇种。
<br/>　　可以说，这十二条小黑蛇的基因本身遗传就很强大，绝对是觉醒蛇的后代。
<br/>　　舒瑶的蛇胎到底是怎么怀上的，这房间内，根本没有一点觉醒蛇的气息存在。
<br/>　　他迫切的想知道一切。
<br/>　　就在这时，套房的门被破开了。
<br/>　　一位年轻男子大步流星的掠了进来，头发无风自动，整个人都非常的亢奋。
<br/>　　“哈哈...我的孩子出世了，我感应到了”
<br/>　　年轻男子瞬间就看到了地板上的变异双头蛇，眼睛闪闪发光：“好啊，看来已经完成变异了，既然还是双头蛇，比我预期的还要强”
<br/>　　咦...
<br/>　　“怎么还有一条蛇”年轻男子看到旁边还有一条金银蛇，顿时看向它，非常的意外。
<br/>　　“不，它不是我的孩子，跟我没有血脉相连”
<br/>　　年轻男子死死的盯着金银蛇，察觉到它的不一般。
<br/>　　涂小安更是意外，这突然冲进来的男子给人非常妖异的感觉，顿时诧异：“好强的蛇类气息”
<br/>　　对，没错，这年轻男子身上有一股很恐怖的蛇类气息，他在说什么。
<br/>　　说双头蛇是他的孩子？
<br/>　　难道....
<br/>　　这人有点眼熟，好像哪里见过。
<br/>　　涂小安灵光一闪，蛇眸朝了一个角落看去，只见一张照片像长了翅膀的飞到了他的面前。
<br/>　　照片上是一对情侣相依偎的场景，那照片上的男孩，不就是眼前这个男人吗。
<br/>　　只是成熟了。
