　　苏莫望着被巨大玄力气罩守护住的巫殇，眉头微皱，对方撑起这么大的气罩，的确是让他没有办法偷袭了。
<br/>　　因为对方将有足够的时间应对了，苏莫叹了口气，看来只能用强力击败对方了。
<br/>　　他没有急着现身，绕过一众游荡的骷髅战士，来到了巫殇的侧方，对方的玄力气罩之前。
<br/>　　他要以最快的速度击败对方，让那些骷髅战士都来不及救援。
<br/>　　这些骷髅战士虽然单体实力不是多强，但胜在数量多，而且还杀不死。
<br/>　　苏莫不想用吞噬战魂吞噬这些骷髅，只能以雷霆之势击败巫殇了。
<br/>　　体内五枚神丹尽皆催动了起来，属性玄力融合，疯狂的涌入手中长剑之中。
<br/>　　这一刻，苏莫催动了十五种剑之战魂，狂暴的剑威瞬间冲散了他身上的气流，让得他的身形暴露而出，
<br/>　　“斩！”厉喝一声，苏莫出手了，瞬间连斩三剑。
<br/>　　霎时之间，三道恐怖的剑气瞬间撕裂了巫殇的防御气罩，向对方袭杀而去。
<br/>　　这三道剑气骇人无比，每一道都长达数万丈，横贯天地，蕴含的劲力惊如汪洋大海，弥漫的锋芒仿佛能撕碎人的灵魂。
<br/>　　三道剑气呈品字形，瞬间跨越百里的距离，临近了巫殇。
<br/>　　而苏莫，连斩三剑之后，也没有收手，身形一闪，如同一道疾驰的闪电，仅随在剑气之后冲了出去。
<br/>　　“什么？”巫殇大惊失色，他不是惊讶苏莫突然出手，因为他时刻都在防备着。
<br/>　　他震惊的是，苏莫这三道剑气的威力，让得他头皮发麻，心中生出无比危险的感觉。
<br/>　　三道剑气已经封死了巫殇的躲避路线，来不及多想，电光火石之间，他一边身形急退，一边奋力还击。
<br/>　　“玉虚神掌！”
<br/>　　巫殇大喝一声，手掌猛然向前一推，一只巨大的玄力手掌横推虚空，破灭一切。
<br/>　　这掌印大如山峰，通体虚幻，但又闪烁着晶莹的光泽，威势亦是惊天动地。
<br/>　　掌印一出，虚空顿时爆炸，承受不住掌印磅礴的威势。
<br/>　　瞬息之间，三道三色剑气和玉虚神掌硬撼在了一起。
<br/>　　轰！轰！轰！
<br/>　　三声巨响震动了整个战界，巨大的玉虚掌印瞬间炸裂，化为了无尽纷乱的气流。
<br/>　　而三道剑气虽然威势有多减弱，但去势丝毫不减，直接向巫殇身上袭杀而去。
<br/>　　“不好！”巫殇见此脸色剧变，之前他受了一点伤，导致实力有所减弱，没想到完全挡不住苏莫的攻击。
<br/>　　好在他急速暴退，现在还有再次还手的机会。
<br/>　　轰！
<br/>　　虚影升腾，两种战魂顿时从巫殇身后升腾而起，强大的气息弥漫八荒。
<br/>　　这两种战魂都不简单，一种是一块一尺大小的绿色玉石，一种是狰狞恐怖的异兽。
<br/>　　此兽高大三丈，乃是人形，头似牛，脸似马，仿佛传闻中的牛头马面一般，只不过全身毛发极长，甚是怪异。
<br/>　　此兽名为冥马兽，也是一种很强大的异兽，非常的少见。
<br/>　　两种战魂乍一出现，巫殇身上的气息瞬间暴涨，绿色的玉质之光伴随着凶煞之气，冲天而起。
<br/>　　“破！”一声怒喝，巫殇再次出掌，又是一记玉虚神掌轰击而出。
<br/>　　轰隆隆！！
<br/>　　这一次，全力爆发的巫殇，终于将苏莫的三道剑气彻底击碎，玉虚掌印和三道剑气全部崩裂，化为漫天气流，涌向八方。
<br/>　　“败吧！”就在此时，蓦地，苏莫冷漠的声音响起。
<br/>　　只见苏莫的身形，仿若是一支疾驰的利箭，身上闪烁着绚丽无比的光泽，洞穿汹涌的气浪，瞬间临近了巫人王。
<br/>　　苏莫身上的光芒，是特殊体质的异像，以及护体的三色玄力。
<br/>　　金刚之体和琉璃宝体，赋予了他超强的防御力，再加上护体玄力，他根本无惧这些攻击余波。
<br/>　　事实上，就算是让苏莫以血肉之躯，硬抗巫殇一掌，他都不会死。
<br/>　　因为他的防御力，已经达到了非常骇人的地步了。
<br/>　　咻！
<br/>　　刺耳的破空声响起，空间被洞穿，苏莫手中宝剑光芒璀璨，直刺巫殇的脑袋，瞬间临近了对方的脑门。
<br/>　　“糟糕！”巫殇再次大惊，苏莫的攻击一泼接一波，让他几乎连喘息之机都没有。
<br/>　　而且两千只骷髅战士，根本来不及过来救援。
<br/>　　凌厉的锋芒还没刺中脑门，巫殇已经感觉脑门的血肉被凌厉的锋芒撕开。
<br/>　　“滚！”
<br/>　　仓促之间，巫殇来不及凝聚招式，只能大手一探，再次向迎面而来的长剑抓去。
<br/>　　铛！
<br/>　　一声剧烈的金铁交鸣之声响起，巫殇的手掌再次和苏莫的长剑接触到了一起。
<br/>　　然而，上一次他能一把抓住苏莫的长剑，这一次却是没有这么好的运气了。
<br/>　　只见巫殇的手掌乍一和剑身接触，便瞬间被震开，一触即分。
<br/>　　这一次，苏莫催动了十五种剑之战魂，以及特殊体质的力量，这一剑的威力比之前偷袭的那一剑，不知强大了几倍。
<br/>　　巫殇想以手掌抓住苏莫的长剑，简直是痴人说梦。
<br/>　　毫无意外，长剑荡开了巫殇的手掌，瞬间刺进了巫殇的脑门之中。
<br/>　　而后，苏莫手中劲力一吐。
<br/>　　嘭！
<br/>　　一声闷响，巫殇的脑袋顿时四分五裂，炸成了碎片。
<br/>　　脑袋炸裂，巫殇的肉身顿时身死，身体向下方坠落而去。
<br/>　　嗖！嗖！
<br/>　　而巫殇的两种战魂，在肉身死亡之后，立刻急速逃窜。
<br/>　　肉身已死，只剩下两种战魂，他的实力暴降，哪还敢逗留分毫？
<br/>　　“逃得了吗？”苏莫冷笑一声，立刻催动了吞噬战魂，强大的吞噬之力立刻汹涌而出。
<br/>　　玉石战魂和冥马兽战魂，受到吞噬之力的影响，顿时速度暴降，几乎停懈不前。
<br/>　　“回来吧！”苏莫大手一抓，瞬间幻化出一只巨大的玄力手掌，一把将两种战魂抓了过来。
<br/>　　而后，他狠狠一捏，几乎将两种战魂捏爆了。
<br/>　　啊！啊！
<br/>　　两种战魂之中，均是传出巫殇凄厉的惨叫声，显得痛苦无比。
<br/>　　随即，苏莫意念一动，立刻将两种战魂收进了玉镯空间之中。
<br/>　　至此，此次挑战结束，真圣碑上‘巫殇’之名，逐渐消失。
<br/>　　整个真圣界内，雅雀无声，众人怔怔的望着这一幕，暗暗震惊。
<br/>　　巫殇败了！
<br/>　　败的比巫人王还要凄惨！
<br/>　　肉身惨死，战魂被擒，一败涂地！
