        诺爸见状，从收营台下拿出一个小药箱递给了诺筱颖，并叮嘱道：“山里蛇多，自个儿还是要多小心点儿。”
<br/>
<br/>        “嗯，我知道了，那我上山去采药了！”诺筱颖接过诺爸手中的药箱，放在背后的竹篓里，然后去了后门，踏上了田间小路，往对面的山头走去。
<br/>
<br/>        回来的这些天，她每天都有帮诺母上山采药，一个山头采完，就去下一个山头，而且一个山比一个山要深。
<br/>
<br/>        大山里没有城市的繁华与聒噪，倒是有大自然的繁盛与灵动。
<br/>
<br/>        诺筱颖蹲在大树下，枯枝绿草里，一边采药，一边听着虫鸣鸟叫，却看不到虫身鸟影。
<br/>
<br/>        今天她运气不错，采到了灵芝和人参，不知不觉中，她已经进入了大山深处，深山里，大树参天，山间有溪水缓流，虽是炎炎夏日，但这里却温暖如春。
<br/>
<br/>        诺筱颖刚采到一株半夏草，抬眸的无意间，在灌木丛中发现了几朵漂亮的深紫色花。远看长得有点儿像虞美人，很漂亮。
<br/>
<br/>        女孩子爱花，是本性。诺筱颖也不意外，既然那花那么漂亮，她想连根一起采回去盆栽。
<br/>
<br/>        当诺筱颖走过去一看，看仔细了后，瞬间吓得脸色都白了。
<br/>
<br/>        这不是虞美人，而是……罂粟花！
<br/>
<br/>        罂粟是外来物种，他们这地方天生是没有的。
<br/>
<br/>        那么，这深山里为什么会有罂粟？
<br/>
<br/>        诺筱颖凝眉想了想，将这几株罂粟给扯了出来，放入背后的竹篓里，用其他药草盖住。
<br/>
<br/>        顺着这条蜿蜒得不成形的山路，诺筱颖继续往前走，直到走到了山顶，站在一块大岩石上眺望着远方。
<br/>
<br/>        这里重峦叠嶂，云雾缭绕，美如世外桃源。
<br/>
<br/>        诺筱颖顺着远景往下看去，山脚下群山围绕之间，还有一大片的红、粉、橙、黄、紫的花田。
<br/>
<br/>        很显然，那大片的花田，不是自然生长出来的。
<br/>
<br/>        那些花……是罂粟？！
<br/>
<br/>        到底是什么人会藏匿在这深山野林里种罂粟？
<br/>
<br/>        诺筱颖忽然间明白了什么，转身便跑。
<br/>
<br/>        下山的速度远比上山的速度要快许多，诺筱颖才跑到半山腰，忽然不知从哪个方向传来了一声“啊”地惨叫。
<br/>
<br/>        这声惨叫，是个男人的声音。
<br/>
<br/>        诺筱颖柳眉微蹙，下意识地躲到了一堆灌木丛里后。
<br/>
<br/>        “啪——嚓——咔——”
<br/>
<br/>        “这条蛇死了吗？”
<br/>
<br/>        “不知道！你快到我背上来，我背你下山去找医生看看。”
<br/>
<br/>        听到这个人的声音后，诺筱颖突然觉得很耳熟。
<br/>
<br/>        “不……我动不了了，脚好麻……而且还很痛！”又是另一个男人的声音。
<br/>
<br/>        诺筱颖眸光微沉，顺着那两个男人的声音找了过去。
<br/>
<br/>        不到十米远的大树下，坐着两个穿着他们苗族服饰的男人。
<br/>
<br/>        诺筱颖见其中一个男人的脚腕上有两个血洞，他的脚边还趴着一条一动也不动了的黑白相间的蛇，她连忙放下背上的竹篓，查看男人的伤口情况。
<br/>
<br/>        “银环蛇一般不会主动攻击人，刚刚一定是你不小心踩到它了。”诺筱颖从竹篓里翻出药箱，从里面拿出一个小木盒，打开小木盒后，里有几粒棕色的小药丸。
<br/>
<br/>        她二话不说，便捏起其中一颗小药丸子，直接塞入了男人的嘴中，命令他干吞下肚。
<br/>
<br/>        随即，诺筱颖又拿出一只未开封的注射器和蛇毒血清，全部准备就绪后，给这男人打了一针。
<br/>
<br/>        两个男人看着诺筱颖面面相觑，但他们不反抗是因为他们心里明白，她这是在救人。
<br/>
<br/>        紧接着，诺筱颖又从竹篓里拿出水壶拧开瓶盖，一边替这男人冲洗小腿上的伤口，还一边从药箱里拿出捣药碗，然后在竹篓里翻出药草放在碗里捣碎了，敷在了这男人的小腿伤口处，并拿纱布帮他固定好。
<br/>
<br/>        “下山后，要去县城里找医生再给你看看。”诺筱颖接着说道，抬眸的无意间，撞上了另一个男人的目光。
<br/>
<br/>        男人黑眸冷冽，脸部轮廓非常精致，他的干涸的嘴唇上和下巴处虽然长满了络腮胡须，但从他脸上的皮肤来看，这个男人大概就只是个二十出头三十岁还不到的年轻男人。
<br/>
<br/>        他们化了妆伪装了原本真实的面目，虽然穿着他们这里的服饰，但是她刚听他们说话的口音，一点都不像他们这儿的本地人。
<br/>
<br/>        “谢谢你，救了我兄弟。”男人替自己身边这位被蛇咬伤的男人谢道。
<br/>
<br/>        诺筱颖越听这声音越觉得耳熟，猛然间想起了什么，伸手便往这男人脸上一挥，一把扯掉了他下巴上那撮络腮胡子。
<br/>
<br/>        “果然又是你！”诺筱颖手里抓着这把假胡子，瞪大了眼睛看着傅夜沉。
<br/>
<br/>        这个男人，还真就阴魂不散了！
<br/>
<br/>        傅夜沉抿了抿唇，腼腆地微微一笑：“嗨，宝贝，我们又见面了！”
<br/>
<br/>        “你两认识啊？”被蛇咬了的韩剑锋一脸惊怔地看了看傅夜沉，又看了看诺筱颖。
<br/>
<br/>        傅夜沉用手肘戳了一下韩剑锋的肩头，一本正经地说道：“还不叫嫂子。”
<br/>
<br/>        “是、是！谢谢嫂子的救命之恩！”韩剑锋连连颔首。
<br/>
<br/>        诺筱颖微微凝眉，懒得理会这两个臭男人，而是随手捡起一旁的树枝，拨动着那条趴在地上半天不见动静的银环蛇。
<br/>
<br/>        “已经死了吗？”诺筱颖自言自语着，正苦恼着要不要将这条蛇带回去做成中药的时候，这条蛇突然间又动了起来。
<br/>
<br/>        韩剑锋看到那条蛇又动了，吓得直往身旁傅夜沉的怀里缩，双手还特别“伪娘”地搂着傅夜沉的脖子。
<br/>
<br/>        这人还真是一朝被蛇咬十年怕井绳！
