　　嗖…嗖…嗖…
<br/>　　云漩剑气、血煞剑气，密布四周，密密麻麻，但二者却配合无间。
<br/>　　自成一红白相间的惊人剑网。
<br/>　　一众剑修，大惊失色。
<br/>　　原本，是他们包围了周遭，包围了萧逸三人。
<br/>　　现今，却轮到他们被剑网包围，逃脱不得。
<br/>　　而最重要的是，萧逸那麻木的恶魔般模样，此刻正在快速收割着一条条性命。
<br/>　　萧逸的剑，再无之前那般横扫八方与威势十足。
<br/>　　有的，仅仅是一劈一挥。
<br/>　　每一次剑落，必是一个剑修陨落。
<br/>　　这等状况，比萧逸横扫一片，秒杀大批剑修，更为让人胆寒。
<br/>　　一个个地杀，对于在场还存在的每一个剑修来说，只能被动等死的境地，显然让人最为煎熬，最为恐慌。
<br/>　　但对萧逸而言，这却是似乎成了本能的存在。
<br/>　　一路杀戮，脸色微变，淡漠如斯。
<br/>　　脸上，始终挂着狰狞之色。
<br/>　　唯一变化的，便是他身后，仿佛映衬出一片滔天血海。
<br/>　　“恶魔…”
<br/>　　“恶魔…”
<br/>　　一声声惊呼，从在场剑修颤抖着的嘴巴中挤出。
<br/>　　在场剑修中，已无绝世强者层次的剑者。
<br/>　　剑光府供奉，明幻学宫、金辰学宫的带队长老，还有少数一些之前突破至绝世强者的剑修，早已身死。
<br/>　　在萧逸如同恶魔般收割性命之时，这些绝世强者便已在试图全力阻拦或者斩杀的途中，反被萧逸斩杀。
<br/>　　剩下来的剑修，顶天在武道大能层次。
<br/>　　不过十数分钟。
<br/>　　原本浩浩荡荡的数千剑修，只剩数百余人。
<br/>　　偌大个万剑碑林，尸横遍野。
<br/>　　尸体的死状，异常可怖，或分尸而亡，或鲜血流尽，只余一具干尸。
<br/>　　从气宗宗主和欧阳烈的角度看去，整个万剑碑林，俨然已成一恐怖而血腥的炼狱修罗场。
<br/>　　这个修罗场的主角，只是一道单薄的年轻身躯，缺嗜血至极。
<br/>　　往日名震中域的成名剑修，或剑道天骄，已无反抗的心思，竟是只余恐慌的双眸，直视萧逸。
<br/>　　“萧逸小贼…不，不，不，萧逸…”
<br/>　　“我等乃是天阙学宫剑堂弟子，你敢杀我们的话，我天阙学宫绝不会放过你。”
<br/>　　“我明幻学宫也一样。”
<br/>　　“……”
<br/>　　一个个原本不可一世，盛气凌人，自觉高人一等的学宫天骄，尽可能地稳着身体的颤抖，发出最后的怒喝。
<br/>　　五大学宫的剑道弟子，自然也有来参加剑域盛事。
<br/>　　只不过那些个首席，全部去了天藏学宫罢了。
<br/>　　来此的学宫弟子，无一不是各学宫顶尖的剑道弟子。
<br/>　　可，这些身份，本就不放在萧逸眼中。
<br/>　　更别说现今的萧逸已然双眸嗜血，宛若疯魔。
<br/>　　“小子。”雪山剑皇南宫白咬了咬牙。
<br/>　　“今日，是老夫不敌你；但今日留一线，日后好相见。”
<br/>　　“老夫今日若有半分不测，你就等着北部范围雪山剑修与你不死不休。”
<br/>　　“哼。”又是数十个年轻剑修子冷哼一声。
<br/>　　“萧逸，我东离剑宫的弟子虽极少在剑域行走。”
<br/>　　“但我东离剑宫，乃是当世三大剑宫之一，你若敢杀我等…”
<br/>　　这数十年轻剑修的话，并未能说完。
<br/>　　无论是各学宫的剑道天骄，还是各势力的顶尖剑道强者…
<br/>　　他们的一句句威胁之言，根本未能让萧逸的脚步或者挥剑的手停下分毫。
<br/>　　嗤…嗤…嗤…
<br/>　　紫电神剑，锋利异常。
<br/>　　仅仅是数分钟，剩余的这数百剑修，已悉数殒命。
<br/>　　死前，他们眼中看到的最后一幕，非是萧逸的身影，而是一片猩红血色。
<br/>　　偌大个万剑碑林，已再无活人。
<br/>　　参与围攻的数千剑修，一个未留。
<br/>　　哦，不，还有两人。
<br/>　　“云渊剑、讳无视，还有萧逸小贼，你们找死。”气宗宗主冷笑一声。
<br/>　　“围困天下剑修，且悉数残杀之。”
<br/>　　“你可知能来剑域的剑修，个个皆是一方大势力的强者或霸主势力的天骄。”
<br/>　　“别说我气宗，就是整个剑域，都不敢如此做。”
<br/>　　“你二人竟敢围困之，萧逸小贼更是残杀之。”
<br/>　　“你三人，就等着被整个中域的各大势力追杀吧。”
<br/>　　气宗宗主冷笑的声音，成功引起了萧逸的注意。
<br/>　　萧逸的目光，看向了远处正与云渊长老二人鏖战的气宗宗主与欧阳烈。
<br/>　　那双嗜血的目光，冰冷至极。
<br/>　　“桀。”萧逸嘴角，缓缓咧开，发出一声轻蔑的‘桀’声。
<br/>　　下一秒，身影一闪，竟是直朝二人而去。
<br/>　　在他眼中，这二人似乎不是闻名当世的绝世强者，而仅是他要击杀的对象。
<br/>　　“无视，动作快些。”云渊长老忽然冷喝一声。
<br/>　　讳执事点了点头。
<br/>　　二人原本一边维持围困万剑碑林的剑网，一边与气宗宗主二人激战。
<br/>　　如今，二人瞬间散去了剑网。
<br/>　　数千剑修已尽皆身死，剑网也没必要存在了。
<br/>　　二人全力出手，气宗宗主与欧阳烈，几乎节节败退，毫无还手之力。
<br/>　　“你二人做什么？”气宗宗主忽然脸色大变。
<br/>　　“你二人还想击杀我，给那小贼当垫脚石？”
<br/>　　“有何不可？”讳执事冷笑一声。
<br/>　　“你和欧阳烈，注定了是萧逸小子的敌人。”
<br/>　　“现在死和日后死，有何区别？”
<br/>　　确实，萧逸今日杀光了在场气宗剑者，还有五痴学宫弟子；气宗宗主和欧阳烈，注定了是萧逸的敌人。
<br/>　　与其今日放虎归山，不如直接击杀，省得日后麻烦。
<br/>　　讳执事，是数十年前便成名的杀神了。
<br/>　　再添两条性命，在他眼中也不过如此。
<br/>　　“你们疯了。”气宗宗主头一次露出了惊慌之色。
<br/>　　下一秒，竟是转身便跑。
<br/>　　“哼。”讳执事冷笑一声，“血刹斩。”
<br/>　　讳执事手中血色利剑，一剑劈出。
<br/>　　一道滔天血色剑气，一闪而过。
<br/>　　远处，气宗宗主还未能跑远，已一口腥血喷出，直接重伤。
<br/>　　恰在此时，萧逸的身影，追上了气宗宗主。
<br/>　　那双泛着血色的眼眸，直让气宗宗主打了个冷颤。
<br/>　　萧逸的脸色，没有丝毫变化。
<br/>　　紫电神剑，轰然落下。
<br/>　　气宗宗主，分尸而死。
<br/>　　堂堂气宗一脉的最强者，剑域三大强者之一，就此凄惨陨落。
