　　一个时辰的时间，很快便是过去。
<br/>　　擂台周围的安静，也逐渐变得热闹了起来，真正的前十擂台战，终于是要开始了。
<br/>　　“前十擂台战！现在开始！胜者晋升，败者出局！”
<br/>　　通天阁老者，站在最高的看台上，声音威严的开口道。
<br/>　　“第一句的对手，北邙学院楚风眠，对阵地魔门冷面魔君！”
<br/>　　这第一句，便是又轮到了楚风眠。
<br/>　　楚风眠身形一动，猛然落入到了擂台之中去。
<br/>　　而就在楚风眠的对面，他的对手，则正是那如今地魔门的第一人，冷面魔君。
<br/>　　楚风眠双眼，已经是开始打量起这冷面魔君，这冷面魔君，在地魔门的地位，几乎相当于是夜天君在北邙学院之中的地位。
<br/>　　甚至是可以说，比起夜天君在北邙学院之中的地位，还要高。
<br/>　　因为在北邙学院之中，夜天君虽然也是第一人，但是在其中也并非是所有人都听他号令，但是冷面魔君，可是地魔门年轻一代中真正的第一人。
<br/>　　地魔门的年轻一代，没有任何一人，没有降服他。
<br/>　　地魔门的武者，可都是桀骜不驯之辈，能够做到这一点，这冷面魔君的实力，也是比起想象之中的，还要恐怖的多。
<br/>　　面对这冷面魔君，楚风眠的心中，没有任何一点的轻视，此人，绝对是一个劲敌。
<br/>　　“你就是楚风眠？斩杀了我们地魔门不少弟子，一个默默无闻的小家伙，居然是偷偷的没少斩杀我们地魔门的弟子，看来你是真的打算跟我们地魔门对抗到底了？”
<br/>　　冷面魔君看着楚风眠，冷冷的开口道。
<br/>　　楚风眠斩杀过不少地魔门弟子，这些事，他现在自然是知晓的一清二楚。
<br/>　　“今日，你不打算跟我们地魔门一个交代吗？”
<br/>　　“交代？”
<br/>　　楚风眠的嘴角，露出几分嘲笑道。
<br/>　　“一群从九魔城里跑出来的小魔头，杀就杀了，还需要给个什么交代？”
<br/>　　“别说是杀了你们几个小魔头，就算是九魔城的那些老东西来了，楚某一样会杀。”
<br/>　　楚风眠言语之中，冰冷到了极致。
<br/>　　面对冷面魔君的话，他的话丝毫没有弱势，甚至是强势到了极点。
<br/>　　地魔门的人，杀就杀了。
<br/>　　敢杀地魔门弟子的人，不少。
<br/>　　但是敢在冷面魔君面前，说出这种话，这简直就是一种挑衅，一种丝毫没有将地魔门放在眼中的挑衅。
<br/>　　下方的无数地魔门弟子，听到楚风眠的话，都是群情激昂起来，一道道杀意，猛然看向楚风眠，恨不得现在就将楚风眠给碎尸万段。
<br/>　　“狂妄！杀了他！”
<br/>　　“杀了我们地魔门弟子，还如此狂妄！这是没有将我们地魔门放在眼中，是在挑衅我们地魔门的威严！”
<br/>　　“杀！跟我们地魔门为敌的人！都要死！”
<br/>　　无数地魔门弟子，疯狂的叫喊着，冲天杀意。
<br/>　　要不是这擂台战，他们不能参与其中，现在他们都恨不得一起出手，将楚风眠轰杀了。
<br/>　　“好，够狂妄！敢跟我们地魔门这样挑衅的狂妄小子，已经是很久没有出现过了，不过楚风眠，越是狂妄，你死的也就越是凄惨。”
<br/>　　冷面魔君冰冷的开口道。
<br/>　　“你的所有奇遇，我都知道，你不过就是得到了颜古龙帝的传承，又在北邙学院的众圣堂中，得到了北邙圣君的传承吗？奇遇虽然不错，但是在我的面前，你，依然只是一个蝼蚁。”
<br/>　　冷面魔君脸色一变，露出几分狰狞的表情来。
<br/>　　“我会将你镇压，将你关在炼狱之中，好好折磨折磨你，那你永远出于无尽的痛苦之中！”
<br/>　　“说大话的本事不小，不过就凭你，你觉得你有可能击败我？”
<br/>　　楚风眠脸色平静道。
<br/>　　“既然你是地魔门培养出来的天才，那我就先杀了你这个小魔头再说！”
<br/>　　“口气不小，我倒是要看看，一会你被我镇压的时候，嘴还有没有这么硬！”
<br/>　　冷面魔君冷笑一声道。
<br/>　　话音未落，他便是猛然出手了。
<br/>　　滔天魔意，一下子便是展现出来，这魔意，浑厚到了一种惊人的地步。
<br/>　　顿时擂台之上，天地变色，这一方擂台的周围，仿佛都变化成为了一方魔域。
<br/>　　天上地下，每一个地方，都是弥漫这魔意。
<br/>　　冷面魔君的身影，一下子便是消失在了这魔意之中，就在这魔意的背后，令人根本看不穿他的身形。
<br/>　　“魔意漫天！”
<br/>　　无尽魔意，顿时轰然冲天而降，向着楚风眠轰击过来。
<br/>　　这种魔意，在冷面魔君的手中，已经是控制到了最为精妙的地步。
<br/>　　显然已经是入魔以深，已经是真正化魔。
<br/>　　这魔意，看似无害，但是一旦是弥漫在楚风眠的周围，冷面魔君便是可以随时催动力量，四面八方向着楚风眠轰击。
<br/>　　他的功法，便是足以潜藏在这魔意之中，可以说只要是还拥有魔意，他就可以永远的躲藏起来，立于不败之地，而且可以从四面八方，随时出手偷袭。
<br/>　　若是在外面，倒是还好，因为可以选择暂时撤退。
<br/>　　但是这这擂台之中，就这么大的一方区域，根本无处可退，只能是硬拼。
<br/>　　比起在外面，在这擂台上的冷面魔君，还要难缠的多。
<br/>　　这魔意，已经是逐渐笼罩着楚风眠的身躯。
<br/>　　一旦是被笼罩其中，那楚风眠可就真的危险了，随时可能会败。
<br/>　　“就这小小的魔道，也来敢来在我面前嚣张？”
<br/>　　面对这魔意袭来，楚风眠站在原地，脸色不动，只是口中，突然猛然咆哮一声。
<br/>　　这一声咆哮，便是极为清澈的龙吟，响彻而起。
<br/>　　龙吟出。
<br/>　　楚风眠身边笼罩的魔意，顿时全部消散，直接全部被打破。
<br/>　　他的龙吟，便是克制这魔意最好的手段，将周围的魔意，全部震碎。
<br/>　　魔意消散。
<br/>　　这冷面魔君的身躯，都被直接震了出来。
<br/>　　“龙！”
<br/>　　战龙之爪，瞬间便是就在楚风眠的面前凝聚，一爪猛然挥了过去。
<br/>　　“魔天爪！”
<br/>　　这冷面魔君，身形直接震出，他的脸色都没有丝毫的变化。
<br/>　　突然出手，他的一双手，顿时化为血红，两道血红色的利爪，跟战龙之爪，猛然相对。
