　　萧逸已然飞离。
<br/>　　方向，正是剑域的中部范围，亦是万剑碑林坐落之地。
<br/>　　一个时辰后。
<br/>　　萧逸路经一大城，飞行的身影陡然停下。
<br/>　　剑域这里，虽武者朴素，建筑简陋，但同样有着一座座大城。
<br/>　　看了眼下方大城，身影一闪，就此落下。
<br/>　　他落下的地方，是一座风刹殿分殿。
<br/>　　大城之内，自然也有上古八殿的分殿坐落于此。
<br/>　　值得一提的是，上古之殿在这里异常显眼。
<br/>　　原因很简单，大城之内一座座建筑简陋朴素，但唯独上古之殿一如既往地庄严、雄伟而肃穆。
<br/>　　剑域这里，有剑域的特点。
<br/>　　剑者，为了清心寡欲，不为外物所影响心境，故每一样东西都异常简朴。
<br/>　　但这并不代表坐落于此的上古之殿也要迎合这里的特色。
<br/>　　上古八殿，这八座曾经誓死捍卫大陆的殿宇，从上古之时，上至总殿，下至每一座分殿，便都是一模一样的外貌。
<br/>　　并不会因为其它什么东西而改变。
<br/>　　上古八殿，本身便是大陆的一份象征。
<br/>　　萧逸恰好遇到风刹殿的分殿，自然便打算停下。
<br/>　　他想查一查剑域这边的事情。
<br/>　　赶路之事，现在倒是暂时不必急了。
<br/>　　虽说剑域，乃是中域有名的地域，其本身范围极大。
<br/>　　按萧逸的估计，路途动辄过百万里。
<br/>　　不过，萧逸全力赶路的话，一日之内便可到达。
<br/>　　之前一月时间，便是太过赶时间了，甚至连查一下剑域里头情况的时间都没有。
<br/>　　他总不能什么都不知道，糊里糊涂地在剑域里四处闯荡。
<br/>　　本身万剑碑林的盛事，也需要一番了解。
<br/>　　进入风刹殿，萧逸取出令牌，直接便进入卷宗室查阅。
<br/>　　他乃分殿主头衔，在这些普通分殿的权限，足够查阅任何情报了。
<br/>　　一个时辰后，他大致明白了过来，也对剑域有一定了解了，便就此离开。
<br/>　　离开卷宗室，迎面走来一老者。
<br/>　　“萧逸分殿主。”老者拱了拱手，行的是平辈之礼。
<br/>　　萧逸亦拱了拱手，打了个招呼。
<br/>　　老者便是这座分殿的负责人，亦是分殿主头衔。
<br/>　　“哦对了。”萧逸忽然想起了什么，笑笑，道，“刚好有事要劳烦分殿主。”
<br/>　　“萧逸分殿主客气了。”老者客气道。
<br/>　　萧逸道，“分殿这里，应当有剑力的修炼法诀。”
<br/>　　“我想用自身任务点，兑换一份。”
<br/>　　“剑力的修炼法诀？”老者闻言，显示一愣，随后玩味一笑。
<br/>　　“怎么？”萧逸见状，面露疑惑之色。
<br/>　　老者，一身布衣，毫无疑问，也是剑域这里的剑者。
<br/>　　另外，殿内的大多工作人员，亦是剑域本土的剑者。
<br/>　　剑域这里的上古之殿，虽外貌不变，但里头武者，显然都是本土的剑者。
<br/>　　老者笑笑，道，“每个来到剑域的外来武者，特别是本身有八殿武者身份的，都会特地兑换一份剑力修炼法诀。”
<br/>　　“所以萧逸分殿主有此要求，老夫早有预料。”
<br/>　　“不过，老夫要告诉萧逸分殿主的是，这东西对外来武者来说，丝毫无用，兑换了也是白兑换。”
<br/>　　“哦？”萧逸皱了皱眉。
<br/>　　老者笑道，“起码至今为止，但凡兑换过剑力法诀的外来武者，还未有一人能修炼出剑力。”
<br/>　　“为何？”萧逸眉头轻皱。
<br/>　　来剑域的外来武者，应该不多。
<br/>　　但这么多年来，一次次的剑域盛事，从外而来的剑修，却绝对不少。
<br/>　　这么多剑修，无一人能成功修炼出剑力？
<br/>　　这不得不让萧逸感到惊讶。
<br/>　　“很简单。”老者回答道，“剑力法诀，只是辅助一些毫无修炼剑力经验的人的东西。”
<br/>　　“但这不是修炼剑力的绝对因素。”
<br/>　　“或者，萧逸分殿主可以这般理解，为何剑力仅仅是剑域独有的力量？”
<br/>　　“为何偌大个中域，别的地域没有剑力的存在？”
<br/>　　“原因在于这片地域？”萧逸沉思问道。
<br/>　　“不错。”老者点点头。
<br/>　　“与其说，这里的剑者，清心寡欲，修炼出剑力。”
<br/>　　“倒不如说，是这里独特的剑力，让得这里的剑者，不得不清心寡欲，剑心纯粹。”
<br/>　　“还请明言。”萧逸拱拱手，道。
<br/>　　老者摇了摇头，“无法明言。”
<br/>　　老者苦笑道，“说实话，萧逸分殿主年纪轻轻，便是总殿那边的传承考核通过者，亦是绝世剑修，老夫自问没那个本事指导你。”
<br/>　　“这修炼之事，也很难明言。”
<br/>　　“不过，萧逸分殿主若在剑域待的时间久些，自己便会明白的。”
<br/>　　“真正让得外来武者难以修炼出剑力的原因，萧逸分殿主自己便会明悟。”
<br/>　　说罢，老者手中光芒一闪，取出一本书籍。
<br/>　　“这是我们分殿本身收藏的，阶品比较高的剑力法诀。”
<br/>　　“萧逸分殿主大可自己揣摩一番。”
<br/>　　“至于任务点，便不需要了，算老夫赠予萧逸分殿主的。”
<br/>　　“这…”萧逸迟疑了一下。
<br/>　　“萧逸分殿主不必客气。”老者笑笑，“不过是一份剑力法诀罢了，还算不得什么重宝。”
<br/>　　“这，谢谢。”萧逸点点头，拱了拱手。
<br/>　　“在下还有事，便不多留了。”
<br/>　　“这本剑力法诀，之后在下通过别的分殿转送回来这边，可行？”
<br/>　　“随萧逸分殿主。”老者点点头。
<br/>　　“告辞了。”萧逸拱拱手，转身离去。
<br/>　　……
<br/>　　出了风刹殿分殿，萧逸御空飞离。
<br/>　　数个时辰后，大概了飞行了二、三十万里路，萧逸忽然停下。
<br/>　　一路上，他都在思索剑力之事。
<br/>　　那位分殿主的话，让得萧逸对剑力这种奇特的力量愈发好奇，也愈发感兴趣。
<br/>　　当然，剑力的威力，本就不可小觑，这也是萧逸打算修习剑力的原因。
<br/>　　嗖…
<br/>　　萧逸身影一闪，从高空落下。
<br/>　　这里是一片荒山野林，不是什么妖兽森林，无半只妖兽，更不会有武者前来历练。
<br/>　　恰好，这里之清净可以让他尝试揣摩一番剑力法诀。
<br/>　　随意盘膝坐下，萧逸取出那位分殿主赠予的剑力法诀，看了起来。
<br/>　　“圣阶法诀。”萧逸自语一声。
<br/>　　半晌，萧逸合起书籍，皱了皱眉。
