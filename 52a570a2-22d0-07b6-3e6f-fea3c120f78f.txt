　　任云都和苏莫聊了很久，苦口婆心的安慰，就是让苏莫不要有什么过激的行为。
<br/>　　直到一个时辰后，任云都才离开苏莫的洞府。
<br/>　　任云都走后，苏莫阴沉着脸，独自盘坐在洞府内低头沉思。
<br/>　　这个尤天韩绝对不能放过，三番两次的挑衅，真当他是泥捏的吗？
<br/>　　只不过，他现在的实力，想杀对方估计还有一些困难，还是要努力提升实力。
<br/>　　片刻之后，苏莫抛却心中的杂念，继续修炼水仙录。
<br/>　　时间缓慢的流逝，他的水仙录功法已经真正的入门，逐渐的精深。
<br/>　　水仙录功法作为尊级中品功法，比苏莫原来修炼的帝级上品功法强大太多了，即便他修炼的火候还很低，依旧被帝级上品功法强大。
<br/>　　随着水仙录的不断精深，苏莫的总体实力亦是缓缓的提升。
<br/>　　……
<br/>　　神武学府之外。
<br/>　　距离巨大的陨落星辰约莫万余里之处，一座低矮的土丘之中，巫人王盘膝坐在土丘下，灵识倾泄而出，时刻查看着神武学府的情况。
<br/>　　他已经在此等待了近一个月，也没有见到苏莫回来，但他只能在此静等。
<br/>　　神武学府门前，不断的有人到来，扬言挑战苏莫，但神武学府的人根本不予理会。
<br/>　　神武学府早已经派了高手坐镇在府门之前，受了府中高层的吩咐，严禁任何不属于神武学府的人进入府内。
<br/>　　“可恶！”巫人王阴沉着脸，他知道苏莫可能已经回到了神武学府，只是他没有发现而已。
<br/>　　毕竟，苏莫隐藏身形极其厉害，上次在大荣城就是以此逃出他的追杀。
<br/>　　就在此刻，巫人王双眉微挑，因为他身上的传讯符有动静了。
<br/>　　手掌一翻，传讯符出现在手中，巫人王立刻意念一探，其内传出尤天韩的声音。
<br/>　　“苏莫已经回到了神武学府，但一直在内府不出！”
<br/>　　巫人王闻听此言，眉头紧皱了起来，沉思了片刻，立刻向对方传讯道：“有没有办法让他出来？我一招就能将他击杀！”
<br/>　　巫人王很有自信，上次在大荣城他若不是大意了，怎么可能被苏莫逃走？
<br/>　　即便苏莫能隐藏身形，但他施展大范围的攻击，隐藏身形也无用。
<br/>　　当然，他是想要擒拿苏莫，并不想一击击杀，因为他还想探知苏莫身上的秘密。
<br/>　　向尤天韩传讯之后，巫人王静静等待对方的回话，不过传讯符寂静下来，再也没有任何反应。
<br/>　　直到半盏茶的功夫之后，传讯符才再次有了反应，嗡嗡震动了起来。
<br/>　　巫人王急忙查看，传讯符中再次传出尤天韩的声音：“学府外不是有很多人要挑战苏莫吗？想办法激苏莫走出学府！”
<br/>　　巫人王沉默了下来，这倒是一个办法，只不过，这样的话定然搞得人尽皆知。
<br/>　　到时候神武学府很多高层都会关注，他想要杀苏莫容易，想要擒走苏莫，好像不太可能。
<br/>　　巫人王犯难了，难道擒拿一个小小的苏莫，还要让族中祭司出手吗？
<br/>　　不过，无论能否擒拿苏莫，他都要试一试，实在不行，就只能让族中祭司出手了。
<br/>　　而且，真圣界就要开启了，他也没有太多的时间耗在此地，必须尽早离开。
<br/>　　随即，巫人王破土而出，离开了土丘。
<br/>　　……
<br/>　　火元峰，洞府内。
<br/>　　正在修炼的苏莫，突然睁开了眼眸，因为一道蟒袍身影，突兀的出现在了他的洞府门前。
<br/>　　蟒袍身影，正是古天意。
<br/>　　“苏莫，不请我进去坐坐？”古天意面带和煦微笑，让人如沐春风。
<br/>　　“古兄请进！”苏莫立刻起身，邀请对方进洞府。
<br/>　　少倾之后，两人在洞府的石桌前坐了下来。
<br/>　　“苏莫，你最近可是名声大震，算是彻底扬名了！”古天意笑道。
<br/>　　“古兄就不必埋汰我了，虚名而已！”苏莫摇头道。
<br/>　　“这可不是虚名，这真圣榜之上，能上榜的人哪一个不是绝世天才，既然你能排在第一，天赋可想而知！”
<br/>　　古天意轻笑道，他的眸中也不禁露出唏嘘之色，现在外面关于苏莫的传言，简直是铺天盖地。
<br/>　　四生战魂、两种特殊体质，这件事简直传的沸沸扬扬。
<br/>　　苏莫默然的摇了摇头，不想与对方闲扯，略一沉吟，道：“古兄，不知你能否将我的两位朋友，从镇魂之地弄出来？”
<br/>　　古天意闻言，面色也是郑重了起来，凝声道：“的确是能弄出来，但关键还是靠你。”
<br/>　　“靠我？我需要做什么？”苏莫疑惑问道。
<br/>　　“执法堂的惩戒是铁律，不可能更改，学府高层也不会打自己的脸，将两人放了！”
<br/>　　古天意没有回答苏莫，而是徐徐讲解，言到此处他稍稍停顿，然后继续道：“想要让聂乾和韩盈离开镇魂之地，唯有十三个大势力之一将两人带走，两人提前脱离神武学府！”
<br/>　　“哦？”
<br/>　　苏莫闻言眼眸一亮，道：“那古兄能否将两人带走，加入皇祖圣朝？”
<br/>　　“不能，我也没有这个资格，除非我父皇和皇叔亲自开口！”古天意摇头说道。
<br/>　　苏莫闻言皱眉，带走两个人而已，还要皇祖圣朝的掌权人亲自开口，没有这么夸张吧？
<br/>　　仿若是知道苏莫心中的想法，古天意又道：“他们若是没有犯事，自然不用这么麻烦，但两人都是戴罪之身，只有十三个大势力中的武圣开口，执法堂才会放人！”
<br/>　　苏莫点了点头，目光直视古天意，道：“那古兄，你能让圣皇陛下或者是贵皇叔出面吗？”
<br/>　　“不能！父皇和皇叔不可能为此等小事出面！”古天意再次摇了摇头。
<br/>　　“需要我做什么？古兄明说吧！”苏莫沉声问道，之前对方说还是要靠他，他知道对方必有深意。
<br/>　　“八个月之后，真圣界开启，你若是能在真圣碑留名，我就能说服父皇出面救出两人！”古天意沉声道。
<br/>　　“真圣碑留名？”苏莫闻言皱眉。
<br/>　　“不错，只要是能在真圣碑留名的天才，将来基本上必成武圣，你若做到了，我能说服父皇帮你出面！”古天意解释道。
<br/>　　苏莫闻言沉默不语，他还有巫族这个大敌，目前一段时间之内，根本不宜离开神武学府。
<br/>　　“苏莫，真圣碑留名，得真圣之气，是你的机会，也是救出聂乾和韩盈的机会，两全其美！”古天意郑重说道，他并不知苏莫和巫族的仇怨。
