        &lt;h3&gt;第三卷第一百六十二章  宁雅去韩国&lt;/h3&gt;
<br/>
<br/>        宁雅收拾好心情，将公司的事情处理完之后，交给田星和钱喵喵就和胡婷夫妇一起去了韩国。
<br/>
<br/>        宁雅等着胡婷夫妇来接她，凌辰自从接了胡婷恶作剧的电话之后，基本从他们的世界消失了一样，和狸猫也没有任何联系。
<br/>
<br/>        狸猫开着车，带着胡婷来接宁雅一起去机场。
<br/>
<br/>        “小雅，好久不见！”狸猫下车替宁雅拿起行李箱放进后备箱。
<br/>
<br/>        “好久不见。你是大忙人，怎么可能和我们这些闲散人士来往呢。”宁雅笑着怼狸猫。
<br/>
<br/>        胡婷和狸猫俩个人腻腻歪歪的，总是让她当灯泡吃狗粮。
<br/>
<br/>        “嘿，谁是大忙人谁清楚啊，你连一起吃饭的时间都没有。”狸猫等宁雅上车，关好门。三个人说说笑笑的朝机场走去。
<br/>
<br/>        “这次我可是跟着你们一起去流浪啊，你们两个要好好照顾我。”狸猫笑嘻嘻的样子看得宁雅想揍他。
<br/>
<br/>        “我们俩个人去流浪，你跟着干什么？那里凉快哪里呆着去。”宁雅也笑骂着。
<br/>
<br/>        “我这里凉快，我这里凉快，狸猫，来我这里。”胡婷唧唧歪歪。
<br/>
<br/>        “你那里太热了，我还是想找小雅，小雅那里凉快。对了，小雅，凌辰那个冰坨子最近有没有联系你？我好久都没有他消息了，好像群里也退了？”狸猫想起凌辰。
<br/>
<br/>        上一次他们几个聚会，拍了张照片，惹得胡婷醋意满天飞，还跑宁雅那里去了。
<br/>
<br/>        宁雅：“最近也有一段时间没有联系呢，之前去了贵州不知道有没有回来。”
<br/>
<br/>        胡婷：“别提凌辰那个渣男了，自己都有女朋友了还撩小雅。”
<br/>
<br/>        “凌辰这几年都没有女朋友，怎么就突然有了女朋友？我咋不知道?”狸猫是知道凌辰对宁雅的心思，只是大家年底都特别忙，他也没有时间去关注凌辰的事情，连上网的时间都没有。
<br/>
<br/>        所以对宁雅和凌辰之间的误会是不知道的，也不知道他那没心没肺的老婆给他们制造了多大的困扰，让俩个没有安全感的人经受着怎么样的情感折磨。
<br/>
<br/>        凌辰那火热的激情突然就掉进了冰窟窿，一下子就冷却了，不但冷却了还慌里慌张的。
<br/>
<br/>        “他确实是有女朋友了，上次晚上我打电话的时候是他女朋友接的，他女朋友还说他正在和她的爸爸在书房谈事情。”宁雅现在的心境已经平静下来了，好好干一番事情才是正经的路子。
<br/>
<br/>        “他前一段时间不是去贵州了吗？难道在贵州找到女朋友了？没有这么快吧？”狸猫有点摸不着头脑，凌辰会突然一下子放弃宁雅和别人谈恋爱？
<br/>
<br/>        宁雅：“不知道呢，最近都没有联系。有女朋友了就祝他幸福吧，毕竟我们都是好朋友。”
<br/>
<br/>        狸猫和胡婷都没有说话，他们是外人，但是他们也能看出来俩个人之间那种奇怪的感情，他们俩个人始终都是处在若即若离的状态，说是不爱吧，俩个人彼此心中都装着对方，说是爱吧，他们谁也不肯前进一步，一直拖到现在，固然家庭环境是一方面，可是为什么俩个人就不努力一下？
<br/>
<br/>        胡婷倒是和狸猫的心思能同步，她伸出了手，将手放在了狸猫的手心。
<br/>
<br/>        还好，他们没有错过。
<br/>
<br/>        首尔
<br/>
<br/>        宁雅是来考察的，她是一门心思的想做一番事情，就像陈子昂那天说的那句话：“靠山山会倒，靠水水会流，只有靠自己才是实实在在的。女人必须有自己的一份事业，无论是上班还是自己创业，你才能活得有价值，活得充实。”
<br/>
<br/>        她的家庭条件是不错，可是她还有弟弟妹妹，父亲能给她这个过气了的嫡长女多少资源？俗话说爱屋及乌啊，有了小阿姨在父亲身边，她，也就是一个女儿而已。
<br/>
<br/>        所以不如自己成长起来，有了自己的事业，有了自己的资源，谁也不能把她怎么样。
<br/>
<br/>        胡婷和狸猫出去玩了，她联系了在韩国的同学，韩国这几年的美容事业蓬勃发展，她就让同学带着走访了好几个美容机构，也联系了好多专家，讨论了后期的发展状况。
<br/>
<br/>        虽然没有时间出去逛一逛，却收获了很多，信心满满。
<br/>
<br/>        她还没有正式开始，她联系的专家答应她开业的时候会来给她撑一段时间，后期也是要看营业状况，这种专家都不会永远在一个机构待下去的。
<br/>
<br/>        谈妥了一个高薪聘请坐诊的专家，专家刚好明年5月份合约到期，对于宁雅给出的条件也很心动，最主要的是这个专家很喜欢宁雅的祖国，对中国的美食那是赞不绝口。
<br/>
<br/>        宁雅算是吃了定心丸，她们营业地点装修也在进行中，营业执照在办理中，她做的不是一个小的美容院，而是一个女子美容医院，先将这个开好，后期还可以开连锁的。
<br/>
<br/>        女人一旦认真起来了，是没有任何事情能阻挡她们前进的脚步。
<br/>
<br/>        在国内，她也利用自己手中的资源，联系了一些有资质的有能力的医师，并且找了一个医院的老总作为她公司的执行总裁，还聘请了一些退休的高级教授，作为医生这个行业，他们是越老越有经验。
<br/>
<br/>        宁雅本身很瘦，短短的一段时间，她又瘦了很多。
<br/>
<br/>        万事俱备只欠东风，装修的进度应该需要几个月的时间，新年过后周末她看着装修，周内上班，等陈子昂的耗材正式推向市场之后她就可以辞职干自己的事情了。
<br/>
<br/>        宁雅对未来充满了希望！
<br/>
<br/>        胡婷和狸猫逛着韩国那美丽的风景，想着留宁雅一个人到处跑，觉得心里有些内疚。
<br/>
<br/>        “狸猫，你说我们俩个出来逛，留宁雅一个人去跑好不好啊？”
<br/>
<br/>        “你又插不上手，你跟着干什么呢？跟着宁雅反而是宁雅的累赘。”狸猫安慰胡婷，反正胡婷也不会和宁雅一样做生意，她在家里呆着就好，过一两年产个小宝宝出来才是幸福无比的事情呢。
<br/>
<br/>        想起小宝宝，他就想起了他的同学----奶爸郭力，他们家里的那个可爱的小宝贝简直爱死个人，他抱着都舍不得松手，他也要赶紧要一个养着玩玩。
<br/>
<br/>        狸猫笑眯眯的想着小宝贝，那一脸幸福的样子让胡婷看着有些纳闷。
<br/>
<br/>        “想啥呢？”
<br/>
<br/>        “想小栖桐呢？”
<br/>
<br/>        “小栖桐？张黎，你胆子肥了是吧？”胡婷的手已经拧住了狸猫的耳朵。
<br/>
<br/>        “放手放手，好歹我是一个作家，我想一想小栖桐怎么了，你都不知道小栖桐那可爱的样子，哎呀，你赶紧的给我生一个。”
<br/>
<br/>        “生一个？”
<br/>
<br/>        “是啊，郭力家的小宝贝啊，才六个月，太惹人爱了，我一看见小宝贝，心里就像是有一跟羽毛划过，软软的，满心欢喜。”
<br/>
<br/>        “好吧，原谅你了，我们也去造一个小栖桐出来。”
<br/>
<br/>        “不管宁雅了？”
<br/>
<br/>        “不管了，让她继续听凌辰的情话去。”
<br/>
<br/>        “不是算了吗？”
<br/>
<br/>        “其实，我觉得他们一定是误会了，我有一次........”胡婷倒豆子一样将那天的情况给狸猫说了。
<br/>
<br/>        “你说你这女人........”狸猫觉得误会闹大了。
