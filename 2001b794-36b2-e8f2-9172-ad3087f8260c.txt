　　回到上海后，叶帝他们倒是没有再去休息，因为马上就要开始世界赛了，由不得他们不紧张起来。
<br/>　　而且现在他们被捧的太高了，几人深知捧的越高、摔的越惨的这个道理，为了不给那些黑粉喷他们的机会，这也由不得他们不继续努力。
<br/>　　好在他们状态都不错，而且大家都知道，只要自己等人不出现太严重的失误，那么小组赛什么的应该都没太大的问题。
<br/>　　而在皇族和雷霆的最终对决中，最终雷霆战队获得了比赛的胜利，若是说单个位置，那么皇族这边可能还略微占据一些优势，毕竟他们有着楚白衣和司马贼这两位世界知名的选手。
<br/>　　但是论整体实力的话，他们这边比之雷霆就要差上一些了，虽然飞鸽几人都在进步，但是毕竟还是有着一些差距的，所以最终的对决以雷霆战队这边3：1的战绩取胜对手。
<br/>　　到底也是正式决定了华夏的三个出线的种子选手，这倒是让无数的粉丝再次激动了起来。
<br/>　　至于对于皇族的淘汰，遗憾大家倒是有一些，不过并没有想象的那么重罢了。
<br/>　　时间慢慢的走着，不知不觉间5天的时间便是过去了，而这个时候六十支来自世界各地的战队也是陆陆续续的到达了上海。
<br/>　　叶帝他们因为距离比赛地点比较近，所以他们倒是没有去住宾馆什么的。
<br/>　　这次的比赛倒是没有以往那般繁琐，需要到很多地方比赛，这一次世界赛完全是在上海这一个地方进行，从小组赛到最终的决赛都是如此。
<br/>　　而在这些战队到达上海的时候，和叶帝他们比较要好的ST这样的战队，也是来过叶帝他们这里做客，时常打个训练赛什么的，双方倒是互有胜负。
<br/>　　当然了虽然荣耀战队的基地确实够大，不过金相赫这群人倒是并没有留在这里居住，地方够了，但是电脑不够啊，大家总归是要训练的，而且双方都有着自己的隐私，所以他们也只是偶尔来个下午一起打打训练赛，然后就在回到居住的宾馆那里了。
<br/>　　“这一次分组咱们两边又分配在了不同的两个小组，说不定最后真的有机会会师决赛呢。”想着昨天夜里的抽签，蒋念笑着说道。
<br/>　　“前提是你们能拿下你们小组的第一，要是第二的话，你们可就要在半决赛遇到我们了。”香蕉不留情面的说道。
<br/>　　“该注意的是你们吧，ST那个战队可是在A组了，以他们现在的状态，说不定还真的能击败你们，到时候拿个小组第一呢，到时候你们恐怕就要在半决赛的时候被淘汰出去了，还是加油吧。”楚南这个时候也是开口调侃了起来。
<br/>　　“ST战队的实力还是摆在那里的，你们可真的要加油了，我们这边虽然有一些强力的队伍，但是对于我们冲击小组第一的威胁应该不是很大，所以如果你们排在小组第二的话，那么咱们肯定会有一个被淘汰出去了，到时候咱们会师决赛的梦想可就要断了。”秦霄跟着开口说道。
<br/>　　“放心吧，这一次就算是拼了老命也是要将ST给拉下马来的，小组赛第一天就有我们的比赛，这一场比赛大家一定要加油，只要赢下这场比赛，接下来咱们就占据先机了。”叶帝对着自家的队友打气。
<br/>　　到了这个时候他可没有了玩笑的心思，毕竟ST战队的实力摆在那里呢，有着小花生这个打野的加盟，他们的实力可是变的更加的强大了，现在就算是叶帝心里也是没有必胜的信心。
<br/>　　“ST战队不仅新来的这个打野强，他们的下路组合coco和newlife两人也是非常的强力，就算是我和老秦两个遇到他们也都是有些棘手。”林萧这个时候跟着开口说道。
<br/>　　“兵来将挡，水来土掩，反正都已经到这个时候了，只有打了才知道，连你们都能打过，我就不信打不赢个ST。”香蕉依旧是无比的自信。
<br/>　　听着香蕉的话，叶帝几人的脸上也都是露出了笑容，反正已经打到这个时候了，自然是没有怯战的道理，恐怕现在ST他们应该也在为自己等人赶到头疼吧。
<br/>　　在世界赛开赛之前这两天，上海这座城市的客流量也是在极具的增加，据统计比平时足足多了十倍有余，本来上海就是一座世界性的大城市，每天的客流量已经非常恐怖了。
<br/>　　然而现在居然是往常的十倍有余，可想而知这一次的世界赛影响力究竟是有多么的大。
<br/>　　街头上经常能够看到一些穿着带着选手头像的半截袖的年轻人，而且每一个人脸上都是洋溢着十足的热情。
<br/>　　就这样在大家的期待中，时间一天一天的走着，缓缓的向着揭幕战的那天到来，而那些没有买到票或者有事情来不了现场观战的人，这个时候也是在网上疯狂的议论了起来。
<br/>　　“马上就要比赛了，揭幕战就是白队打ST，这个真的是太给力了。”
<br/>　　“只要赢了ST，咱们就有机会在最终的决赛上打内战了，大魔王你们一定要加油啊。”
<br/>　　“ST现在还是很棘手的，有着小花生的加盟，他们这个队伍的实力变的更加的强大了，夏季赛更是以全胜的战绩夺冠，希望大魔王他们能够顶住压力。”
<br/>　　“666666666666，终于要打世界赛了，这一天等了太久了，今年绝对是我荣耀蓝白的天下。”
<br/>　　“加油加油，虽然不能去现场观战，但是我们会在直播间里为你们加油助威的，现场的兄弟一定要让其他赛区的人知道咱们华夏的热情，大魔王他们登场的时候，你们要往死里欢呼。”
<br/>　　“没错，以往咱们在国外的时候可是没少受到他们的冷遇，虽然咱们是礼仪之邦，但是怎么着也要尽一尽地主之宜的，让他们也感受一下，尤其是那些鬼子。”
<br/>　　论坛上、微博上无数的粉丝在那里议论着，这一刻大家都在等待着世界赛的到来。
<br/>　　而就在大家的期待下，世界赛开赛日终于是缓缓来到。
