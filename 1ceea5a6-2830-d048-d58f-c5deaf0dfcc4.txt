　　见天虚宗弟子全部离开，苏莫也是松了口气，终于能安安心心的猎杀魔物了。
<br/>　　不过，他刚才吞噬了大量的攻击和玄力，并不想全部浪费，现在则需要炼化。
<br/>　　转头看向吕罡，苏莫沉声道：“吕罡，现在我已经夺到了优先权，现在要静修片刻，你来帮我猎杀魔物！”
<br/>　　“啊！”
<br/>　　满脸怔怔的吕罡，闻言顿时回过神来，诧异的问道：“我帮你猎杀？”
<br/>　　“不错，我分你一成骨刺！”苏莫点头说道，他虽然以前心中对吕罡有些微词，但现在已经全消了。
<br/>　　此人刚才见天虚宗众人要联手对付他，敢于前来帮忙，说明为人还是不错的。
<br/>　　当然，并不是说对方一个想要帮忙的举动，苏莫就分给对方一成的骨刺。
<br/>　　而是苏莫不想亲自动手，猎杀魔物将会耗费大量的时间，他不想浪费时间。
<br/>　　这个时间，他还是用在修炼上比较好，让吕罡来当他的打手。
<br/>　　“一成的骨刺！”吕罡闻言惊讶，这个比例虽然听起来很小，但实则并不小。
<br/>　　因为正常情况下来说，他若想得到魔物的骨刺，最后也是和其他道宗的弟子联手。
<br/>　　只有召集了十几个人以上，或者是有非常厉害的人在，才能夺得猎杀魔物的机会。
<br/>　　所以，正常情况下，他根本分不到一成的骨刺。
<br/>　　就比如天虚宗的那些人，总共二十多人一起，最后每个人分到的骨刺，别说一成了，连半成都没有。
<br/>　　所以，苏莫愿意分出一成的骨刺，对于吕罡来说，已经非常的大方了。
<br/>　　“嗯，你若不愿意就算了！”苏莫点头，并没有强行要求。
<br/>　　“愿意，当然愿意！”
<br/>　　吕罡心中暗喜，忙不迭的答应了下来，道：“你去静修吧！猎杀魔物的事，全部交给我了！”
<br/>　　“好！”苏莫微微一笑，随即身形飞到不远处降落了下来，就地盘膝而坐，闭上了眼眸。
<br/>　　众人见此，暗暗惊讶，苏莫还真是大方，居然愿意分出一成的骨刺。
<br/>　　没有人敢上前争夺猎杀魔物的权利，连武仁和二十多名天虚宗弟子都败了，其他人更是没有实力与苏莫争夺。
<br/>　　银袍青年等几名道宗弟子，脸色均是难看了下来，心中暗暗后悔。
<br/>　　吕罡只是做出了一个要帮忙的举动，连出手都没出手，居然就得了这么大的便宜。
<br/>　　而他们，却是错过了这个机会。
<br/>　　可是，现在后悔已经没有用了。
<br/>　　不多时，再次有魔物出现了，足足三只魔物，实力颇为不弱，相当于一般的武帝境九重武者。
<br/>　　不过，吕罡怎么说也是天才，战力堪比武尊境强者，斩杀这三只魔物，只用了不到五招。
<br/>　　而苏莫，此刻已经进入了修炼状态。
<br/>　　他的体内玄力滚滚，如同大河涌动，均是他吞噬而来的玄力。
<br/>　　苏莫极力的炼化，将这些玄力炼化为自己的玄力，最后汇于神丹之中。
<br/>　　时间缓缓流逝，苏莫完全沉浸在了修炼之中，没有去管吕罡猎杀了多少魔物。
<br/>　　良久之后，他将所有玄力炼化，他的修为也提升到了武帝境九重的巅峰，实力再次增加了不少。
<br/>　　睁开眼眸，看了一眼正在与两只魔物大战的吕罡，苏莫再次闭上了眼眸，极力淬炼玄力，巩固修为。
<br/>　　啊！
<br/>　　又过去了几个时辰之后，突然一声惨叫声响起，苏莫睁开眼眸，便看到吕罡的身形不断暴退。
<br/>　　只见吕罡的肩膀上，一大块血肉已经被撕了下来，鲜血潺潺而流。
<br/>　　而在其前方，一只身高五丈、全身漆黑的人形魔物，正在扬天咆哮，气势摄人。
<br/>　　这只魔物的实力很强，接近一般的武尊境二重武者，所以根本不是吕罡所能对付的。
<br/>　　苏莫见此，意念一动，十方灵蛇剑阵急速袭出，三下五除二便将这只魔物绞杀，身躯都被斩成了十数块。
<br/>　　“多谢！”
<br/>　　吕罡见此，遥望苏莫，沉声道谢。
<br/>　　苏莫微微点了点头，便再次闭上了眼眸，继续淬炼玄力，巩固修为。
<br/>　　而后，吕罡便将魔物的骨刺收取，继续等待魔物出现，然后一一猎杀。
<br/>　　接下来的时间之中，苏莫依旧是安静修炼，偶尔出现吕罡对付不了的魔物，他便出手击杀。
<br/>　　好在这魔渊的魔物，并没有什么超强的存在，也就偶尔有两三只比较厉害，但在苏莫的全力袭杀之下，依旧被击杀。
<br/>　　时间飞快流逝，眨眼间便过去五日。
<br/>　　五日的时间，吕罡猎杀了不下五百只魔物，获得了大量的骨刺，所得颇丰。
<br/>　　而苏莫，只是偶尔出手了十余次，一直在安心修炼。
<br/>　　这个时候，很多其它势力的人，已经暗暗焦急了起来，五日时间已经不短了，而苏莫和吕罡居然完全没有停手的意思。
<br/>　　但是，众人虽然焦急，却也没有人敢上去抢，苏莫就只是盘坐的不远处，就是极大的震慑。
<br/>　　就这样，又过去了三日，很多人都等的不耐烦了起来。
<br/>　　这时，一名身穿紫衣、背负古剑的俊朗青年，缓缓的飞到了苏莫的身前。
<br/>　　“苏莫，你们已经霸占了魔渊八日，是不是该让位了？”紫衣青年沉声问道。
<br/>　　苏莫闻言，缓缓的睁开了眼眸，打量了一番紫衣青年，摇头道：“还早，再过几日再说吧！”
<br/>　　苏莫不可能让位，有吕罡帮他猎杀魔物，他只需安心修炼便可，经过八日的修炼，他的修为淬炼的已经差不多了。
<br/>　　只需再淬炼几日，差不多就能尝试冲击武尊之境了。
<br/>　　所以，在他修为淬炼完成，根基巩固之前，他是不可能让位的。
<br/>　　“还要过几日？”紫衣青年闻言皱眉，脸色很不好看，他已经等了很久了。
<br/>　　他乃是御剑谷的弟子，之前天虚宗弟子包场之时，他已经等了五天，现在苏莫包场他也等了八天，前前后后已经快半个月了。
<br/>　　“打败我，你便能占据魔渊！”苏莫淡淡的说道。
<br/>　　“你……！”紫衣青年闻言怒急，他若是有把握打败苏莫，岂会等到现在？
<br/>　　深深的吸了口气，紫衣青年极力压下心中的怒气，实力不如苏莫，他只能等待。
<br/>　　同时，他心中盘算，若是再过三日，苏莫还不走，说不得只能一战了。
<br/>　　嗖！嗖！嗖！
<br/>　　就在此时，蓦地，天际的尽头响起刺耳的破空声，三道流光急速而来。
<br/>　　紫衣青年转头一看，顿时笑了起来，因为武仁居然又回来了，身边还带着两名高手。
